Bedienungsanleitung zur GlobalTIS Installation


1.  Ausblick und Hintergrund

1.1 Einleitung

Für die Opel Partner stellt TIS 2000 eine wichtige Informationsquelle für den Servicebereich dar.  Technische Dokumentationen sowie Software für Tester und Steuergeräte (ECU) werden über regelmäßige DVD Updates aktualisiert.

Eine Überarbeitung von TIS 2000 steht nun als neue Anwendung über das Internet zur Verfügung.  Die Vorzüge sind offensichtlich: aktuelle Informationen und das Vermeiden von Installationsproblemen.  Als Zwischenlösung bis zum Erscheinen der Internet-Applikation wird diese als DVD-Version zur lokalen Installation angeboten.  Somit wird die Abhängigkeit von der Internetverfügbarkeit umgangen.

Zusätzlich wurde das frühere TIS 2000 dahingehend verändert, daß eine kompaktere Informationssuche schneller zu dem gewünschten Dokument führt.   Ein überarbeiteter Verzeichnisbaum zeigt nicht nur die Reparatur- und Ein-/Ausbauanleitungen sondern zugleich die zutreffenden Feldabhilfen, Prüfanleitungen und Stromlaufpläne.

1.2  Fehlende Funktionalitäten

Die neue Applikation ersetzt noch nicht vollständig TIS 2000.  In der Anfangsphase wird die SPS Datenbank und die Updates der Diagnosesoftware für den Tech und das MDI (Tech 2 Nachfolger) enthalten sein.  Während dieser Zeit kann sich der Nutzer mit dem neuen System vertraut machen.

Bis Ende 2009 wird die dann noch fehlende Information wie z. B. Serviceinformation, Stromlaufpläne, Arbeitswerte, Prüfanleitungen und alle anderen Dokumente integriert werden.  Das ist dann zugleich das Ende der aktuellen TIS 2000 Anwendung.

2.  Benutzerhandbuch

2.1  Installation (Server oder Standalone PC)

Wie bei TIS 2000 muss die DVD auf dem Server oder dem lokalen PC (Standalone) installiert werden.  Dennoch gibt es einige Unterschiede in der Handhabung zu beachten.  Ein bedeutender Unterschied ist die Notwendigkeit der Registrierung.  Das bedeutet, daß GlobalTIS nur verwendet werden kann nachdem sich der Anwender erfolgreich als zugelassener Nutzer ausweisen kann.  Einzelheiten finden Sie in Kapitel 2.1.4.

2.1.1  Vor der Installation:

  • Der Nutzer benötigt für eine spätere Installation Administrator Rechte für diesen PC.
  • Vor der Installation muß überprüft werden, daß das Windows System mit den neuesten Servicepacks aktualisiert wurde.
  • Das Installationsverzeichnis für GlobalTIS darf nicht vorhanden sein oder muß leer sein.  Der Verzeichnisname darf keinerlei Sonderzeichen enthalten.
  • Im Falle eines angeschlossenen USB-Hardware Keys muß dieser vor der Installation abgezogen werden.  Anderenfalls wird MS Windows automatisch die Treibersoftware installieren, was negative Auswirkungen auf das spätere Arbeiten hat.  Ein falscher Treiber würde daraufhin installiert.

2.1.2 Während der Installation:

  • Sie werden nach der Sprache gefragt. Ihre getroffene Auswahl bezieht sich nur auf die Installationsprozedur. Die Spracheinstellung für die Benutzeroberfläche und die Dokumente wird an einer anderen Stelle vorgenommen.
  • Sie werden nach Angaben zum 'Port' gefragt z. B. "Transbase Server Port". Ohne einen triftigen Grund sollten diese nicht leichtfertig geändert werden.
  • Sie werden nach dem Namen des Hosts gefragt.Dieser Wert kann bei einem Standalone 'localhost' bleiben, bei einer Serverinstallation muß dieser Wert auf den Computernamen des Server geändert werden (mit rechter Maustaste auf dem Desktop "My Computer" wählen -> Eigenschaften -> Computername auswählen).
  • Sie werden nach dem "Mail Server" gefragt.  Diese Angaben werden beim direkten Versenden von Feedbackformularen benötigt - auf Ausdrucke für Faxversand kann dann verzichtet werden.  Standardmäßig sollten diese Werte nicht geändert werden.

2.1.3  Nach der Installation, aber vor dem ersten Start:

Nach erfolgreicher Installation startet der Nutzer die Anwendung mit: Start -> Programme -> GlobalTIS -> GlobalTIS.  Falls ein Hardware Key vorhanden ist sollte er jetzt an den PC angeschlossen werden.  Dies hat unbedingt vor dem ersten Start aber nach der Installation zu erfolgen.

 

2.1.4 Registrierung:

Nur zugelassene Nutzer können mit GlobalTIS arbeiten.  Die Autorisierung wird während des Registrierungsvorgangs erlangt oder mittels vorhandenem HWK.  Letzteres ist der einfachste und schnellste Zugangsweg.

2.1.4.1  HWK Prozedur:

Bevor GlobalTIS das erste Mal gestartet wird

  • schließen Sie den HWK an den Standalone PC oder
  • den HWK mit dem Server (nur WIN2000 und XP) verbinden

Im Gegensatz zu TIS 2000 darf der HWK nicht an einen Client-PC angeschlossen werden.  Die Seriennummer des HWK wird automatisch zum Lizenzschlüssel umgewandelt und somit erfolgt die Autorisierung.  Bitte beachten Sie, daß der HWK bzw. dessen Seriennummer nur einmalig zu einem Lizenzschlüssel umgewandelt und somit verwendet werden kann.  Der HWK bleibt auch nach dessen Konvertierung für TIS 2000 weiterhin voll funktionsfähig.

2.1.4.2  Windows 2003 Server:

Die Konvertierung des HWK auf einem PC mit Windows 2003 funktioniert nicht ordnungsgemäß.  Daher ist entweder eine Online-Registrierung oder eine Manuelle Registrierung per Fax erforderlich (siehe unten).

2.1.4.3  Softwareschlüssel Prozedur:

Ist kein HWK vorhanden wird der Nutzer automatisch nach Start der Anwendung durch den Registrierungsprozess geleitet.  Jedes der mit * markierten Felder muß mit den entsprechenden Daten befüllt werden.  Am Ende der Registrierung kann der Nutzer zwischen einer Online-Registrierung oder einer manuellen Registrierung wählen.  Im Falle einer Online-Registrierung werden die Nutzer-spezifischen Daten elektronisch zu einem zentralen Server geleitet, wo ein Lizenzschlüssel erzeugt und zurückgeschickt wird.  Dieser Lizenzschlüssel ist verschlüsselt und muss in die dafür vorgesehenen Felder des "Lizenzschlüssel" eingetragen werden.

Im Falle einer manuellen Registrierung erstellt das System automatisch ein PDF-Dokument.  Es kann als E-Mail Anhang zu der angegebenen Adresse geschickt werden, oder auch als Fax.  E-Mail ist jedoch die aufgrund der schnelleren Bearbeitung bevorzugte Form.

2.1.5  Abonnements (Subscriptionen) für eine Marke

Im Zuge der Anmeldung erhält der Händler 1 Abonnement für genau 1 Marke.  Für jede weitere Marke muss die Registrierung erweitert werden indem der vorgenannte Registrierungsprozess erneut durchlaufen wird.  Nachdem das Abonnement erweitert wurde muß sich der Nutzer ausloggen und die Anwendung erneut starten, damit die entsprechenden Anwendungen anwählbar sind.

2.1.6  Ändern des Abonnements

Reine Adressänderungen des Abonnenten dürfen ausschließlich per Fax übermittelt werden.  Dieses funktioniert derzeit nicht online.  Sofern ein Abonnement für zusätzliche Marken benötigt wird muß die vorhandene Registrierung geändert werden.  Hierzu bitte GlobalTIS starten und die Registermarke "Admin" wählen.  In diesem Bildschirm können Sie nun ein weiteres Abonnement auswählen und eine "Manuelle" oder "Online Registrierung" über die entsprechende Schaltfläche wählen.

Sie verfahren gleichermaßen, wenn Sie nicht genügend Lizenzen haben und deren Anzahl erhöhen müssen.  Diese Zahl begrenzt die Anzahl der Nutzer, die gleichzeitig am Server angemeldet sein können.  In den meisten Fällen erfordert dieses Feld keine Änderung.

2.1.7  Aktualisierung von GlobalTIS

Eine neue GlobalTIS Version auf DVD stellt ein Update einer bereits bestehenden Installation dar.  Legen Sie die DVD in das entsprechende DVD Laufwerk ein.  Die Aktualisierung startet bei eingeschalteter "Autoplay" Option automatisch.  Wenn diese Funktion auf Ihrem System nicht freigeschaltet ist, müssen Sie die setup.exe von Ihrem GlobalTIS DVD Wurzelverzeichnis ausführen.  GlobalTIS sucht nicht automatisch nach Updates auf der DVD beim Starten von GlobalTIS.

2.1.8  Starten der Anwendung

Die Anwendung kann durch das Klicken auf den entsprechenden Eintrag im Menü "Start" gestartet werden.  Der Standardeintrag steht unter "Start" - "Programme" - "GlobalTIS" - "GlobalTIS".

2.1.8.1  Einen Client starten

Für den Start der Client-Anwendung wird nur der "Internet Explorer" benötigt.  Im Adressfeld wird der Servername eingegeben, z. B. http://server-name:port/tis2web.

Beispiel: http://asterix:9080/tis2web

Der Port ist typischerweise "9080" oder Sie geben den Wert ein, der auch während der Serverinstallation bei "Tomcat" Ports eingegeben wurde.
Die so zusammengesetzte Adresse sollten Sie sich als Lesezeichen in den "Favoriten" Ihres Browsers sichern.  GlobalTIS sollte nur mit Microsoft Internet Explorer genutzt werden.

2.1.8.2 Server starten

Sobald Sie einen Server starten müssen Sie sich mit einem Nutzernamen anmelden.  Beim ersten Start nach der Installation müssen Sie einen neuen Nutzer Ihrer Wahl anlegen.  Bei jedem späteren Start können Sie diesen auswählen oder auch einen neuen eingeben.

2.1.9  Java Runtime Umgebung

Diese wird benötigt für die Module

  • Übertragung Diagnosesoftware
  • Steuergeräteprogrammierung (SPS)
  • Zugangsberechtigung
  • Schnappschuß
  • Tech2 Ansicht
  • Techline Druck

Ist auf dem PC bereits die Javaversion 1.4x oder höher installiert, wird die Java Anwendung beim Starten der GlobalTIS Module automatisch heruntergeladen:   Dies ist ein einmaliger Vorgang bzw. wird erneut durchgeführt wenn eine neuere Version auf dem Server verfügbar ist.
Ist keine oder eine ältere Java Version installiert, muß zuerst die neue Version installiert werden.  Die Java Version kann in der GlobalTIS Hilfe heruntergeladen werden: Allgemein -  Anwendungen herunterladen

2.1.10  Java Proxy Einstellungen

Falls die Java Anwendung nicht startet, kann eine Änderung in der Konfiguration der "Java Web Start" Anwendung erforderlich sein.  Die Proxy Einstellungen müssen deaktiviert werden.
'Datei' - 'Einstellungen' - 'Allgemein'.  Wählen Sie "Keine" für Proxies

2.2  Bildschirmeinstellungen

Um die Anwendung effizient und einfach zu bedienen sind optimale Bildschirmeinstellungen von höchster Wichtigkeit.  Zwei Einstellungen sind hierfür besonders zu beachten:

Wählen Sie eine angemessene Größe für die Dokumentanzeige .  Hierzu wählen Sie die Symbolfläche "Einstellungen" und ändern den Wert "Höhe Dokumentanzeige" bis das Dokument die komplette Seite einnimmt.

2.3  Spracheinstellungen

Die Bediensprache von GlobalTIS wird durch die entsprechende Einstellung im Browser vorgegeben.  Zum Wechseln der Sprache öffnen Sie das Browsermenu "Werkzeuge" und wählen "Internet Optionen" und die Schaltfläche "Sprache..."  Folgen Sie den Anweisungen.
Damit die Änderung wirksam wird muß der Nutzer GlobalTIS schließen und erneut starten.

2.4  Browsereinstellungen

Derzeit ist die Anwendung für eine Bildschirmauflösung von 1024 * 768 Pixel und die Textgröße "kleiner" ausgelegt. (Internet Explorer "Ansicht" - "Textgröße")  Werden andere Einstellungen bevorzugt wird gegebenenfalls eine Bildlaufleiste eingeblendet, um das gesamte Dokument ansehen zu können.

2.5  Sessionmanagement

In TIS 2000 gibt es die Funktionalität des "Sessionmanagements".  Es ermöglicht mehreren eingewählten Nutzern das Arbeiten in voneinander unabhängigen Fensterumgebungen.  Hierbei werden alle Einstellungen wie Fahrzeugtyp, Modelljahr und Suchkriterien beibehalten während ein anderer eingewählter Nutzer Informationen abfragt.

GlobalTIS verfügt über einen vergleichbaren Mechanismus unter Verwendung einer anderen Technik.  Wenn ein anderer Nutzer mit GlobalTIS arbeiten möchte aber bisherige Einstellungen nicht verwerfen möchte, öffnet er lediglich ein weiteres Fenster im Internet Explorer und wählt sich mit seiner Kennung ein.

2.6  Unterschiede zum TIS 2000

  • Die Bedienung erfolgt vorwiegend durch Schaltflächen.
  • Die Fahrzeugmarke muß nach der Installation von TIS2Web nur einmalig festgelegt werden.
  • Die Startseite umfaßt weniger Anwendungsschaltflächen als Sie dies bisher von TIS 2000 gewohnt waren.
Die Fahrzeugmarke muß nach der Installation von TIS2Web nur einmalig festgelegt werden.  Hierzu TIS2Web starten und in der rechten oberen Ecke auf das Symbol "Einstellungen" klicken um einen der Einträge auszuwählen.  Dieser Wert wird zukünftig verwendet, auch nach Beenden des Programms.

Weitere wichtige Erläuterungen finden Sie in der "Hilfefunktion" in der rechten oberen Ecke.


2.7  Bedeutung der Schaltflächen "Nächste" und "Zurück"

Die Schaltflächen "Nächste" und "Zurück" des Internet Explorers funktionieren für die Navigation innerhalb von GlobalTIS nicht.  Sie dürfen daher nicht verwendet werden.  Stattdessen verwenden Sie bitte das entsprechende Symbol in der Symbolleiste.

 

2.8  Übertragung Diagnosesoftware, Schnappschuß und Steuergeräteprogrammierung

Die bekannten Anwendungen wurden im wesentlichen aus TIS 2000 übernommen und in die Webtechnologie integriert.  Daher kann davon ausgegangen werden, daß es einer weiteren Einführung hier nicht bedarf.

2.9 Unterstützung

Im Falle von Problemen während der Installation oder der Arbeit mit der Anwendung können Sie sich an das Kundeninformationszentrum DAC wenden.

2.10 Rückmeldung

Für eventuell auftretende Fehler bei der Arbeit mit GlobalTIS und der Informationssuche steht Ihnen eine einfache Art der Rückmeldung an uns zur Verfügung.  Ein installiertes Rückmeldeformular ist bereits mit der entsprechenden Information vorausgefüllt, welches Sie lediglich mit der Fehlerbeschreibung und Ihren Kontaktdaten vervollständigen.  Mit der Schaltfläche "Senden" wird dann automatisch ein E-Mail erzeugt und an uns geschickt.  Ein separates Fax oder E-Mail ist nicht erforderlich.

3. Löschen/Deinstallation

Es gibt 2 mögliche Vorgehensweisen, um GlobalTIS von Ihrem PC zu löschen.

  1. Die Deinstallation kann mit "Start" - " Programme" - "GlobalTIS" - "Deinstallation" gestartet werden.
  2. GlobalTIS kann ebenso über 'Start' - Einstellungen - Systemsteuerung - Hinzufügen/Löschen entfernt werden.  Suchen Sie den Eintrag "GlobalTIS" und wählen die Schaltfläche "Ändern/entfernen".

Einige Informationen über die vorgenommenen Einstellungen während der Installation bleiben auf dem Rechner gespeichert.  Wie beispielsweise die Informationen, die Sie während der Installation und des Registrierungsprozesses eingegeben haben.  Sie können im Falle einer erneuten Installation verwendet werden und erleichtern diese somit.  Wählen Sie jedoch "Einstellungen ignorieren" während einer GlobalTIS Installation, gehen alle vorher gemachten Registrierungsdaten verloren und Sie müssen die Lizenzprozedur erneut durchlaufen.
