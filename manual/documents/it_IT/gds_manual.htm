Informazioni generali

Il GDS è ideato per l'uso da parte di personale di servizio addestrato per diagnosticare e riparare sistemi elettronici di automobili.  È stato fatto tutto il possibile per offrire un'informazione tecnica completa ed accurata, basata sull'informazione di servizio industriale disponibile al momento della pubblicazione.  Ci si riserva comunque il diritto di apportare modifiche in qualsiasi momento senza informativa.

Per familiarizzarsi con GDS e con le sue funzioni, nonché con le sue modalità d'uso, si prega di leggere le guide dell'utente prima di mettere in funzione il GDS.

GDS offre le seguenti funzioni:

  • Lettura di codici guasto diagnostici
  • Lettura di informazioni di identificazione ECU
  • Lettura e visualizzazione grafica del Parametro Elenco dati
  • Registrazione e memorizzazione di dati snapshot
  • Funzioni Controllo
  • Funzioni Configurazione e Azzeramento
  • Protocollo diagnostico di registrazione e memorizzazione
  • Memorizzazione di sessioni diagnostiche per veicolo
Importante:
La programmazione SPS dei controller veicolo non è supportata in GDS. Per la programmazione Pass Through SPS con il MDI usare l'applicazione SPS TIS2Web / GlobalTis.

3. Installazione GDS

Panoramica componenti GDS:

Il software GDS contiene i seguenti moduli che saranno installati inizialmente ed aggiornati frequentemente in certi periodi. [I]

  • GDS Base System
    Software di base GDS installato sul PC
  • Deliverable GDS:
    Pacchetti di dati diagnostici per modelli di veicolo che eseguono le procedure diagnostiche con i controller veicolo.
  • GDS Smart Start:
    Agente di applicazione che mette a confronto le versioni dei deliverable GDS Base System, GDS e il server del software MDI Manager tra TIS2Web/GlobalTIS e l'installazione locale. Nel caso siano disponibili nuovi aggiornamenti, vengono richiesti per il download.
  • Software MDI Manager (applicazione separata):
    Applicazione per aggiornamento e configurazione della periferica MDI (interfaccia diagnostico multiplo)


[I]

Installazione:

GDS può essere installato da un DVD di installazione separato o direttamente come download da TIS2Web / GlobalTIS.

Installazione dal DVD GDS:

  • Il PC dev'essere avviato in modalità Amministratore
  • Avvia eseguibile d'installazione nel menù iniziale del DVD
  • Seleziona la lingua dell'agente d'installazione
  • Configurazioni e dati GDS esistenti possono essere riutilizzati nel caso un'installazione GDS precedente sia rivelata sul PC.
  • Accetta il contratto di licenza (richiesto per procedere)
  • Seleziona la directory di installazione per GDS
  • Il processo di installazione GDS si avvia
  • Il software MDI Manager verrà installato nel caso non ci sia nessun software MDI Manager o ce ne sia uno vecchio installato sul PC. Per l'installazione iniziale del software MDI Manager, l'hardware MDI deve essere connesso durante il processo d'installazione. Per ulteriori dettagli fare riferimento al manuale MDI.
  • GDS può essere avviato dal simbolo del desktop o attraverso Start/Programmi/GDS dopo il completamento del processo d'installazione. 
  • Se richiesto configura il firewall per consentire la comunicazione di GDS con la periferica MDI.
  • GDS Smart Start si avvia, il processo di aggiornamento mostrato verrà eseguito prima che GDS venga avviato.

Installazione da TIS2Web / GlobalTIS:


[II]

Importante: Durante l'installazione da TIS2Web circa 140 MB di dati vengono scaricati da internet.
  • Il PC deve funzionare in modalità Amministratore
  • Avvia GDS sull'Home Page TIS2Web / GlobalTIS [II]
  • GDS Smart Start viene scaricato dal server TIS2Web, a seconda della connettività internet questo potrebbe impiegare un momento.
  • Seleziona la directory di installazione per GDS.
  • Inserisci la password internet (solo se viene usato un server proxy per accedere ad internet)
  • Se richiesto configura il firewall per consentire la comunicazione di GDS con la periferica MDI.
  • Quando l'installazione GDS Smart Start comincia, potrebbe impiegare un momento.
  • Avvio di GDS Smart Start, il processo di aggiornamento di GDS verrà eseguito. [III]
  • Quando il download dei file di installazione è finito, la versione GDS e la versione software MDI Manager vengono mostrate e possono essere scaricate accettando il dialogo richiesto. Con il pulsante Mostra progresso del download, la barra di stato del progresso di aggiornamento può essere mostrata.
  • Il software MDI Manager verrà installato nel caso non ci sia un software MDI Manager installato sul PC o se esiste una versione più vecchia del software MDI Manager. Per l'installazione iniziale del software MDI Manager l'hardware MDI deve essere connesso durante il processo d'installazione. Per ulteriori dettagli fare riferimento al manuale MDI.
[III] [IV]

Aggiornamento di GDS via Smart Start:

Lancia GDS da TIS2Web/GlobalTIS o come applicazione standalone tramite il menu Start di Windows. Prima del caricamento di GDS, Smart Start viene lanciato per il confronto delle seguenti versioni dell'installazione GDS PC con il contenuto sul server TIS2Web / GlobalTIS:

  • GDS Smart Start
  • GDS Base System
  • MDI Manager

Smart Start richiede nuovi aggiornamenti all'utente per il download, se nuove versioni sono disponibili sul server. [IV]

 
Importante: I processi di aggiornamento descritti richiedono che il PC funzioni sempre in modalità Amministratore.

Rollback dell'installazione di GDS Base System

L'installazione di GDS può essere riportata alla versione installata precedentemente, nel caso un aggiornamento di GDS causi problemi operativi.

Per eseguire la funzione di rollback, vai su START ->PROGRAMMI-> GDS -> GDS ROLLBACK. Un prerequisito della funzione rollback è che una versione di GDS più vecchia dev'essere installata sul PC.

Sistema di controllo licenza GDS (licenza)

Il sistema di controllo licenza GDS impedisce l'uso non autorizzato di GDS. Per rinnovare la licenza GDS è necessario lanciare GDS da TIS2Web o GlobalTIS. La licenza GDS deve essere rinnovata dopo 7 giorni se l'applicazione è stata lanciata esclusivamente in modalità standalone durante tale periodo. 
Se la licenza GDS è scaduta, un messaggio di errore chiede all'utente di lanciare l'applicazione GDS da TIS2Web/GlobalTIS per rinnovare la licenza.

Scarica e aggiorna i deliverable di GDS

I deliverable di GDS sono applicazioni diagnostiche che controllano la comunicazione e i servizi diagnostici tra GDS e il controller veicolo. I deliverable di GDS sono rilasciati per modello e saranno aggiornati stabilmente su TIS2Web / GlobalTIS.

GDS Smart Start identifica automaticamente se nuove o aggiornate versioni di deliverable sono disponibili su TIS2Web / GlobalTIS quando il processo di aggiornamento è innescato. I nuovi deliverable devono innanzitutto essere scaricati su una directory locale del PC prima di poter essere installati in GDS. Gli aggiornamenti di deliverable già installati saranno scaricati automaticamente e dovranno solo essere installati nel database principale di GDS.


Download iniziale e aggiornamento dei deliverable GDS

  • Lancia GDS da TIS2Web/GlobalTIS o come applicazione standalone tramite il menu Start di Windows.
  • Seleziona il pulsante Aggiungi nuovo pacchetto diagnostico dalla schermata login di GDS.
  • Premi il pulsante Aggiorna elenco deliverable dal server. [V-1]. Pacchetti disponibili sul server ma non ancora scaricati sono mostrati con il simbolo. 
  • Segna e scarica i pacchetti deliverable [V-2]. La funzione filtro aiuta a cercare deliverable specifici. I pacchetti deliverable scaricati sono mostrati con il simbolo. 
  • Installa i pacchetti scaricati nel database principale GDS; i pacchetti installati vengono mostrati nella finestra di applicazione di destra [V-3].


[V]

Aggiorna deliverable GDS installati nel database principale:

  • Clicca sul pulsante Aggiorna pacchetto diagnostico nello schermata login GDS [VI-1]. GDS mette a confronto le versioni dei deliverable GDS installati con i deliverable disponibili sul server e scarica i pacchetti aggiornati.
  • I pacchetti scaricati vengono mostrati per l'installazione nella finestra in basso a sinistra. Seleziona il pulsante Installa o Installa tutti per cominciare l'installazione. [VI-2].


[VI]:

Aggiornamento deliverable rollback

Il processo di rollback è in generale lo stesso processo descritto nel capitolo iniziale Scarica e aggiorna i deliverable di GDS

  • Seleziona il pulsante Aggiungi nuovo pacchetto diagnostico dalla schermata login di GDS.

Disinstallazione di GDS:

GDS può essere disinstallato avviando il seguente programma:

START -> PROGRAMMI -> GDS ->GDS DISINSTALLAZIONE

Cambia la lingua di GDS / Impostazioni internazionali:

GDS usa le impostazioni di lingua del sistema operativo del PC. Non c'è nessun menù in GDS per cambiare le impostazioni di lingua. Si prega di effettuare i seguenti passi per cambiare la lingua del GDS.

  • Apri il PANNELLO DI CONTROLLO di Windows con questa sequenza: START -> IMPOSTAZIONI -> PANNELLO DI CONTROLLO
  • Apri il menù OPZIONI INTERNAZIONALI E DELLA LINGUA
  • Seleziona il paese desiderato in STANDARD E FORMATI
  • Aprire il menu AVANZATE. Seleziona le impostazioni di lingua desiderate nella finestra a tendina per PROGRAMMI NON UNICODE
  • Configura le OPZIONI INTERNAZIONALI per cambiare le unità mostrate in GDS.

4. Guida introduttiva

AVVISO:
Prima di eseguire GDS, è importante osservare le seguenti precauzioni:

  • Usare una versione antiquata potrebbe danneggiare i moduli veicolo. Il GDS e l'interfaccia diagnostico multiplo devono avere il software più recente.
  • Assicurati che la batteria del veicolo sia completamente carica Il voltaggio della batteria per GDS dovrebbe essere tra 12 e 14 volt.
  • Un caricabatteria però non deve essere connesso al veicolo durante l'uso del MDI.
  • Assicurati che le connessioni dei cavi siano fissate. Un cavo disconnesso POTREBBE causare un errore del controller.
  • Nel caso usi un computer portatile/altra periferica display (PDA ecc.) per GDS, assicurati che l'alimentatore sia correttamente connesso. Se alimentato da AC e il cavo di alimentazione si disconnette, potrebbe interrompere la programmazione e causare danni al modulo di controllo. Se il portatile funziona con la sua alimentazione interna (batterie), allora assicurati che sia sufficientemente carica per completare il processo di GDS.
AVVISO
Se dev'essere programmato un ECU, dev'essere installato nel veicolo prima di iniziare questo processo. Assicurati che la batteria del veicolo sia completamente carica.
Importante
Allo scopo di ridurre il potenziale per la perdita di segnale, il MDI dovrebbe essere configurato con l'opzione di comunicazione più stabile alla vostra posizione.Potete scegliere tra USB, Ethernet o Ethernet Wireless.

Procedura di avvio di GDS:

GDS può essere avviato attraverso TIS2Web/GlobalTIS cliccando sull'icona GDS. Oppure, GDS può essere avviato in modalità stand-alone attraverso il menù Start di Windows.

  1. Per avviare clicca sul simbolo GDS dal menù principale TIS2Web/GlobalTIS. Se si avvia GDS in modalità stand-alone, clicca su START/PROGRAMMI/GDS
  2. Avvio di GDS Smart Start, il processo di aggiornamento mostrato verrà eseguito. Se una nuova versione di GDS o MDI Manager è disponibile sul server, verranno richiesti il download e l'installazione sul PC. Clicca Continua, per avviare il download o l'installazione [IV]. Se GDS viene avviato in modalità stand-alone, nessun aggiornamento è possibile.
  3. La Pagina Login di GDS viene caricata.

5. Navigazione generale

Menù a tendina 

Ci sono cinque menù a tendina disponibili da GDS [VII]

Nota: Questi menù saranno solo selezioni di menù per le versioni per PC di GDS. Ci sarà anche un formato touch screen, in cui si potrà accedere ai menù con un pulsante pop-up.

  • Il menù File ha la voce Esci per chiudere GDS.
  • Il menù Sessione ha due selezioni: Salva il log di sessione corrente e Aggiungi commenti alla sessione...
    Aggiungi commenti alla sessione...: l'informazione può essere inserita dalla tastiera del tuo computer e salvata nella Gestione del log di sessione.
  • La Selezione Strumenti ha cinque selezioni: Avvia traccia, Riproduzione snapshot, Crea rapporto, Mostra stato del download..., Amministrazione utente
    Nuovi utenti possono essere creati cliccando sul simbolo alla pagina Amministrazione utente  .
  • Finestra: Nel menù Preferenze, può essere selezionata la configurazione dello schermo, ad es. modalità schermo intero, modalità PC o touch screen.
  • Il selezione Guida ha tre selezioni:
    Sommario Guida mostra tutti i file guida GDS disponibili.  Informazioni su GDS mostra la versione di GDS e Build ID.  Mostra la versione MDI mostra versioni software e hardware di MDI.  Ti potrebbe essere richiesta questa informazione quando contatti il nostro Customer Support Center.


[VII]

Barra degli strumenti dell'applicazione diagnostica

Ci sono diverse selezioni della barra degli strumenti che diventano disponibili dopo essere arrivati alla visualizzazione della navigazione di sistema. [VIII]

  • Il Registra snapshot (cerchio rosso), è disponibile solo dopo essere arrivati alla visualizzazione della navigazione di sistema, ed è usato per registrare uno snapshot del veicolo. Durante la registrazione di uno snapshot, questa selezione non è disponibile.
  • Il Interrompi la registrazione di snapshot (quadrato blu) appare mentre si sta registrando uno snapshot del veicolo. Questa opzione viene selezionata per interrompere la registrazione dello snapshot e salvare il file.
  • Il simbolo Screenshot viene selezionato per acquisire il contenuto della schermata mostrata e memorizzarla in una posizione sul computer.
  • Il Zoom indietro aumenta le dimensioni del carattere. L'immagine mostrata viene ridotta ogni volta che si clicca sul mouse.
  • La selezione di Zoom avanti aumenta le dimensioni del carattere. L'immagine mostrata viene aumentata ogni volta che si clicca sul mouse.
  • Il pulsante Esci chiude l'applicazione diagnostica attiva.


[VIII]

Tasti freccia

I tasti freccia sono situati in fondo all'elenco parametri. I tasti freccia possono essere usati per la seguente navigazione:


[IX]:

  • La freccia Sposta in su può essere usata per spostare in su una cartella nell'elenco parametri ogni volta che viene selezionata.
  • La freccia Sposta in giù può essere usata per spostare in giù una cartella nell'elenco parametri ogni volta che viene selezionata.

Altri tasti

Anche altri tre tasti sono situati sul fondo di tutte le schermate. Questi tasti possono essere usati per le seguenti funzioni:

  • Il Avvia ti permette di avviare un'applicazione.
  • Il Azione precedente può essere usato per spostarsi alla schermata precedente. Questa selezione sarà disponibile solo quando c'è una schermata a cui ritornare.
  • Il Prossima azione può essere usata per spostarsi alla schermata seguente. Questa selezione sarà disponibile solo se tutte le selezioni necessarie sono state fatte nella schermata corrente.

Inserire informazione veicolo/MDI

1. Dal menù principale di GDS, seleziona un utente per il login. Se non è stato creato nessun utente, può essere creato un nuovo utente o può essere selezionato l'utente di default. (Vedi sezione Menù a tendina)

2. Dopo che la tua scelta di accesso è stata selezionata, verrà mostrata la schermata Menù di identificazione. Da questa schermata sono disponibili diverse scelte:


[X]

  • [X-1] Selezione MDI: Tutti i MDI disponibili sulle varie reti nella concessione vengono mostrati. Puoi selezionare il MDI che stai usando per effettuare la diagnostica del veicolo. Per la connessione MDI, evidenzia un MDI disponibile cliccandoci sopra e seleziona Connetti. Seleziona Disconnetti per disconnettere il MDI. Allega il nuovo MDI al veicolo e seleziona Connetti. Seleziona Aggiorna per verificare l'esistenza di nuovi MDI sulle reti disponibili. Per continuare in GDS senza alcun MDI connesso, attiva la casella di spunta Naviga senza MDI.
  • [X-2] Voce VIN: Il campo del numero identificazione veicolo per immissione manuale o identificazione VIN automatica dal controller veicolo. (connessione MDI richiesta). Ulteriori commenti associati a questo VIN / veicolo possono essere aggiunti nel campo di testo sottostante.
  • [X-3] Sezione Identificazione veicolo: seleziona Marca, Modello e Anno modello prima di continuare.
  • [X-4] Cronologia diagnostica del veicolo: Se un veicolo è stato diagnosticato precedentemente con questo sistema GDS, il VIN e l'informazione associata saranno memorizzati nella cronologia del veicolo. Seleziona il veicolo da diagnosticare dall'elenco e continua direttamente al Home page veicolo. Una funzione di filtro del testo gratuita è disponibile

3. Il Home page veicolo verrà mostrato. From this screen various vehicle data is displayed including: entered vehicle information, previous vehicle diagnosis, session comments, session log reports, and stored data

Home page veicolo

Il Home page veicolo consiste di diverse aree:

[XI-1]: Il Informazione Veicolo elenca tutte le informazioni sul veicolo, comprese Marca, Modello, Anno modello ecc.
Questo elenco viene aggiornato quando il veicolo è diagnosticato e verrà salvato con ogni sessione. Il Elimina selezione opzioni veicolo viene usato per eliminare la selezione del contenuto di un veicolo precedente. Evidenzia la riga da eliminare dall'Elenco del contenuto del veicolo e seleziona Elimina attrezzature.

[XI-2]: Commenti di sessione viene usato per aggiungere commenti utente in formato libero su una sessione diagnostica. Clicca su Aggiungi un commento per aggiungere ulteriori informazioni riguardanti una sessione. L'informazione può essere inserita sotto le aree Utente, Dati o Commento dalla tua tastiera. Per eliminare commenti precedenti, evidenzia l'informazione sotto le aree Utente, Dati o Commento e clicca su Elimina commento selezionato.

[XI-3]: Rapporti log di sessione può essere usato per una stampa dell'informazione diagnostica del veicolo. Visualizzazione completa fornisce un rapporto completo di tutta la diagnostica effettuata sul veicolo. Rapporto semplice fornisce un riepilogo diagnostico del veicolo.

[XI-4]: Dati Snapshot elenca tutti i dati snapshot memorizzati del veicolo. Verranno mostrati il nome utente, il nome file snapshot, la data in cui è stato fatto lo snapshot, e i commenti. Una riproduzione snapshot può essere mostrata evidenziando l'elemento e selezionando Rivedi. Elementi singoli possono essere eliminati evidenziando l'elemento e selezionando Elimina.
[XI-6]: Esporta salva una fotografia sul proprio PC locale. È possibile inviare i dati della fotografia per ulteriore analisi alla persona di riferimento.

[XI-5]: Il menù Applicazioni eseguite fornisce una cronologia della sessione diagnostica.

Prossima azione viene usato per continuare la diagnosi del veicolo selezionato.

Navigazione Sistema - Selezione Applicazione Diagnostica

Una volta selezionato Prossima azione dal Home page veicolo, la visualizzazione della Navigazione Sistema verrà mostrata [XII].

Seleziona il percorso di navigazione che vuoi investigare sul veicolo. La navigazione può essere effettuata con entrambe le frecce di navigazione; la freccia destra/sinistra apre/chiude una sottocartella, il pulsante Avvio avvia l'applicazione diagnostica selezionata. La navigazione è possibile anche con il mouse cliccando sui simboli + dell'albero di navigazione o con i tasti freccia della tastiera del PC.

Continua la selezione di cartelle per il percorso desiderato fino a raggiungere un'applicazione diagnostica, che ha un'icona di applicazione.

Riproduzione Snapshot

La funzione Riproduzione Snapshot consente la revisione di dati snapshot del veicolo registrati in precedenza. Questa informazione può essere usata per l'analisi dei problemi del veicolo e per la rilevazione di condizioni di errore trigger. La funzione grafica può essere usata per mettere a confronto parametri per vedere se una componente funziona correttamente. Gli snapshot registrati possono essere selezionati per la visualizzazione con i tre seguenti metodi:

  • Selezionando un file snapshot dal Home page veicolo e cliccando sulla Selezione revisione.
  • Cliccando due volte sullo snapshot selezionato dal Home page veicolo.
  • Entrando nella selezione Strumenti dalla selezione Menù e selezionando Riproduzione snapshot. Poi, apri la Gestione Snapshot. Fai doppio click sul file snapshot che vuoi visualizzare o clicca sulla selezione Carica Snapshot e seleziona un file snapshot.

Indipendentemente dal metodo di selezione, una finestra pop-up File caricato verrà mostrata dalla schermata Riproduzione Snapshot. Seleziona OK dalla finestra pop-up.

Funzioni Riproduzione Snapshot [XIX-1]

Ci sono diverse funzioni di riproduzione snapshot che sono usate per riprodurre snapshot registrati

La funzione Play viene usata per riprodurre lo snapshot in avanti cliccando su questa selezione.   Un segnalibro si sposterà attraverso lo schermo quando questa selezione viene usata.   Tutte le selezioni in quest'area diventano grigie quando sono attive.
La funzione Pausa viene usata per mettere in pausa la riproduzione dello snapshot. Quando questa selezione viene cliccata una seconda volta, la riproduzione dello snapshot ricomincerà dal punto in cui era stata interrotta.
La funzione Stop viene usata per interrompere la riproduzione dello snapshot. Lo snapshot si avvierà dall'inizio, se Play viene selezionato dopo aver usato la funzione Stop.

Controllo Tempo [XIX-2]

Controllo tempo viene usato per controllare la riproduzione snapshot. Questa funzione consente la selezione di una specifica cornice temporale, la presentazione di diverse visualizzazioni ed il controllo della velocità di presentazione. Le seguenti funzioni possono essere selezionate direttamente sotto Controllo tempo.

La selezione Segnalibro ti consente di aggiungere un commento specifico nel display corrente e ritornare su questo punto in futuro. Quando viene selezionata questa opzione, una finestra pop-up viene mostrata. Inserisci un nome del segnalibro e seleziona OK.
Il Notepad può essere selezionato per aggiungere commenti al file di riproduzione snapshot nel punto di tempo corrente. Quando viene selezionata questa opzione, una finestra pop-up viene mostrata. Inserisci un commento per il file snapshot e seleziona OK.
Il Carica Snapshot (cartella blu) apre un file snapshot e lo prepara per ulteriore analisi. Se si desidera visualizzare una foto esportata, è necessario decomprimere prima l'archivio *.zip.  Questa funzione è disponibile anche cliccando su un file snapshot disponibile dal Gestione Snapshot.
La freccia in giù mostra tutte le opzioni Controllo tempo quando viene selezionata. Anche opzioni singole possono essere selezionate cliccando sull'elemento desiderato.
La velocità di presentazione snapshot può essere controllata selezionando questa opzione.   Possono essere selezionate velocità basse di 1/8X, 1/4X, e 1/2X.   Anche la velocità in tempo reale può essere selezionata a 1.   Anche velocità più alte di 2X, 4X e 8X possono essere ottenute usando questa selezione.

Selezioni di visualizzazione [XIX-3]

Ogni selezione di visualizzazione offre un modo diverso di mostrare i dati del veicolo per l'analisi. Le seguenti funzioni possono essere selezionate sotto Visualizzazioni disponibili:

  • Visualizzazione Elenco Parametri - mostra informazione in arrivo dal file snapshot. Il valore corrente dei parametri viene mostrato sullo schermo.
Importante
Il nome del parametro verrà mostrato nella lingua originale usata per l'acquisizione se nessuna traduzione è disponibile per il nome.
  • Visualizzazione Log - contiene tutta l'informazione del file snapshot. Tutti i valori del parametro vengono elencati nell'ordine in cui sono stati registrati. Parametri specifici possono essere aggiunti o rimossi da una watchlist cliccando sull'elemento appropriato.
  • Visualizzazione Stato - mostra i valori di tutti i parametri in una specifica cornice temporale. Questi valori possono essere usati per il confronto con la visualizzazione trigger.
  • Visualizzazione Trigger - mostra tutti i trigger del sistema e dell'utente definiti. Questa funzione può essere usata quando c'è una combinazione di fattori implicati in un problema diagnostico.  Quando si incontra una condizione Trigger, verrà mostrato sullo schermo un triangolo di segnalazione.
    • Carica i trigger - Ti consente di caricare un file contenente definizioni trigger.
    • Salva i trigger - Ti consente di salvare il trigger sul disco rigido del tuo computer.
    • Rimuovi Trigger - Ti consente di rimuovere il trigger corrente. Non sarà più presente.
    • Modifica Trigger - mostra una finestra pop-up dove puoi modificare i limiti e i parametri contenuti in un trigger.
    • Aggiungi Trigger - mostra una finestra pop-up per configurare un trigger.
  • Visualizzazione DTC - Dopo che la funzione Play e questa opzione sono state selezionate, questa funzione apre l'elenco di DTC registrati.
  • Visualizzazione Grafico di Riga - mostra dati che possono essere visualizzati sull'asse temporale. Da questa visualizzazione, parametri specifici possono essere selezionati per la rappresentazione grafica controllando la casella sotto Selezionato. In fondo al grafico ci sono quattro funzioni: Zoom avanti (aumenta dimensioni), Zoom indietro (riduci dimensioni), Segnalibri e Metti in pausa Scorrimento. Si può accedere a ciascuna di queste funzioni cliccandoci sopra.

7. RISOLUZIONE DEI PROBLEMI

Problemi di Connessione MDI

Se le connessioni MDI sono disgiunte o non correttamente connesse al veicolo/computer, una finestra pop-up verrà mostrata. Effettua i seguenti passi:

  • Assicurati che il software Gestione MDI non sia in esecuzione mentre si sta eseguendo GDS.
  • Assicurati che il MDI sia correttamente connesso al veicolo e al computer
  • Usa un riconosciuto "idoneo cavo USB" da un'altra unità MDI, se disponibile. Effettua tutte le connessioni e riprova.
  • Usa un riconosciuto "idoneo MDI cavo di connessione DLC" da un'altra unità MDI, se disponibile. Effettua tutte le connessioni e riprova.
  • Se la condizione persiste, contatta Customer Support.
  • Assicurati che il firewall non blocchi la comunicazione GDS/MDI.

Conflitto Software GDS/MDI Manager

Il software GDS e il software MDI Manager non possono essere in esecuzione allo stesso tempo. Se i due programmi software sono eseguiti allo stesso tempo, potrebbe derivare un conflitto di software. La selezione Dettagli fornirà possibile informazione riguardante l'errore. Potrebbe esserti chiesto da Customer Support di fornire questa informazione se il problema persiste.

Se il conflitto di software continua ad avvenire, potrebbe essere mostrata una finestra pop-up "Si è verificato un errore irreversibile" il software di gestione MDI. Arresta il software di gestione MDI e torna al programma GDS. Se il problema persiste dopo essere entrati in GDS, contatta Customer Support.
