Informação geral

O GDS é concebido para a utilização de pessoal de serviço com formação, para o diagnóstico e a reparação de sistemas electrónicos automóveis.  Cada tentativa foi efectuada para fornecer informações técnicas completas e precisas, com base na informação disponível de fábrica, no momento da publicação.  No entanto, reserva-se o direito de efectuar alterações, em qualquer altura, sem aviso prévio.

Antes de começar a trabalhar com GDS e para se ambientar com GDS, o seu âmbito e respectivo funcionamento, leia atentamente os Manuais de Utilização .

GDS proporciona as seguintes capacidades:

  • Leitura do Código de Avaria de Diagnóstico
  • Leitura da informação de identificação do ECU
  • Leitura e apresentação gráfica da Lista de Dados do Parâmetro
  • Gravação e armazenamento dos dados de visualização geral
  • Funções de Controlo
  • Funções de Configuração e de Reposição
  • Protocolo de diagnóstico de gravação e armazenamento
  • Armazenamento das sessões de diagnóstico por veículo
Importante:
SPS a programação de controladores do veículo não é suportada no GDS Para a programação SPS Pass Through com MDI utilize a aplicação TIS2Web / GlobalTIS SPS.

3. Instalação GDS

GDS Descrição Geral do Componente:

O software GDS contém os seguintes módulos, os quais serão inicialmente instalados e actualizados com regularidade. [I]

  • GDS Base System
    Software base GDS instalado no PC
  • GDS Entregues:
    Pacotes de dados de diagnóstico para modelos de veículos que executam rotinas de diagnóstico com os controladores do veículo.
  • GDS Smart Start:
    O Agente Aplicacional que compara as versões de GDS Base System, GDS Entregas e MDI Manager o Servidor de software entre TIS2Web/GlobalTIS e a instalação local. No caso de existirem novas actualizações estas serão apresentadas.
  • MDI Manager Software (Aplicação separada):
    Aplicação para a actualização e configuração do MDI Dispositivo (Interface de Diagnóstico Múltiplo)


[I]

Instalação:

GDS pode ser instalado separadamente a partir do DVD ou directamente através de um downlaod a partir de TIS2Web / GlobalTIS.

Instalação a partir do DVD GDS:

  • O PC deverá ser iniciado no modo Admin
  • Inicie o executável de instalação, no menu Iniciar do DVD
  • Seleccione o idioma para o agente de instalação
  • A configuração e os dados existentes GDS podem ser novamente utilizados caso seja detectada uma anterior instalação GDS no PC.
  • Aceite o contrato de licença (necessário para continuar)
  • Seleccione o directório de instalação para GDS
  • GDS É iniciado o processo de instalação
  • O software MDI Manager será instalado, caso não esteja instalado o software MDI Manager no PC, ou este seja antigo. Para a instalação inicial do software MDI Manager é necessário que o hardware MDI esteja ligado durante o processo de instalação. Para mais informações consulte o manual MDI.
  • GDS pode ser iniciado através do símbolo no ambiente de trabalho, ou através de Iniciar/Programas/GDS, após ter concluído o processo de instalação. 
  • Caso seja necessário, configure a firewall para permitir a comunicação GDS com o dispositivo MDI.
  • GDS Smart Start é iniciado e o processo de actualização é apresentado antes de GDS ser iniciado.

Instalação a partir de TIS2Web / GlobalTIS:


[II]

Importante: Durante a instalação a partir de TIS2Web são transferidos, aproximadamente, 140 MB de dados através da internet.
  • O PC deve estar em modo Admin
  • Iniciar a Página Principal GDS em TIS2Web / GlobalTIS [II]
  • GDS Smart Start é transferido a partir do servidor TIS2Web e, dependendo da ligação à internet, poderá demorar alguns instantes.
  • Seleccione o directório de instalação para GDS.
  • Introduza a Palavra-Passe da Internet (no caso de o proxy ser utilizado para aceder à internet)
  • Caso seja necessário, configure a firewall para permitir a comunicação GDS com o dispositivo MDI.
  • Quando a instalação de GDS Smart Start é iniciada, poderá demorar alguns instantes.
  • Iniciar GDS Smart Start, o processo de actualização GDS será executado. [III]
  • Quando a transferência dos ficheiros de instalação terminar, a versão GDS e a versão do software MDI Manager será apresentada e pode ser transferida ao aceitar a caixa de diálogo que é apresentada. Pode ser apresentada a barra de estado do progresso de actualização, através do botão Apresentar o Progresso da Transferência.
  • O software MDI Manager será instalado, caso não esteja instalado o software MDI Manager ou exista uma versão antiga do software MDI Manager. Para a instalação inicial do software MDI Manager é necessário que o hardware MDI esteja ligado durante o processo de instalação. Para mais informações consulte o manual MDI.
[III] [IV]

Actualização de GDS através Smart Start:

Inicie o GDS quer a partir do TIS2Web/GlobalTIS ou como aplicação autónoma através do menu iniciar do Windows. Antes de carregar o GDS, o agente Smart Start (Arranque Inteligente) é iniciado e compara as versões seguintes da instalação GDS do PC com o conteúdo do servidor TIS2Web/GlobalTIS:

  • GDS Smart Start
  • GDS Base System
  • MDI Manager

Smart Start apresenta novas actualizações ao utilizador sempre que existam novas versões disponíveis no servidor. [IV]

 
Importante: Os processos de actualização descritos necessitam que o PC esteja sempre em modo Admin.

Reversão da Instalação de GDS Base System

A instalação GDS pode ser revertida para a versão previamente instalada, caso a actualização GDS provoque problemas operacionais.

Para executar a função de reversão, vá a INICIAR ->PROGRAMAS -> GDS -> GDS REVERSÃO. Um pré-requisito da função de reversão é a instalação anterior de uma versão antiga GDS no PC.

Sistema de Controlo de Licença GDS (Concessão)

O sistema de controlo da licença GDS previne a utilização não autorizada de GDS. Para a concessão GDS é necessário iniciar o GDS a partir do TIS2Web ou GlobalTIS. A concessão GDS necessita de renovação após 7 dias se a aplicação tiver sido executada durante esse período de tempo apenas no modo autónomo. 
Se a concessão GDS expirar, uma mensagem de erro avisa o utilizador para lançar a aplicação GDS a partir do TIS2Web/GlobalTIS para renovação da concessão.

Transferir a Actualizar os Materiais a entregar GDS

GDS Os materiais a entregar são aplicações de diagnóstico que controlam os serviços de comunicação e de diagnóstico entre GDS e o controlador do veículo. Os materiais a entregar GDS são lançados por modelo e estão permanentemente a ser actualizados em TIS2Web / GlobalTIS.

GDS Smart Start identifica automaticamente se uma nova actualização de versão dos materiais a entregar se encontra disponível em TIS2Web / GlobalTIS quando o processo de actualização é activado. Os novos materiais a entregar necessitam de ser transferidos para um directório local no PC, antes de puderem ser instalados em GDS. As actualizações de materiais a entregar já instaladas serão automaticamente transferidas e apenas necessitam de ser instaladas na Base de Dados (BD) Principal GDS.


Transferência inicial e Actualização dos Materiais a entregar GDS

  • Inicie o GDS quer a partir do TIS2Web/GlobalTIS ou como aplicação autónoma através do menu iniciar do Windows.
  • Seleccione o botão Adicionar um Novo Pacote de Diagnóstico a partir do Ecrã de Início de Sessão GDS.
  • Prima o botão Actualizar a lista de material e entregar a partir do servidor. [V-1]. No servidor disponível, mas ainda não transferido, os pacotes são apresentados com o símbolo. 
  • Marque e transfira os pacotes de materiais a entregar [V-2]. A função do filtro ajuda a efectuar uma procura por determinados materiais a entregar. Os pacotes transferidos são apresentados com o símbolo. 
  • Instale os pacotes transferidos na Base de Dados (BD) Principal GDS; os pacotes instalados são apresentados na janela da aplicação à direita [V-3].


[V]

Actualizar Materiais de entrega Instalados GDS na Base de Dados (BD) principal:

  • Clique no botão Actualizar Pacote de Diagnóstico no ecrã de Início de Sessão GDS [VI-1]. GDS compara as versões dos materiais GDS instalados com a dos materiais disponíveis no servidor, e transfere os pacotes actualizados.
  • Os pacotes transferidos são apresentados para instalar na janela inferior da esquerda. Seleccione o botão Instalar ou Instalar todos para iniciar a instalação. [VI-2].

Reversão da Actualização do Material a entregar

No geral, o processo de reversão é o mesmo processo descrito no capítulo Transferência e Actualização Inicial dos Materiais a entregar GDS

  • Seleccione o botão Adicionar um Novo Pacote de Diagnóstico a partir do Ecrã de Início de Sessão GDS.
  • Prima o botão Actualizar a lista de material e entregar a partir do servidor. [V-1]

Desinstalação de GDS:

GDS pode ser desinstalado ao iniciar o seguinte programa:

INICIAR -> PROGRAMAS -> GDS ->GDS DESINSTALAÇÃO

Alterar o Idioma de GDS / Definições Regionais:

GDS utiliza o idioma definido no sistema operativo do PC. Não existe qualquer menu no GDS para alterar a definição do idioma. Execute os seguintes passos para alterar o idioma de GDS.

  • Abra o PAINEL DE CONTROLO do Windows pela seguinte sequência: INICIAR -> DEFINIÇÕES -> PAINEL DE CONTROLO
  • Abra o menu OPÇÕES REGIONAIS E DE IDIOMA
  • Seleccione o país pretendido em PADRÕES E FORMATOS
  • Abra o menu AVANÇADAS. Seleccione o idioma pretendido na caixa pendente para os PROGRAMAS UNICODE
  • Configure as OPÇÕES REGIONAIS para alterar a apresentação das unidades em GDS.

4. Introdução

NOTA:
Antes de executar GDS, é importante que as seguintes precauções sejam observadas:

  • A utilização de uma versão desactualizada poderá provocar danos nos módulos do veículo. O GDS e a Interface de Diagnóstico Múltiplo devem possuir o software mais recente.
  • Certifique-se de que a bateria do veículo se encontra totalmente carregada A tensão da bateria para GDS deverá ser entre 12 e 14 volts.
  • Não deve ser ligado um carregador da bateria ao veículo durante a utilização de MDI.
  • Certifique-se de que as ligações do cabo são seguras. Um cabo desligado PODE provocar uma falha do controlador.
  • Quando utilizar um computador portátil/outro dispositivo (por exemplo, um PDA) no GDS, certifique-se de que a fonte de alimentação se encontra correctamente ligada. Se o cabo de alimentação for desligado poderá provocar a interrupção do programa e provocar danos no módulo de controlo. Se o computador portátil estiver a funcionar com a fonte de alimentação externa (baterias), certifique-se de que esta se encontra convenientemente carregada para terminar o processo GDS.
NOTA
Se uma ECU estiver para ser programada, deverá ser instalada no veículo antes de iniciar este processo. Certifique-se de que a bateria do veículo se encontra totalmente carregada
Importante
Com o objectivo de reduzir a perda de sinal, o MDI deve ser configurado para a opção de comunicação mais estável da sua zona.Pode escolher entre USB, Ethernet e Ethernet sem fios.

Procedimento de Arranque GDS:

GDS pode ser iniciado através de TIS2Web/GlobalTIS ao efectuar um clique no ícone GDS. Em alternativa, GDS pode ser iniciado em modo autónomo através do menu Iniciar do Windows.

  1. Para iniciar clique no símbolo GDS a partir do menu principal de TIS2Web/GlobalTIS. Se iniciar GDS em modo autónomo, clique em INICIAR/PROGRAMAS/GDS
  2. Ao Iniciar GDS Smart Start, o processo de actualização apresentado será executado. Se estiver disponível uma nova versão no servidor de GDS ou MDI Manager, a mesma será apresentada para transferência e instalação no PC. Clique em Contínua, para iniciar a transferência ou a instalação [IV]. Não é possível efectuar uma actualização, caso GDS seja iniciado em modo autónomo.
  3. É carregada a Página de Início de Sessão de GDS.

Barra de Ferramentas da Aplicação de Diagnóstico

Existem diversas selecções da barra de ferramentas, as quais ficam disponíveis assim que aceder à vista do sistema de navegação. [VIII]

  • O Gravar Visualização Geral (Círculo Vermelho), estará apenas disponível depois de aceder à vista do sistema de navegação e é utiliza-se na gravação de uma visualização geral do veículo. Esta selecção encontra-se indisponível enquanto uma visualização geral estiver a ser guardada.
  • O (Quadrado Azul) Parar a gravação da Visualização Geral aparece durante a gravação da visualização geral do veículo. Esta opção é seleccionada para parar a gravação da visualização geral e guardar o ficheiro.
  • O símbolo Captura de ecrã é seleccionado para capturar o conteúdo do ecrã e guardá-lo numa localização do computador.
  • O Reduzir aumenta o tamanho do tipo de letra. A imagem apresentada é reduzida de cada vez que clicar no rato.
  • A selecção Ampliar aumenta o tamanho do tipo de letra. A imagem apresentada é aumentada de cada vez que clicar no rato.
  • O botão Sair fecha a aplicação de diagnóstico activa.


[VIII]

Teclas de Seta

As teclas de seta encontram-se situadas na parte inferior da lista do parâmetro. As teclas de seta podem ser utilizadas para a seguinte navegação:


[IX]:

  • A Seta Deslocar para Cima permite deslocar uma pasta para cima na lista do parâmetro, de cada vez que é seleccionada.
  • A Seta Deslocar para Baixo permite deslocar uma pasta para baixo na lista do parâmetro, de cada vez que é seleccionada.
  • A Seta Deslocar para a Direita pode ser utilizada para abrir um nível na árvore de Navegação.
  • A Seta Deslocar para a Esquerda pode ser utilizada para fechar um nível na árvore de Navegação. Esta selecção não será apresentada se estiver no primeiro ecrã de um procedimento.

Outras teclas

Existem igualmente mais três teclas localizadas na parte inferior de todos os ecrãs. Estas teclas têm as seguintes funções:

  • A Iniciar permite iniciar uma aplicação.
  • A Retroceder Acção pode ser utilizada para retroceder para o ecrã anterior. Esta selecção apenas estará disponível quando existir um ecrã para onde possa retroceder.
  • A Próximo Acção pode ser utilizada para se deslocar para o próximo ecrã. Esta selecção apenas estará disponível caso tenham sido efectuadas selecções ecrã actual.

Introduzir Veículo/Informação MDI

1. Seleccione um utilizador para iniciar sessão a partir do Menu Principal GDS. Caso não tenha sido criado um utilizador, pode ser criado um novo utilizador ou seleccionar o utilizador predefinido. (Consulte a secção do menus suspensos)

2. Após ter seleccionado a sua escolha de acesso, será apresentado o ecrã do Menu de Identificação. Encontram-se disponíveis várias opções neste ecrã:


[X]

  • [X-1] Selecção MDI: São apresentadas todas as MDI disponíveis nas diversas redes do concessionário. Pode seleccionar a MDI que está a utilizar para efectuar os diagnósticos de veículos. Para a ligação MDI clique em cima para realçar uma MDI disponível, e seleccione Ligação. Seleccione Desligar para desligar MDI. Anexe o novo MDI ao veículo e seleccione Ligação. Seleccione Actualizar para pesquisar por novos MDI nas redes disponíveis. Para continuar em GDS sem nenhuma ligação MDI, active a caixa de verificação Navegar sem MDI.
  • [X-2] Entrada VIN: Campo para a entrada manual, ou automática, do Número de Identificação do Veículo do controlador do veículo. (MDI ligação necessária). Podem ser adicionados comentários adicionais associados a este VIN / Veículo, no campo de texto abaixo.
  • [X-3] Identificação do Veículo Secção: antes de prosseguir, seleccione a Marca, o Modelo e o Ano do Modelo.
  • [X-4] Histórico do Diagnóstico do Veículo: Se um veículo tiver sido previamente diagnosticado com este sistema GDS, o VIN e a informação associada serão guardadas no histórico do veículo. A partir da lista, seleccione o veículo a diagnosticar e siga directamente para Página Principal do Veículo. Encontra-se disponível uma função do filtro de texto livre

3. Será apresentado o Página Principal do Veículo. From this screen various vehicle data is displayed including: entered vehicle information, previous vehicle diagnosis, session comments, session log reports, and stored data

Página Principal do Veículo

O Página Principal do Veículo tem por base diversas áreas:

[XI-1]: O Informações do Veículo apresenta toda a informação do veículo, incluindo a Marca, Modelo, Ano do Modelo, etc.
A lista é actualizada assim que o veículo é diagnosticado e será guardada com cada sessão. O Elimina a selecção da opção do veículo é utilizado para eliminar uma selecção anterior do conteúdo do veículo. Realce a linha a eliminar, da Lista do Conteúdo do Veículo, e seleccione Eliminar Equipamento.

[XI-2]: Comentários da Sessão é utilizado para adicionar comentários do utilizador de uma forma livre sobre uma sessão de diagnóstico. Clique em Adicionar um Comentário para adicionar informações relativas à sessão. Aa informações podem ser introduzida no contexto do Utilizador, Dados ou Comentários, a partir do teclado. Para eliminar comentários anteriores, realce a informação das áreas do Utilizador, Dados ou Comentários e clique em Eliminar Comentário Seleccionado.

[XI-3]: Relatórios do Registo da Sessão pode ser utilizado para imprimir a informação do diagnóstico do veículo. Vista Completa fornece um relatório exaustivo sobre todos os diagnósticos efectuados ao veículo. Relatório Simples fornece um resumo do diagnóstico do veículo.

Reprodução da Visualização Geral

A função Reprodução da Visualização Geral permite rever previamente os dados gravados da visualização geral do veículo. Esta informação pode ser utilizada para analisar os problemas do veículo e para detectar condições de erro do activador. A função do gráfico pode ser utilizada para comparar parâmetros, com o objectivo de verificar se o componente está a funcionar correctamente. As gravações das visualizações gerais podem ser seleccionadas para uma visualização dos seguintes três métodos:

  • Seleccionar um ficheiro de visualização geral a partir de Página Principal do Veículo e clicar na selecção de Revisão.
  • Efectue um duplo clique na visualização geral seleccionada, a partir de Página Principal do Veículo.
  • Entrar na selecção Ferramentas a partir da selecção Menu e seleccionar Reprodução da Visualização Geral. Em seguida abra o Gestor da Visualização Geral. Efectue um duplo clique no ficheiro de visualização geral que pretende visualizar, na selecção Carregar Visualização Geral, e seleccione um ficheiro de visualização geral.

Independentemente do método de selecção, a janela emergente do Ficheiro Carregado será apresentada a partir do ecrã Reprodução da Visualização Geral. Seleccione OK na janela emergente.

Funções de Reprodução da Visualização Geral [XIX-1]

Existem diversas funções de reprodução da visualização geral, as quais são utilizadas na reprodução das visualizações gerais gravadas.

A função Reproduzir é utilizada na reprodução da visualização geral.   Quando esta selecção for utilizada, um marcador deslocar-se-á ao longo do ecrã.   Quando activas, todas as selecções nesta área ficarão cinzentas.
A função Pausa é utilizada para interromper a reprodução da visualização geral. Assim que clicar uma segunda vez sobre a selecção, a visualização geral irá novamente recomeçar no ponto onde tinha sido interrompida.
A função Parar é utilizada para parar a reprodução da visualização geral. A visualização geral irá começar do início, caso a função de reprodução seja utilizada após a função Parar.

Controlo de Tempo [XIX-2]

Controlo de Tempo é utilizado para controlar a reprodução da visualização geral. Esta função permite a selecção de um frame de tempo específico, a apresentação de diferentes visualizações e o controlo da velocidade do ecrã. As seguintes funções podem ser seleccionadas directamente sob Controlo de Tempo.

A selecção do Marcador permite-lhe adicionar um determinado comentário no ecrã actual e regressar posteriormente a este ponto. Quando este ponto é seleccionado é apresentada uma janela emergente. Introduza um nome do Marcador e seleccione OK.
O Bloco de Notas pode ser seleccionado para adicionar comentários ao ficheiro de reprodução da visualização geral, nesse preciso momento. Quando este ponto é seleccionado é apresentada uma janela emergente. Introduza um comentário para a visualização geral e seleccione OK.
A função Carregar Visualização Geral (pasta azul) abre um ficheiro de visualização geral e prepara-o para análises adicionais. Se pretender visualizar uma imagem instantânea exportada, descompacte primeiro o arquivo *.zip correspondente.  Esta função encontra-se igualmente disponível ao clicar num ficheiro de visualização geral disponível a partir de Gestor da Visualização Geral.
A Seta para Baixo apresenta todas as opções de Controlo de Tempo, quando seleccionada. As opções individuais podem ser igualmente seleccionadas através de um clique no item pretendido.
A Velocidade do Ecrã de Visualização Geral pode ser controlada a partir desta opção.   Podem ser seleccionadas as velocidades de 1/8X, 1/4X, e 1/2X.   A velocidade em tempo real pode ser seleccionada em 1 igualmente.   Velocidades mais elevadas de 2X, 4X e 8X podem igualmente ser alcançadas ao utilizar esta selecção.

Ver Selecções [XIX-3]

Cada Selecção de Vista proporciona diferentes formas de apresentação dos dados do veículo para análise. As seguintes funções podem ser seleccionadas sob Vistas Disponíveis.

  • Vista da Lista do Parâmetro - apresenta a informação recebida do ficheiro de visualização geral. O valor actual dos parâmetros é apresentado no ecrã.
Importante
O nome do parâmetro será apresentado no idioma original em que foi capturado, caso o nome não esteja traduzido.
  • A Vista do Registo contém toda a informação do ficheiro de visualização geral. Todos os valores dos parâmetros são apresentados pela ordem de gravação. Os Parâmetros específicos podem ser adicionados ou removidos da lista de observação efectuando um clique no item correspondente.
  • Vista do Estado - Apresenta os valores de todos os parâmetros num determinado período de tempo. Estes valores podem ser utilizados para comparar com os da vista do activador.
  • Vista do Activador - apresenta todos os activadores definidos do sistema e dos utilizadores. Esta função pode ser utilizada quando existe uma combinação de factores num diagnóstico.  Quando a condição do Activador for alcançada será apresentado um triângulo de atenção no ecrã.
    • Carregar Activadores - Permite carregar um ficheiro que contém definições de activadores.
    • Guardar Activadores - Permite guardar o activador para a unidade de disco rígido do computador.
    • Remover Activador - Permite a remoção do activador actual. Nunca mais será apresentado.
    • Editar Activador - apresenta uma janela emergente que lhe permite os limites e os parâmetros do activador.
    • Adicionar Activador - uma janela emergente é apresentada para configurar um activador.
  • Vista DTC - Depois da função Reproduzir ter sido seleccionada, esta função abre a lista de DTCs gravadas.
  • Vista do Gráfico de Linhas - os dados são visualizados sobre o eixo de tempo. Nesta vista podem ser seleccionados os parâmetros específicos para o gráfico, marcando a caixa abaixo Seleccionada. Na parte inferior do gráfico existem quatro funções: Ampliar (aumentar tamanho), Reduzir (diminuir tamanho), Marcadores, e Interromper Deslocação. Cada uma destas funções pode ser acedida efectuando um clique sobre ela.

7. RESOLUÇÃO DE PROBLEMAS

MDI Problemas de Ligação

Será apresentada uma janela emergente, no caso de se perderem as ligações MDI ou caso estas não estejam devidamente ligadas ao veículo/computador. Execute os seguintes passos:

  • Certifique-se de que o software do Gestor MDI não se encontra em execução ao mesmo tempo que o GDS.
  • Certifique-se de que o MDI se encontra devidamente ligado ao veículo e ao computador
  • Utilize um cabo USB adequado "e em bom estado" de outro computador MDI se disponível. Efectue todas as ligações e tente novamente.
  • Utilize um cabo de ligação DLC conhecido "e em bom estado MDI " de outro computador MDI se disponível. Efectue todas as ligações e tente novamente.
  • Se a situação persistir, contacte Customer Support.
  • Certifique-se de que a firewall não bloqueia a comunicação GDS/MDI.

GDS/MDI Manager Conflito de Software

O software GDS e o software MDI Manager não podem ser executados em simultâneo. Caso ambos os programas estejam a ser executados em simultâneo, é provável que ocorra um conflito de software. A selecção de Detalhes fornecerá possíveis informações sobre o erro. Poderá ser contactado pelo Customer Support para fornecer esta informação, caso o problema persista.

Se o conflito de software persistir, poderá ser apresentada uma janela emergente "Ocorreu um erro fatal" no software MDI. Encerre o software Manger MDI e volte para o programa GDS. Se ainda assim o problema persistir GDS, contacte o Customer Support.
