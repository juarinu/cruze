Remarques générales

Le GDS doit être utilisé par les employés de maintenance qualifiés pour effectuer des diagnostics et réparer les systèmes électroniques des automobiles.  Nous nous sommes efforcés de fournir des informations techniques complètes et exactes en fonction des informations de maintenance de l'usine disponibles à la date de publication.  Toutefois, des modifications peuvent être effectuées à tout moment sans préavis.

Pour vous familiariser avec GDS et ses fonctions, lisez les guides d'utilisation avant d'utiliser GDS.

GDS fournit les fonctions suivantes :

  • Lecture du code de dépannage des diagnostics
  • Lecture des informations d'identification ECU
  • Lecture et affichage graphique du paramètre de liste de données
  • Enregistrer et stocker les données de capture instantanée
  • Fonctions de contrôle
  • Fonctions de configuration et de réinitialisation
  • Enregistrer et stocker le protocole de diagnostics
  • Stockage des sessions de diagnostics par véhicule
Important:
La programmation SPS des contrôleurs de véhicule n'est pas prise en charge dans GDS. Pour la programmation SPS Pass Through avec la MDI, utilisez l'application TIS2Web / GlobalTIS SPS.

3. Installation de GDS

Présentation des composants GDS :

Le logiciel GDS contient les modules suivants qui seront installés initialement et mis à jour régulièrement à certaines périodes. [I]

  • GDS Base System
    Logiciel de base GDS installé sur le PC
  • Livrables GDS:
    Packages de données de diagnostics pour les modèles de véhicules qui exécutent les diagnostics avec les contrôleurs de véhicule.
  • GDS Smart Start:
    Agent application qui compare les versions de GDS Base System, livrables GDS et le serveur logiciel MDI Manager entre TIS2Web/GlobalTIS et l'installation locale. S'il existe des mises à jour, un message vous demande de les télécharger.
  • Logiciel MDI Manager (application distincte):
    Application pour la mise à jour et la configuration de l'unité MDI (Multiple Diagnostic Interface)


[I]

Installation :

GDS peut être installé depuis un DVD d'installation distinct ou directement en le téléchargeant depuis TIS2Web / GlobalTIS.

Installation depuis le DVD GDS :

  • Le PC doit être démarré en mode d'administration.
  • Lancez l'exécutable d'installation dans le menu de démarrage DVD.
  • Sélectionnez la langue de l'agent d'installation.
  • La configuration et les données existantes GDS peuvent être réutilisées si une installation précédente de GDS est détectée sur le PC.
  • Acceptez le contrat de licence (obligatoire pour pouvoir continuer).
  • Sélectionnez le répertoire d'installation de GDS.
  • L'installation de GDS démarre.
  • Le logiciel MDI Manager sera installé si aucun logiciel MDI Manager n'est installé sur le PC ou qu'un ancien logiciel s'y trouve. Pour l'installation initiale du logiciel MDI Manager, le matériel MDI doit être connecté au cours de l'installation. Pour plus d'informations, voir le manuel MDI.
  • GDS peut être lancé par le symbole de bureau ou via Démarrer/Programmes /GDS à la fin de l'installation. 
  • Si nécessaire, configurez le pare-feu pour permettre la communication GDS avec l'unité MDI.
  • GDS Smart Start démarre. La procédure de mise à jour affichée sera exécutée avant le démarrage de GDS.

Installation depuis TIS2Web / GlobalTIS:


[II]

Important: Au cours de l'installation depuis TIS2Web, 140 Mo de données environ sont téléchargées via Internet.
  • Le PC doit fonctionner en mode administration.
  • Démarrez GDS dans la page d'accueil TIS2Web / GlobalTIS. [II]
  • GDS Smart Start est téléchargé depuis le serveur TIS2Web ; selon la connexion Internet, l'opération peut prendre un certain temps.
  • Sélectionnez le répertoire d'installation de GDS.
  • Entrez le mot de passe Internet (uniquement si le serveur proxy est utilisé pour l'accès Internet).
  • Si nécessaire, configurez le pare-feu pour permettre la communication GDS avec l'unité MDI.
  • L'installation de GDS Smart Start peut prendre un certain temps.
  • Lorsque vous lancez GDS Smart Start, la mise à jour GDS est exécutée. [III]
  • Une fois le téléchargement des fichiers d'installation terminée, la version GDS et la version du logiciel MDI Manager s'affichent et peuvent être téléchargées en acceptant le message. Vous pouvez afficher la barre d'état de l'avancement de la mise à jour avec le bouton Afficher l'avancement du téléchargement.
  • Le logiciel MDI Manager sera installé si aucun logiciel MDI Manager n'est installé sur le PC ou si une ancienne version du logiciel MDI Manager existe. Pour l'installation initiale du logiciel MDI Manager, le matériel MDI doit être connecté au cours de l'installation. Pour plus d'informations, voir le manuel MDI.
[III] [IV]

Mise à jour de GDS via Smart Start.:

Lancer GDS à partir de TIS2Web/GlobalTIS ou, en tant qu'application Standalone, du menu Démarrer de Windows. Avant le chargement de GDS, l'agent Smart Start est lancé et compare les versions d'installation de GDS sur le PC avec le contenu TIS2Web / GlobalTIS Server suivants :

  • GDS Smart Start
  • GDS Base System
  • MDI Manager

Smart Start demande à l'utilisateur de télécharger les nouvelles mises à jour si de nouvelles versions sont disponibles sur le serveur. [IV]

 
Important: Les mises à jour décrites nécessitent de faire fonctionner le PC en mode administration.

Annulation de l'installation GDS Base System

Vous pouvez annuler l'installation de GDS pour restaurer la version précédente si la mise à jour de GDS génère des problèmes.

Pour annuler l'installation, accédez à DEMARRER ->PROGRAMMES -> GDS -> ANNULER GDS. Une ancienne version de GDS doit être installée sur le PC pour pouvoir effectuer l'annulation.

Système de gestion des licences GDS (Lease)

Le système de contrôle de licence GDS empêche l'utilisation non autorisée de GDS. Pour renouveler la licence GDS, GDS doit être démarré à partir de TIS2Web ou GlobalTIS. La licence GDS doit être renouvelée au bout de 7 jours si l'application n'est utilisée qu'en mode Standalone pendant cette durée. 
Dès que la licence GDS est expirée, un message d'erreur invite l'utilisateur à démarrer l'application GDS à partir de TIS2Web/GlobalTIS afin de renouveler la licence.

Téléchargement et mise à jour des livrables GDS

Les livrables GDS sont des applications de diagnostics qui contrôlent les services de communication et de diagnostics entre GDS et le contrôleur de véhicule.. Des livrables GDS sont publiés pour chaque modèle et seront mis à jour en permanence sur TIS2Web / GlobalTIS.

GDS Smart Start détermine automatiquement si de nouvelles versions ou des mises à jour des livrables sont disponibles sur TIS2Web / GlobalTIS lorsque la mise à jour a lieu.. Vous devez télécharger les livrables dans un répertoire local sur le PC avant de les installer dans GDS. Les mises à jour des livrables déjà installés seront téléchargées automatiquement et doivent simplement être installées dans la base de données principale GDS.


Téléchargement et mise à jour initiaux des livrables GDS

  • Lancer GDS à partir de TIS2Web/GlobalTIS ou, en tant qu'application Standalone, du menu Démarrer de Windows.
  • Sélectionnez le bouton Ajouter un nouveau package de diagnostics dans l'écran de connexion GDS.
  • Appuyez sur le bouton Mettre à jour la liste des livrables depuis le serveur. [V-1]. Sur le serveur, les packages disponibles mais pas encore téléchargés s'affichent avec le symbole. 
  • Marquez et téléchargez les packages de livrables. [V-2]. La fonction de filtrage permet de rechercher des livrables. Les packages de livrables téléchargés s'affichent avec le symbole. 
  • Installez les packages téléchargés dans la base de données principale GDS ; les packages installés figurent dans la fenêtre d'application de droite. [V-3].


[V]

Mise à jour des livrables installés GDS dans la base de données principale:

  • Cliquez sur le bouton Mettre à jour un package de diagnostics dans l'écran de connexion GDS. [VI-1]. GDS compare les versions des livrables GDS installés aux livrables disponibles sur le serveur et télécharge les packages mis à jour.
  • Les packages téléchargés à installer figurent dans la fenêtre inférieure gauche. Sélectionnez le bouton Installer ou Installer tout pour lancer l'installation. [VI-2].


[VI]:

Annuler la mise à jour des livrables

En règle générale, l'annulation est identique à celle décrite dans le chapitre Téléchargement et mise à jour initiaux des livrables GDS.

  • Sélectionnez le bouton Ajouter un nouveau package de diagnostics dans l'écran de connexion GDS.
  • Appuyez sur le bouton Mettre à jour la liste des livrables depuis le serveur. [V-1]
  • Marquez et téléchargez les anciennes versions de packages de livrables appropriées. La fonction de filtrage permet de rechercher des livrables.
  • Installez les packages téléchargés dans la base de données principale GDS.

Désinstallation de GDS:

Vous pouvez désinstaller GDS en lançant le programme suivant :

DEMARRER -> PROGRAMME -> GDS ->GDS DESINSTALLATION

Changement de la langue GDS / Paramètres régionaux:

GDS utilise le paramètre de langue du système d'exploitation du PC. GDS ne contient aucun menu pour changer le paramètre de langue. Procédez comme suit pour changer la langue de GDS.

  • Ouvrez le PANNEAU DE CONFIGURATION de Windows comme suit : DÉMARRER -> PARAMÈTRES -> PANNEAU DE CONFIGURATION
  • Ouvrez le menu OPTIONS REGIONALES ET LINGUISTIQUES
  • Sélectionnez le pays dans STANDARDS ET FORMATS
  • Cliquez sur l'onglet OPTIONS AVANCÉES Sélectionnez le paramètre de langue approprié dans la liste déroulante PROGRAMMES NON UNICODE.
  • Configurez les OPTIONS REGIONALES pour changer les unités affichées dans GDS.

4. Mise en route

MISE EN GARDE:
Avant d'utiliser GDS, il est important de tenir compte des informations suivantes :

  • L'utilisation d'une version obsolète peut endommager les modules du véhicule. GDS et Multiple Diagnostic Interface doivent correspondre au dernier logiciel.
  • Assurez-vous que la batterie du véhicule est complètement chargée. La tension de la batterie pour GDS doit être comprise entre 12 et 14 volts.
  • Aucun chargeur de batterie ne doit être connecté au véhicule lorsque vous utilisez la MDI.
  • Vérifiez que les câbles de connexion sont branchés correctement. Un câble déconnecté PEUT entraîner le dysfonctionnement du contrôleur.
  • Si vous utilisez un ordinateur portable/un autre périphérique d'affichage (PDA, etc.) pour GDS, vérifiez que l'alimentation électrique est correctement connectée. Si l'alimentation secteur et le cordon d'alimentation sont déconnectés, la programmation s'interrompt et le module de contrôle peut s'endommager. Si l'ordinateur portable fonctionne sur la batterie, vérifiez que la charge est suffisante pour terminer le processus GDS.
MISE EN GARDE
Si vous devez programmer une unité ECU, elle doit être installée dans le véhicule avant de lancer ce processus. Assurez-vous que la batterie du véhicule est complètement chargée.
Important
Pour réduire la perte potentielle du signal, configurez MDI pour utiliser l'option de communication la plus stable sur votre site.Vous pouvez choisir USB, Ethernet ou Wireless Ethernet.

Procédure de démarrage GDS:

GDS peut être démarré via TIS2Web/GlobalTIS en cliquant sur l'icône GDS. Vous pouvez également démarrer GDS en mode autonome via le menu Démarrer de Windows.

  1. Pour commencer, cliquez sur le symbole GDS dans le menu principal TIS2Web/GlobalTIS.. Si vous démarrez GDS en mode autonome, cliquez sur DEMARRER/PROGRAMMES/GDS.
  2. Lorsque vous lancez GDS Smart Start, la mise à jour affichée s'exécute. Si une nouvelle version GDS ou MDI Manager est disponible sur le serveur, un message vous demande de la télécharger et de l'installer sur le PC. Cliquez sur Continuer, pour démarrer le téléchargement ou l'installation [IV]. Si vous lancez GDS en mode autonome, aucune mise à jour n'est possible.
  3. La page de connexion GDS se charge.

5. Navigation générale

Menus déroulants 

Il existe cinq menus déroulants dans GDS. [VII]

Remarque: Ces menus ne correspondent qu'à des sélections de menu pour les versions PC de GDS. Il existera également un format d'écran tactile permettant d'accéder aux menus via un bouton contextuel.

  • Le menu Fichier contient l'entrée Quitter pour fermer GDS.
  • Le menu Session contient deux options : Enregistrer le journal de session en cours et Ajouter des commentaires à la session...
    Ajouter des commentaires à la session... : vous pouvez saisir les informations depuis le clavier de l'ordinateur et les enregistrer dans le gestionnaire de journal de session.
  • Sélection Outils contient cinq options : Lancer la trace, Lecture de capture instantanée, Créer un rapport, Afficher l'état de téléchargement..., Administration des utilisateurs
    Vous pouvez créer des utilisateurs en cliquant sur le symbole dans la page Administration des utilisateurs.  .
  • Fenêtre : dans le menu Préférences, vous pouvez sélectionner la configuration d'écran (mode Plein écran, PC ou Écran tactile).
  • Aide contient trois options :
    Sommaire de l'aide affiche tous les fichiers d'aide GDS.  A propos de GDS affiche la version et l'ID de build GDS.  Afficher la version MDI affiche les versions logicielle et matérielle MDI.  Cette information peut vous être demandée si vous contactez notre Customer Support Center.


[VII]

Barre d'outils de l'application de diagnostics

Il existe plusieurs sélections de barre d'outils disponibles lorsque vous accédez à la vue de navigation du système. [VIII]

  • Enregistrer la capture instantanée (cercle rouge) est disponible uniquement dans la vue de navigation du système et permet d'enregistrer une capture instantanée de véhicule. Lorsque vous enregistrez une capture instantanée, cette option n'est pas disponible.
  • Arrêter l'enregistrement de capture instantanée (carré bleu) apparaît lors de l'enregistrement d'une capture instantanée. Cette option permet d'arrêter l'enregistrement de la capture instantanée et d'enregistrer le fichier.
  • Le symbole Capture instantanée est sélectionné pour capturer le contenu de l'écran affiché et le stocker sur l'ordinateur.
  • Zoom arrière augmente la taille de la police. La taille de l'image affichée diminue chaque fois que vous cliquez.
  • La sélection Zoom avant augmente la taille de la police. La taille de l'image affichée augmente chaque fois que vous cliquez.
  • Le bouton Quitter ferme l'application de diagnostics active.


[VIII]

Touches fléchées

Les touches fléchées se trouvent au bas de la liste des paramètres. Les touches fléchées peuvent être utilisées pour la navigation suivante :


[IX]:

  • La flèche Vers le haut permet de remonter d'un dossier dans la liste des paramètres chaque fois que vous la sélectionnez.
  • La flèche Vers le bas permet de descendre d'un dossier dans la liste des paramètres chaque fois que vous la sélectionnez.
  • La flèche Vers la droite permet d'ouvrir un niveau dans l'arborescence de navigation.
  • La flèche Vers la gauche permet de fermer un niveau dans l'arborescence de navigation. Cette sélection ne s'affiche pas si vous vous trouvez dans le premier écran d'une procédure.

Autres touches

Trois autres touches se trouvent au bas de tous les écrans. Ces touches peuvent être utilisées comme suit :

  • Lancer permet de lancer une application.
  • Action précédente permet de revenir à l'écran précédent. Cette sélection n'est disponible que s'il existe un écran précédent.
  • L'action Action suivante permet de passer à l'écran suivant. Cette sélection n'est disponible que si toutes les sélections nécessaires ont été faites dans l'écran en cours.

Saisie des informations de véhicule/MDI

1. Dans le menu principal GDS, sélectionnez un utilisateur pour vous connecter. Si aucun utilisateur n'a été créé, vous pouvez en créer un ou sélectionner l'utilisateur par défaut. (Voir le chapitre des menus déroulants)

2. Après avoir sélectionné votre choix d'accès, l'écran Menu d'identification s'affiche. Cet écran contient plusieurs options :


[X]

  • [X-1] Sélection MDI: Toutes les MDI disponibles dans les réseaux chez le distributeur s'affichent. Vous pouvez sélectionnez la MDI que vous utilisez pour effectuer les diagnostics de véhicule. Pour la connexion MDI, sélectionnez une MDI disponible en cliquant dessus et sélectionnez Se connecter. Sélectionnez Se déconnecter pour vous déconnecter de la MDI. Associez une nouvelle MDI au véhicule et sélectionnez Se connecter. Sélectionnez Actualiser pour rechercher les nouvelles MDI dans les réseaux disponibles Pour continuer dans GDS sans MDI connectée, cochez la case Naviguer sans MDI.
  • [X-2] Entrée NIV: Champ de numéro d'identification du véhicule pour la saisie manuelle ou identification du NIV automatique depuis le contrôleur de véhicule. (Connexion MDI nécessaire). Vous pouvez ajouter des commentaires sur le NIV/Véhicule dans le champ de texte en dessous.
  • [X-3] Section Identification du véhicule: Sélectionnez la marque, le modèle et l'année du modèle pour continuer.
  • [X-4] Historique des diagnostics du véhicule: Si un véhicule a déjà subi des diagnostics avec ce système GDS, le NIV et les informations associées seront stockés dans l'historique du véhicule. Dans la liste, sélectionnez le véhicule pour lequel vous voulez établir des diagnostics et continuez directement vers Page d'accueil du véhicule. Une fonction de filtrage de texte de format libre est disponible.

3. Page d'accueil du véhicule s'affiche. From this screen various vehicle data is displayed including: entered vehicle information, previous vehicle diagnosis, session comments, session log reports, and stored data

Page d'accueil du véhicule

Page d'accueil du véhicule comporte plusieurs zones :

[XI-1]: Informations du véhicule contient toutes les informations du véhicule, notamment la marque, le modèle, l'année du modèle, etc.
Cette liste est mise à jour au cours des diagnostics du véhicule et elle est enregistrée avec chaque session. Supprimer la sélection d'option de véhicule permet de supprimer une sélection de contenu de véhicule précédente. Sélectionnez la ligne à supprimer dans la liste de contenu de véhicule et sélectionnez Supprimer l'équipement.

[XI-2]: Commentaires sur la session permet d'ajouter des commentaires utilisateur de format libre sur une session de diagnostics. Cliquez sur Ajouter un commentaire pour ajouter des informations sur une session. Vous pouvez entrer des informations dans la zone Utilisateur, Données ou Commentaire depuis le clavier. Pour supprimer les commentaires précédents, sélectionnez les informations dans la zone Utilisateur, Données ou Commentaire et cliquez sur Supprimer le commentaire sélectionné.

[XI-3]: Rapports de journaux de sessions permet d'imprimer les informations de diagnostics du véhicule. Vue complète fournit un rapport complet de tous les diagnostics effectués sur le véhicule. Rapport simple fournit un résumé des diagnostics du véhicule.

[XI-4]: Donnée de capture instantanée liste toutes les données de capture instantanée véhicule stockées. Le nom de l'utilisateur, le nom du fichier de capture instantanée, la date de la capture et les commentaires s'afficheront. Vous pouvez lire la capture instantanée en sélectionnant l'élément et Vérifier. Vous pouvez supprimer des éléments individuellement en les sélectionnant et en choisissant Supprimer.
[XI-6]: La fonction Exporter enregistre un instantané sur le disque dur de votre ordinateur local. Vous pouvez ensuite transmettre les données d'instantané à la personne concernée en vue d'une analyse complémentaire.

[XI-5]: Le menu Applications exécutées fournit un historique de session de diagnostics.

Action suivante permet de poursuivre les diagnostics du véhicule sélectionné.


[XI]

Navigation dans le système - Sélection de l'application de diagnostics

Après avoir sélectionné Action suivante dans Page d'accueil du véhicule, la vue Navigation dans le système s'affiche. [XII].

Sélectionnez le chemin de navigation à analyser sur le véhicule. La navigation peut être effectuée avec les flèches de navigation. Les flèches Droite/Gauche ouvrent/ferment un sous-dossier. Le bouton Lancer démarre l'application de diagnostics sélectionnée. Vous pouvez également utiliser la souris pour naviguer en cliquant sur les symboles + de l'arborescence de navigation ou en utilisant les touches fléchées du clavier du PC.

Continuez de sélectionner les dossiers du chemin approprié jusqu'à ce que vous atteigniez une application de diagnostics qui comporte une icône d'application.

Après avoir lancé une application de diagnostics, la vue de navigation dans le système disparaît et elle est remplacée par une application de diagnostics.  Après avoir utilisé l'application de diagnostics, sélectionnez Quitter sur le côté supérieur droit de la vue de l'application de diagnostics.  Cela permet de fermer l'application de diagnostics et de revenir à la vue de navigation dans le système.

Lecture de capture instantanée

Le bouton Lecture de capture instantanée permet de vérifier les données de capture instantanée précédentes du véhicule. Ces informations peuvent être utilisées pour analyser les incidents du véhicule et détecter les conditions d'erreur du déclencheur. La fonction graphique permet de comparer les paramètres pour déterminer si un composant fonctionne correctement. Vous pouvez sélectionner les captures instantanées enregistrées pour les afficher des trois manières suivantes :

  • En sélectionnant un fichier de capture instantanée depuis Page d'accueil du véhicule et en cliquant sur l'option Vérifier.
  • En cliquant deux fois sur la capture instantanée dans Page d'accueil du véhicule.
  • En entrant la sélection Outils depuis la sélection Menu et en sélectionnant Lecture de capture instantanée. Ensuite, ouvrez Gestionnaire de capture instantanée. Cliquez deux fois sur le fichier de capture instantanée à afficher ou cliquez sur l'option de chargement de capture instantanée et sélectionnez un fichier de capture instantanée.

Quelque soit la méthode de sélection, une fenêtre indépendante Fichier chargé s'affiche depuis l'écran Lecture de capture instantanée. Sélectionnez OK dans la fenêtre indépendante.

Fonctions de lecture de capture instantanée [XIX-1]

Vous pouvez utiliser plusieurs fonctions de relecture de capture instantanée pour lire les captures instantanées enregistrées

La fonction Lecture permet de faire avancer la capture instantanée en cliquant sur cette sélection.   Un marqueur traverse l'écran lorsque vous utilisez cette sélection.   Toutes les sélections de cette zone deviennent grises lorsqu'elles sont actives.
La fonction Suspendre permet de suspendre la lecture de la capture instantanée. Si vous cliquez une seconde fois sur cette sélection, la lecture de la capture instantanée reprend là où elle s'est arrêtée.
La fonction Arrêter permet d'arrêter la capture instantanée. La capture instantanée démarre depuis le début si vous sélectionnez Lecture après avoir utilisé la fonction Arrêter.

Contrôle de temps [XIX-2]

Contrôle de temps permet de contrôler la lecture de la capture instantanée. Cette fonction permet de sélectionner une période, d'afficher différentes vues et de contrôler la vitesse d'affichage. Vous pouvez sélectionner les fonctions suivantes directement sous Contrôle de temps.

L'option Signet permet d'ajouter un commentaire dans l'écran en cours et de revenir à ce point. Si vous sélectionnez cette option, une fenêtre indépendante s'affiche. Entrez un nom de signet et sélectionnez OK.
Vous pouvez sélectionner le bloc-notes pour ajouter des commentaires au fichier de lecture de la capture instantanée à ce stade. Si vous sélectionnez cette option, une fenêtre indépendante s'affiche. Entrez un commentaire pour le fichier de capture instantanée et sélectionnez OK.
Charger la capture instantanée (dossier bleu) affiche un fichier de capture instantanée pour effectuer d'autres analyses. Pour afficher un instantané exporté, vous devez d'abord décompresser l'archive *.zip correspondante.  Cette fonction est également disponible en cliquant sur un fichier de capture instantanée dans Gestionnaire de capture instantanée.
La flèche Bas affiche toutes les options Contrôle de temps lorsque vous la sélectionnez. Vous pouvez également sélectionner des options individuellement en cliquant dessus.
Sélectionnez cette option pour contrôler la vitesse d'affichage de la capture instantanée.   Vous pouvez sélectionner les vitesses lentes 1/8X, 1/4X, et 1/2X.   Vous pouvez également sélectionner la vitesse réelle 1.   Vous pouvez utiliser les vitesses rapides 2X, 4X et 8X en utilisant cette sélection.

Sélections de vues [XIX-3]

Chaque sélection de vue permet d'afficher différemment les données d'un véhicule pour les analyser. Vous pouvez sélectionner les fonctions suivantes sous Vues disponibles.

  • Vue de la liste des paramètres : affiche les informations provenant du fichier de capture instantanée. La valeur en cours des paramètres est affichée.
Important
Le nom du paramètre s'affiche dans la langue d'origine utilisée pour la capture si aucune traduction n'est disponible pour le nom.
  • Vue de journal : contient toutes les informations du fichier de capture instantanée. Toutes les valeurs des paramètres s'affichent dans leur ordre d'enregistrement. Vous pouvez ajouter des paramètres à une liste de surveillance ou supprimer en cliquant dessus.
  • Vue d'état : affiche les valeurs de tous les paramètres dans une période donnée. Vous pouvez utiliser ces valeurs pour effectuer une comparaison avec la vue des déclencheurs.
  • Vue des déclencheurs : affiche tous les déclencheurs système et définis par l'utilisateur. Vous pouvez utiliser cette fonction lorsqu'il existe une combinaison de facteurs impliqués dans un problème de diagnostics.  Lorsqu'une condition de déclenchement existe, un triangle de précaution s'affiche.
    • Charger les déclencheurs - Permet de charger un fichier contenant des définitions de déclencheurs.
    • Enregistrer les déclencheurs - Permet d'enregistrer le déclencheur sur le disque dur de l'ordinateur.
    • Supprimer le déclencheur - Permet de supprimer le déclencheur en cours définitivement.
    • Modifier le déclencheur - Affiche une fenêtre indépendante dans laquelle vous pouvez modifier les limites et les paramètres qui figurent dans un déclencheur.
    • Ajouter un déclencheur - Affiche une fenêtre indépendante pour configurer un déclencheur.
  • Vue DTC - Lorsque vous sélectionnez la fonction Lecture et cette option, cette fonction ouvre la liste des DTC enregistrés.
  • Vue Courbe - affiche les données pouvant être visualisées par rapport à l'axe de temps. Dans cette vue, vous pouvez sélectionner des paramètres de graphique en cochant la case sous Sélectionné. Au bas du graphique figurent quatre fonctions : Zoom avant (augmenter la taille), Zoom arrière (diminuer la taille), Signets et Suspendre le défilement. Chacune de ces fonctions est accessible en cliquant dessus.

7. DEPANNAGE

Problèmes de connexion MDI

Si les connexions MDl sont lâches ou incorrectes sur le véhicule/l'ordinateur, une fenêtre indépendante s'affiche. Procédez comme suit :

  • Vérifiez que le gestionnaire MDl n'est pas actif lorsque GDS est actif.
  • Vérifiez que l'interface MDI est correctement connectée au véhicule ou à l'ordinateur
  • Utilisez un "câble USB en bon état" d'une autre unité MDI, si disponible. Effectuez toutes les connexions et réessayez.
  • Utilisez un " câble de connexion DLC MDI en bon état" d'une autre unité MDI, si disponible. Effectuez toutes les connexions et réessayez.
  • Si l'incident persiste, contactez le Customer Support.
  • Vérifiez que le pare-feu ne bloque pas les communications GDS/MDI.

Conflit avec le GDS/MDI Manager

Le logiciel GDS et le MDI Manager ne peuvent pas être actifs simultanément. S'ils le sont, un conflit se produit. L'options Informations fournit des informations sur l'erreur. Si l'incident persiste, le Customer Support peut vous demander de fournir ces informations.

Si le conflit logiciel persiste, la fenêtre indépendante "Une erreur irrémédiable s'est produite" s'affiche dans le gestionnaire MDI. Fermez le gestionnaire MDI et revenez dans le programme GDS. Si l'incident persiste après être revenu au programme GDS, contactez le Customer Support.
