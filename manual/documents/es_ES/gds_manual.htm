Información general

GDS ha sido concebido para que lo utilice personal de servicio con formación para diagnosticar y reparar sistemas electrónicos de automoción.  Hemos concentrado todos nuestros esfuerzos en proporcionar la información técnica más completa y precisa basada en la información de servicio de fábrica disponible en el momento de la publicación.  No obstante, nos reservamos el derecho a realizar cambios en cualquier momento sin previo aviso.

Antes de poner GDS en funcionamiento, lea atentamente la presente guía del usuario para familiarizarse con GDS y sus funciones, así como su funcionamiento.

GDS proporciona las siguientes prestaciones:

  • Lectura de códigos de avería (DTC)
  • Lectura de información de identificación de la ECU
  • Lectura y pantalla gráfica de los parámetros de la lista de datos
  • Registro y almacenamiento de datos de instantáneas
  • Funciones de control
  • Funciones de configuración y de restablecimiento
  • Protocolo de diagnóstico de almacenamiento y de registro
  • Almacenamiento de sesiones de diagnóstico por vehículo
Importante:
La programación SPS de los controladores de vehículo no tiene soporte en GDS. Para la programación Pass Through de SPS con la MDI, utilice la aplicación TIS2Web / GlobalTIS de SPS.

3. Instalación de GDS

Visión general de los componentes de GDS:

El software de GDS contiene los módulos siguientes que se instalarán inicialmente y se actualizarán con frecuencia en determinados periodos. [I]

  • GDS Base System
    Software básico de GDS instalado en el PC
  • Entregables de GDS:
    Los paquetes de datos de diagnóstico de los modelos de vehículos que ejecutan las rutinas de diagnóstico con los controladores de vehículos.
  • GDS Smart Start:
    Agente de aplicaciones que compara las versiones de GDS Base System, de entregables de GDS y del servidor de software MDI Manager entre TIS2Web/GlobalTIS y la instalación local. En caso de que haya nuevas actualizaciones disponibles, el sistema solicitará que las descargue.
  • Software MDI Manager (aplicación separada):
    Aplicación para actualizar y configurar el dispositivo MDI (interfaz de diagnóstico múltiple)


[I]

Instalación:

GDS se puede instalar desde un DVD de instalación aparte o directamente como descarga desde TIS2Web / GlobalTIS.

Instalación desde el DVD de GDS:

  • El PC debe iniciarse en modo de administración
  • Abra el ejecutable de instalación en el menú de inicio del DVD
  • Seleccione el idioma del agente de instalación
  • La configuración y los datos de GDS existentes se pueden volver a utilizar en caso de que se detecte en el PC una instalación de GDS anterior.
  • Acepte el contrato de licencia (necesario para continuar)
  • Seleccione el directorio de instalación de GDS
  • Se inicia el proceso de instalación de GDS
  • Se instalará el software MDI Manager, en caso de que no haya un software MDI Manager instalado en el PC o de que la versión instalada sea más antigua. Para la instalación inicial del software MDI Manager, es preciso conectar el hardware de MDI durante el proceso de instalación. Para obtener más información, consulte el manual de MDI.
  • GDS puede abrirse mediante el símbolo de escritorio o siguiendo la ruta Inicio/Programas/GDS después de acabar el proceso de instalación. 
  • Si es preciso, configure el servidor de seguridad para permitir la comunicación de GDS con el dispositivo de MDI.
  • GDS Smart Start se inicia y el proceso de actualización visualizado se ejecutará antes de iniciar GDS.

Instalación desde TIS2Web / GlobalTIS:


[II]

Importante: Durante la instalación desde TIS2Web, se descargan alrededor de 140 MB de datos de Internet.
  • El PC debe ejecutarse en modo de administración
  • Inicie GDS en la página principal de TIS2Web / GlobalTIS [II]
  • GDS Smart Start se descarga desde el servidor TIS2Web; según la velocidad de conexión a Internet, la descarga puede tardar varios minutos.
  • Seleccione el directorio de instalación de GDS.
  • Especifique la contraseña de Internet (sólo si se utiliza un servidor proxy para obtener acceso a Internet)
  • Si es preciso, configure el servidor de seguridad para permitir la comunicación de GDS con el dispositivo de MDI.
  • Cuando empiece la instalación de GDS Smart Start, el proceso puede tardar varios minutos.
  • Al iniciar GDS Smart Start, se ejecutará el proceso de actualización de GDS. [III]
  • Cuando haya terminado de descargar los archivos de instalación, aparecerán la versión de GDS y la versión de software de MDI Manager y se podrán descargar aceptando el cuadro de diálogo que así lo solicite. Pulse el botón Mostrar progreso de la descarga para visualizar la barra de estado de progreso de la actualización.
  • El software de MDI Manager se instalará en caso de que no haya un software de MDI Manager instalado en el PC o si hay una versión antigua del software MDI Manager. Para la instalación inicial del software MDI Manager, es preciso conectar el hardware de la MDI durante el proceso de instalación. Para obtener más información, consulte el manual de MDI.
[III] [IV]

Actualización de GDS vía Smart Start:

Inicie GDS bien desde TIS2Web / GlobalTIS, o como aplicación individual, mediante el menú de inicio de Windows. Antes de cargar GDS, se inicia el agente Smart Start y compara las siguientes versiones de la instalación GDS en el PC con el contenido del servidor TIS2Web / GlobalTIS:

  • GDS Smart Start
  • GDS Base System
  • MDI Manager

Si hay nuevas versiones disponibles en el servidor, Smart Start solicitará al usuario que descargue las actualizaciones. [IV]

 
Importante: Los procesos de actualización descritos necesitan que el PC se ejecute siempre en modo de administración.

Reversión de la instalación de GDS Base System

Se puede revertir la instalación de GDS a la versión anterior instalada, en caso de que una actualización de GDS cause problemas operativos.

Para ejecutar la función de reversión, vaya a INICIO ->PROGRAMAS -> GDS -> GDS ROLLBACK. Como requisito previo a la función de reversión, debe haber una versión de GDS más antigua instalada en el PC.

Sistema de control de licencia de GDS

El sistema de control de licencias de GDS evita el uso no autorizado de GDS. Para renovar la licencia de GDS es necesario iniciar GDS desde TIS2Web o GlobalTIS. La licencia de GDS deberá renovarse a los 7 días si la aplicación se inició solamente en modo individual durante dicho periodo. 
Si la licencia de GDS ha expirado, aparecerá un mensaje de error indicándole al usuario que inicie la aplicación GDS desde TIS2Web / GlobalTIS para renovar la licencia.

Descarga y actualización de los entregables de GDS

Los entregables de GDS son aplicaciones de diagnóstico que controlan los servicios de comunicación y diagnóstico entre GDS y el controlador de vehículos. Los entregables de GDS se publican por modelo y se actualizan de forma permanente en TIS2Web / GlobalTIS.

GDS Smart Start detecta automáticamente si hay versiones de entregables nuevas o actualizadas disponibles en TIS2Web / Global TIS cuando se desencadena el proceso de actualización. Primero, los nuevos entregables deben descargarse en un directorio local del PC, antes de que se puedan instalar en GDS. Las actualizaciones de los entregables ya instalados se descargarán automáticamente y sólo requerirán la instalación en la base de datos maestra de GDS.


Descarga y actualización inicial de los entregables de GDS

  • Inicie GDS bien desde TIS2Web / GlobalTIS, o como aplicación individual, mediante el menú de inicio de Windows.
  • Seleccione el botón Agregar un nuevo paquete de diagnóstico de la pantalla de inicio de sesión de GDS.
  • Pulse el botón Actualizar la lista de entregables del servidor. [V-1]. En el servidor, los paquetes que estén disponibles pero que no se hayan descargado todavía se marcan con un símbolo. 
  • Marque y descargue los paquetes entregables [V-2]. La función de filtro ayuda a buscar entregables específicos. Los paquetes entregables descargados se visualizan con el símbolo. 
  • Instale los paquetes descargados en la base de datos maestra de GDS; los paquetes instalados se visualizan en la ventana derecha de la aplicación [V-3].


[V]

Actualización de los entregables de GDS instalados en la base de datos maestra:

  • Haga clic en el botón Actualizar paquete de diagnóstico en la pantalla de inicio de sesión de GDS [VI-1]. GDS compara las versiones de los entregables de GDS instalados con los entregables disponibles en el servidor y descarga los paquetes actualizados.
  • Los paquetes descargados se visualizan para su instalación en la ventana inferior izquierda. Seleccione el botón Instalar o Instalar todo para empezar la instalación. [VI-2].


[VI]:

Reversión de la actualización de un entregable

En general, el proceso de reversión es el mismo que el descrito en el capítulo Descarga y actualización inicial de los entregables de GDS

  • Seleccione el botón Agregar un nuevo paquete de diagnóstico de la pantalla de inicio de sesión de GDS.
  • Pulse el botón Actualizar la lista de entregables del servidor. [V-1]
  • Marque y descargue las versiones del paquete de entregables más antiguo deseado. La función de filtro ayuda a buscar entregables específicos.
  • Instale los paquetes descargados en la base de datos maestra de GDS.

Desinstalación de GDS:

GDS se puede desinstalar iniciando el programa siguiente:

INICIO -> PROGRAMAS -> GDS ->GDS DE-INSTALLATION

Cambio del idioma de GDS / Configuración regional:

GDS utiliza la configuración de idioma del sistema operativo del PC No hay menú en GDS para cambiar la configuración del idioma. Ejecute los pasos siguientes para cambiar el idioma de GDS.

  • Abra al PANEL DE CONTROL de Windows siguiendo estos pasos: Inicio -> Configuración -> PANEL CONTROL
  • Abra el menú CONFIGURACIÓN REGIONAL Y DE IDIOMA
  • Seleccione el país deseado en ESTÁNDARES Y FORMATOS
  • Seleccionar el menu Avanzado. Seleccione la configuración de idioma que desee en el cuadro desplegable PROGRAMAS QUE NO SON UNICODE
  • Configure las OPCIONES REGIONALES para cambiar las unidades visualizadas en GDS.

4. Introducción

AVISO:
Antes de ejecutar GDS, es importante tener en cuenta las siguientes precauciones:

  • Si utiliza una versión no actualizada, los módulos de vehículo pueden resultar dañados. GDS y la interfaz de diagnóstico múltiple han de tener el último software.
  • Asegúrese de que la batería del vehículo esté totalmente cargada El voltaje de la batería para GDS debe estar entre 12 y 14 voltios.
  • No obstante, no debe haber un cargador de batería conectado al vehículo cuando utilice MDI.
  • Asegúrese de que las conexiones de cable estén seguras. Un cable desconectado PUEDE provocar un error de controlador.
  • Cuando utilice un equipo portátil u otro dispositivo de pantalla (PDA, etc.) para GDS, asegúrese de que el suministro eléctrico esté correctamente conectado. Si la electricidad es CA y el cable eléctrico se desconecta, la programación puede interrumpirse y provocar daños en el módulo de control. Si el portátil funciona con su propia fuente de alimentación interna (baterías), asegúrese de que esté correctamente cargado para completar el proceso de GDS.
AVISO
Si debe programarse una ECU, deberá instalarse en el vehículo antes de iniciar este proceso. Asegúrese de que la batería del vehículo esté totalmente cargada.
Importante
A fin de reducir el potencial de pérdida de señal, configure MDI en la opción de comunicación más estable para su ubicación.Puede elegir entre USB, Ethernet o Ethernet inalámbrica.

Procedimiento de inicio de GDS:

Se puede iniciar GDS mediante TIS2Web/GlobalTIS haciendo clic en el icono de GDS. O bien, GDS se puede iniciar en modo autónomo a través del menú Inicio de Windows.

  1. Para empezar, haga clic en el símbolo de GDS en el menú principal de TIS2Web/GlobalTIS. Si inicia GDS en modo autónomo, haga clic en INICIO/PROGRAMAS/GDS
  2. Al iniciar GDS Smart Start, se ejecutará el proceso de actualización visualizado. Si hay una nueva versión de GDS o MDI Manager disponible en el servidor, el sistema solicitará su descarga e instalación en el PC. Haga clic en Continuar, para empezar la descarga o instalación [IV]. Si se inicia GDS en modo autónomo, no será posible ninguna actualización.
  3. Se carga la página de inicio de sesión de GDS.

    Agregar comentarios a la sesión...: la información se puede introducir desde el teclado del PC y guardar en el administrador de registros de sesiones.
  • Herramientas tiene cinco opciones seleccionables: Iniciar el rastreo, Reproducir instantáneas, Crear informe, Mostrar el estado de la descarga..., Administración de usuarios
    Se pueden crear usuarios nuevos haciendo clic en el símbolo de la página Administración de usuarios  .
  • Ventana: en el menú Preferencias, se puede seleccionar la configuración de la pantalla, por ejemplo, el modo de pantalla completa o el modo de pantalla táctil o PC.
  • Selección de ayuda tiene tres opciones seleccionables:
    Contenido de la ayuda visualiza todos los archivos de ayuda de GDS disponibles.  Acerca de GDS visualiza la versión de GDS y el ID de compilación.  Mostrar la versión de la MDI visualiza las versiones de hardware y de software de MDI.  Es posible que se le solicite esta información cuando se ponga en contacto con nuestro Customer Support Center.


[VII]

Barra de herramientas de la aplicación de diagnóstico

Varias selecciones de la barra de herramientas pasan a estar disponibles cuando se llega a la vista de navegación del sistema. [VIII]

  • El Registrar instantánea (círculo rojo) sólo está disponible después de llegar a la vista de navegación del sistema y se utiliza para registrar la instantánea de un vehículo. Mientras se registra una instantánea, esta selección no está disponible.
  • El Detener la grabación de la instantánea (cuadrado azul) aparece mientras se registra una instantánea de vehículo. Esta opción se selecciona para detener la grabación de la instantánea y guardar el archivo.
  • El símbolo Captura de pantalla se selecciona para capturar el contenido de la pantalla visualizada y almacenarlo en una ubicación del PC.
  • El Alejar aumenta el tamaño de la fuente. La imagen visualizada se reduce cada vez que se hace clic con el mouse (ratón).
  • La selección de Acercar aumenta el tamaño de la fuente. La imagen visualizada se incrementa cada vez que se hace clic con el mouse (ratón).
  • El botón Salir cierra la aplicación de diagnóstico activa.


[VIII]

  • Acción anterior se puede utilizar para acceder a la pantalla anterior. Esta selección sólo estará disponible cuando haya una pantalla a la que ir hacia atrás.
  • La Acción Siguiente se puede utilizar para ir a la pantalla siguiente. Esta selección sólo estará disponible si se han efectuado todas las selecciones necesarias en la pantalla actual.

Introduciendo información del vehículo/MDI

1. Desde el menú principal de GDS, seleccione un usuario para iniciar sesión. Si no se ha creado ningún usuario, se puede crear uno nuevo o seleccionar uno predeterminado. (Consulte los menús desplegables de la sección)

2. Después de hacer su elección de acceso, se visualizará la pantalla del menú de identificación. En esta pantalla dispone de varias opciones:


[X]

  • [X-1] Selección de la MDI: Se visualizan todos los MDI disponibles en las diversas redes del vendedor. Puede seleccionar el MDI que está utilizando para realizar diagnósticos de vehículos. Para la conexión de MDI, seleccione un MDI disponible haciendo clic en él y seleccione Conectarse. Seleccione Desconectar para desconectar la MDI. Conecte la MDI al vehículo y seleccione Conectarse. Seleccione Actualizar para comprobar las nuevas MDI en las redes disponibles. Para seguir en GDS sin ninguna MDI conectada, active la casilla de verificación Desplazarse sin MDI.
  • [X-2] Entrada de VIN: Campo para la entrada manual del número de identificación del vehículo o identificación VIN automática del controlador del vehículo. (Conexión de MDI obligatoria). Se pueden agregar comentarios adicionales asociados a este VIN/Vehículo en el campo de texto de abajo.
  • [X-3] Sección de Identificación del vehículo: seleccione Marca, Modelo y Año antes de continuar.
  • [X-4] Historial de diagnósticos del vehículo: Si se ha diagnosticado previamente un vehículo con este sistema GDS, el VIN y su información asociada se almacenarán en el historial del vehículo. Seleccione el vehículo de la lista que se vaya a diagnosticar y acceda directamente a la Página principal del vehículo. Hay disponible una función de filtro de texto libre

3. Se visualizará la Página principal del vehículo. From this screen various vehicle data is displayed including: entered vehicle information, previous vehicle diagnosis, session comments, session log reports, and stored data

Página principal del vehículo

La Página principal del vehículo está formada por varias áreas:

[XI-1]: El Información del vehículo enumera toda la información del vehículo, incluidos la marca, el modelo y el año, etc.
Esta lista se va actualizando conforme se va diagnosticando el vehículo y guardando con cada sesión. La Selección de la opción de eliminación de vehículo se utiliza para eliminar una selección de contenido de vehículo anterior. Resalte la fila que se eliminará de la lista de contenido del vehículo y seleccione Eliminar equipamiento.

[XI-2]: Comentarios de la sesión se utiliza para agregar comentarios de usuario sin formato acerca de una sesión de diagnóstico. Haga clic en Agregar un comentario para agregar información adicional sobre una sesión. Se puede introducir información en las áreas Usuario, Datos o Comentario desde el teclado. Para eliminar comentarios anteriores, resalte la información de las áreas Usuario, Datos o Comentario y haga clic en Eliminar el comentario seleccionado.

[XI-3]: Informes del registro de sesión se puede utilizar para imprimir una copia de la información de diagnóstico del vehículo. Vista completa proporciona un informe completo de todos los diagnósticos efectuados en el vehículo. Informe básico proporciona un resumen del diagnóstico del vehículo.

Reproducir instantáneas

La función Reproducir instantáneas permite revisar los datos de instantáneas del vehículo registrados anteriormente. Esta información se puede utilizar para analizar los problemas del vehículo y detectar condiciones desencadenantes de error. La función de gráfico se puede utilizar para comparar parámetros y ver si un componente funciona correctamente. Se pueden seleccionar instantáneas registradas para visualizarlas siguiendo uno de los tres métodos siguientes:

  • Seleccionando un archivo de instantáneas en la Página principal del vehículo y haciendo clic en la selección Revisar.
  • Haciendo doble clic en la instantánea seleccionada en la Página principal del vehículo.
  • Introduciendo la selección Herramientas de la selección Menú y seleccionando Reproducir instantáneas. A continuación, abra el Administrador de instantáneas. Haga doble clic en el archivo de instantáneas que desee ver o haga clic en la selección Cargar instantánea y seleccione un archivo de instantáneas.

Independientemente del método de selección, se visualizará una ventana emergente de Archivo cargado en la pantalla Reproducir instantáneas. Seleccione Aceptar en la ventana emergente.

Funciones de reproducción de instantáneas [XIX-1]

Existen varias funciones de reproducción de instantáneas que se utilizan para reproducir instantáneas registradas

La función Reproducir se utiliza para avanzar la reproducir la instantánea haciendo clic en esta selección.   Un marcador se moverá por la pantalla cuando se utilice esta selección.   Todas las selecciones de esta área se vuelven de color gris cuando están activas.
La función Pausa se utiliza para poner en pausa la reproducción de la instantánea. Cuando se hace clic por segunda vez en esta selección, la instantánea retoma la reproducción allí dónde se interrumpió.
La función Detener se utilizar para detener la reproducción de instantáneas. La instantánea empezará desde el principio si se selecciona Reproducir después de utilizar la función Detener.

Control de la hora [XIX-2]

Control de la hora se utiliza para controlar la reproducción de instantáneas. Esta función permite seleccionar un maco de tiempo específico, visualizar diferentes vistas y controlar la velocidad de visualización. Las funciones siguientes se pueden seleccionar directamente desde Control de la hora.

La selección de marcadores permite agregar un comentario específico en la pantalla actual y regresar a este punto en el futuro. Cuando se selecciona esta opción, se visualiza una ventana emergente. Introduzca un nombre de marcador y seleccione Aceptar.
Se puede seleccionar el bloc de notas para agregar comentarios al archivo de reproducción de instantáneas en el punto del tiempo actual. Cuando se selecciona esta opción, se visualiza una ventana emergente. Introduzca un comentario para el archivo de instantáneas y seleccione Aceptar.
Cargar instantáneas (carpeta azul) abre un archivo de instantáneas y lo prepara para análisis avanzados. Si quiere ver una instantánea exportada, primero debe descomprimir el archivo *.zip correspondiente.  Esta función también está disponible haciendo clic en el archivo de instantáneas disponible en Administrador de instantáneas.
La flecha hacia abajo visualiza todas las opciones de Control de la hora cuando se selecciona. También se pueden seleccionar opciones individuales haciendo clic en el elemento que desee.
Se puede controlar la velocidad de visualización de instantáneas seleccionando esta opción.   Se pueden seleccionar velocidades lentas de 1/8X, 1/4X y 1/2X.   La velocidad en tiempo real también se puede seleccionar en 1.   También se pueden obtener velocidades más rápidas de 2X, 4X y 8X utilizando esta selección.

Importante
El nombre del parámetro se mostrará en el idioma original utilizado para la captura si no hay una traducción disponible para el nombre.
  • La vista de registros contiene toda la información del archivo de instantáneas. Todos los valores de los parámetros se muestran en una lista en el orden en que se registraron. Se pueden agregar o suprimir parámetros específicos de una lista de vigilancia haciendo clic en el elemento adecuado.

7. SOLUCIÓN DE PROBLEMAS

Problemas de conexión de la MDI

Si las conexiones de la MDI están flojas o no están bien conectadas al vehículo/PC, se abrirá una ventana emergente. Ejecute los pasos siguientes:

  • Asegúrese de que el software del administrador de MDI no esté ejecutándose al mismo tiempo que se ejecuta GDS.
  • Asegúrese de que la MDI esté correctamente conectada al vehículo y al PC
  • Utilice, si está disponible, un "cable USB bueno" de otra unidad de MDI. Realice todas las conexiones y vuelva a intentar la operación.
  • Utilice un "cable de conexión DLC MDI bueno" de otra unidad MDI, si está disponible. Realice todas las conexiones y vuelva a intentar la operación.
  • Si la condición sigue estando presente, póngase en contacto con Customer Support.
  • Asegúrese de que el servidor de seguridad no bloquee la comunicación GDS/MDI.

Conflicto de software GDS/MDI Manager

Los softwares de GDS y MDI Manager no pueden ejecutarse al mismo tiempo. Si ambos programas de software se ejecutan al mismo tiempo, puede producirse un conflicto de software. La selección Detalles proporcionará información posible sobre este error. Si el problema persiste, el Customer Support podrá pedirle esta información.

Si el conflicto de software continúa, es posible que el software del administrador de MDI abra una ventana emergente "Se ha producido un error fatal". Apague el software del administrador de MDI y regrese al programa de GDS. Si el problema continúa después de entrar en GDS, póngase en contacto con Customer Support.
