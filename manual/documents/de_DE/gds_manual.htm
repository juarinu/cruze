Allgemein

Das GDS dient zur Diagnose und Reparatur von Kfz-Elektroniksystemen durch geschulte Servicemitarbeiter.  Es wurden alle Anstrengungen unternommen, um die vorliegenden technischen Informationen auf Grundlage der zum Zeitpunkt der Drucklegung werkseitig verfügbaren Daten so vollständig und exakt wie möglich wiederzugeben.  Dennoch bleibt das Recht auf Änderungen ohne Vorankündigung jederzeit vorbehalten.

Bevor Sie das GDS in Betrieb nehmen, machen Sie sich bitte mit dem GDS sowie dessen Möglichkeiten vertraut, indem Sie dieses Benutzerhandbuch durchlesen.

GDS bietet die folgenden Hauptfunktionen:

  • Fehlercode auslesen
  • Steuergerätekennung auslesen
  • Datenlistenparameter auslesen und grafisch anzeigen
  • Schnappschuss aufzeichnen und speichern
  • Stellgliedtests
  • Konfiguration und Zurücksetzen von Steuergeräten
  • Aufzeichnung von Diagnoseprotokollen
  • Speichern von Daten aus Diagnosesitzungen
Wichtig:
Die SPS Programmierung von Fahrzeugsteuergeräten erfolgt nicht mit GDS. Die Steuergeräte Programmierung und Konfiguration erfolgt mittels MDI und TIS2Web/GlobalTIS SPS.

3. GDS Installation

Übersicht über die GDS Komponenten:

Die GDS Software besteht aus folgenden Modulen, welche installiert und in regelmäßigen Abständen aktualisiert werden. [I]

  • GDS Basissystem
    Installierte GDS Basissoftware auf dem PC
  • GDS Applikationspakete:
    Diagnosedatenpakete für Fahrzeugmodelle, welche die Diagnoseroutinen mit den Steuergeräten ausführen
  • GDS Smart Start:
    Dienstprogramm welches neue GDS Basissystemversionen, neue GDS Applikationspakete sowie neue MDI Manager Versionen auf TIS2Web/Global TIS ermittelt und dem Anwender zum Herunterladen anbietet. Liegen neue Versionen vor, werden diese dem Anwender zum Herunterladen angeboten.
  • MDI Manager Software (separate Anwendung):
    Anwendung zur Aktualisierung und Konfiguration des MDI (Multiple Diagnostic Interface)


[I]

Neuinstallation:

GDS kann von einer separaten Installations-DVD installiert werden oder direkt aus TIS2Web/Global TIS.

Installation von GDS DVD:

  • Das Betriebssystem muss mit Administrator Rechten gestartet sein
  • Installation über Auswahlmenü starten
  • Sprache auswählen
  • Sollte GDS bereits zuvor installiert gewesen sein, können die vorhandenen Einstellungen übernommen werden
  • Lizenzvereinbarungen bestätigen
  • Installationsverzeichnis wählen
  • GDS Installation beginnt
  • Falls keine oder eine ältere MDI Manager Software installiert ist, wird diese durch eine neue Installation erstellt bzw. ersetzt.  Bei einer Neuinstallation muss ein MDI bereitgehalten und nach Aufforderung angeschlossen werden. Für weitere Informationen zur MDI Installation bitte das MDI Handbuch verwenden.
  • Nach Abschluss der Installation kann GDS direkt über das Desktop Symbol oder über START/PROGRAMME/GDS gestartet werden. 
  • Gegebenenfalls die Firewall konfigurieren, damit die GDS Kommunikation nicht blockiert wird.
  • GDS Smart Start wird gestartet, die angezeigten Aktualisierungsschritte werden durchlaufen und GDS wird abschließend gestartet.

Installation über TIS2Web/Global TIS:


[II]

Wichtig: Bei der Installation über TIS2Web werden ca. 140 MB Daten über das Internet heruntergeladen.
  • Das Betriebssystem muss mit Administratorrechten gestartet sein.
  • GDS in TIS2Web/Global TIS Startseite aufrufen. [II]
  • GDS Smart Start wird vom Server geladen, dieses kann abhängig von der Internetverbindung längere Zeit dauern.
  • Installationsverzeichnis für GDS wählen.
  • Eingabe des Internetzugangspasswortes (nur bei Verwendung eines Proxyservers).
  • Gegebenenfalls die Firewall konfigurieren, damit die GDS Kommunikation nicht blockiert wird.
  • GDS Smart Start wird installiert, dieser Vorgang dauert einen Augenblick.
  • GDS Smart Start wird gestartet, die angezeigten Aktualisierungsschritte werden durchlaufen . [III]
  • Nach dem Herunterladen der Dateien wird die Installation einer neuen GDS Version und ggf. einer neuen MDI Manager Software angeboten. Diese kann durch Bestätigen der angebotenen Dialogbox heruntergeladen werden. Über die untere Schaltfläche Herunterladen anzeigen kann jeweils der Fortschritt angezeigt werden.
  • Falls keine oder eine ältere MDI Manager Software installiert ist, wird diese durch eine neue Installation erstellt bzw. ersetzt. Bei einer Neuinstallation muss ein MDI bereitgehalten und nach Aufforderung angeschlossen werden. Für weitere Informationen zur MDI Installation bitte das MDI Handbuch verwenden.

GDS Aktualisierung über Smart Start:

GDS entweder über TIS2Web/GlobalTIS oder als Standalone über das Windows Start Menü starten. Bevor GDS geladen wird, vergleicht der gestartete SmartStart Agent die folgenden Versionen der GDS Installation auf dem PC mit dem entsprechenden Inhalt auf TIS2Web/GlobalTIS:

  • GDS Smart Start
  • GDS Basissystem
  • MDI Manager

Liegen neue Versionen vor, werden diese dem Anwender zum Herunterladen angeboten. [IV]

 
Wichtig: Die Ausführung beschriebener Aktualisierungsprozesse erfordert stets, dass der PC mit Administrator Rechten gestartet wird.

GDS Basissysteminstallation zurücknehmen

Sollte eine GDS Aktualisierung zu Problemen führen, kann bei Bedarf eine GDS Aktualisierung zurückgenommen werden.

Dieses erfolgt unter dem Menüpunkt START -> PROGRAMME -> GDS -> GDS ROLLBACK. Voraussetzung für diese Funktion ist, dass zuvor eine ältere GDS Version installiert war.

GDS Lizenz- und Nutzungsberechtigungssystem

GDS ist mit einem Lizenzsystem versehen, welches die nicht autorisierte Verwendung von GDS verhindert. Zur Erneuerung der GDS Nutzungsberechtigung ist das Starten der GDS Anwendung von TIS2Web oder GlobalTIS aus erforderlich. Die GDS Nutzungsberechtigung muß nach spätestens 7 Tagen erneuert werden, sofern die Anwendung während dieser Zeit nur im Standalone Modus gestartet wurde. 
Ist die GDS Nutzungsberechtigung erloschen, fordert eine Fehlermeldung den Nutzer auf, diese durch den Start der GDS Anwendung von TIS2Web/GlobalTIS aus zu erneuern.

Diagnose Applikationspakete laden und aktualisieren

Applikationspakete sind die GDS Diagnoseanwendungen, welche für die Kommunikation mit den Fahrzeugsteuergeräten benötigt werden.. Die Applikationspakete werden pro Modell freigegeben und auf TIS2Web/Global TIS aktualisiert.

Im Gegensatz zur Tech2 Aktualisierung überprüft GDS Smart Start automatisch, ob neue oder aktualisierte Versionen vorliegen.. Neue Diagnosepakete (z.B. bei Neueinführung eines Fahrzeuges) müssen zunächst in ein lokales Verzeichnis geladen werden und anschließend in GDS installiert werden. Aktualisierungen von bereits installierten Diagnosepaketen werden automatisch heruntergeladen und zur Installation angeboten.


Neue Diagnosepakete erstmalig laden

  • GDS entweder über TIS2Web/GlobalTIS oder als Standalone über das Windows Start Menü starten.
  • Im GDS Anmeldebildschirm Neue Diagnosepakete hinzufügen wählen
  • Schaltfläche Liste der Diagnosepakete aktualisieren wählen. [V-1]. Auf dem Server vorhandene aber noch nicht heruntergeladene Pakete werden mit dem Symbol angezeigt. 
  • Applikationspakete markieren und herunterladen [V-2]. Unter dem Menüpunkt Suchfilter können hierzu spezifische Suchkriterien eingegeben werden.  Heruntergeladene Pakete werden mit dem Symbol angezeigt. 
  • Heruntergeladene Diagnosepakete in GDS installieren bis die Daten in dem rechten Fenster unter Hauptdatenbank angezeigt werden [V-3].


[V]

Vorhandene Diagnosepakete aktualisieren:

  • Im GDS Anmeldebildschirm die Schaltfläche Diagnosepakete aktualisieren betätigen. [VI-1]. GDS vergleicht nun die vorhandenen Daten mit den Daten auf dem Server und lädt ggf. Aktualisierungen zur Installation herunter.
  • Die heruntergeladenen Datenpakete werden im unteren Fenster zur Installation in GDS angezeigt. Zur Installation die Schaltfläche Installieren oder ggf. Alles Installieren betätigen [VI-2].


[VI]:

Aktualisierung von Diagnosepaketen zurücknehmen

Die grundsätzliche Vorgehensweise entspricht den Schritten wie unter Punkt Neue Diagnosepakete erstmalig laden beschrieben.

  • Im GDS Anmeldebildschirm Neue Diagnosepakete hinzufügen wählen
  • Schaltfläche Liste der Diagnosepakete aktualisieren wählen. [V-1]
  • Gewünschte ältere Diagnosepakete markieren und herunterladen. Unter dem Menüpunkt Suchfilter können hierzu spezifische Suchkriterien eingegeben werden.
  • Installieren der heruntergeladenen Pakete in die GDS Hauptdatenbank.

Deinstallation von GDS:

GDS kann unter folgenden Punkt deinstalliert werden:

START -> PROGRAMME -> GDS ->GDS DEINSTALLATION

Umstellung der GDS Sprache / Regionsoptionen:

GDS nutzt die Spracheinstellungen des PC-Betriebssystems. GDS selbst verfügt über kein Menü zum Ändern der Spracheinstellung. Folgende Schritte sind durchzuführen, um die Sprache für GDS zu wechseln.

  • Öffnen des Fensters SYSTEMSTEUERUNG: START -> EINSTELLUNGEN -> SYSTEMSTEUERUNG
  • Menü REGIONS- UND SPRACHOPTIONEN öffnen
  • Unter STANDARDS UND FORMATE das gewünschte Land wählen.
  • Öffnen des Menüs ERWEITERT. Die gewünschte Sprache in der Auswahlbox SPRACHE FÜR PROGRAMME, DIE UNICODE NICHT UNTERSTÜTZEN auswählen.
  • Zum Einstellen der angezeigten Einheiten bitte mit Anpassen... ändern.

4. Inbetriebnahme

ACHTUNG:
Vor Ausführung von GDS ist es wichtig folgende Punkte zu beachten:

  • Die Verwendung einer veralteten GDS Version kann Schäden an den Fahrzeugsteuergeräten verursachen.  Es ist sicherzustellen, dass das GDS und MDI stets aktualisiert wird.
  • Es ist sicherzustellen, dass die Fahrzeugbatterie voll aufgeladen ist. Die Fahrzeugnetzspannung sollte 12-14V betragen.
  • Während des Betriebes eines Batterieladegeräts darf kein MDI angeschlossen sein.
  • Überprüfen Sie die Kabelverbindungen. Ein lockeres Kabel KANN Schäden am Steuergerät verursachen.
  • Wenn ein Notebook Computer für GDS verwendet wird, ist sicherzustellen, dass das Notebook Netzgerät permanent verbunden ist. Ein Verlust der Spannungsversorgung des Notebooks über das Netzteil während einer Programmierung kann Schäden am Steuergerät verursachen.  Falls das Notebook über Akku betrieben wird, ist sicherzustellen, dass der Akku ausreichend geladen ist, um den GDS Arbeitsvorgang durchzuführen.
ACHTUNG
Die zu programmierenden Steuergeräte müssen im Fahrzeug eingebaut sein bevor sie mit GDS verbunden werden. Es ist sicherzustellen, dass die Fahrzeugbatterie voll aufgeladen ist.
Wichtig
Um die Gefahr eines Signalverlustes zu minimieren ist sicherzustellen, dass mit dem MDI mit einer stabilen Verbindung kommuniziert wird.Das MDI kann mit USB, Ethernet oder Wireless Ethernet verbunden werden.

GDS Inbetriebnahme:

GDS wird durch die GDS Schaltfläche in TIS2Web/Global TIS gestartet. Alternativ kann GDS auch als Standalone Applikation durch das Windows Startmenü aufgerufen werden.

  1. Zum Starten des Programms die GDS Schaltfläche in TIS2Web/Global TIS betätigen. Zum Starten von GDS als Standalone Anwendung das Windows Start Menü START/PROGRAMME/GDS ausführen.
  2. GDS Smart Start wird gestartet, der angezeigte Aktualisierungsprozess wird ausgeführt. Falls eine neue GDS oder MDI Version auf dem TIS2Web Server oder der GlobalTIS Installation verfügbar ist, wird diese zum Download angeboten. Zum Fortfahren und Installieren der Software die Schaltfläche OK betätigen. [IV]. Falls GDS als Standalone Anwendung gestartet wird, kann keine Überprüfung nach neuen Softwareversionen erfolgen.
  3. GDS Anmeldeseite wird geladen.

5. Allgemeine Bedienung

Menüleiste 

Die GDS Menüleiste besteht aus fünf Bereichen. [VII]

Hinweis: Die Menüleiste wird nur in der PC-Darstellung angezeigt. Die Anzeige kann auch in den Touchscreenbetrieb umgeschaltet werden, die Menüanzeige erscheint dann in einem hervorgehobenen Fenster.

  • Über das Menü Datei kann GDS über den Befehl Exit geschlossen werden.
  • Das Menü Diagnosesitzung besitzt 2 Einträge: Speichern der Diagnosesitzung und Kommentar zur Diagnosesitzung zufügen.
    Kommentar zur Diagnosesitzung zufügen: Information können über die Tastatur eingegeben und im Session Log Manager gespeichert werden.
  • Das Menü Werkzeuge enthält die fünf Einträge: Report erzeugen, Aufzeichnung starten, Datenaufzeichnungsansicht, Aktualisierungsstatus anzeigen, Benutzerverwaltung.
    Neue Nutzer können in der Benutzerverwaltung durch betätigen der Schaltfläche eingerichtet werden  .
  • Im Menü Fenster/Einstellungen können die Bildschirmeinstellungen wie Vollbildschirm oder Touchscreen sowie PC-Ansicht konfiguriert werden.
  • Das Hilfe Menü enthält drei Einträge: Hilfe Inhalt, Über GDS sowie MDI-Versionsinformation.
    Das Menü Hilfe Inhalt führt zu den GDS Hilfe Dateien.  Über GDS zeigt die GDS Version und Produktnummer an.  Die MDI-Versionsinformationen zeigt die MDI Softwareversion sowie die Hardwareversion des MDI an.  Die Informationen werden gegebenenfalls bei einem Anruf im Helpdesk benötigt.


[VII]

Diagnose Anwendungen - Werkzeugmenüleiste

Sobald die Diagnoseanwendungen gestartet werden, erscheint im oberen Bildschirmbereich eine Werkzeugauswahlliste. [VIII]

  • Die Schnappschussaufnahme kann über die Schaltfläche mit dem roten Kreis gestartet werden. Während einer Schnappschussaufnahme ist die Schaltfläche inaktiv.
  • Die Schnappschussaufnahme kann über das blaue Quadrat gestoppt werden. Die Schaltfläche wird aktiv, sobald eine Aufzeichnung gestartet wird. Mit der Anwahl kann eine Aufzeichnung angehalten und gespeichert werden.
  • Mit dem Datenaufzeichnungssymbol kann der aktuelle Bildschirm aufgenommen werden und in einem beliebigen Verzeichnis auf dem PC gespeichert werden.
  • Mit der Verkleinerungsfunktion wird die Schriftgröße reduziert. Jeder Mausklick bewirkt eine verkleinerte Darstellung.
  • Die Vergrößerungsfunktion vergrößert die Schriftgröße. Das Bild wird mit jedem Mausklick größer dargestellt.
  • Mit der Schaltfläche Beenden wird die jeweilige Diagnoseanwendung geschlossen.


[VIII]

Pfeil Schaltflächen

Die Pfeilschaltflächen sind am unteren Bildschirmrand positioniert.  Die Pfeilschaltflächen können für folgende Navigation verwendet werden.


[IX]:

  • Mit der Schaltfläche Hoch kann in der Parameterliste eine Ebene hoch gegangen werden.
  • Mit der Schaltfläche Runter kann in der Parameterliste eine Ebene nach unten gegangen werden.
  • Mit der Schaltfläche Rechts kann eine aktive Navigationsebene geöffnet werden.
  • Mit der Schaltfläche Links kann eine aktive Navigationsebene geschlossen werden. Diese Schaltfäche ist deaktiviert, wenn Sie sich am Anfangsbildschirm einer Prozedur befinden.

Weitere Schaltflächen

Drei weitere Schaltflächen werden am unteren Rand aller Seiten angezeigt. Diese werden für folgende Funktionen genutzt:

  • Die Schaltfläche Starten startet eine in der Systemnavigation markierte Diagnoseanwendung.
  • Die Schaltfläche Zurück wird verwendet, um in GDS eine Ebene zurück zu navigieren. Die Schaltfläche ist nur aktiv, wenn eine Zurücknavigation möglich ist.
  • Die Schaltfläche Weiter wird verwendet um in GDS eine Ebene weiter zu navigieren. Die Schaltfläche ist nur aktiv, wenn alle notwendigen Angaben innerhalb eines Bildschirms gemacht wurden.

Eingabe der Fahrzeug / MDI Information

1. Auswahl eines Benutzers im GDS Anmeldebildschirm. Falls noch kein eigener Nutzer angelegt wurde, kann mit dem user, default die Anmeldung durchgeführt werden. (Siehe auch zuvor die Beschreibung Menü)

2. Nach Bestätigen mit WEITER erscheint der Identifizierungsbildschirm in der nächsten GDS Ebene. Die folgenden Eingabemöglichkeiten können hier durchgeführt werden:


[X]

  • [X-1] MDI Auswahl: Im MDI Auswahlfenster werden die zur Verbindung verfügbaren MDI über die bestehenden Netzwerkverbindungen aufgelistet. Um GDS mit dem MDI zu verbinden, ist das entsprechende Gerät im Fenster zu markieren und die Schaltfläche Verbinden zu betätigen. Über die Schaltfläche Trennen kann die Verbindung zum MDI unterbrochen werden und ggf. ein anderes MDI verbunden werden. Verbinden Sie ein anderes MDI mit dem Fahrzeug und wählen Verbinden. Mit der Schaltfläche Aktualisieren können neu im Netzwerk angemeldete MDIs angezeigt werden. Um GDS ohne MDI Verbindung zu betreiben, das Feld Ohne MDI-Verbindung navigieren markieren.
  • [X-2] Eingabe der Fahrzeugidentifikationsnummer (FIN): Manuelle Eingabe oder automatisches Lesen der Fahrzeugidentifikationsnummer (FIN). (Verbindung zum MDI erforderlich). Zusätzliche Kommentare zu dem Fahrzeug können im darunterliegenden Feld eingetragen werden, so z.B. das Kennzeichen.
  • [X-3] Fahrzeugidentifikation: Auswahl der Marke, des Modells und des Modelljahres.
  • [X-4] Diagnosesitzungen: Falls ein Fahrzeug zuvor mit diesem GDS System diagnostiziert worden ist, werden die Fahrgestellnummer und die dazugehörigen Informationen der Diagnosesitzungen gespeichert. Nach Auswahl des gewünschten Fahrzeuges kann direkt die nächste GDS Ebene aufgerufen werden, da die Fahrzeugkonfiguration mit übernommen wird. Ein Textfilter kann bei der Suche des Fahrzeuges verwendet werden.

3. Die Fahrzeugstartseite wird in der nächsten Ebene angezeigt.  Auf diesem Bildschirm werden verschiedene fahrzeugspezifische Informationen, wie im folgenden Kapitel beschrieben, angezeigt.

Fahrzeugstartseite

Die Fahrzeugstartseite enthält folgende Bereiche:

[XI-1]: Die Fahrzeugübersicht enthält die spezifischen Ausstattungsmerkmale des Fahrzeugs.
Die Liste wird mit jeder Diagnosesitzung aktualisiert. Mit der Schaltfläche Löscht gewählte Fahrzeugkonfiguration können Ausstattungsmerkmale gelöscht werden. Zu löschende Zeile markieren und Schaltfläche wählen.

[XI-2]: In dem Feld Kommentare können beliebige Informationen zur Diagnosesitzung eingegeben werden. Diese können über die Schaltfläche Kommentar hinzufügen eingegeben werden.  Informationenen können in den Rubriken Nutzer, Daten oder Kommentar über die Tastatur eingegeben werden. Einmal eingegebene Kommentare können markiert und mit der Schaltfläche Markierten Kommentar löschen gelöscht werden.

[XI-3]: Über Druck der Diagnosesitzung können Diagnoseberichte erzeugt und ausgedruckt werden. Komplettbericht erzeugt einen umfassenden Diagnosebericht des Fahrzeugs. Kurzbericht erzeugt einen zusammenfassenden Diagnosebericht des Fahrzeugs.

[XI-4]: Im Bereich Schnappschussaufzeichnungen werden die für dieses Fahrzeug aufgenommenen Schnappschüsse angezeigt. Nutzername, Schnappschussdatei, Datum des Schnappschusses und Kommentare werden angezeigt. Ein markierter Schnappschuss kann mit Abspielen angezeigt werden. Einzelne Teile können mit Löschen markiert und gelöscht werden.
[XI-6]: Export speichert einen Schnappschuss auf Ihrem PC. Diesen Schnappschuss können Sie zur weiteren Analyse per E-Mail an die entsprechende Person versenden.

Datenaufzeichnung

Die Datenaufzeichnungsfunktion ermöglicht die Wiedergabe von zuvor aufgezeichneten Daten z.B. zur Weitergabe an das Technische Assistance Center. Die Dateninformationen können zur Analyse von Fahrzeugproblemen sowie Setzbedingungen von Fehlercodes verwendet werden. Die grafische Anzeige kann verwendet werden, um bei der Analyse von Komponenten Parameter zu vergleichen. Aufgenommene Datenaufzeichnungen können über die folgenden zwei Möglichkeiten geöffnet werden:

  • Auswahl der Datenaufzeichnungsdatei in der Fahrzeugstartseite im Bereich Gespeicherte Dateien durch Markieren der Aufzeichnung.
  • Mit der Schaltfläche Überprüfen bzw. Doppelklick auf die Datei wird die Datenaufzeichnungsanwendung gestartet.
  • In der Menüleiste Werkzeuge Datenaufzeichnungsansicht bzw. Start Aufzeichnung wählen. Anschliessend den Datenaufzeichnungsmanager öffnen. Den gewünschten Schnappschuss doppelklicken oder Lade Schnappschussauswahl mit anschließender Schnappschusswahl betätigen.

Unabhängig von der Auswahlmethode erscheint ein Bestätigungsfenster, sobald die Datei vollständig geladen ist. Bitte mit OK bestätigen.

Wiedergabe der Datenaufzeichnung [XIX-1]

Die Daten können in verschiedenen Anzeigefunktionen wiedergegeben werden

Die Wiedergabe wird mit der WIEDERGABE Schaltfläche gestartet. Der graue Positionsbalken zeigt den Abspielfortschritt.   Der graue Positionsbalken zeigt den Abspielfortschritt.   Alle aktiven Schaltflächen werden grau dargestellt.

Zeitsteuerung  [XIX-2]

Im Menü Zeitsteuerung wird die Wiedergabe der Datenaufzeichnungen gesteuert. Die verfügbaren Funktionen erlauben die Wiedergabe von bestimmten Abschnitten der Aufzeichnung, die Auswahl bestimmter Darstellungsoptionen sowie die Veränderung der Geschwindigkeit der Wiedergabe. Die folgenden Funktionen können ausgewählt werden:

Mit der Lesezeichenfunktion können in der Datenaufzeichnung Lesezeichen mit Kommentar gesetzt werden, welche später wieder aufgerufen werden können.  Wenn die Funktion gewählt wird, erscheint ein Eingabefenster für den Lesezeichenkommentar, in das ein beliebiger Text eingegeben werden kann. Einen Lesezeichennamen vergeben und mit OK bestätigen.
Mit der Kommentarfunktion kann eine Aufzeichnungsdatei mit einem Kommentar versehen werden. Wenn die Funktion gewählt wird, erscheint ein Eingabefenster für den Lesezeichenkommentar, in das ein beliebiger Text eingegeben werden kann. Geben Sie einen Kommentar ein und bestätigen mit OK.
Mit Dateiaufzeichnung laden (blauer Ordner) wird eine Datei zur weiteren Betrachtung geladen. Wenn Sie einen zuvor exportierten Schnappschuss ansehen möchten, entpacken Sie zuvor das entsprechende Archiv *.zip.  Die Dateien können auch durch Doppelklicken auf die Datei im Datenaufzeichnungsmanager geladen werden.
Mit der Taste Pfeil nach unten können alle Optionen zur Zeitanzeige angezeigt werden. Eigene Optionen können durch Klick auf den entsprechenden Eintrag gewählt werden.
Die Wiedergabegeschwindigkeit der Datenaufzeichnung kann mit dieser Funktion geändert werden.   Für Zeitlupenwiedergabe stehen die Wiedergabegeschwindigkeiten 1/8x, 1/4x, und 1/2x zur Verfügung.   Der Wert für die Echtzeitwiedergabe beträgt 1.   Zeitraffer wird mit 2x, 4x, und 8x ausgewählt.

7. FEHLERSUCHE

MDI Verbindungsprobleme

Falls das MDI nicht mit GDS kommunizieren kann, wird ein Warnhinweis angezeigt. Zur Fehlerbehebung folgende Schritte ausführen:

  • Es ist sicherzustellen, dass die MDI Manager Software nicht zeitgleich mit GDS betrieben wird.
  • Überprüfen Sie die korrekte Verbindung des MDI zwischen Fahrzeug und Computer.
  • Das USB-Kabel durch Quertausch mit funktionierenden MDI-Einheiten überprüfen. Überprüfen Sie alle Verbindungen und versuchen es erneut.
  • Das MDI DLC Verbindungskabel durch Quertausch mit funktionierenden MDI-Einheiten überprüfen. Überprüfen Sie alle Verbindungen und versuchen es erneut.
  • Falls der Fehler weiterhin besteht, wenden Sie sich an das Technische Assistance Center.
  • Überprüfen Sie, daß die Firewall die GDS-/MDI-Kommunikation nicht behindert.

Konflikt zwischen GDS/MDI Manager

GDS Software und MDI Manager Software können nicht gleichzeitig laufen/gestartet sein. Bei gleichzeitiger Benutzung beider Programme kann ein Konflikt auftreten. Über die Schaltfläche Details können Sie mögliche Fehlerursachen anzeigen. Sollte dieses Problem weiterhin bestehen, werden diese Information gegebenenfalls vom Technical Assistance Center angefordert.

Bei Fortbestehen dieses Softwarekonflikts wird die Meldung "Fatal error has occured" von der MDI Manager Software ausgegeben. Beenden Sie die MDI Manager Software und wechseln Sie zur GDS Anwendung. Sollte das Problem auch nach dem Wechseln ins GDS bestehen, wenden Sie sich an das Technical Assistance Center.
