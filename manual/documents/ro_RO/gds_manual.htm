Informaţii generale

GDS este conceput pentru a fi utilizat de către personalul de service instruit pentru diagnosticul şi repararea sistemelor electronice din domeniul auto.  S-a făcut tot posibilul pentru a oferi informaţii tehnice complete şi precise, bazate pe informaţiile de service din fabrică disponibile în momentul publicării.  În orice caz, se rezervă dreptul de a face oricând modificări, fără notificare prealabilă.

Pentru a vă familiariza cu GDS, funcţionalităţile şi modul său de utilizare, vă rugăm să citiţi în întregime Ghidurile utilizatorului înainte de a lucra cu GDS.

GDS oferă următoarele funcţionalităţi:

  • Citire coduri de eroare
  • Citire informaţii de identificare ECU
  • Citire şi afişare grafică listă de parametri
  • Înregistrare şi memorare date instantaneu
  • Funcţii de control
  • Funcţii de configurare şi resetare
  • Înregistrare şi memorare protocol diagnostic
  • Memorare sesiuni de diagnostic per vehicul
Important:
Programarea SPS pentru controlerele vehiculelor nu este suportată în GDS. Pentru programarea SPS Pass Through cu MDI utilizaţi aplicaţia TIS2Web / GlobalTIS SPS.

3. Instalare GDS

Prezentare generală GDS:

Aplicaţia software GDS conţine următoarele module care vor fi instalate iniţial şi actualizate frecvent în anumite perioade. [I]

  • GDS Base System
    Software-ul de bază GDS instalat pe computer
  • Produse finale GDS:
    Pachete de date de diagnostic pentru modelele de vehicule care execută rutinele de diagnostic împreună cu controlerele vehiculelor.
  • GDS Smart Start:
    Agentul aplicaţiei care compară versiunile de software Server pentru GDS Base System, Produse finale GDS şi MDI Manager între TIS2Web/GlobalTIS şi instalarea locală. În cazul în care sunt disponibile actualizări noi, veţi fi anunţat să le descărcaţi.
  • Software-ul MDI Manager (aplicaţie separată):
    Aplicaţia pentru actualizare şi configurare MDI Device (Interfaţă de diagnostic multiplă)


[I]

Instalare:

GDS poate fi instalat de pe un DVD de instalare separat sau direct descărcat din TIS2Web / GlobalTIS.

Instalarea de pe DVD-ul GDS:

  • Computerul trebuie pornit în modul Admin
  • Lansaţi executabilul de instalare din meniul de pornire al DVD-ului
  • Alegeţi limba pentru agentul de instalare
  • Datele şi configurarea GDS existente pot fi reutilizate în cazul în care este detectată o instalare GDS anterioară pe computer.
  • Acceptaţi acordul de licenţiere (obligatoriu pentru a continua)
  • Alegeţi directorul de instalare pentru GDS
  • Procesul de instalare GDS începe
  • Software-ul MDI Manager va fi instalat în cazul în care pe computer nu este instalat un software MDI Manager mai vechi. Pentru instalarea iniţială a software-ului MDI Manager, componenta hardware MDI trebuie să fie conectată în timpul procesului de instalare. Pentru mai multe detalii, consultaţi manualul MDI.
  • După terminarea procesului de instalare, GDS poate fi lansat în execuţie de la simbolul de pe ecran sau din Start/Programe/GDS. 
  • Dacă este necesar, configuraţi firewall-ul pentru a permite comunicarea GDS cu dispozitivul MDI.
  • La pornirea GDS Smart Start, procesul de actualizare afişat va fi executat înainte ca GDS să fie lansat în execuţie.

Instalarea din TIS2Web / GlobalTIS:


[II]

Important: În timpul instalării din TIS2Web, aproximativ 140 MB de date vor fi descărcate prin Internet.
  • Computerul trebuie să ruleze în modul Admin
  • Porniţi GDS din Pagina de pornire TIS2Web / GlobalTIS [II]
  • GDS Smart Start este descărcat de pe serverul TIS2Web, în funcţie de viteza conexiunii la Internet, acest lucru poate dura.
  • Alegeţi directorul de instalare pentru GDS.
  • Introduceţi parola de Internet (numai dacă pentru accesul la internet este utilizat un proxy server)
  • Dacă este necesar, configuraţi firewall-ul pentru a permite comunicarea GDS cu dispozitivul MDI.
  • Când începe instalarea GDS Smart Start, poate dura.
  • La lansarea GDS Smart Start, va fi executat procesul de actualizare GDS. [III]
  • Când descărcarea fişierelor de instalare este încheiată, se afişează versiunea de software GDS şi versiunea MDI Manager şi pot fi descărcate prin acordarea acceptului în dialogul respectiv. Prin intermediul butonului Arată progres descărcare se poate afişa bara de stare pentru progresul actualizării.
  • Software-ul MDI Manager va fi instalat în cazul în care nu există un software MDI Manager instalat pe computer sau dacă există o versiune mai veche a MDI Manager. Pentru instalarea iniţială a software-ului MDI Manager, componenta hardware MDI trebuie să fie conectată în timpul procesului de instalare. Pentru mai multe detalii, consultaţi manualul MDI.
[III] [IV]

Actualizare GDS prin Smart Start:

Lansaţi aplicaţia GDS din TIS2Web/GlobalTIS sau ca aplicaţie independentă cu ajutorul meniului de pornire din Windows. Înainte de încărcarea aplicaţiei GDS, se lansează agentul de pornire inteligentă care compară următoarele versiuni ale instalării aplicaţiei GDS pe PC cu conţinutul serverului TIS2Web /GlobalTIS:

  • GDS Smart Start
  • GDS Base System
  • MDI Manager

Smart Start anunţă utilizatorul că noi actualizări trebuie descărcate, dacă pe server sunt disponibile versiuni noi. [IV]

 
Important: Procesele de actualizare descrise necesită întotdeauna funcţionarea computerului în modul Admin.

Derularea înapoi a instalării GDS Base System

Instalarea GDS poate fi derulată înapoi până la versiunea instalată anterior, în cazul în care o actualizare GDS produce probleme operaţionale.

Pentru a executa funcţia de derulare înapoi, mergeţi la START ->PROGRAME ->GDS -> DERULARE ÎNAPOI GDS O cerinţă obligatorie a funcţiei de derulare înapoi este existenţa unei versiuni GDS mai vechi instalate pe computer.

Sistem control licenţă GDS (Licenţă)

Sistemul de control al licenţei GDS previne utilizarea neautorizată a GDS. Pentru a reînnoi licenţa aplicaţiei GDS trebuie să lansaţi aplicaţia din TIS2Web sau din GlobalTIS. Aplicaţia GDS necesită reînnoirea licenţei după 7 zile dacă în această perioadă s-a lansat numai în modul independent. 
Dacă licenţa GDS a expirat, un mesaj de eroare solicită utilizatorului lansarea aplicaţiei GDS din TIS2Web/GlobalTIS pentru reînnoirea licenţei.

Descărcare şi actualizare Produse finale GDS

Produsele finale GDS sunt aplicaţii de diagnostic care controlează serviciile de comunicare şi diagnostic dintre GDS şi controlerul vehiculului. Produsele finale GDS au versiuni per model şi vor fi permanent actualizate în TIS2Web / GlobalTIS.

GDS Smart Start identifică automat dacă versiuni noi sau actualizate la produsele finale sunt disponibile pe TIS2Web / GlobalTIS când este declanşat procesul de actualizare. Produsele finale noi trebuie descărcate mai întâi în directorul local de pe computer înainte de a putea fi instalate în GDS. Actualizările produselor finale deja instalate vor fi descărcate automat şi trebuie doar să fie instalate în baza de date master GDS.


Descărcarea iniţială şi actualizarea produselor finale GDS

  • Lansaţi aplicaţia GDS din TIS2Web/GlobalTIS sau ca aplicaţie independentă cu ajutorul meniului de pornire din Windows.
  • Alegeţi butonul Adăugare pachet diagnostic nou din ecranul de autentificare GDS.
  • Apăsaţi butonul Actualizare listă produse finale de pe server. [V-1]. Pachetele disponibile pe server şi care nu au fost descărcate încă sunt afişate cu simbolul. 
  • Marcaţi şi descărcaţi pachetele de produse finale [V-2]. Funcţia de filtrare ajută la căutarea unor anumite produse finale. Pachetele de produse finale descărcate sunt afişate cu simbolul. 
  • Instalaţi pachetele descărcate în baza de date master GDS; pachetele instalate sunt afişate în fereastra din dreapta a aplicaţiei [V-3].


[V]

Actualizare produse finale GDS instalate în baza de date master:

  • Faceţi clic pe butonul Actualizare pachet diagnostic din ecranul de autentificare GDS [VI-1]. GDS compară versiunile produselor finale GDS instalate cu produsele finale disponibile pe server şi descarcă pachetele actualizate.
  • Pachetele descărcate sunt afişate pentru instalare în fereastra din stânga jos. Selectaţi butonul Instalare sau Instalare toate pentru a începe instalarea. [VI-2].


[VI]:

Derulare înapoi actualizare produs final

Procesul de derulare înapoi este în general acelaşi proces descris în capitolul Descărcarea iniţială şi actualizarea produselor finale GDS

  • Alegeţi butonul Adăugare pachet diagnostic nou din ecranul de autentificare GDS.
  • Apăsaţi butonul Actualizare listă produse finale de pe server. [V-1]
  • Marcaţi şi descărcaţi versiunile dorite de pachete de produse finale mai vechi. Funcţia de filtrare ajută la căutarea unor anumite produse finale.
  • Instalaţi pachetele descărcate în baza de date master GDS.

Dezinstalarea GDS:

GDS poate fi dezinstalat prin pornirea următorului program:

START -> PROGRAME -> GDS ->DEZINSTALARE GDS

Modificarea limbii GDS / Setărilor regionale:

GDS utilizează setările de limbă ale sistemului de operare din computer. Nu există un meniu în GDS pentru modificarea setărilor de limbă. Pentru a modifica limba utilizată în GDS, efectuaţi următorii paşi.

  • Deschideţi PANOUL DE CONTROL Windows cu următoarea secvenţă: START -> SETTINGS (SETĂRI)-> CONTROL PANEL (PANOU DE CONTROL)
  • Deschideţi meniul OPŢIUNI REGIONALE ŞI DE LIMBĂ
  • Selectaţi ţara dorită în STANDARDE ŞI FORMATE
  • Deschideţi meniul ADVANCED (AVANSAT). Selectaţi limba dorită din caseta derulantă pentru PROGRAME NON UNICODE
  • Configuraţi OPŢIUNI REGIONALE pentru a modifica unităţile de măsură afişate în GDS.

4. Pornirea

AVERTISMENT:
Înainte de utilizarea GDS, este important să luaţi următoarele măsuri de precauţie:

  • Utilizarea unei versiuni învechite poate deteriora modulele vehiculului. GDS şi Interfaţa de diagnostic multiplă trebuie să aibă ultima versiune de software.
  • Asiguraţi-vă că bateria vehiculului este complet încărcată Tensiunea bateriei pentru GDS trebuie să fie între 12 şi 14 volţi.
  • În orice caz, la vehicul nu trebuie să fie conectat un încărcător de baterie când utilizaţi MDI.
  • Asiguraţi-vă că sunt bine fixate conexiunile cablului. Un cablu deconectat POATE duce la defectarea controlerului.
  • La utilizarea unui laptop/alt dispozitiv de afişare (PDA etc) pentru GDS, asiguraţi-vă că sursa de alimentare este corect conectată. Dacă alimentarea se face cu un adaptor şi un cablu de alimentare se deconectează, este posibil sa ducă la întreruperea programului şi la deteriorarea modulului de control. Dacă laptopul funcţionează de la sursa de alimentare internă (baterii), asiguraţi-vă că sunt corespunzător încărcate pentru a încheia procesul GDS.
AVERTISMENT
Dacă trebuie programată o unitate ECU, aceasta trebuie instalată în vehicul înainte de începerea acestui proces. Asiguraţi-vă că bateria vehiculului este complet încărcată.
Important
Pentru a reduce riscul pierderii de semnal, MDI trebuie configurat pe opţiunea pentru cea mai stabilă comunicare în locaţia dumneavoastră.Puteţi alege dintre variantele USB, Ethernet sau Ethernet fără fir.

Procedura de pornire GDS:

GDS poate fi pornit din TIS2Web / GlobalTIS făcând clic pe pictograma GDS. Alternativ, GDS poate fi lansat în modul independent din meniul Start din Windows.

  1. Pentru a începe, faceţi clic pe simbolul GDS din meniul principal TIS2Web / GlobalTIS. Dacă porniţi GDS în modul independent, faceţi clic pe START/PROGRAME/GDS
  2. La lansarea GDS Smart Start, va fi executat procesul de actualizare afişat. Dacă pe server este disponibilă o nouă versiune GDS sau MDI Manager, veţi fi anunţat să o descărcaţi şi să o instalaţi în computer. Faceţi clic pe Continuare, pentru a începe descărcarea sau instalarea [IV]. Dacă GDS este lansat în modul independent, nu este posibilă nicio actualizare.
  3. Pagina de autentificare GDS este încărcată.

  • [X-3] Secţiunea Identificare vehicul: selectaţi marca, modelul şi anul modelului înainte de a continua.
  • [X-4] Istoric diagnostic vehicul: Dacă un vehicul a avut anterior diagnosticul efectuat cu acest sistem GDS, NIV-ul şi informaţiile asociate vor fi memorate în istoricul vehiculului. Selectaţi din listă vehiculul la care urmează a fi efectuat diagnosticul şi continuaţi direct la Pagina de pornire vehicul. Este disponibilă o funcţie de filtrare după un text liber.

Redare instantaneu

Funcţia Redare instantaneu permite examinarea datelor instantaneu vehicul înregistrare anterior. Această informaţie poate fi folosită la analiza problemelor vehiculului şi detectarea condiţiilor de eroare declanşator. Funcţia grafică poate fi utilizată pentru a compara parametrii pentru a vedea dacă o componentă funcţionează corespunzător. Instantaneele înregistrate pot fi selectate pentru vizualizare prin următoarele trei metode:

  • Selectarea unui fişier instantaneu din Pagina de pornire vehicul şi făcând clic pe opţiunea Examinare.
  • Faceţi dublu clic pe instantaneul selectat din Pagina de pornire vehicul.
  • Intrarea în opţiunea Instrumente din opţiunea Meniu şi selectând Redare instantaneu. Apoi, deschideţi Manager instantaneu. Faceţi dublu clic pe fişierul instantaneu pe care doriţi să-l vizualizaţi sau faceţi clic pe opţiunea Încărcare instantaneu şi alegeţi un fişier instantaneu.

Indiferent de metoda de selecţie, din ecranul Redare instantaneu va fi afişată o fereastră pop-up Fişier încărcat. Selectaţi OK în fereastra pop-up.

Funcţii de redare instantaneu [XIX-1]

Există câteva funcţii de redare instantaneu care sunt utilizate pentru a reda instantaneele înregistrate

Funcţia Redare este utilizată pentru a reda instantaneul în faţă făcând clic pe această opţiune.   Când este utilizată această opţiune, pe ecran se va mişca transversal un marker.   Toate opţiunile din această zonă devin gri atunci când sunt active.
Funcţia Pauză este utilizată pentru a face o pauză la redarea unui instantaneu. Când se face clic a doua oară pe acestă opţiune, redarea instantaneului va porni din locul în care a rămas.
Funcţia Stop este utilizată pentru a opri redarea unui instantaneu. Instantaneul va porni de la început dacă după utilizarea funcţiei Stop se selectează din nou funcţia Redare.

Control timp [XIX-2]

Control timp este utilizată pentru a controla redarea instantaneului. Această funcţie permite selecţia unui anumit cadru de timp, afişarea diferitelor vizualizări şi controlul vitezei de afişare. Următoarele funcţii pot fi selectate direct din Control timp.

Opţiunea Semn de carte vă permite să adăugaţi un anumit comentariu în afişajul curent şi să reveniţi la acest punct în viitor. Când este selectată această opţiune, este afişată o fereastră pop-up. Introduceţi un nume pentru semnul de carte şi selectaţi OK.
Notepad poate fi selectat pentru a adăuga comentarii la fişierul de redare instantaneu în momentul curent de timp. Când este selectată această opţiune, este afişată o fereastră pop-up. Introduceţi un comentariu pentru fişierul instantaneu şi selectaţi OK.
Încărcare instantaneu (folderul albastru) deschide un fişier instantaneu şi îl pregăteşte pentru analize ulterioare. Dacă doriţi să vedeţi un snapshot exportat, mai întâi dezarhivaţi arhiva corespunzătoare *.zip.  Acestă funcţie este disponibilă şi făcând clic pe fişierul instantaneu disponibil din Manager instantaneu.
Când este selectată săgeata jos, se afişează toate opţiunile Control timp. De asemenea, pot fi selectate opţiuni individuale făcând clic pe elementul dorit.
Viteza de afişare instantaneu poate fi controlată prin selectarea acestei opţiuni.   Pot fi selectate viteze mai mici de 1/8X, 1/4X şi 1/2X.   De asemenea, poate fi selectată viteza în timp real la 1.   Cu ajutorul acestei selecţii pot fi atinse şi viteze mai mari de 2X, 4X şi 8X.

Selecţie vizualizare [XIX-3]

Fiecare Selecţie vizualizare oferă un mod diferit de afişare a datelor vehiculului pentru analiză. Următoarele funcţii pot fi selectate din Vizualizări disponibile.

  • Vizualizare listă parametri - afişează informaţiile ce sosesc din fişierul instantaneu. Valoarea curentă a parametrilor este afişată pe ecran.
Important
Numele parametrului va fi afişat în limba originală utilizată pentru captură dacă nu există o traducere disponibilă pentru acel nume.
  • Vizualizarea Jurnal, care conţine toate informaţiile din fişierul instantaneu. Toate valorile parametrilor sunt listate în ordinea în care au fost înregistrate. Anumiţi parametri pot fi adăugaţi sau eliminaţi dintr-o listă de urmărire făcând clic pe elementul corespunzător.
  • Vizualizarea Stare - arată valorile tuturor parametrilor într-un anumit cadru de timp. Aceste valori pot fi utilizate pentru compararea cu vizualizarea declanşator.
  • Vizualizarea Declanşator - afişează toţi declanşatorii de sistem şi pe cei definiţi de utilizator. Această funcţie poate fi utilizată când există o combinaţie de factori implicaţi într-o problemă de diagnostic.  Când este îndeplinită o condiţie de declanşare, pe ecran va fi afişat un triunghi de atenţionare.
    • Încărcare declanşatori - Vă permit să încărcaţi un fişier care conţine definiţiile de declanşare.
    • Salvare declanşatori - Vă permite salvarea declanşatorului pe unitatea de stocare informaţii a computerului.
    • Eliminare declanşator - Vă permite să eliminaţi declanşatorul curent. Acesta nu va mai fi prezent.
    • Editare declanşator - afişează o fereastră pop-up unde puteţi modifica limitele şi parametrii conţinuţi într-un declanşator.
    • Adăugare declanşator - afişează o fereastră pop-up pentru a configura un declanşator.
  • Vizualizare cod de eroare - După selectarea funcţiei Redare şi alegere acestei opţiuni, această funcţie deschide lista codurilor de eroare înregistrate.
  • Vizualizare grafic linie - afişează date care pot fi vizualizate pe axa timpului. Din această vizualizare, pot fi selectaţi anumiţi parametri pentru reprezentare grafică prin bifarea casetei de sub Selectate. În partea de jos a graficului există patru funcţii: Transfocare spre interior (măreşte dimensiunea), Transfocare spre exterior (micşorează dimensiunea), Semne de carte şi Pauză defilare. Fiecare din aceste funcţii poate fi accesată făcând clic pe ea.

7. DEPANAREA

Probleme de conexiune MDI

Dacă conexiunile MDI se pierd sau nu sunt corect conectate la vehicul/computer, va fi afişată o fereastră pop-up. Efectuaţi următorii paşi:

  • Asiguraţi-vă că aplicaţia software MDI Manager nu rulează în acelaşi timp când este efectuată GDS.
  • Asiguraţi-vă că MDI este corect conectată la vehicul şi la computer.
  • Utilizaţi un "cablu USB" despre care sunteţi sigur că este bun de la altă unitate MDI dacă este disponibil. Efectuaţi toate conexiunile şi încercaţi din nou.
  • Utilizaţi un "cablu de conexiune DLC MDI" cunoscut ca fiind bun de la altă unitate MDI dacă este disponibil. Efectuaţi toate conexiunile şi încercaţi din nou.
  • Dacă condiţiile sunt încă prezente, contactaţi Customer Support.
  • Asiguraţi-vă că firewall-ul nu blochează comunicarea GDS/MDI.

Conflict software GDS/MDI Manager

Aplicaţia software GDS şi aplicaţia software MDI Manager nu pot rula în acelaşi timp. Dacă ambele programe software sunt rulate în acelaşi timp, poate rezulta un conflict software. Opţiunea Detalii va oferi informaţiile posibile privitor la eroare. Dacă problema persistă, vi se poate solicita de la Customer Support să furnizaţi aceste informaţii.

Dacă se continuă apariţia conflictului software, poate fi afişată o fereastră pop-up "A apărut o eroare fatală" de aplicaţia software MDI Manager. Închideţi aplicaţia software MDI Manager şi reveniţi în programul GDS. Dacă problema este încă prezentă după intrarea în GDS, contactaţi Customer Support.
