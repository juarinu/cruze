Descripción de DTC

DTC B3984: Identificador de entorno del dispositivo 1 no programable

Descripción del circuito/sistema

Cuando se programan y configuran ciertos módulos durante la instalación, éstos memorizan un identificador de inmovilizador específico que es exclusivo de ese vehículo. El identificador de inmovilizador se utiliza para impedir el intercambio de módulos entre vehículos. El módulo de control de la carrocería (BCM) es que el alberga el identificador del inmovilizador. El cuadro de instrumentos, el módulo de control del freno electrónico (EBCM) y el módulo de control HVAC informan al identificador del inmovilizador durante su proceso de configuración. Durante el servicio del vehículo, cada uno de estos módulos compara su inmovilizador identificador respecto al BCM. Si los identificadores coinciden, se permite el arranque del vehículo. Si no coincide un identificador del inmovilizador, se desactiva el arranque del vehículo.

Condiciones para el funcionamiento de DTC

El encendido esté en la posición RUN o ACCESSORY.

Condiciones para ajustar el DTC

Se ha programado un identificador con entorno incorrecto o no es programable.

Acción realizada cuando se registra el DTC
    • Se encenderá el indicador de seguridad del cuadro de instrumentos.
    • Se desactivará el arranque del motor.

Condiciones para el borrar el DTC

Un DTC actual se borrará cuando el módulo obtenga un identificador de entorno.

Verificación del circuito/sistema

Realice el procedimiento de Programación y configuración en el módulo de control correspondiente. Compruebe que no vuelve a activarse el DTC tras la programación.

⇒Si el DTC se reposiciona, sustituir el módulo de control involucrado.

Instrucciones de reparación

Realice Verificación de la reparación por diagnóstico después de completar el procedimiento de diagnóstico.
