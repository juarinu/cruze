Descripción y funcionamiento del antirrobo de contenido (CTD)

Cuando está activado, el sistema antirrobo de contenido está diseñado para impedir que se robe el contenido del vehículo encendiendo las bocinas y las luces exteriores durante unos 30 segundos cuando se detecta una entrada no autorizada en el vehículo. No obstante, el sistema antirrobo de contenido no afecta al arranque del motor.

Con el sistema antirrobo de contenido activado, se puede considerar como acceso no autorizado cualquiera de los siguientes:

    • Entrada no autorizada en el área bajo el capó
    • Acceso no autorizado al compartimento trasero
    • Al abrir cualquier puerta sin utilizar el comando DESBLOQUEO desde un transmisor de entrada sin llave
    • Tras volver a conectar la batería, si la batería se desconectó con el sistema antirrobo de contenido armado

Los componentes del sistema antirrobo de contenidos son los siguientes:

    • Módulo electrónico de la carrocería (BCM)
    • Módulo del receptor de función remota/receptor de control remoto de la cerradura de la puerta
    • Indicador de seguridad
    • Interruptores de puerta abierta
    • Interruptor de maletero abierto
    • Interruptor de capó abierto

Activación del sistema antirrobo de contenido

Utilice el siguiente procedimiento para activar el sistema:

  1. Coloque la palanca de cambio en P (estacionamiento).
  2. Apague el encendido.
  3. Abra una de las puertas.

    4. Nota: El sistema no está activado si las puertas se bloquean manualmente; debe usarse el interruptor del cierre centralizado de las puertas o el transmisor de entrada sin llave para activar el sistema antirrobo de contenidos.

  4. Cierre las puertas con el interruptor del cierre centralizado o con el botón LOCK (Bloqueo) del transmisor. El sistema está en el modo de espera y no pondrá en marcha el temporizador de activación hasta que se cierren todas las puertas.
  5. El sistema empezará la secuencia de activación inmediatamente después de que se cierre la última puerta. Si se utiliza el transmisor de entrada sin llave para activar el sistema después de cerrar las puertas del vehículo, la secuencia de activación empezará en cuanto se reciba el comando LOCK (Bloqueo) del transmisor.
  6. Si se pulsa dos veces el botón LOCK (Bloqueo) del transmisor de entrada sin llave se derivará la función de activación retardada y se forzará la activación del sistema.
  7. Si el vehículo está equipado con bloqueos pasivos, el vehículo iniciará la secuencia de activación cuando se deje de detectar un transmisor de entrada sin llave.

Bloqueo de las puertas del vehículo sin activar el sistema antirrobo de contenidos.

Se puede bloquear las puertas del vehículo sin activar el sistema antirrobo de contenido. Si utilice los seguros manuales de las puertas o una llave para bajar los seguros del vehículo, las puertas quedarán bloqueadas pero no se activará el sistema antirrobo.

Desactivación de un sistema activado/apagado de la alarma

Si el interruptor del cierre centralizado o el transmisor de entrada con llave ha solicitado la activación del sistema, tendrá que desactivarse.

Nota: Desconectar la batería o quitar fusibles no deshabilita el modo de activación, ya que el BCM guarda el estado de modo del sistema antirrobo de contenidos en la memoria.

    • Para desactivar el sistema antirrobo de contenido en el modo de espera, realice lo siguiente:
       - Pulse uno de los interruptores del cierre centralizado.
       - Pulse el botón UNLOCK (Desbloqueo) del transmisor de entrada sin llave.
       - Meta una llave válida en el contacto y póngala en la posición ON.
    • Para desactivar el sistema antirrobo de contenido en el modo activado (sin suceso) o cuando se dispara (durante un suceso de alarma):
       - Pulse el botón UNLOCK (Desbloqueo) del transmisor de entrada sin llave.
       - Meta una llave válida en el contacto y póngala en la posición ON

Descripción del circuito del sistema antirrobo de contenidos

A continuación figura una descripción de cada uno de los componentes utilizados en el sistema antirrobo de contenido:

Módulo electrónico de la carrocería

El sistema antirrobo de contenidos es una función interna del BCM que utiliza datos de serie e información diversa de las entradas de los interruptores para gestionar las funciones del sistema antirrobo de contenidos. Cuando el BCM detecta una entrada no autorizada, activa las bocinas y las luces exteriores. El BCM tiene 4 modos básicos (desactivado, en espera, activado y alarma) para el funcionamiento del sistema antirrobo de contenidos. Los distintos modos se describen a continuación.

  1. El BCM mantiene al sistema antirrobo de contenidos en modo desactivado hasta que se detectan las siguientes condiciones:
    2. • Llave de encendido en la posición OFF.
    • Puertas bloqueadas con el interruptor del cierre centralizado o con el botón LOCK (Bloqueo) del transmisor.
  2. El BCM entra en el modo de espera cuando se detectan las siguientes condiciones. Si hay alguna puerta abierta cuando se solicita el modo de activación, el modo de espera no pone en marcha el temporizador hasta que se cierra la última puerta.
  3. Cuando se cierra la última hora, se enciende un temporizador de 15 segundos. Cuando la cuenta del temporizador llega a su fin, el BCM entra en el modo de activación. Después de este período, cualquier entrada forzada dispara el modo de alarma.
  4. Cuando el BCM detecta una entrada forzada, este pasa al modo de alarma. El BCM enciende las bocinas y las luces exteriores durante 30 segundos. Después transcurre un plazo de tres minutos en que la bocina deja de sonar. Si no se detectan otras intrusiones después de este período, la bocina sigue sin sonar. El sistema debe desactivarse o cancelar la condición de intrusión tras el período para que el sistema pueda salir del modo de alarma.

Módulo del receptor de función remota/receptor de control remoto de la cerradura de la puerta

El sistema de entrada sin llave puede activar y desactivar el sistema antirrobo de contenido. Cuando el módulo de entrada sin llave recibe una señal de bloqueo o desbloqueo de la puerta del transmisor, el módulo de entrada sin llave envía un mensaje al BCM a través de datos de serie para que se ejecuten las funciones de activación/desactivación que correspondan.

Indicador de seguridad

El BCM enciende el LED de seguridad del panel de instrumentos superior. El sistema antirrobo de contenidos utiliza el LED de seguridad para informar al conductor del estado del sistema antes de activarlo.

Interruptores de puerta abierta

El sistema antirrobo de contenido utiliza los interruptores de puerta abierta como indicador de estado para encender la alarma. El módulo de control de la carrocería supervisa los interruptores de puerta abierta a través de una señal de entrada discreta procedente de cada uno de los interruptores. Si el BCM recibe una señal indicando el estado abierto de una puerta cuando el sistema antirrobo de contenidos está activado, el BCM activará la alarma.

Interruptor de capó abierto

El sistema antirrobo de contenido utiliza el interruptor de puerta abierta como indicador de estado para encender la alarma. El BCM supervisa el interruptor de capó abierto a través de una señal de entrada discreta del interruptor. Si el BCM recibe una señal indicando que se ha abierto el capó cuando el sistema antirrobo de contenidos está activado, el BCM activará la alarma.

Interruptor de maletero abierto

El sistema antirrobo de contenido utiliza el interruptor de maletero abierto como indicador de estado para encender la alarma. El BCM supervisa el interruptor de maletero entreabierto a través de una señal de entrada discreta del interruptor. Si el BCM recibe una señal indicando que se ha abierto el maletero cuando el sistema antirrobo de contenidos está activado, el BCM activará la alarma.

Señales de entrada

El BCM supervisa las siguientes señales de entrada del sistema antirrobo de contenidos:

    • Los interruptores de puerta abierta
    • Los botones LOCK/UNLOCK (Bloqueo/desbloqueo) del transmisor de entrada sin llave; un mensaje procedente del módulo del receptor de función remota/receptor de control remoto de la cerradura de la puerta
    • El estado del inmovilizador. El BCM utiliza el estado del inmovilizador para desactivar el sistema o silenciar una alarma cuando se utiliza la llave correcta para arrancar el vehículo
    • El interruptor de maletero abierto
    • El interruptor de capó abierto

Salidas

El BCM controla los siguientes elementos del sistema antirrobo de contenidos:

    • El relé de la bocina
    • Las luces exteriores
