Descripción de DTC

DTC U2103 00: Menos controladores de los programados

Descripción de circuito

Los módulos conectados a los circuitos de datos de serie GMLAN controlan las comunicaciones de datos serie durante el funcionamiento normal del vehículo. Entre los módulos se intercambian comandos e información operativa. Los módulos disponen de información programada sobre los mensajes que es necesario intercambiar en los circuitos de datos serie para cada una de las redes virtuales. El módulo receptor también supervisa los mensajes y utiliza algunos mensajes periódicos como indicación de la disponibilidad del módulo transmisor.

Se activará el DTC U2103 si se detectan menos módulos en el bus antes de que los módulos puedan aprender los ID de otros módulos. Si el bus de comunicación está abierto o si se pierde la alimentación de corriente del módulo cuando el vehículo está en modo OFF, y después se establece en el modo RUN, se activará el DTC U2103 en los módulos específicos.

Condiciones para el funcionamiento de DTCs
    • La tensión alimentada a los módulos está dentro del intervalo operativo de tensión normal.
    • El modo de potencia del vehículo requiere que se produzca la comunicación de datos serie.
    • El DTC U2100 no tiene un estado actual.

Condiciones para ajustar el DTC

No se ha recibido un mensaje supervisado periódicamente que incluye la disponibilidad del módulo transmisor.

Acción realizada cuando se registra el DTC

El módulo utiliza un valor por defecto para el parámetro faltante.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio sin que se repita una avería.

Ayuda al diagnóstico
    • El diagnóstico de este DTC se realiza a través del síntoma o un DTC adicional, consulte Síntomas - Comunicaciones de datos o DTC U0100-U02FF .
    • Es probable que un estado intermitente se deba a un cortocircuito en los circuitos de datos de serie GMLAN. Utilice el procedimiento DTC U0100-U02FF para aislar un estado intermitente.
