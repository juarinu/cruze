Descripción del circuito/sistema

Los sensores de objeto son sensores de 3 cables que se utilizan para determinar la distancia entre el vehículo y un objeto de interés. El módulo de alarma de objeto suministra 8 V a los sensores de objeto mediante el circuito de referencia de 8 V y ofrece masa mediante el circuito de referencia baja. El módulo de alarma de objeto dispara los sensores en un ciclo secuencial. Después de que cada sensor transmita, el módulo de alarma de objeto utiliza el anillo del sensor recibido a través del circuito de señal para calcular la distancia y posicionar un objeto.

Condiciones para la activación de DTC
    • Encendido ON
    • El estacionamiento asistido está activado. La activación tiene lugar al iniciar la marcha atrás.

Condiciones para ajustar el DTC

B0958 01, B0959 01, B0960 01 o B0961 01

El módulo de alarma de objeto ha detectado que la tensión del circuito sensor es superior a 11,5 V.

B0958 06, B0959 06, B0960 06 o B0961 06

El módulo de alarma de objeto ha detectado que la tensión del circuito sensor es menor a 0,5 V.

B0958 08, B0959 08, B0960 08 o B0961 08

El módulo de alarma de objeto ha recibido una señal inválida.

B0958 21, B0959 21, B0960 21 o B0961 21

El sensor de objetos no determina ningún cambio en la posición de un objeto mientras el vehículo está en movimiento.

B0958 3A, B0959 3A, B0960 3A o B0961 3A

El módulo de alarma de objeto determina que se ha instalado un sensor de tipo incorrecto.

Acción realizada cuando el DTC se ajusta
    • El estacionamiento asistido está desactivado.
    • El indicador de estacionamiento asistido en el cuadro del tablero de instrumentos se activará.

Condiciones para el borrar el DTC

La anomalía que dio lugar al código de avería (DTC) ya no está presente.

Verificación del circuito/sistema
  1. Verificar que no hay un DTC B138A 03 o B138A 07.
    2. ⇒Si existe un DTC, consulte DTC B138A .
  2. Verificar que no hay un DTC B0958 01, B0959 01, B0960 01 o B0961 01.
    3. ⇒ Si hay un DTC, comprobar el sensor apropiado para ver si está contaminado por barro, hielo o nieve. Si no se diagnostica contaminación, consulte las pruebas del sistema/circuito.
  3. Verificar que no hay un DTC B0958 21, B0959 21, B0960 21 o B0961 21.
    4. ⇒ Si hay un DTC, comprobar el sensor apropiado para ver si está contaminado por barro, hielo o nieve. Si no se diagnostica contaminación, cambiar el sensor.
  4. Verificar que no hay un DTC B0958 3A, B0959 3A, B0960 3A o B0961 3A.
    5. ⇒ Si hay un DTC, sustituir el sensor apropiado con un sector del tipo correcto.

Comprobación del circuito/sistema
  1. Encendido desactivado durante 30 s, desconectar el conector del mazo de cables en el sensor apropiado de objeto posterior.
  2. Pruebe con menos de 5 Ω entre el terminal 2 del circuito referencia baja y masa.
    3. ⇒Si es mayor que el valor especificado, compruebe si el circuito de referencia baja tiene una resistencia abierta/alta. Si según la comprobación el circuito está bien, sustituya el módulo de la alarma de objetos K41.
  3. Encendido conectado, activar el estacionamiento asistido, comprobar que haya 7,5-9,0 V entre el terminal 1 del circuito de referencia de 8 V y masa.
    4. ⇒Si es menor que el margen especificado, compruebe si hay un circuito abierto/resistencia alta en el circuito de referencia 8 V. Si según la comprobación el circuito está bien, sustituya el módulo de la alarma de objetos K41.
  4. Encendido desactivado, retirar el sensor de objetos que está fallando y cambiar su emplazamiento con otro sensor de objeto posterior que esté funcionando correctamente. Conectar el conector del mazo de cables en ambos sensores.
  5. Encendido conectado, verificar que el DTC se ha reposicionado para la nueva ubicación donde está instalado el sensor que tiene un fallo de funcionamiento.
    6. ⇒Si el DTC no se ha reposicionado para la nueva ubicación, comprobar el terminal 3 del circuito de señal para ver si existe un cortocircuito de tensión, un cortocircuito a masa o una resistencia alta/abierta. Si según la comprobación el circuito está bien, sustituya el módulo de la alarma de objetos K41.
    ⇒Si el DTC se reposiciona para la nueva ubicación en la que se encuentra el sensor con el fallo de funcionamiento, sustituir el sensor de objeto posterior apropiado.
