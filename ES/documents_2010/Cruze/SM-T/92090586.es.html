1. Mal funcionamiento del control de velocidad

Descripción del circuito/sistema

El interruptor del control de velocidad es una entrada al módulo de control del motor (ECM). El ECM supervisa los circuito de señal de los interruptores de encendido/apagado del control de velocidad, establecer/deceleración y reanudar/acelerar para detectar cuándo se ha activado un interruptor de la función del control de velocidad. El ECM detecta una señal de voltaje predeterminado al activarse un interruptor de la función del control de velocidad, ya que cada interruptor posee un valor de voltaje diferente.

Condiciones para el funcionamiento de DTC
    • El encendido está en la posición ON.
    • El interruptor de crucero está en ON.

Condiciones para ajustar el DTC
    • El ECM detecta una señal de voltaje inválida en el circuito de señal de conexión del control de velocidad.
    • La condición anterior se da durante más de 2 segundos.
    • El ECM ejecuta este diagnóstico cada 0,05 segundos.

Acciones realizadas cuando el DTC se ajusta
    • El testigo luminoso de avería (MIL) no se iluminará.
    • El sistema de control de crucero está desactivado.

Condiciones para el borrar el DTC
    • El estado responsable para ajustar el DTC ya no existe.
    • Se borrará un historial de DTC después de que hayan pasado 40 ciclos de encendido sin fallos.

Verificación del circuito/sistema

Con el motor encendido, mientras gira el volante de dirección hasta sus topes, observe el parámetro de la herramienta de diagnóstico Interruptor de control de velocidad, mientras pulsa cada uno de los interruptores de control de velocidad. El parámetro deberá alternar entre encender, apagar, reanudar, configurar a medida que se pulsa el interruptor correspondiente.

Comprobación del circuito/sistema
  1. Con el encendido OFF, desconecte el conector del mazo de cables del interruptor del control de velocidad.
  2. Con el encendido conectado, compruebe B+ entre el terminal 2 del circuito B+ y masa.
    3. ⇒Si es inferior al valor especificado, compruebe si el circuito B+ tiene un cortocircuito a masa o una resistencia abierta/alta.
  3. Verifique con la herramienta de diagnóstico que el parámetro del interruptor de ENCENDIDO/APAGADO del control de velocidad muestra APAGADO.
    4. ⇒Si no se corresponde con el valor especificado, compruebe si el terminal 1 del circuito de señal tiene un cortocircuito a voltaje. Si la prueba del circuito sale bien, sustituya el ECM.
  4. Monte un cable puente con fusible de 3A entre el terminal 1 del circuito de señal y el terminal 2 del circuito de B+. Verifique con la herramienta de diagnóstico que el parámetro del interruptor de ENCENDIDO/APAGADO del control de velocidad muestra ENCENDIDO.
    5. ⇒Si no se corresponde con el valor especificado, compruebe si el terminal 1 del circuito de señal tiene un cortocircuito a masa o una resistencia abierta/alta. Si la prueba del circuito sale bien, sustituya el ECM.
  5. Verifique con la herramienta de diagnóstico que el parámetro del interruptor Cancelar control de velocidad muestra Inactivo.
    6. ⇒Si no se corresponde con el valor especificado, compruebe si el terminal 3 del circuito de señal tiene un cortocircuito a voltaje. Si la prueba del circuito sale bien, sustituya el BCM.
  6. Monte un cable puente con fusible de 3A entre el terminal 3 del circuito de señal y el terminal 2 del circuito de B+. Verifique con la herramienta de diagnóstico que el parámetro del interruptor Cancelar control de velocidad muestra Activo.
    7. ⇒Si no se corresponde con el valor especificado, compruebe si el terminal 3 del circuito de señal tiene un cortocircuito a masa o una resistencia abierta/alta. Si la prueba del circuito sale bien, sustituya el BCM.
  7. Si las pruebas de todos los circuitos son normales, compruebe o sustituya el interruptor del control de velocidad.

Comprobación de componentes
  1. Con el encendido OFF, desconecte el conector del mazo de cables del interruptor del control de velocidad.
  2. Con el interruptor del control de velocidad activado, mida la resistencia entre los terminales 2 y 1 ó 2 y 3 del interruptor Cancelar en el lado del interruptor del control de velocidad del conector. Active individualmente y mantenga presionado cada interruptor de la función del control de velocidad y compare la lectura de resistencia con los valores recogidos en la tabla del interruptor correspondiente.
    3. ⇒Si cualquiera de las mediciones de resistencia de los interruptores individuales no está dentro de los valores especificados, sustituya el interruptor del control de velocidad.
