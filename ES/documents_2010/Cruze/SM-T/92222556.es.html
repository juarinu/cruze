Descripción de DTC

DTC P1793 71: Datos no válidos sobre el funcionamiento del sensor de velocidad del vehículo

Descripción del circuito/sistema

El módulo de control del freno electrónico (EBCM) envía información sobre el sensor de velocidad de la rueda al módulo de control del motor (ECM). Los datos se envían al ECM a través de un bus de datos en serie.

Condiciones para el funcionamiento de DTC

El encendido está en la posición ON.

Condiciones para ajustar el DTC

El ECM detecta una señal no válida desde el módulo de control del freno electrónico.

Acción realizada cuando se registra el DTC

El DTC P1793 es un DTC de tipo C.

Condiciones para el borrar el DTC

El DTC P1793 es un DTC de tipo C.

Comprobación del circuito/sistema
  1. Observe los registros de fallos/cuadro congelado con la herramienta de diagnóstico.
  2. Borre los DTC con una herramienta de diagnóstico.
  3. Encendido desconectado durante 30 s.
  4. Conduzca el vehículo dentro de la condiciones para que se ejecute el DTC. También podría conducir el vehículo dentro de las condiciones que se haya observado del registro de fallos/cuadro congelado.
    5. ⇒Si el DTC P1793 falla durante este encendido, sustituya el módulo de control del freno electrónico (EBCM) K17.
