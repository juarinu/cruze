Descripción del circuito/sistema

El módulo de control del motor (ECM) utiliza el circuito de control de conexión del generador para controlar la carga del generador en el motor. Un controlador de alta en el ECM aplica una tensión al regulador de tensión. Esto controla el regulador de tensión para que coloque el circuito de campo en Activado y Desac. El ECM monitoriza el estado del circuito de control de conexión del generador. El ECM debería detectar una tensión baja en el circuito de control de conexión del generador cuando el encendido está conectado y el motor está desactivado, o cuando el sistema de carga tiene un fallo de funcionamiento. Con el motor funcionando, el ECM debería detectar tensión alta en el circuito de control de conexión del generador. El ECM realiza pruebas de encendido y funcionamiento para determinar el estado del circuito de control de conexión del generador.

Condiciones para el funcionamiento de DTC
    • Con el encendido conectado y motor apagado, para la prueba de encendido conectado.
    • El motor está funcionando durante la prueba de funcionamiento.

Condiciones para ajustar el DTC
    • Durante la prueba de encendido conectado, el ECM detecta alta tensión en el circuito de control de conexión del generador durante 5 segundos.
    • Durante la prueba de funcionamiento, el ECM detecta baja tensión en el circuito de control de conexión del generador durante 15 segundos.

Acción realizada cuando se registra el DTC

El DTC P0621 es un código de avería de tipo C.

Condiciones para el borrar el DTC

El DTC P0621 es un código de avería de tipo C.

Comprobación del circuito/sistema
  1. Comprobar que se ilumina una luz de prueba entre el terminal 1 X1 del generador B+ G13 y masa.
    2. ⇒Si la luz de prueba no se ilumina, pruebe el circuito B+ para ver si hay un cortocircuito a tierra o una resistencia abierta/alta.
  2. Con el encendido desactivado, desconecte el conector del mazo de cables X2 del generador G13.
  3. Encendido conectado, comprobar si hay menos de 1 V entre el terminal 1 X2 del circuito de control y masa.
    4. ⇒Si es superior al margen especificado, compruebe si el circuito de control tiene un cortocircuito a voltaje. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  4. Con el motor en marcha, compruebe si hay más de 3,5 V entre el terminal 1 X2 del circuito de control y masa.
    5. ⇒ Si el valor es inferior al margen especificado, compruebe si el circuito de control tiene un cortocircuito a masa o un circuito abierto/resistencia alta. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  5. Si la comprobación del circuito da buenos resultados, sustituya el generador G13.
