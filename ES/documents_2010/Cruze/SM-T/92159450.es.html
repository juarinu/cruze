Programación y configuración del módulo de control de asiento calefactado

El módulo de control de calefacción de asientos debe estar programado con el software de calibración adecuado. El módulo almacena y usa esta información, sin embargo, si no está debidamente configurado con el software de calibración correcto, el módulo de control de calefacción de asientos no controlará todas las funciones del vehículo adecuadamente.

Asegúrese de que existen las condiciones siguientes para preparar la programación del módulo de control de calefacción de asientos:

  1. La batería está completamente cargada.
  2. Los conectores del enlace de datos de MDI están asegurados.
  3. La herramienta de diagnóstico dispone de la versión de software más actual.
  4. Todos los módulos y dispositivos desconectados se reconectan de forma segura.
  5. Todo este procedimiento se ha revisado antes de proceder.

No reprogramar el módulo de control de calefacción de asientos a menos que así lo determine un procedimiento de servicio o un boletín de servicio.

Programación del módulo de control de calefacción de asientos

Para programar y configurar un nuevo o existente módulo de control de calefacción de asientos sustituto, lleve a cabo el siguiente procedimiento:

  1. Con una herramienta de diagnóstico, acceda al Sistema de programación de servicio (SPS) y siga las instrucciones que aparecen en la pantalla. Consultar Sistema de programación de servicio (SPS) .
  2. En la pantalla de controladores soportados por SPS, seleccione Módulo de control de calefacción del asiento delantero HVSMF - Programación y siga las instrucciones que aparecen en la pantalla.

Si el módulo de control de calefacción del asiento delantero HVSMF no acepta el programa, verifique que todas las conexiones entre la herramienta de exploración y el módulo de control de calefacción de asientos están sujetas.

Nota: :Una vez terminada la programación, realizar lo siguiente para evitar futuros diagnósticos erróneos:

  1. Poner el encendido en OFF durante 10 segundos.
  2. Abra y cierre la puerta.
  3. Conectar la herramienta de diagnóstico al conector de enlace de datos.
  4. Encendido conectado, motor desconectado.
  5. Con una herramienta de diagnóstico, borre todos los DTC de todos los módulos.
