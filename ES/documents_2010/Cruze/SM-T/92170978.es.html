1. La herramienta de diagnóstico no se comunica con el dispositivo GMLAN de baja velocidad

Descripción del circuito/sistema

Los módulos conectados a los circuitos de datos de serie GMLAN de alta y baja velocidad controlan comunicaciones de datos serie durante el funcionamiento normal del vehículo. Entre los módulos se intercambian comandos e información operativa. Los módulos disponen de información almacenada previamente sobre los mensajes que es necesario intercambiar en los circuitos de datos serie para cada una de las redes virtuales. Se supervisan los mensajes y, además, el módulo receptor utiliza algunos mensajes periódicos como indicación de la disponibilidad del módulo transmisor. Todos los mensajes contienen el número de identificación del módulo transmisor.

El módulo de control de la carrocería (BCM) activa los circuitos de datos de serie de reactivación de accesorios y reactivación de datos de serie GMLAN de alta velocidad aplicando tensión cuando la llave de encendido está en ACC, ON o START. El circuito de reactivación de datos de serie GMLAN de alta velocidad reactiva el módulo de control del freno electrónico (EBCM) para la comunicación por bus y el circuito de datos de serie de reactivación de accesorios reactiva el módulo de control del motor (ECM) y el módulo de control del cambio (TCM).

Condiciones para el funcionamiento de DTC
    • El voltaje del sistema está entre 9-16 voltios.
    • El modo de potencia del vehículo requiere que se produzca la comunicación de datos serie.

Condiciones para ajustar el DTC

No se ha recibido un mensaje supervisado periódicamente que incluye la disponibilidad del módulo transmisor.

Acción realizada cuando se registra el DTC

El módulo utiliza un valor por defecto para los parámetros faltantes.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio de 50 sin que se repita una avería.

Ayuda al diagnóstico
    • Utilice la lista de Descripciones de DTC anterior para determinar el módulo que no comunica.
    • Si hay varios módulos que no comunican, elija el más cercano al conector de enlace de datos (DLC).
    • Use Referencias de enlace de datos para determinar si el módulo usa comunicaciones de datos de serie GMLAN de alta o baja velocidad
    • Algunos módulos pueden no tener protección interna para salidas de tensión específicas y pueden abrir un fusible de fuente de tensión de encendido o tensión positiva de batería. Si está abierto un fusible de entrada de tensión y no se encuentra ningún cortocircuito en ese circuito, antes de sustituir el módulo asegúrese de que ningún circuito de tensión de salida del módulo presenta un cortocircuito a masa.
    • Este diagnóstico puede utilizarse para cualquier módulo que deba comunicar con datos de serie GMLAN de alta o baja velocidad siempre que el vehículo esté equipado con la opción que utiliza ese módulo.

Comprobación del circuito/sistema

Nota: si hay más de un módulo que no comunica, utilice Esquemas de comunicación de datos para determinar cuál es el módulo más cercano al DLC. Comience el diagnóstico con ese módulo.

Los identificadores de conector y terminal serán diferentes en función del módulo individual que esté comprobándose. Utilice la referencia del esquema y la referencia de vista final de conector para determinar a qué conector y terminal acceder.

  1. Mediante la descripción de DTC y la ayuda de diagnóstico, determine el módulo que no está comunicando y que debería comprobarse.
  2. Con el encendido OFF, desconecte el conector del mazo de cables del módulo que no se está comunicando.
  3. Compruebe que existen menos de 2 ohmios entre los circuitos a masa del módulo y la masa.
    4. ⇒Si es mayor que el intervalo especificado, compruebe si el circuito de toma de tierra tiene una resistencia abierta/alta.

    Nota: Sólo los módulos GMLAN de alta velocidad tienen un circuito de reactivación de datos de serie cableado. Consulte los esquemas de módulos para identificar qué módulos tienen un circuito tipo despertador. Si el módulo que no está comunicando no tiene un circuito de reactivación de datos de serie, continúe en el paso 5.

  4. Con el encendido conectado, compruebe que se ilumina una lámpara de pruebas entre el circuito de reactivación de datos de serie GMLAN de alta velocidad O el circuito de datos de serie de reactivación de accesorios y la masa.
    5. ⇒Si la luz de prueba no se ilumina, compruebe si el circuito tiene una resistencia abierta/alta. Si la prueba de circuitos sale bien, sustituya el BCM.
  5. Con el encendido conectado, compruebe que se ilumina la lámpara de pruebas entre todos los circuitos B+ y la masa.
    6. ⇒Si la luz de prueba no se ilumina, pruebe el circuito B+ para ver si hay un cortocircuito a tierra o una resistencia abierta/alta. Si el fusible del circuito está abierto, compruebe también si las salidas de voltaje positivo del módulo están cortocircuitados a tierra. Si la prueba de circuitos sale bien, sustituya el módulo adecuado.
  6. Verifique que se ilumina una lámpara de pruebas entre todos los circuitos de tensión de encendido y la masa.
    7. ⇒Si la luz de pruebas no se ilumina, pruebe los circuitos de tensión de encendido para ver si hay un cortocircuito a masa o una resistencia abierta/alta. Si el fusible del circuito está abierto, compruebe también si las salidas de voltaje positivo del módulo están cortocircuitados a tierra. Si la prueba de circuitos sale bien, sustituya el módulo adecuado.
  7. Compruebe que existe menos de 1 Ω en los circuitos de datos de serie GMLAN entre el módulo que no comunica y el módulo que muestra el DTC.
    8. ⇒Si el valor es superior al especificado, compruebe si los circuitos de datos de serie GMLAN tienen una resistencia abierta/alta.
  8. Si todos los circuitos están bien, sustituya el módulo que no se está comunicando.
