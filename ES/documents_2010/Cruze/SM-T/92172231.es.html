Descripción de DTC

DTC P0513: Llave del inmovilizador incorrecta

Descripción del circuito/sistema

El módulo de control de la carrocería (BCM) envía el código de seguridad al módulo de control del motor (ECM) a través del circuito de datos en serie. El ECM envía entonces una señal al BCM. Tanto el ECM como el BCM realizan un cálculo en la recepción de esta señal. Si la respuesta calculada procedente del BCM es igual al cálculo realizado por el ECM, el ECM permitirá el arranque del vehículo.

Condiciones para el funcionamiento de DTC

El encendido esté en la posición RUN o ACCESSORY.

Condiciones para ajustar el DTC

La respuesta calculada procedente del BCN no es igual al cálculo del ECM.

Acción realizada cuando se registra el DTC
    • Se encenderá el indicador de seguridad del cuadro de instrumentos.
    • Se desactivará el arranque del motor.

Condiciones para el borrar el DTC
    • Un DTC P0513 actual se borrará cuando se haya recibido un cálculo válido.
    • Se borrará del historial de DTC después de 40 ciclos consecutivos de encendido sin fallos o cuando se use una herramienta de diagnóstico para borrar los DTCs.

Ayuda al diagnóstico

La llave referenciada en el descriptor del DTC no hace referencia a la llave del vehículo. En este caso, la llave hace referencia a la respuesta a la señal.

Verificación del circuito/sistema

Nota: Debido a las comunicaciones normales de sistema entre el BCM y el ECM, todos los códigos B de inmovilización deben diagnosticarse antes del diagnóstico de los códigos P.

Llevar a cabo el procedimiento de reaprendizaje del inmovilizador. Consultar Programación de componentes del sistema del inmovilizador . El ECM debe completar con éxito el aprendizaje de la contraseña y el vehículo debe arrancar.

⇒Si el ECM no completa la programación o si el arranque del vehículo sigue desactivado, sustituya el módulo de control del motor K20.

Instrucciones de reparación

Realice Verificación de la reparación por diagnóstico después de completar el procedimiento de diagnóstico.

    • Referencias de módulos de control para sustituir, configurar y programar el ECM.
    • Para los códigos P consulte Lista de códigos de diagnóstico de avería (DTC) - Vehículo .
