Reparaciones y comprobaciones necesarias tras un impacto

Información de servicio

Advertencia : Los sistemas obstruidos pueden resultar dañados en caso de un impacto. Para ayudar a evitar lesiones y garantizar que se sustituyen todas las piezas que lo requieren:

   • Sustituya cualquier sistema de cinturón de seguridad que estuviera siendo utilizado durante un impacto suficientemente grave como para activar cualquier dispositivo de sujeción automático, tales como airbags y los pretensores de cinturón de seguridad. Esto no sólo incluye los sistemas de cinturón de seguridad que estuvieran siendo utilizados por personas adultas, sino también los sistemas de cinturón de seguridad empleados para asegurar sujeciones para niños, soportes infantiles y asientos auxiliares, incluyendo sistemas de TRINQUETES DE BLOQUEO y anclajes de sujeción superiores.
   • Sustituya cualquier sistema de cinturón de seguridad que tenga componentes rotos, desgastados o dañados. Esto no sólo incluye sistemas de cinturón de seguridad para adultos, sino también sujeciones para niños integradas y componentes de sistema de trinquete de bloqueo, si los hubiera.
   • Sustituya cualquier sistema de cinturón de seguridad si observa las palabras "SUSTITUIR" o "PRECAUCIÓN", o si se observa una etiqueta amarilla. No sustituya un cinturón de seguridad si sólo está visible la etiqueta de precaución de asiento para niños.
   • Sustituya cualquier sistema de cinturón de seguridad si tiene dudas acerca de su estado. Esto no sólo incluye los sistemas de cinturón de seguridad para adultos, sino también sujeciones para niños incorporadas, componentes de sistemas de TRINQUETE DE BLOQUEO y cualquier sistema de sujeción utilizado para asegurar soportes infantiles, fijaciones para niños o asientos auxiliares.
        NO sustituya componentes individuales del sistema de cinturón de seguridad de vehículos que han sufrido un impacto tal como se ha descrito anteriormente. Sustituya siempre todo el sistema de cinturón de seguridad con el conjunto de retractor, guía y hebilla, que incluye el material de conexión y el trinquete de bloqueo.

Tras un impacto menor en el que no se activó el dispositivo de sujeción automático, puede no ser necesario sustituir el sistema de cinturón de seguridad a menos que algunas de las piezas estén rotas, desgastadas o dañadas.
