Descripción del circuito/sistema

Los asientos calefactados del conductor y el acompañante están controlados por un módulo único de asiento calefactado situado debajo del cojín del asiento del conductor. Ambos asientos calefactados están controlados por interruptores separados de asiento calefactado que están situados en el control HVAC. Cuando se pulsa un interruptor de asiento calefactado, el control HVAC envía un mensaje vía LIN-Bus al módulo de control HVAC. Entonces el módulo de control HVAC sirve como enlace para transmitir el mensaje al módulo de asiento calefactado mediante datos en serie GMLAN que indican el comando sobre el asiento calefactado. Entonces, en respuesta a esta señal, el módulo de asiento calefactado aplica tensión positiva de la batería mediante el circuito para suministrar tensión a los elementos correspondientes de calefacción del asiento. Entonces el módulo del asiento calefactado devuelve un mensaje de datos en serie al módulo de control del HVAC para que transmita la información al control HVAC para que o bien encienda o apague el indicador de temperatura apropiado.

Verificación del circuito/sistema
  1. Encendido conectado, observe el parámetro Sw. del asiento calefactado del conductor con HVAC en la herramienta de diagnóstico a la vez que pulsa el interruptor de asiento CALEFACTADO del conductor. Las lecturas deberían indicar cambio entre Inactivo y Activo.
    2. ⇒Si no es el valor especificado, sustituir el control HVAC A26.
  2. Observar el HVAC Pass. en la herramienta de diagnóstico Parámetro Sw. de asiento calefactado a la vez que se pulsa el interruptor de asiento CALEFACTADO del acompañante. Las lecturas deberían indicar cambio entre Inactivo y Activo.
    3. ⇒Si no es el valor especificado, sustituir el control HVAC A26.
  3. Si todas lecturas de la herramienta de diagnóstico son normales, no hay ningún DTC, y el asiento calefactado del conductor y/o el acompañante están inoperativos, sustituir el módulo de asiento calefactado K29.
