Descripción y funcionamiento del dispositivo de apertura de la puerta del garaje

El dispositivo de apertura de la puerta del garaje puede funcionar con un código fijo o alternante. El código alternante es un sistema que permite que el código que el receptor del cliente recibe del dispositivo de apertura de la puerta del garaje cambie cada vez que se use dicho dispositivo dentro del alcance operativo del receptor. La programación del código alternante requiere que el cliente pulse un botón de programación/memorización en el receptor del dispositivo de apertura de su casa. Este botón suele estar situado en la unidad receptora debajo de una tapa (de la luz), en el extremo de la unidad. El cliente debe seguir las instrucciones del fabricante del dispositivo de apertura de la puerta del garaje para programar/memorizar el receptor para aceptar el Sistema remoto doméstico universal como dispositivo de apertura autorizado para su unidad. Una vez programados juntos el receptor y el dispositivo de apertura de la puerta del garaje, se establece un código y se crea otro nuevo para cada transmisión nueva. El software del receptor reconoce el dispositivo de apertura y acepta el código nuevo.

El dispositivo de apertura de la puerta del garaje es compatible con la mayoría de transmisores, pero no con todos los tipos y marcas.

El dispositivo de apertura de la puerta del garaje es un transmisor que funciona en la banda entre 288 y 399 MHz. La potencia y alcance del transmisor está limitado para cumplir con las leyes que rigen la generación de interferencias de radiofrecuencias. El transmisor es programado por el usuario para que acepte la señal generada por sus transmisores.

El dispositivo de apertura de la puerta del garaje tiene 3 botones que pueden programarse para combinaciones individuales de transmisores/receptores, pudiéndose controlar hasta 3 dispositivos de apertura, puertas de seguridad, sistemas de iluminación, etc. Cada botón representa una parte del código del transmisor que funciona por separado respecto a cualquier otro botón y que puede considerarse como un transmisor independiente. El funcionamiento consiste en pulsar un botón para activar el transmisor correspondiente.
