1. Mal funcionamiento del control de velocidad

Descripción del circuito/sistema

El interruptor del control de velocidad es una entrada al módulo del panel de instrumentos (IPM). El IPM supervisa los interruptores de activación/desactivación, ajuste/inercia, reanudación/aceleración y cancelación del control de velocidad mediante el circuito de señal del interruptor del regulador de velocidad con el fin de detectar si el conductor ha solicitado alguna función de dicho regulador. Al accionarse uno de estos interruptores, el IPM detecta una señal de tensión concreta en el circuito de señal mencionado. El módulo de control del motor (ECM) recibe la función solicitada del interruptor del control de velocidad procedente del IPM mediante un mensaje de datos serie. El módulo de integración trasero (RIM) supervisa el interruptor de cancelación del control de velocidad para detectar cuándo desea el conductor cancelar el control de velocidad. El ECM recibe la solicitud de cancelación del control de velocidad de RIM mediante un mensaje de datos serie.

Verificación del circuito/sistema

Con el motor encendido, mientras gira el volante de dirección hasta sus topes, observe el parámetro de la herramienta de diagnóstico Interruptor de control de velocidad, mientras pulsa cada uno de los interruptores de control de velocidad. El parámetro deberá alternar entre encender, apagar, reanudar y configurar a medida que se pulsa el interruptor correspondiente.

Comprobación del circuito/sistema
  1. Con el encendido desactivado, desconecte el conector X2 del mazo de cables del interruptor multifunción/intermitentes.
  2. Con el encendido conectado, compruebe que existe B+ entre el terminal 1 del circuito de referencia de voltaje y masa.
    3. ⇒Si es inferior al valor especificado, compruebe si el circuito de referencia de tensión tiene un cortocircuito a masa o una resistencia abierta/alta. Si las comprobaciones del circuito son normales, sustituya el IPM.
  3. Compruebe si existe B+ entre el terminal 1 del circuito de referencia de voltaje y el terminal 4 del circuito de señal.
    4. ⇒Si es inferior al valor especificado, compruebe si el circuito de señal tiene un cortocircuito de voltaje o una resistencia abierta/alta. Si las comprobaciones del circuito son normales, sustituya el IPM.
  4. Con el encendido desactivado, desconecte el conector del mazo de cables X3 en el IPM.
  5. Compruebe si existe una resistencia infinita entre el terminal 20 del circuito de señal y masa.
    6. ⇒Si es inferior al valor especificado, compruebe si el circuito de señal tiene un cortocircuito a masa.
  6. Si las pruebas de todos los circuitos son normales, compruebe o sustituya el interruptor multifunción/intermitentes.

Comprobación de componentes

Importante: El interruptor del control de velocidad debe estar activado para medir correctamente la resistencia de los interruptores -ESTABLECER, +REINICIAR, Cancelar, Separación arriba, Separación abajo.

Interruptor multifunción/intermitentes

  1. Con el encendido desactivado, desconecte el conector X2 del mazo de cables del interruptor multifunción/intermitentes.
  2. Conecte un DMM entre el terminal 1 y el terminal 4 en el interruptor multifunción/intermitentes.
  3. Con el interruptor del control de velocidad activado, mida la resistencia mientras activa individualmente y mantiene presionado cada uno de los interruptores de la función del control de velocidad, y compare las lecturas de resistencia con los valores recogidos en la tabla del interruptor correspondiente.
    4. ⇒Si cualquiera de las mediciones de resistencia de los interruptores individuales no está dentro de los valores especificados, sustituya el interruptor multifunción/intermitentes.
