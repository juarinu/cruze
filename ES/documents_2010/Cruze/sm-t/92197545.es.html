Descripción del circuito/sistema

El módulo de control del HVAC detecta un escape de gas de un sensor de calidad del aire. El sensor de calidad del aire es un sensor de 3 cables con un circuito de tensión de encendido, un circuito de masa y un circuito de señal.

La información es una señal de modulación por ancho de pulso (PWM) generada en el pasador de salida.

El módulo de control del HVAC evalúa la información del sensor de calidad del aire y cierra la solapa de recirculación mientras está en modo automático, tan pronto como la concentración de agentes contaminantes exceda un valor predeterminado.

Condiciones para el funcionamiento de DTC
    • Encendido conectado.
    • El módulo de control del HVAC está Activado.

Condiciones para ajustar el DTC

B3843 02

No se detecta ninguna señal PWM y la señal es baja.

B3843 05

No se detecta ninguna señal PWM y la señal es alta.

B3843 08

El sensor envía una señal PWM al 5% indicando que hay un cortocircuito o un circuito abierto en el sensor interno.

Acción realizada cuando se registra el DTC

El modo de recirculación automático está desactivado.

Condiciones para el borrar el DTC

La anomalía que dio lugar al DTC ya no está presente.

Verificación del circuito/sistema

Verifique que no se ha registrado el DTC B3843 08.

⇒Si existe un DTC, sustituir el sensor de calidad del aire B108.

Comprobación del circuito/sistema
  1. Con el encendido OFF, desenchufe el conector del mazo de cables del sensor de calidad B108.
  2. Compruebe si hay menos de 5 Ω entre el terminal 2 del circuito de masa y la masa.
    3. ⇒Si es mayor que el margen especificado, compruebe si hay un circuito abierto/resistencia alta en el circuito de masa.
  3. Con el encendido en ON, verifique que una luz de comprobación se ilumina entre el terminal 1 del circuito de encendido y masa.
    4. ⇒Si la luz de prueba no se ilumina, pruebe el circuito de encendido para ver si hay un cortocircuito a tierra o un circuito abierto/resistencia alta.
  4. Verificar que el parámetro Sensor de olor de la herramienta de diagnóstico es superior a 95%.
    5. ⇒Si es inferior al intervalo especificado, compruebe si el circuito de señales tiene un cortocircuito a masa. Si según la comprobación el circuito está bien, sustituya el módulo de control HVAC K33.
  5. Instalar un cable puente con fusible de 3 A entre el siguiente terminal 3 del circuito de señal y el terminal 2 del circuito de masa. Verificar que el parámetro Sensor de olor de la herramienta de diagnóstico es menos de 5%.
    6. ⇒Si el valor es superior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a voltaje o un circuito abierto/resistencia alta. Si según la comprobación el circuito está bien, sustituya el módulo de control HVAC K33.
  6. Si tras la comprobación todos los circuitos están normales, sustituir el sensor de calidad del aire B108.
