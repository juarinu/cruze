1. Mal funcionamiento del control de velocidad

Descripción del circuito/sistema

El interruptor del regulador de velocidad es una entrada en el módulo electrónico de la carrocería (BCM). El BCM controla los circuitos de señal del interruptor de encendido/apagado, control de velocidad/deceleración y reanudar/acelerar para detectar cuando se ha activado un interruptor de control de velocidad de crucero. El BCM detecta una señal de voltaje predeterminada cuando se activa un interruptor de función de control de velocidad de crucero determinado, en el que cada interruptor tiene un valor de voltaje diferente.

Condiciones para el funcionamiento de DTC
    • El encendido está en la posición ON.
    • El interruptor de crucero está en ON.

Condiciones para ajustar el DTC
    • El ECM detecta una señal de voltaje inválida en el circuito de señal de conexión del control de velocidad.
    • La condición anterior se da durante más de 2 segundos.
    • El ECM ejecuta este diagnóstico cada 0,05 segundos.

Acciones realizadas cuando el DTC se ajusta
    • El testigo luminoso de avería (MIL) no se iluminará.
    • El sistema de control de crucero está desactivado.

Condiciones para el borrar el DTC
    • El estado responsable para ajustar el DTC ya no existe.
    • Se borrará un historial de DTC después de que hayan pasado 40 ciclos de encendido sin fallos.

Verificación del circuito/sistema

Con el motor encendido, mientras gira el volante de dirección hasta sus topes, observe el parámetro de la herramienta de diagnóstico Interruptor de control de velocidad, mientras pulsa cada uno de los interruptores de control de velocidad. El parámetro deberá alternar entre encender, apagar, reanudar y configurar a medida que se pulsa el interruptor correspondiente.

Comprobación del circuito/sistema
  1. Con el encendido desconectado, desenchufe el conector del mazo de cables del interruptor izquierdo de control del volante K20.
  2. Con el encendido conectado, compruebe B+ entre el circuito de referencia de voltaje y la masa.
    3. ⇒Si es inferior al valor especificado, compruebe si el circuito de referencia de tensión tiene un cortocircuito a masa o una resistencia abierta/alta. Si la prueba del circuito sale bien, sustituya el BCM.
  3. Compruebe B+ entre el circuito de referencia de voltaje y el circuito de señal.
    4. ⇒Si es inferior al valor especificado, compruebe si el circuito de señal tiene un cortocircuito de voltaje o una resistencia abierta/alta. Si la prueba del circuito sale bien, sustituya el BCM.
  4. Con el encendido desactivado, desconecte el conector del mazo de cables X3 en el K9 de BCM.
  5. Compruebe si hay resistencia infinita entre el terminal 9 del circuito de señales y la masa.
    6. ⇒Si es inferior al valor especificado, compruebe si el circuito de señal tiene un cortocircuito a masa.
  6. Si todas las pruebas de circuitos son normales, compruebe o sustituya el interruptor izquierdo de control del volante K20.
