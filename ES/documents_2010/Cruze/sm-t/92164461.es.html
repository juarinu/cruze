Descripción y funcionamiento del centro de información del conductor (DIC)

En la parte inferior central del cuadro de instrumentos hay instalada una pantalla adicional. Su función es ofrecer información adicional, como un cuentakilómetros o códigos de error/mensajes de advertencia. Consultar Descripción y funcionamiento del testigo/mensaje de advertencia .

Hay 4 funciones de interruptor para el centro de información del conductor:

    • Menú
    • Establecer/Borrar
    • Arriba
    • Abajo

Menú

El botón de menú se utiliza para navegar entre el menú de información del vehículo y el menú de información de viaje/combustible. Navegar entre los siguientes parámetros del vehículo pulsando repetidamente el botón de menú.

Arriba y abajo

Los botones de arriba y abajo se utilizar para navegar entre los menús. Navegar entre los siguientes parámetros del menú pulsando repetidamente el botón de abajo

Menú de información del vehículo

    •  Menú de unidades
    • Advertencia de velocidad

Menú de información de viaje/combustible

    • Viaje 1
    • Rango de combustible
    • Economía de combustible media
    • Economía de combustible instantánea
    • Velocidad media del vehículo

Rango de combustible

Este mensaje indica la distancia estimada que el vehículo puede recorrer bajo su economía de combustible actual y las condiciones del nivel de combustible desde la última conexión de la batería al cuadro de instrumentos. El rango de combustible se calcula utilizando la ecuación: Rango de combustible = Distancia de rango/combustible de rango utilizado.

    • Distancia de rango = La distancia acumulada viajada desde la última vez que se reajustó este valor.
    • Combustible utilizado = El combustible acumulado entregado desde la última vez que se reajustó este valor

El módulo de control del monitor (ECM) envía el valor del rango de combustible al cuadro de instrumentos. El cuadro de instrumentos recibe un mensaje de datos serie con la información del combustible. El valor del rango de combustible se conserva mientras el encendido está desactivado y se puede cambiar entre unidades Europa-SI, USA y GB accediendo al menú Unidades del centro de información del conductor. La pantalla de rango de combustible no se puede reajustar. La indicación BAJO aparece cuando el rango de combustible cae por debajo de un valor predeterminado.

Economía de combustible media

La economía de combustible media se calcula utilizando la ecuación: Economía de combustible media = Distancia/Combustible

    • Distancia = La distancia acumulada viajada desde la última vez que se reajustó este valor
    • Combustible = El combustible acumulado consumido desde la última vez que se reajustó este valor

El ECM envía el mensaje de datos serie de economía de combustible media al cuadro de instrumentos. El valor de este modo se conserva mientras el encendido está desactivado y se puede cambiar entre unidades del sistema imperial y unidades del sistema médico seleccionando en el menú de opciones del centro de información del conductor.

Economía de combustible instantánea

El ECM envía el mensaje de datos serie de economía de combustible instantáneos al cuadro de instrumentos. El valor de este modo se puede cambiar entre unidades del sistema imperial y unidades del sistema métrico seleccionando en el menú de opciones del centro de información del conductor.

Velocidad media del vehículo

La velocidad media se calcula utilizando la ecuación: Velocidad media = Distancia/Tiempo de velocidad media

    • Distancia = La distancia acumulada viajada desde la última vez que se reajustó este valor
    • Encendido conectado = El encendido acumulado a tiempo desde la última vez que se ajustó este valor

Menú de unidades

El modo de unidad se utiliza para alternar entre unidades Europa-SI, USA y GB y se puede acceder mediante el botón Menú del centro de información del conductor y el botón Ajustar/Reajustar del centro de información del conductor.

Cuentakilómetros

El cuadro de instrumentos muestra el cuentakilómetros del vehículo en el centro de información del conductor. El ECM envía un mensaje de recuento de rodadura de distancia por GMLAN al módulo de control de la carrocería (BCM). El BCM utiliza esta información para calcular el cuentakilómetros del vehículo. Este valor del cuentakilómetros se envía entonces al cuadro de instrumentos por GMLAN. El cuadro de instrumentos no calcula el cuentakilómetros. El cuentakilómetros muestra millas o kilómetros según se seleccione en el menú Unidades.

El valor del cuentakilómetros está almacenado en varios módulos. El cuadro de instrumentos es un módulo de almacenamiento secundario para el cuentakilómetros, mientras que el BCM hace de almacenamiento principal y de acumulador.

Además de memorizar el valor del cuentakilómetros del vehículo, el cuadro de instrumentos y el BCM guardan el número VIN. Para asegurar estos módulos, se realizan comprobaciones de software y la información memorizada del cuentakilómetros no puede moverse ni transferirse de un vehículo a otro.

Si los VINs no coinciden, el cuadro de instrumentos entrará en modo de error y mostrará "----" (guiones). Ante una falta de coincidencia de VINs en una distancia calibrada, el cuadro de instrumentos "bloqueará" la indicación del cuentakilómetros y sólo mostrará guiones, incluso si la falta de coincidencia de VINs es posteriormente correcta. La única manera de borrar o "desbloquear" el cuadro de instrumentos es realizar un evento de programación del BCM utilizando SPS.

Idioma

El centro de información del conductor puede mostrar varios idiomas, según el ajuste de idioma de la radio. El cuadro de instrumentos recibe un mensaje de datos serie con la información sobre el idioma de la radio.
