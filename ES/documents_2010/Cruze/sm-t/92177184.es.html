Definición

Condiciones de servicio: Encendido ON

Detección automática de acompañante, estado actual de ocupación del asiento delantero

 

Vacío

La herramienta de diagnóstico muestra Desconocido, Vacío, Ocupado o Reservado. Este es el estado del asiento del acompañante, ocupado o vacío, detectado por el sistema de presencia del acompañante.

Estado de fallo de detección automática de acompañante

 

No hay avería

La herramienta de diagnóstico muestra "Ningún fallo", "Fallo presente", o "Falta de correspondencia de vehículo". Este es el estado interno del sistema de presencia del acompañante monitorizado por el SDM.

Sistema de detección automática de ocupante activado

 

Activado

La herramienta de diagnóstico muestra si el sistema de detección del pasajero está habilitado.

Sistema de detección automática de ocupante programado

 

Programado

La herramienta de diagnóstico muestra "Programado" o "No programado". Se muestra programado si la presencia del acompañante está adecuadamente programada.

Número de pieza del modelo base

 

Varía

La herramienta de diagnóstico muestra el número de pieza del SDM que se almacena en la memoria no volátil.

Sensor del cinturón de seguridad del conductor activado

 

Activado

La herramienta de diagnóstico muestra si el sensor del cinturón de seguridad en el cierre del cinturón de seguridad está activado.

Sensor del cinturón de seguridad del conductor programado

 

Programado

La herramienta de diagnóstico muestra "Programado" o "No programado". Aparece"Programado" si el SDM está adecuadamente programado para el sensor de cinturón de seguridad del asiento del conductor.

Estado de "Programado" del sensor de posición del asiento del conductor

 

Programado

La herramienta de diagnóstico muestra "Programado" o "No programado". Aparece "Programado" si el SDM está adecuadamente programado para el sensor de posición del asiento del conductor.

Estado del interruptor de posición de asiento del conductor hacia delante

 

Varía

La herramienta de diagnóstico muestra, "Hacia delante", "Hacia atrás". Este parámetro indica la posición del asiento del conductor.

Estado de "Activado" del sensor de posición del asiento del conductor

 

Activado

La herramienta de diagnóstico muestra si el sensor de posición del asiento del conductor está activado.

Número de pieza del modelo final

 

Varía

La herramienta de diagnóstico muestra el número de pieza del modelo final. La clave principal son los cuatro últimos dígitos del número de modelo final. El número de modelo final es único para cada calibración/configuración y se almacena en una memoria no volátil del SDM.

Estado del indicador de abrochar cinturón de seguridad del conductor

 

Off (Desac.)

La herramienta de diagnóstico muestra si el indicador de cinturón de seguridad está iluminado. El indicador se iluminará inicialmente si el cinturón de seguridad no está abrochado.

Clasificación del acompañante

--

Vacío

La herramienta de diagnóstico muestra Desconocido, Vacío, Niño, Adulto pequeño o Adulto grande. Es el estado de aquello que ocupa el asiento del acompañante, tal como lo detecta el sistema de presencia del acompañante.

Estado de abrochado de cinturón de seguridad de asiento del conductor de fila delantera

 

Desabrochado

La herramienta de diagnóstico muestra "Abrochado" o "Desabrochado". Este es el estado del interruptor del cierre del cinturón de seguridad del conductor en el cierre del cinturón de seguridad del conductor.

Estado de abrochado de cinturón de seguridad de asiento del acompañante de fila delantera

 

Desabrochado

La herramienta de diagnóstico muestra "Abrochado" o "Desabrochado". Este es el estado del interruptor del cinturón de seguridad del acompañante en el cierre del cinturón de seguridad del acompañante.

ESS 1-8 activado

 

Activado

La herramienta de diagnóstico indica Activado o Desactivado. Esto es el estado del sensor.

ESS 1-8 programado

 

Programado

La herramienta de diagnóstico muestra "Programado" o "No programado". Se muestra "Programado" si el SDM tiene el sensor programado.

ESS Tipo 1-8

 

Varía

La herramienta de diagnóstico muestra "No utilizado", "Sensor delantero LF", "Sensor delantero RF", "Sensor de impacto lateral izquierdo fila 1", "Sensor de impacto lateral derecho fila 1", "Sensor de impacto lateral izquierdo fila 2", "Sensor de impacto lateral derecho fila 2". Este es el tipo de sensor de SIR.

Circuito 1-18 activado

 

Activado

La herramienta de diagnóstico muestra "Activado" o "Desactivado". Es el estado del módulo del generador de gas, capaz del desligue cuando está activado.

Circuito 1-18 programado

 

Programado

La herramienta de diagnóstico muestra "Programado" o "No programado". Aparece Programado si el SDM tiene programado el número de bucle con el módulo de generador de gas presente conectado.

Resistencia de bucle 1-18

 

Varía

La herramienta de diagnóstico muestra la resistencia del circuito con el módulo de generador de gas conectado.

Tipo de circuito 1-18

 

Varía

La herramienta de diagnóstico muestra "No utilizado", "Etapa 1 LF", "Etapa 1 RF", "Etapa 2 LF", "Etapa 2 RF", "Pretensor LF", "Pretensor RF", "Airbag lateral LF", "Airbag lateral RF", "Etapa 1 izda techo", "Etapa 1 dcha techo", "Etapa 2 izda techo"," Etapa 2 dcha techo", "Etapa 3 izda techo", o "Etapa 3 dcha techo". Es el tipo de bucle.

Estado del indicador de advertencia del sistema de protección de ocupantes

  

La herramienta de diagnóstico muestra si se ilumina el indicador de advertencia del sistema de presencia del acompañante. El indicador se iluminará dependiendo de si en el asiento del acompañante se sienta un niño o un adulto.
