Descripciones de DTC

Códigos de avería B101E 43: Error de programa ECU EEPROM

Códigos de avería B101E 4B: Calibración no programada

Descripción del circuito/sistema

Cuando el encendido está conectado, el sistema de presencia de acompañante realiza una auto comprobación para verificar si existe cualquier fallo crítico de funcionamiento dentro del sistema. Durante el modo de arranque el sistema de presencia de acompañante comprueba si hay ajustados datos válidos de calibración. Si no se encuentra una calibración válida el sistema de presencia de acompañante desactiva el sistema.

Condiciones para la activación de DTC

Tensión del sistema entre 9-16 V

Condiciones para ajustar el DTC

Los datos de calibración ajustados no se programan en el sistema de presencia de acompañante.

Acción realizada cuando el DTC se ajusta

Las funciones del sistema de presencia de acompañante están desactivadas.

Condiciones para el borrar el DTC
    • La anomalía que dio lugar al código de avería (DTC) ya no está presente.
    • El historial de DTC se borra después de 100 ciclos de encendido sin averías.
