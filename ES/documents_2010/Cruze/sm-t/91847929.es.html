Comprobación de cargas previas del sistema de presencia del acompañante

Introducción

El Sistema de presencia de acompañante (PPS) es un sistema calibrado que requiere comprobar las precargas en cada región cuando se ha sustituido o reparado el sistema. Si se han reparado, reinstalado o retirado por algún motivo el cojín interior del asiento del acompañante, el revestimiento inferior del asiento o la almohadilla calefactora, el procedimiento que sigue comprobará las precargas en el PPS. No puede probarse la precarga del módulo del PPS hasta que se desbloquee, utilizando programación mediante SPS. Un módulo de PPS de sustitución de servicio no requiere un procedimiento de desbloqueo antes de una prueba inicial de precarga. Una vez realizada una prueba de precarga, el módulo de PPS se bloqueará independientemente de la presencia de un estado de 'pasa' o 'no pasa'. Si es necesario volver a probar un módulo, debe desbloquearse primero con programación mediante SPS. Antes de comenzar, lea estos procedimientos cuidadosamente y en su integridad. Para más información sobre el PPS, véase Descripción y funcionamiento del sistema SIR .

Nota: 

   • El PPS no funcionará correctamente si el PPS detecta una precarga en una región.
   • Es necesario realizar el Comprobación del sistema de diagnóstico - Vehículo tras completar con éxito el procedimiento de prueba de precarga para asegurarse de que el sistema funciona correctamente.



El PPS fallará la comprobación de precarga cuando una región detecte una precarga/presión en ésta. Si una comprobación de precarga resulta fallida, la herramienta de diagnóstico mostrará un gráfico como se ilustra a continuación. Un estado Sí indica qué región(es) presentan una condición de precarga. Consulte el siguiente gráfico y la ilustración anterior para identificar la región fallida.

Procedimiento de prueba del sistema de precarga del sistema de detección del pasajero (módulo nuevo)
  1. Vaciar el asiento del acompañante delantero exterior.
  2. Con la herramienta de diagnóstico, llevar a cabo el procedimiento de Precarga del sistema de presencia de acompañante.
  3. Si la comprobación falla, ajuste el revestimiento inferior del asiento y frote la cubierta para asegurar que no existe una presión no deseada en el área especificada. Continúe con la prueba de precarga del sistema de detección del pasajero (módulo existente) para volver a comprobar tras realizar el ajuste.
  4. Una vez completada con éxito la prueba, llevar la llave de contacto a OFF.
  5. Llevar a cabo Comprobación del sistema de diagnóstico - Vehículo tras completar con éxito el procedimiento de prueba de precarga para asegurarse de que el sistema funciona correctamente.

Procedimiento de prueba del sistema de precarga del sistema de detección del pasajero (módulo existente)
  1. Con Sistema de programación de servicio (SPS) desbloquear el módulo de PPS.
  2. Vaciar el asiento del acompañante delantero exterior.
  3. Con la herramienta de diagnóstico, llevar a cabo el procedimiento de Precarga del sistema de presencia de acompañante.
  4. Si la prueba falla, ajustar el revestimiento inferior del asiento y masajear la cubierta para asegurarse de que no haya presente una presión no deseada en el área especificada, y reiniciar la Prueba de precarga del sistema de presencia de acompañante (Módulo existente).
  5. Una vez completada con éxito la prueba, llevar la llave de contacto a OFF.
  6. Llevar a cabo Comprobación del sistema de diagnóstico - Vehículo tras completar con éxito el procedimiento de prueba de precarga para asegurarse de que el sistema funciona correctamente.
