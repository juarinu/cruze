Descripción del circuito/sistema

Se proporciona tensión de encendido directamente al testigo luminoso de avería (MIL). El módulo de control del motor (ECM) conecta el MIL poniendo a masa el circuito de control de MIL. Podría haber un MIL encendido con el interruptor de encendido en ON y el motor apagado.

Funcionamiento MIL

El MIL está ubicado en el grupo de panel de instrumentos.

Función MIL

    • El MIL informa al conductor de que ha ocurrido una avería y de que el vehículo debería repararse lo antes posible.
    • El MIL se ilumina durante la comprobación de las bombillas y del sistema.
    • Se almacenará un DTC si el diagnóstico solicita un MIL.

Iluminación MIL

    • El MIL se iluminará con el interruptor de encendido conectado y el motor parado.
    • El MIL se apagará cuando el motor se arranque.
    • El MIL permanecerá activado si el sistema de autodiagnóstico ha detectado una avería.
    • El MIL se apagará cuando desaparezca la avería.
    • Si el MIL se ilumina y, a continuación, el motor se cala, el MIL permanecerá iluminado mientras el interruptor de encendido esté conectado.
    • Si el MIL no se ilumina y el motor se cala, el MIL no se iluminará hasta que el interruptor de encendido se desconecte y, a continuación, se vuelva a conectar.

Verificación del circuito/sistema

Con el encendido conectado, encienda y apague el MIL del cuadro de instrumentos P16 con la herramienta de diagnóstico. El MIL deberá encenderse y apagarse cuando se le ordene.

Comprobación del circuito/sistema
  1. Con el encendido desconectado, desconecte el conector del mazo de cables X2 en el módulo de control del motor K20.
  2. Con el encendido conectado, observe el testigo luminoso de avería (MIL).
    3. ⇒Si el MIL sigue iluminado, compruebe el terminal 48 del circuito de control de MIL para ver si existe un cortocircuito a masa entre el ECM y el cuadro de instrumentos P16. Si la comprobación del circuito es normal, sustituya el cuadro de instrumentos P16.
    ⇒Si el MIL se apaga, sustituya el módulo de control del motor K20.

Instrucciones de reparación

Realice Verificación de la reparación por diagnóstico después de completar el procedimiento de diagnóstico.

    • Consulte Sustitución del cuadro de instrumentos
    • Referencias de módulos de control para sustituir, configurar y programar el ECM.
