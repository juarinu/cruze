Descripción de DTC

Códigos de avería C056E 47: Número de identificación del vehículo (VIN) no programado

Condiciones para la activación de DTC
    • El encendido está en la posición ON.
    • La tensión de encendido sea superior a 8 voltios.

Condiciones para ajustar el DTC

El EBCM detecte un fallo de calibración interno.

Acción realizada cuando el DTC se ajusta

El EBCM activa el sistema antibloqueo de los frenos (ABS) / el sistema de control de tracción (TCS) / el sistema de mejora de la estabilidad del vehículo (VSES) durante todo el ciclo de encendido.

Condiciones para el borrar el DTC
    • El EBCM elimina el DTC tan pronto como se elimine el fallo.
    • El EBCM automáticamente borra el DTC de historial cuando no se detecta un DTC actual en 40 ciclos de conducción consecutivos.
