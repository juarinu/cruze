Descripción del circuito/sistema

El módulo de control del sensor de objetos traseros recibe tensión de encendido desde el bloque de fusibles del panel de instrumentos. Cuando la transmisión está en marca atrás se activa el sistema, se suministra tensión de encendido a través del interruptor de las luces de marcha atrás. Los sensores de alarma de objetos se alimentan con circuitos de señal referenciados a partir de 8 voltios por los circuitos de referencia baja y de módulo de control del sensor de objetos traseros. El módulo de alarma de objetos se alimenta con una fuente de 8 voltios y con referencia baja desde el módulo. El circuito de control de alarma es un circuito de 8 voltios que se modula mediante ancho de pulsos de forma baja para activar la alarma.

Ayuda al diagnóstico

El sistema de estacionamiento asistido puede realizar una autoprueba del sensor de detección de objetos. Mientras que está en el modo de autoprueba del sensor el módulo de estacionamiento asistido verificará las entradas de datos de los sensores de detección de objetos. El módulo de estacionamiento asistido indica el estado del sensor usando la alarma del sistema. Si ambas señales del sensor están siendo recibidas por el módulo de estacionamiento asistido trasero 1 se activará un tono único breve. Cuando una o ambas de las señales del sensor no están siendo recibidas, el módulo responderá activando la alarma con 1 tono repetido para un fallo del sensor izquierdo y 2 tonos repetidos para un fallo del sensor derecho. Si ambos sensores fallan el módulo responderá activando la alarma con 1 tono repetido 3 veces, después 2 tonos repetidos 3 veces y luego se repite la secuencia.

Para solicitar la autoprueba del sensor de objetos complete los siguientes pasos:

  1. Conecte el encendido.
  2. Aplique y libere el freno de estacionamiento 3 veces en un plazo de 10 segundos tras conectar el encendido.
  3. El sistema estará en el modo de autoprueba del sensor de objetos.
  4. Para salir del modo de autoprueba del sensor de detección de objetos desconecte el encendido.

Verificación del circuito/sistema
  1. Con el encendido activado, y el área detrás del vehículo limpia de cualquier objeto que pudiera activar la alarma de estacionamiento asistido, ponga la marcha atrás. La alarma de objetos debería emitir 1 tono breve.
    2. ⇒Si no es el valor especificado, consulte la prueba del circuito del sistema de la alarma de objetos.
  2. Realice el procedimiento de autoprueba del sensor de detección de objetos para verificar los datos que el sensor envía al módulo. Consulte Ayudas para el diagnóstico.
    3. ⇒ Si el modo de autoprueba indica que cualquier entrada del sensor ha fallado, consulte la prueba del circuito del sensor de alarma de objetos.

Comprobación del circuito/sistema

Prueba del circuito del sistema de alarma de objetos

  1. Con el encendido desconectado, desconecte el enchufe del mazo de cables del módulo de control del sensor de objetos traseros.
  2. Compruebe que haya menos de 1,0 Ω entre el terminal 16 de circuito de toma de tierra y masa.
    3. ⇒Si es mayor que el valor especificado, compruebe si el circuito de toma de tierra tiene una resistencia abierta/alta.
  3. Con el encendido conectado, verifique que una luz de comprobación se ilumina entre el terminal 8 del circuito de encendido y masa.
    4. ⇒Si la luz de prueba no se ilumina, compruebe si en el circuito de encendido hay un cortocircuito a masa o una resistencia abierta/alta.
  4. Con el freno de mano puesto y el vehículo en REVERSE (Marcha atrás), verifique que una luz de comprobación se enciende entre el terminal 5 del circuito de control de las luces de marcha atrás y masa.
    5. ⇒Si la luz de prueba no se ilumina, compruebe si en el circuito de control de las luces de marcha atrás hay un cortocircuito a masa o una resistencia abierta/alta.
  5. Con el encendido desconectado, desconecte el enchufe del mazo de cables del módulo de control del sensor de objetos traseros.
  6. Desemborne el conector del mazo de cables en el módulo de alarma de objetos.
  7. Compruebe si hay menos de 1,0 Ω entre el terminal 3 del circuito referencia baja y la masa.
    8. ⇒Si es mayor que el valor especificado, compruebe si el circuito de referencia baja tiene una resistencia abierta/alta.
  8. Con el encendido conectado, compruebe si hay de 7,5 a 8,2 V entre el terminal 1 del circuito de referencia de 8 V y masa.
    9. ⇒Si es inferior al intervalo especificado, compruebe si el circuito de referencia de 8V tiene un cortocircuito a masa o una resistencia abierta/alta. Si según la comprobación el circuito está bien, sustituya el módulo de control del sensor de objetos traseros.
    ⇒Si el valor es superior al margen especificado, compruebe si el circuito de referencia de 8V tiene un cortocircuito a voltaje. Si según la comprobación el circuito está bien, sustituya el módulo de control del sensor de objetos traseros.
  9. Conecte una luz de comprobación entre el terminal 2 del circuito de señal y masa. La lámpara de pruebas no debería encenderse.
    10. ⇒Si se enciende la luz de comprobación, compruebe si el circuito de señal tiene un cortocircuito a voltaje. Si según la comprobación el circuito está bien, sustituya el módulo de control del sensor de objetos traseros.
  10. Compruebe la resistencia infinita entre el terminal 2 del circuito de señal y masa.
    11. ⇒Si es inferior al valor especificado, compruebe si el circuito de señal tiene un cortocircuito a tierra. Si según la comprobación el circuito está bien, sustituya el módulo de control del sensor de objetos traseros.
  11. Con el encendido desconectado, coloque un objeto a menos de 0,4 m del parachoques trasero.
  12. Conecte una luz de comprobación entre el terminal 1 del circuito de referencia de 8 V y el terminal 2 del circuito de señal.
  13. Con el encendido conectado y el freno de mano aplicado, y con la marcha atrás puesta, la luz de comprobación debería iluminarse.
    14. ⇒Si la luz de comprobación no se enciende, compruebe si hay un circuito abierto/resistencia alta en el circuito de señal. Si según la comprobación el circuito está bien, sustituya el módulo de control del sensor de objetos traseros.
  14. Si tras la comprobación todos los circuitos están normales, sustituya el módulo de alarma de objetos.

Prueba del circuito del sensor de alarma de objetos

  1. Con el encendido desconectado, desconecte el enchufe del mazo de cables del sensor de alarma de objetos.
  2. Compruebe si hay menos de 1,0 Ω entre el terminal 2 del circuito referencia baja y la masa.
    3. ⇒Si es mayor que el valor especificado, compruebe si el circuito de referencia baja correspondiente tiene una resistencia abierta/alta.
  3. Con el encendido conectado, el freno de mano aplicado y la marcha atrás puesta, compruebe si hay 7,5-8,2 V entre el terminal 1 del circuito de señal del sensor de alarma de objetos y masa.
    4. ⇒Si el valor es inferior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a masa o un circuito abierto/resistencia alta. Si según la comprobación el circuito está bien, sustituya el módulo de control del sensor de objetos traseros.
    ⇒Si es superior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a voltaje. Si según la comprobación el circuito está bien, sustituya el módulo de control del sensor de objetos traseros.
  4. Si tras la comprobación todos los circuitos están normales, sustituya el sensor de alarma de objetos correspondiente.
