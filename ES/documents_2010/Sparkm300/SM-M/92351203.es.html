Descripción del circuito/sistema

Los sensores de velocidad de guiñada y del acelerómetro lateral están combinados en un mismo sensor, externo al módulo de control electrónico de los frenos (EBCM). El sistema de mejora de la estabilidad del vehículo (VSES) utiliza la entrada del acelerómetro lateral para calcular la velocidad de guiñada deseada. El sensor de aceleración lateral / velocidad de guiñada se comunica con el EBCM mediante circuitos discretos de datos serie de un Bus CAN de alta velocidad y un Bus CAN de baja velocidad.

Condiciones para la activación de DTC

El encendido está en la posición ON.

Condiciones para ajustar el DTC
    • El sensor combinado de velocidad de guiñada / aceleración lateral indique un fallo interno.
    • El sensor combinado de velocidad de guiñada / aceleración lateral memorice un valor de compensación erróneo.
    • El sensor de velocidad de guiñada / de aceleración lateral esté mal montado.

Acción realizada cuando el DTC se ajusta
    • El EBCM desactiva el sistema de mejora de la estabilidad del vehículo (VSES) durante todo el ciclo de encendido.
    • Es posible que en el módulo de la pantalla de información se visualice un indicador de aviso.

Condiciones para el borrar el DTC
    • La anomalía que dio lugar al código de avería (DTC) ya no está presente.
    • El EBCM borra el DTC del historial cuando no detecta un DTC actual en 40 ciclos de conducción consecutivos.

Comprobación del circuito/sistema
  1. Con el encendido DESCONECTADO, desenchufe el conector del mazo de cables en el sensor de velocidad de guiñada / aceleración lateral.
  2. Con el encendido DESCONECTADO, compruebe que haya una resistencia inferior a 1 ohmio entre el terminal del circuito de referencia de baja tensión 6 y la toma de masa.
    3. ⇒Si es mayor que el intervalo especificado, compruebe si el circuito de baja referencia tiene una resistencia abierta/alta.
  3. Con el encendido CONECTADO, mida la tensión de encendido entre el terminal del circuito de tensión de encendido 5 y la toma de masa.
    4. ⇒Si el valor no está dentro del intervalo especificado, compruebe el circuito de tensión de encendido para ver si hay algún cortocircuito a masa o alguna interrupción / resistencia alta.
  4. Con la llave de contacto en posición OFF, desenchufe el conector del cableado en el EBCM.
  5. Con el encendido CONECTADO, compruebe que haya una resistencia inferior a 1 voltio entre los terminales que se relacionan a continuación y la toma de masa.
    6. • Terminal 12 del circuito de datos serie del bus CAN de alta velocidad
    • Terminal 24 del circuito de datos serie del bus CAN de baja velocidad
    ⇒Si el valor es superior al especificado, compruebe el circuito de datos serie correspondiente para ver si hay algún cortocircuito a tensión.
  6. Con el encendido DESCONECTADO, compruebe si hay una resistencia infinita entre los terminales que se relacionan a continuación y la toma de masa.
    7. • Terminal 12 del circuito de datos serie del bus CAN de alta velocidad
    • Terminal 24 del circuito de datos serie del bus CAN de baja velocidad
    ⇒Si no tiene el valor especificado, compruebe el circuito de datos serie correspondiente para ver si hay algún cortocircuito a masa.
  7. Compruebe que haya una resistencia inferior a 1 ohmio entre los terminales del circuito de señal correspondiente, que se relacionan a continuación:
    8. • Terminal 12 del circuito de datos serie del bus CAN de alta velocidad, en el conector del mazo de cables del EBCM, y terminal 3, en el conector del mazo de cables del sensor de velocidad de guiñada / aceleración lateral.
    • Terminal 24 del circuito de datos serie del bus CAN de baja velocidad, en el conector del mazo de cables del EBCM, y terminal 1, en el conector del mazo de cables del sensor de velocidad de guiñada / aceleración lateral.
    ⇒Si el valor es superior al especificado, compruebe el circuito de datos serie correspondiente para ver si hay alguna interrupción / resistencia alta.
  8. Compruebe si hay una resistencia infinita entre el terminal del circuito de datos series del bus CAN de alta velocidad 12 y el terminal del circuito de datos serie del bus CAN de baja velocidad 24.
    9. ⇒Si no tiene el valor especificado, repare el cortocircuito entre los circuitos de datos serie.
  9. Si la comprobación de todos los circuitos indica resultados normales, sustituya el sensor de velocidad de guiñada / aceleración lateral. Si se vuelve a activar el DTC, sustituya el EBCM.
