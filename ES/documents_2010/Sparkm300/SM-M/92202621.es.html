Programación y configuración del módulo de interfaz de comunicación

Cada uno de los siguientes procedimientos de servicio requiere que se programe o configure la radio en el módulo de interfaz de comunicación.

Sustitución del módulo de interfaz de comunicación

Importante: 

   • El módulo de interfaz de comunicación (CIM) dispone de un conjunto específico de números únicos que vinculan el módulo a un vehículo específico. Estos números, una identificación de estación de 10 dígitos (STID) y un número de serie electrónico de 11 dígitos (ESN) o una identificación electrónica del fabricante de 18 dígitos (MEOD) son utilizados por OnStar® y la red celular para identificar al vehículo específico. Como estos números son exclusivos para el número de identificación del vehículo que distingue a cada vehículo, estas piezas NO se deben cambiar con las de cualquier otro vehículo. Tras sustituir el CIM, es esencial reconfigurar el sistema OnStar®. El no reconfigurar el sistema resultará en visitas adicionales del cliente para que se realicen reparaciones. NO mantenga pulsado el botón con un punto blanco del teclado ya que esto no reiniciará esta versión del sistema OnStar® y puede causar que se active un DTC, un LED rojo o servicios OnStar® limitados o incompletos. Se requiere el uso de la aplicación TIS 2 Web SPS y Tech 2 para realizar el procedimiento de configuración y ajuste del CIM para este vehículo. El procedimiento de configuración y ajuste es ahora un proceso de dos pasos que permite una activación automatizada por parte del OnStar Center, sin que el técnico tenga que pulsar ningún botón.
   • El servicio OnStar® puede tardar hasta 24 horas hasta estar completamente activado tras realizar los procedimientos de configuración TIS 2 Web y Tech 2. Realice el procedimiento de activación del OnStar® solamente una vez. Varios intentos de realizar el procedimiento pueden resultar en un retraso del proceso de activación.
   • El procedimiento de configuración Tech 2 debe empezar inmediatamente tras completar la programación. Cualquier retraso entre los procedimientos puede provocar una configuración o activación incompletas del CIM.

  1. Conecte el Tech 2 al vehículo.
  2. Conecte el terminal del sistema de información Techline (TIS) al Tech 2.
  3. Seleccione la opción de programación para la activación de OnStar® usando el sistema de programación de servicio (SPS).
  4. Tras terminar la programación de la activación de OnStar®, desemborne el terminal TIS del Tech 2 y realice el procedimiento de configuración del CIM/OnStar usando el Tech 2.
  5. El idioma por defecto para el nuevo CIM es el inglés. Es posible cambiar el idioma a francés o español gracias a las funciones especiales del sistema de programación de servicio (SPS) y Tech 2. La capacidad de cambio de idioma también depende de la generación del nuevo módulo OnStar.

Informe al cliente de que el servicio OnStar® puede tardar hasta 24 horas hasta estar completamente activado tras realizar los procedimientos de configuración TIS 2 Web y Tech 2.
