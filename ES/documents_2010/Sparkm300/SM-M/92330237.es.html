Descripción del circuito/sistema

El control HVAC vigila el sensor de temperatura del refrigerante del A/C con un circuito de baja referencia y un circuito de señal de 5 V. El control HVAC vigila la caída de voltaje en el sensor, que es proporcional a la temperatura. Cuando las temperaturas del refrigerante son frías, la resistencia del sensor es elevada y las señales de tensión son altas. Cuando las temperaturas del refrigerante son calientes, la resistencia del sensor es baja y las señales de tensión son bajas.

Condiciones para la activación de DTC

El encendido se pone en ON.

Condiciones para ajustar el DTC
    • El control HVAC detecta que el circuito de señal del sensor está a menos de -30°C (-22 °F).
    • El control HVAC detecta que el circuito de señal del sensor está a más de 120°C (248°F).

Acción realizada cuando el DTC se ajusta
    • El sistema opera utilizando un valor por defecto.
    • El DTC parpadeará en la pantalla del control HVAC.

Condiciones para el borrar el DTC
    • Un DTC se borrará cuando se vuelva a poner en marcha el motor.
    • Un DTC se borrará cuando se pulse el interruptor AUTO.
    • Un DTC se borrará cuando se gire el mando de control del ventilador hasta 0.

Comprobación del circuito/sistema
  1. Con el encendido en OFF, desconecte el conector del mazo de cables del sensor de temperatura del refrigerante del A/C.
  2. Compruebe que hay menos de 10 Ω entre el terminal 2 del circuito de referencia baja y la toma de masa.
    3. ⇒Si es mayor que el intervalo especificado, compruebe si el circuito de baja referencia tiene una resistencia abierta/alta.
  3. Con el encendido en ON, compruebe si hay 4,8--5,2 V entre el terminal 1 del circuito de señal y el terminal 2 del circuito de referencia baja.
    4. ⇒Si el valor es inferior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a masa o un circuito abierto/resistencia alta. Si según la comprobación el circuito está bien, sustituya el control HVAC K33.
    ⇒Si es superior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a voltaje. Si según la comprobación el circuito está bien, sustituya el control HVAC K33.
  4. Si tras la comprobación todos los circuitos están normales, sustituya el sensor de temperatura del refrigerante del A/C B34E.
