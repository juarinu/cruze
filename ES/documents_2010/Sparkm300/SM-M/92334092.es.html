Programación y configuración del módulo de control del cambio — JF405E

Los siguientes procedimientos de servicio requieren de una acción de programación o configuración para completar la reparación.

Sustitución del módulo de control de la transmisión (TCM)

Si se sustituye el TCM, debe realizarse su programación. Consultar Sistema de programación de servicio (SPS) .

Reprogramación del TCM

Si se reprograma el TCM, debe realizarse su reprogramación. Consultar Sistema de programación de servicio (SPS) .

Asegúrese de que existen las siguientes condiciones para preparar o reprogramar el TCM:

    • La batería está completamente cargada.
    • El interruptor de encendido debe estar en la posición de MARCHA.
    • La conexión del cable del equipo Techline en el conector de enlace de datos (DLC) es segura.

Programe el TCM usando el software más reciente que concuerde con el vehículo. Consulte las instrucciones actualizadas para el usuario del equipo Techline.

Si el TCM no se programa, proceda del siguiente modo:

    • Asegúrese de que la conexión del TCM es correcta.
    • Inspeccione el equipo Techline para asegurarse de que la versión de software es la más reciente.
    • Intente programar el TCM. Si aún así el TCM no se puede programar correctamente, sustituya el TCM.
