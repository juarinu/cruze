Descripción y funcionamiento del cuadro de instrumentos

Comprobación de las pantallas

Algunas funciones del cuadro de instrumentos se comprueban cuando se conecta el encendido para verificar que están funcionando correctamente. Lo siguiente se produce cuando se conecta el encendido:

    • El indicador del ABS se ilumina brevemente.
    • El indicador de la batería se ilumina brevemente.
    • El indicador de freno se ilumina brevemente.
    • El indicador de puerta abierta se ilumina brevemente.
    • El indicador de nivel de combustible bajo se ilumina brevemente.
    • El indicador de presión del aceite se ilumina brevemente.

Ciclo

El cuentakilómetros de viaje se puede acceder a través de la función Arriba/Abajo del interruptor del centro de información del conductor en la parte superior del cuadro de instrumentos.

Rango de combustible

Este mensaje indica la distancia estimada que el vehículo puede recorrer bajo su economía de combustible actual y las condiciones del nivel de combustible desde la última conexión de la batería al cuadro de instrumentos.

El rango de combustible se calcula utilizando la ecuación:

Rango de combustible = Rango

Distancia/rango de combustible utilizado.

    • Distancia de rango = La distancia acumulada viajada desde la última vez que se reajustó este valor
    • Combustible utilizado = El combustible acumulado entregado desde la última vez que se reajustó este valor

El módulo de control del monitor (ECM) envía el valor del rango de combustible al cuadro de instrumentos. El cuadro de instrumentos recibe un mensaje de datos serie con la información del combustible.

El valor de rango de combustible se retiene cuando el encendido está desconectado. La pantalla de rango de combustible no se puede reajustar. La indicación BAJO aparece cuando el rango de combustible cae por debajo de un valor predeterminado.

Velocidad media del vehículo

La velocidad media se calcula utilizando la ecuación:

Velocidad media = Distancia/Tiempo de velocidad media.

    • Distancia = La distancia acumulada viajada desde la última vez que se reajustó este valor
    • Encendido conectado = El encendido acumulado a tiempo desde la última vez que se ajustó este valor

Indicador de nivel de combustible

El cuadro de instrumentos muestra el nivel de combustible basándose en la información recibida del ECM. El ECM convierte los datos provenientes de los sensores de nivel de combustible en una señal de nivel de combustible. El ECM envía la señal de nivel de combustible mediante una señal CAN-Bus de baja velocidad al cuadro de instrumentos. Si el nivel de combustible cae por debajo del 11% el cuadro de instrumentos conecta el indicador de nivel bajo de combustible. La galga de combustible indica vacío por defecto si:

    • El ECM detecta un fallo de funcionamiento en el circuito del sensor del nivel de combustible.
    • El cuadro de instrumentos detecta una pérdida de la comunicación de datos en serie con el ECM.

Velocímetro

El cuadro de instrumentos muestra la velocidad del vehículo basándose en la información recibida del ECM. El ECM envía la información de la velocidad del vehículo mediante una señal CAN-Bus de baja velocidad al cuadro de instrumentos.

El velocímetro está por defecto en 0 km/h (0 mph) si:

El cuadro de instrumentos detecta una pérdida de la comunicación de datos en serie con el ECM.

Cuentarrevoluciones

El cuadro de instrumentos muestra las revoluciones del motor basándose en la información recibida del ECM. El ECM convierte los datos provenientes del sensor de revoluciones del motor en una señal de revoluciones el motor. El ECM envía la información del régimen de revoluciones del motor mediante una señal CAN-Bus de baja velocidad al cuadro de instrumentos.

El tacómetro indica por defecto 0 R.P.M. si:

    • El ECM detecta un fallo de funcionamiento del circuito del sensor de revoluciones del motor.
    • El cuadro de instrumentos detecta una pérdida de la comunicación de datos en serie con el ECM.
