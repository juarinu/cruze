Descripción del sistema del sensor de picado

Descripción del circuito/sistema

El sistema del sensor de picado permite que el módulo de control del motor (ECM) controle la sincronización del encendido para obtener el máximo recorrido, protegiendo al mismo tiempo el motor de unos niveles de detonación que podrían dañarlo. El ECM utiliza el sistema del sensor de picado para comprobar si existe un ruido anormal en el motor que podrían indicar que se está produciendo una detonación, también llamada golpeteo de chispa.

Descripción del sensor

El sistema de sensor de picado utiliza un sensor de respuesta plana. El sensor utiliza una tecnología de cristal piezoeléctrico que produce una señal de tensión de CA de amplitud y frecuencia variables de acuerdo con la vibración del motor o el nivel de ruido. La amplitud y la frecuencia dependen del nivel de picado detectado por el sensor de picado. El sensor de picado está conectado al módulo de control del motor (ECM) por un circuito de señal y un circuito de referencia baja. Ambos circuitos del sensor de picado están protegidos de las interferencias electromagnéticas mediante un circuito de masa aislante. El circuito de masa aislante se conecta a masa a través del ECM.

El ECM aprende un nivel de ruido mínimo, o ruido de fondo, al ralentí desde el sensor de picado y utiliza valores calibrados para los demás regímenes del motor. El módulo de control utiliza el nivel de ruido mínimo para calcular un canal de ruido Una señal del sensor de picado normal está dentro del canal de ruido. Como la carga y el régimen del motor cambian, los parámetros superiores e inferiores del canal de ruido varían para ajustarse a la señal del sensor de picado normal, manteniendo la señal dentro del canal. Para determinar qué cilindros golpetean, el ECM sólo utiliza la información de señal del sensor de picado cuando todos los cilindros están cerca del punto muerto superior de la carrera de encendido. Si el picado se produce, el ECM detecta que la señal está fuera del canal de ruido.

Si el ECM detecta que se está produciendo un picado, retrasará la sincronización del encendido para intentar eliminar el picado. El ECM siempre intenta ajustarse de nuevo a un nivel de compensación cero o sin retardo de encendido. Una señal del sensor de picado anormal permanece fuera del canal de ruido o no se producirá. El diagnóstico de los sensores de picado está calibrado para detectar errores en los circuitos del sensor de picado en el ECM, el cableado del sensor de picado o la salida de voltaje del sensor de picado. Algunos diagnósticos también se calibran para detectar un ruido constante proveniente del exterior, como un componente que se ha aflojado o roto o un ruido mecánico excesivo del motor.
