Descripción de DTC

DTC C0179 54:  Alta temperatura en el sistema

Descripción del circuito/sistema

El módulo de control electrónico de los frenos (EBCM) controla la duración y el número de veces que se activa el sistema de antibloqueo de los frenos o el sistema de control de tracción, para calcular una temperatura estimada de los componentes de los frenos, a fin de prevenir el sobrecalentamiento.

Condiciones para la activación de DTC

Encendido ON.

Condiciones para ajustar el DTC

El EBCM detecte que la temperatura de los frenos es demasiado alta.

Acción realizada cuando el DTC se ajusta
    • El EBCM desactiva el sistema de control de tracción (TCS) / sistema de mejora de la estabilidad del vehículo (VSES).
    •  Es posible que en el módulo de la pantalla de información se visualice un indicador de aviso.

Condiciones para el borrar el DTC
    • El EBCM calcule que los frenos se han enfriado lo suficiente.
    • El EBCM borra el DTC del historial cuando no detecta un DTC actual en 40 ciclos de conducción consecutivos.

Ayuda al diagnóstico

El DTC C0179 no indica que se haya producido un fallo de funcionamiento. Este DTC indica que el sistema ABS o el sistema de control de tracción se ha activado demasiadas veces, lo cual podría provocar altas temperaturas.

Verificación del circuito/sistema

Verifique que el DTC se establece fuera de las Condiciones para el Funcionamiento y Ajuste. Como la mayoría de las activaciones de este DTC son provocadas por un frenado excesivo o una utilización excesiva del sistema de control de tracción, pregunte al conductor en qué condiciones se activa el DTC.

⇒Si el DTC se activa de acuerdo con las Condiciones de funcionamiento y activación, el sistema de frenos está funcionando correctamente.
