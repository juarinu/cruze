Descripción del circuito/sistema

La detección de fallos internos se realiza dentro del módulo de control. No hay circuitos externos implicados.

Condiciones para la activación de DTC

El microprocesador ejecuta el programa para detectar un fallo interno cuando se ordena la entrada de alimentación. El único requisito es que exista tensión y masa. Este programa se ejecuta incluso si la tensión no está dentro del intervalo operativo válido.

Condiciones para ajustar el DTC
    • El módulo de control detecta un fallo de escritura interno.
    • El módulo de control detecta un fallo de suma de control interno.

Acción realizada cuando el DTC se ajusta

El microprocesador rechace todas las entradas adicionales.

Condiciones para el borrar el DTC
    • La anomalía que dio lugar al código de avería (DTC) ya no está presente.
    • Se borrará un historial de DTC al cabo de 100 ciclos de encendido sin fallos.
