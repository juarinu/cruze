1. Interruptor de control de tracción inoperativo

Descripción del circuito/sistema

El módulo de control electrónico de los frenos (EBCM) monitoriza el interruptor del sistema de control de tracción. Cuando se pulse el interruptor del sistema de control de tracción una vez, el EBCM detectará que el sistema de control de tracción está desactivado. Cuando se pulse el interruptor del sistema de control de tracción otra vez, el EBCM detectará que el control de estabilidad está desactivado. El EBCM enviará una señal al cuadro de instrumentos para que ENCIENDA el indicador de Control de tracción desactivado y Control de estabilidad desactivado, para informar al conductor de la desactivación.

Condiciones para la activación de DTC
    •  Encendido ON.
    •  La tensión de encendido sea superior a 8,0 voltios.

Condiciones para ajustar el DTC

El EBCM detecte que el interruptor del sistema de control de tracción (TCS) / VSES está activado durante 20 segundos o más.

Acción realizada cuando el DTC se ajusta
    • El EBCM almacena el DTC.
    • Es posible que el ABS / el TCS / el sistema de mejora de la estabilidad del vehículo (VSES) funcionen como lo hacían anteriormente.

Condiciones para el borrar el DTC
    • Dejen de darse las condiciones que dieron lugar a la activación del DTC.
    • El EBCM borra el DTC del historial cuando no detecta un DTC actual en 40 ciclos de conducción consecutivos.

Referencia de herramienta de diagnóstico

Referencias del módulo de control para información de herramienta de diagnóstico

Verificación del circuito/sistema
  1. Con el encendido CONECTADO, con una herramienta de exploración conectada, ACTIVE y DESACTIVE la comprobación de la pantalla del IPC. El indicador del sistema Stabilitrak debe ENCENDERSE y APAGARSE.
    2. ⇒Si el indicador no se ENCIENDE y APAGA, sustituya el IPC.
  2. Observe el parámetro "interruptor de control de tracción" en la herramienta de diagnóstico del EBCM mientras pulsa y suelta el interruptor de control de tracción. El valor debería cambiar entre Activo e Inactivo.

Comprobación del circuito/sistema
  1. Con el encendido DESCONECTADO, desenchufe el conector del mazo de cables del interruptor del sistema de control de tracción.
  2. Compruebe si hay menos de 1 ohmio entre el terminal 4 del circuito de masa y masa.
    3. ⇒Si es superior al intervalo especificado, compruebe si el circuito de masa tiene una resistencia abierta/alta.
  3. Con el encendido CONECTADO, compruebe que en la herramienta de exploración el parámetro Interruptor del sistema de control de tracción esté desactivado.
    4. ⇒Si no tiene el valor especificado, compruebe si el terminal 3 del circuito de señales tiene un cortocircuito a masa.
  4. Monte un cable de puente con fusible de 3A entre el terminal 3 del circuito de señal y el terminal 4 del circuito de masa. Compruebe que en la herramienta de exploración el parámetro Interruptor del sistema de control de tracción esté activado.
    5. ⇒Si no se corresponde con el valor especificado, compruebe si el circuito de señal tiene un cortocircuito con la tensión, está abierto o tiene una resistencia alta.
  5. Si la comprobación de todos los circuitos indica resultados normales, compruebe o sustituya el interruptor del sistema de control de tracción. Si se vuelve a activar el DTC, sustituya el EBCM.

Comprobación de componentes
  1. Con el encendido DESCONECTADO, desenchufe el conector del mazo de cables del interruptor del sistema de control de tracción.
  2. Compruebe si la resistencia es infinita entre el terminal de señal 3 el terminal 4 de masa con el interruptor en la posición de apertura.
    3. ⇒Si no tiene el valor especificado, sustituya el interruptor del sistema de control de tracción.
  3. Compruebe que haya una resistencia inferior a 3 ohmios entre el terminal de señal 3 y el terminal de toma de masa 4, con el interruptor cerrado.
    4. ⇒Si el valor es superior al especificado, sustituya el interruptor del sistema de control de tracción.
