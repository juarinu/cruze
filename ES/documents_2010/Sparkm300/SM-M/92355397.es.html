Descripción del circuito/sistema

El sensor de nivel de combustible cambia la resistencia según el nivel de combustible. El módulo de control del motor (ECM) monitoriza el circuito de señal del sensor del nivel de combustible para determinar el nivel de combustible. Cuando el depósito de combustible está lleno, la resistencia del sensor es baja y el ECM detecta una tensión de señal baja. Cuando el depósito de combustible está vacío, la resistencia del sensor es alta y el ECM detecta una tensión de señal alta. El ECM utiliza el circuito de señal del sensor de nivel de combustible para calcular el porcentaje del combustible restante en el depósito. El ECM envía la información sobre el nivel de combustible al cuadro de instrumentos (IPC) con el fin de mostrar el nivel de combustible.

Condiciones para la activación de DTC
    • El encendido está en la posición ON.
    • El DTC se ejecuta continuamente cuando se cumplen las anteriores condiciones.

Condiciones para ajustar el DTC

P0462

El ECM detecta que la tensión de la señal del sensor de nivel de combustible es menor de 0,2004 V.

P0463

El ECM detecta que la tensión de la señal del sensor de nivel de combustible es superior a 4,8485 V.

Acción realizada cuando el DTC se ajusta

Los DTCs P0462 y P0463 son de tipo E.

Condiciones para el borrar el DTC

Los DTCs P0462 y P0463 son de tipo E.

Verificación del circuito/sistema

Con el encendido conectado, verifique en la herramienta de diagnóstico que el parámetro del nivel de combustible - ECM está entre 0,5-2,5 V y que varía con el nivel de combustible.

Comprobación del circuito/sistema
  1. Con el encendido desactivado, desconecte el conector del mazo de cables del sensor de nivel de combustible B46.
  2. Con el encendido desconectado, compruebe que hay menos de 1 Ω entre el terminal 3 del circuito de referencia baja y masa.
    3. ⇒Si es mayor que el intervalo especificado, compruebe si el circuito de baja referencia tiene una resistencia abierta/alta. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  3. Con el encendido conectado, verifique que el parámetro del sensor de nivel de combustible en la herramienta de diagnóstico es superior a 0,2 V.
    4. ⇒Si es inferior al intervalo especificado, compruebe si el terminal 4 del circuito de señales tiene un cortocircuito a masa. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  4. Monte un cable de puente con fusible de 3A entre el terminal 4 del circuito de señales y el terminal 3 del circuito de referencia baja. Verifique que el parámetro del sensor de nivel de combustible en la herramienta de diagnóstico es inferior a 4,85 V.
    5. ⇒Si es superior al valor especificado, compruebe si el circuito de señales tiene un cortocircuito a tensión o una resistencia abierta/alta. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  5. Si según la comprobación todos los circuitos están bien, compruebe o sustituya el sensor de nivel de combustible B46.

Comprobación de componentes
  1. Con la llave de contacto en la posición OFF, desmonte la bomba de combustible G12 y verifique que no haya obstrucciones que interfieran con dicho transmisor.
    2. ⇒Si hay presente cualquier interferencia, repare la interferencia del emisor de combustible.
  2. Con el encendido desactivado, desconecte el conector del mazo de cables del sensor de nivel de combustible B46.
  3. Barra el sensor en toda su extensión mientras comprueba que haya 30-270 Ω entre el terminal 4 de señal y el terminal 3 de circuito de referencia baja.
    4. ⇒Si no se encuentra dentro del intervalo especificado o está errático, sustituya el sensor de nivel de combustible B46.
