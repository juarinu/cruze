Descripción y funcionamiento del sistema antirrobo (CTD)

Cuando está activado, el sistema antirrobo de contenido está diseñado para impedir que se robe el contenido del vehículo encendiendo las bocinas y las luces exteriores durante unos 30 segundos cuando se detecta una entrada no autorizada en el vehículo. No obstante, el sistema antirrobo de contenido no afecta al arranque del motor.

Con el sistema antirrobo de contenido activado, se puede considerar como acceso no autorizado cualquiera de los siguientes:

    • Entrada no autorizada en el área bajo el capó
    • Acceso no autorizado al compartimento trasero
    • Al abrir cualquier puerta sin utilizar el comando DESBLOQUEO desde un transmisor de entrada sin llave

Los componentes del sistema antirrobo de contenidos son los siguientes:

    • Módulo del receptor de función remota/receptor de control remoto de la cerradura de la puerta
    • Indicador de seguridad
    • Interruptores de puerta abierta
    • Interruptor de maletero abierto
    • Interruptor de capó abierto

Activación del sistema antirrobo de contenido

Utilice el siguiente procedimiento para activar el sistema:

  1. Apague el encendido.
  2. Cierre todas las puertas.

    3. Nota: El sistema no está activado si las puertas se bloquean manualmente; debe usarse el transmisor de entrada sin llave para activar el sistema antirrobo de contenidos.

  3. Cierre las puertas pulsando el botón de bloqueo (LOCK) del transmisor. La secuencia de armado empezará tan pronto como el transmisor reciba el comando LOCK (Bloqueo).

Bloqueo de las puertas del vehículo sin activar el sistema antirrobo de contenidos.

Se puede bloquear las puertas del vehículo sin activar el sistema antirrobo de contenido. Usar la llave para bloquear las puertas cerrará el vehículo, pero no activará el sistema antirrobo.

Desactivación de un sistema activado/apagado de la alarma

Si se ha usado el transmisor del mando a distancia de las puertas para solicitar la activación del sistema, éste deberá desactivarse.

Nota: Abrir la puerta con la llave correcta o insertar la llave en el encendido y colocarla en la posición de activación no deshabilita el modo de armado, puesto que el módulo del dispositivo antirrobo no recibirá las señales de estos circuitos.

Para desactivar el sistema antirrobo de contenido en el modo de espera, realice lo siguiente:

   • Pulse el botón UNLOCK (Desbloqueo) del transmisor de entrada sin llave.
   • Desconecte la batería o tire de los fusibles.

Descripción del circuito del sistema antirrobo de contenidos

A continuación figura una descripción de cada uno de los componentes utilizados en el sistema antirrobo de contenido:

Módulo del receptor de función remota/receptor de control remoto de la cerradura de la puerta

El sistema de entrada sin llave puede activar y desactivar el sistema antirrobo de contenido. Cuando el módulo de entrada sin llave recibe una señal de bloqueo o desbloqueo de la puerta del transmisor, el módulo de entrada sin llave envía un mensaje al bloque de fusibles IP (bloque de unión inteligente) para que se ejecuten las funciones de activación/desactivación que correspondan.

Indicador de seguridad

El módulo de entrada sin llave ilumina el LED de seguridad en el grupo del panel de instrumentos (IPC) El sistema antirrobo de contenidos utiliza el LED de seguridad para informar al conductor del estado del sistema antes de activarlo.

Interruptores de puerta abierta

El sistema antirrobo de contenido utiliza los interruptores de puerta abierta como indicador de estado para encender la alarma.

El módulo de entrada sin llave supervisa los interruptores de puerta abierta a través de una señal de entrada discreta procedente de cada uno de los interruptores. Si el módulo de entrada sin llave recibe una señal indicando el estado abierto de una puerta cuando el sistema antirrobo de contenidos está activado, el módulo de entrada sin llave activará la alarma.

Interruptor de capó abierto

El sistema antirrobo de contenido utiliza el interruptor de puerta abierta como indicador de estado para encender la alarma.

El módulo de entrada sin llave supervisa el interruptor de capó abierto a través de una señal de entrada discreta del interruptor. Si el módulo de entrada sin llave recibe una señal indicando que se ha abierto el capó cuando el sistema antirrobo de contenidos está activado, el módulo de entrada sin llave activará la alarma.

Interruptor de maletero abierto

El sistema antirrobo de contenido utiliza el interruptor de maletero abierto como indicador de estado para encender la alarma.

El módulo de entrada sin llave supervisa el interruptor de maletero abierto a través de una señal de entrada discreta del interruptor. Si el módulo de entrada sin llave recibe una señal indicando que se ha abierto el maletero cuando el sistema antirrobo de contenidos está activado, el módulo de entrada sin llave activará la alarma.

Señales de entrada

El módulo de entrada sin llave supervisa las siguientes señales de entrada del sistema antirrobo de contenidos:

    • Los interruptores de puerta abierta
    • El interruptor de maletero abierto
    • El interruptor de capó abierto

Salidas

El módulo de entrada sin llave controla lo siguiente en el sistema antirrobo de contenidos:

    • Señal de bloqueo/desbloqueo de puerta del transmisor de entrada sin llave
    • El relé de la bocina
    • Las luces exteriores
