Descripción de DTC

Códigos de avería U0073: Bus de comunicación con el módulo de control desactivado

Descripción del circuito/sistema

Los circuitos de datos de serie son buses de datos de serie utilizados para comunicar información entre los módulos de control. Los circuitos de datos de serie también se conectan directamente al conector de enlace de datos (DLC).

Condiciones para el funcionamiento de DTCs
    • La tensión de alimentación en los módulos está en el intervalo operativo normal.
    • El modo de potencia del vehículo requiere comunicaciones de datos serie.

Condiciones para ajustar el DTC

El módulo que ha activado el DTC ha intentado establecer comunicaciones en los circuitos de datos de serie en más de 3 ocasiones.

Acción realizada cuando los DTCs se ajustan
    • El módulo suspende la transmisión de todos los mensajes.
    • El módulo utiliza valores por defecto para todos los parámetros recibidos en los circuitos de datos de serie.
    • El módulo impide que se activen todos los demás códigos de avería de comunicación.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio de 50 sin que se repita una avería.

Verificación del circuito/sistema
  1. Consulte Referencias de enlace de datos para determinar el sistema de datos de serie que se utiliza para un módulo determinado.
  2. Este DTC no puede recuperarse con un estado actual. El diagnóstico se realiza utilizando el síntoma. Consultar La herramienta de diagnóstico no se comunica con el dispositivo GMLAN de alta velocidad .
