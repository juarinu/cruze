Parámetros de datos de la herramienta de diagnóstico del módulo de la interfaz de comunicación con el vehículo

Parámetro

Estado del sistema

Valor previsto

Descripción

Condiciones de servicio: Encendido ON

Fecha:

--

Numérico

Indica DD/MM/AAAA. Esta es la fecha de fabricación del VCIM.

Señal del GPS

--

Sí

La herramienta de diagnóstico indica "Sí" ó "No". Indica "Sí", si el VCIM ha recibido una señal del GPS.

Intensidad de la señal

--

68

La herramienta de diagnóstico indica 0 - 100. Esta es la intensidad de la señal del VCIM.

Tipo de señal

--

Digital

La herramienta de diagnóstico indica "Digital" o "Analógica". Este es el tipo de señal que recibe el VCIM.

Tiempo

--

Numérico

La herramienta de diagnóstico indica "HH:MM:SS". Esta es la hora actual.

ID del transmisor-receptor

--

Varía

La herramienta de diagnóstico indica un valor de 2 - 5 dígitos. Este es el número de identificación del transmisor-receptor.

Controles de las señales de salida de la herramienta de diagnóstico del módulo de la interfaz de comunicación con el vehículo

Control de salida

Descripción

Configuración de adquisición

Permite seleccionar entre diferentes opciones de sistemas de tecnología celular digital: AMPS (sistema avanzado de telefonía móvil), CDMA (acceso múltiple por división de código), PCS (servicio personal de comunicaciones), GSM (sistema global de comunicaciones móviles). Seleccione CDMA porque Verizon proporciona la tecnología celular para el OnStar, que utiliza el sistema CDMA.

Datos del B1000

Muestra los datos registrados cuando se activó el DTC B1000 en el VCIM.

LED verde

El VCIM enciende el LED verde del espejo retrovisor interior cuando se selecciona ACTIVAR.

Prueba de llamada telefónica

Cuando se selecciona ACTIVAR, el VCIM solicita al OnStar que se conecte con el Centro OnStar.

Actualización de la PRL

El VCIM realiza una actualización de la lista de itinerancia preferida (PRL) cuando se selecciona esta opción.

Programar SID - NID local digital

Seleccione la SID (ID de emisoras) y el NID (dispositivo de interfaz de red) según su país.

Programar número de teléfono

El VCIM proporciona el número de ID del teléfono móvil y el número de directorio móvil para este vehículo. Utilice las teclas de flecha para cambiar el número de teléfono, si fuera necesario.

LED rojo

El VCIM enciende el LED rojo del espejo retrovisor interior cuando se selecciona ACTIVAR.

Seleccionar idioma

Escoja una opción entre 3 idiomas diferentes: Inglés, francés canadiense y español.

Configuración del VCIM

La pantalla indica: Lleve el vehículo a un sitio exterior, abierto, y asegúrese de que el motor esté en funcionamiento. Al aplicar este procedimiento de configuración del VCIM, el sistema realizará automáticamente una llamada al Centro de llamadas de OnStar. Informe al asesor del Centro de llamadas de que se ha instalado un VCIM nuevo. Tenga a mano la STID y el ESN del VCIM nuevo para dárselo al asesor. Si desea información sobre el Procedimiento de reconfiguración, consulte Programación y configuración del módulo de interfaz de comunicación.
