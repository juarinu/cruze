Descripción del circuito/sistema

Módulos conectados al circuito de datos serie controlan comunicaciones de datos de serie durante el funcionamiento normal del vehículo. Entre los módulos se intercambian comandos e información operativa. Conectar una herramienta de diagnóstico al conector de enlace de datos (DLC) permite la comunicación con los módulos para fines de diagnóstico.

Las provisiones de comunicación con el módulo de control del motor (ECM), módulo de control transeje (TCM), módulo de control del inmovilizador y el acceso sin llave a distancia (RKE) /módulo de arranque del motor a distancia (RES) están ubicados en el conector de enlace de datos (DLC). El DLC está ubicado bajo el panel de instrumentos (I/P). El DLC del pasador 16 para la aplicación SIM2K-D52 ofrece las señales de datos serie en los terminales 7 (ECM, TCM, inmovilizador) y 8 (RKE, RES). El DLC se utiliza para enlazar una herramienta de diagnóstico al circuito de datos serie del módulo de control. A continuación se indican algunos usos comunes del scanner:

    • Identificación de códigos de diagnóstico de avería (DTC) guardados
    • Lectura del flujo de datos serie
    • Cómo borrar los DTCs
    • Lectura de los datos del cuadro congelado guardado

Ayuda al diagnóstico
    • A veces, cuando está diagnosticando un problema de un cliente específico o después de una reparación, puede que vea que hay presente un código U de historial. Sin embargo, no hay un estado "actual" o "activo" asociado. Se pueden registrar códigos U de pérdida de comunicación como éstos por diversas razones. Muchas veces resultan obvios para el operador y el técnico del vehículo o no tienen síntomas asociados. Eventualmente se puede borrar por sí mismos automáticamente después de una serie de ciclos de encendido sin fallos. Este problema se podría atribuir sobre todo a una de las situaciones siguientes:
       - Un módulo de control del circuito de comunicación de datos se desconectó cuando el circuito de comunicación estaba activo.
       - Durante el diagnóstico se interrumpió la alimentación de uno o más módulos.
       - Se produjo un problema de batería baja, de modo que varios módulos de control dejan de comunicarse cuando el voltaje de la batería cae por debajo de un cierto umbral.
       - Se restauró la alimentación de batería al vehículo y no todos los módulos de control del circuito de comunicación se reiniciaron al mismo tiempo.
       - Si aparece un código U de pérdida de comunicación en el historial sin una razón aparente, lo más probable es que esté asociado a una de las situaciones anteriores. Todos son problemas temporales y no se deberían interpretar nunca como una avería intermitente, haciendo que tenga sustituir una pieza.
    • No sustituya un módulo de control que informe sobre un código U. Éste identifica qué módulo de control se debe diagnosticar por un problema de comunicación.
    • Debe haber comunicación entre algún módulo y la herramienta de diagnóstico con el sistema de datos serie de GMLAN o CAN no operativo, ya sea de alta o baja velocidad. Esta condición se debe a que algún módulo utiliza ambos sistemas: GMLAN/CAN y KWP2000.
    • Una interrupción en el terminal 5 de circuito de masa del DLC permitirá que la herramienta de diagnóstico funcione, pero no comunicará con el vehículo. Al intentar comunicarse la herramienta de diagnóstico con el vehículo, se mostrará un mensaje "Módulo CANdi no detectado".

Comprobación del circuito/sistema

    Nota: Use los esquemas para identificar lo siguiente:

       • Los módulos de control con los que está equipado el vehículo
       • Ubicaciones del módulo de control en el circuito de datos serie KWP2000
       • Los módulos de control B+, encendido, masa y terminales del circuito de datos serie

  1. Intente establecer comunicación con un módulo de control en el circuito de datos serie KWP2000, comenzando por uno fácilmente accesible. Determine el módulo que no se está comunicando. Consultar Referencias de enlace de datos .
  2. Con el encendido OFF, desenchufe el conector del mazo de cables en el módulo de control que no se está comunicando.
  3. Pruebe con menos de 10 Ω entre cada terminal del circuito de masa y masa.
    4. ⇒Si es superior al intervalo especificado, compruebe si el circuito de masa tiene una resistencia abierta/alta.
  4. Si está equipada, verifique que la lámpara de prueba se ilumina entre cada terminal del circuito B+ y la toma de tierra.
    5. ⇒Si la luz de prueba no se ilumina, pruebe el circuito B+ para ver si hay un cortocircuito a tierra o una resistencia abierta/alta. Si el fusible del circuito está abierto, compruebe si los circuitos de control del módulo de control están cortocircuitados a tierra. Si la prueba de circuitos sale bien, sustituya el módulo de control.
  5. Si está equipado, encendido ON, verifique que la luz de prueba entre cada terminal del circuito de encendido y toma de tierra.
    6. ⇒Si la luz de prueba no se ilumina, compruebe si en el circuito de encendido hay un cortocircuito a masa o una resistencia abierta/alta. Si el fusible del circuito está abierto, compruebe si los circuitos de control del módulo de control están cortocircuitados a tierra. Si la prueba de circuitos sale bien, sustituya el módulo de control.
  6. Con el encendido en OFF durante 60 segundos, compruebe que haya resistencia infinita entre el terminal DLC indicado a continuación y masa:
    7. •  Terminal 7 del DLC para el ECM, TCM y la unidad inmovilizadora
    •  Terminal 8 del DLC para RKE/RES
    ⇒Si es inferior al intervalo especificado, compruebe si en el circuito de datos de serie existe un cortocircuito a masa.
  7. Compruebe si hay menos de 1 V entre el terminal del DLC indicado a continuación y la masa:
    8. •  Terminal 7 del DLC para el ECM, TCM y la unidad inmovilizadora
    •  Terminal 8 del DLC para RKE/RES
    ⇒Si es superior al intervalo especificado, compruebe si el circuito de datos de serie presenta un cortocircuito con la tensión.
  8. Con el encendido OFF durante 60 segundos, compruebe si hay menos de 5 Ω entre el terminal del módulo de control del circuito de datos de serie y el terminal DLC indicado a continuación:
    9. •  Terminal 7 del DLC para el ECM, TCM y la unidad inmovilizadora
    •  Terminal 8 del DLC para RKE/RES
    ⇒Si es superior al intervalo especificado, compruebe si el circuito de datos serie tiene resistencia abierta/alta.
  9. Si todos los circuitos están bien, sustituya el módulo de control que no se está comunicando.
  10. Repita los pasos 1-9 para el siguiente modulo de control accesible fácilmente hasta que se aísle una condición.
