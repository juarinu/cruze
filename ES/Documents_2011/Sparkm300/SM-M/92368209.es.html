Programación y configuración del módulo de interfaz de comunicación

Cada uno de los siguientes procedimientos de servicio requiere que se programe o configure la radio en el módulo de interfaz de comunicación.

Sustitución del módulo de interfaz de comunicación

Nota: 

   • Es de vital importancia utilizar el módulo de la interfaz de comunicación de repuesto únicamente en el vehículo para el que fue solicitado.
   • El imcumplimiento de los siguientes procedimientos tendrá como resultado un LED rojo, un registro de DTC(s) y servicios OnStar® limitados o incompletos
   • No es preciso presionar el botón OnStar® del centro de asistencia OnStar® para completar el procedimiento.
   • Puede ser preciso un período de 24 horas para que el servicio OnStar® quede completamente activado tras la realización de los procedimientos.
   • Realice sólo una vez el siguiente procedimiento. Intentar el procedimiento varias veces puede resultar en un retraso del proceso de activación.
   • Para activar totalmente un módulo OnStar®, deben completarse los procedimientos de solicitud de configuración y activación en todos los vehículos con y sin una suscripción OnStar® activa para garantizar que la unidad se ha configurado correctamente para el vehículo en el que ha sido instalado, así como para actualizar el OnStar® con la información de unidad correcta.
   • Para iniciar la función Paso a paso, antes debe conducirse el vehículo durante 8-16 km (5-10 millas) para permitir que el sistema autocalibre la función de estimación.
   • Se requiere el uso de una interfaz de herramienta de diagnóstico compatible con J 2534 para realizar el procedimiento de configuración y activación OnStar® (ONSU) compatible con SPS.
   • En vehículos equipados con prestaciones TTY, es necesario realizar un ciclo de potencia del Módulo de la interfaz OnStar® (OTIM) y conmutar el modo TTY tras completar los procedimientos de instalación y configuración del módulo de interfaz de comunicación.

  1. Conecte la herramienta de exploración al vehículo.
  2. Conecte el terminal del sistema de información Techline (TIS) a la herramienta de diagnóstico.
  3. Seleccione la opción de configuración y activación de OnStar® (ONSU) usando el sistema de programación de servicio (SPS).
  4. El idioma predeterminado del nuevo módulo de interfaz de comunicación es el inglés. Para cambiar el idioma a francés o español, acceda a la herramienta de diagnóstico.
  5. Informe al cliente que el servicio OnStar® puede requerir un período de 24 horras para quedar completamente activado.
