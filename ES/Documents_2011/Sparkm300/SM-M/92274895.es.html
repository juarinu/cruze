Procedimiento de pantallazos de la herramienta de diagnóstico

Uso de instantáneas en el sistema de diagnóstico global (GDS)

Vista de conjunto

Una instantánea es un registro de lo que estaba recibiendo un módulo de control en el vehículo para información cuando se realizó la instantánea. Una instantánea puede utilizarse para analizar los datos durante el tiempo en el que una condición del vehículo es actual. Esto le permitirá centrarse en hacer que se produzca la condición, en lugar de intentar ver todos los datos como anticipación del fallo. Para grabar una instantánea en GDS, seleccione Grabar instantánea de las opciones de la parte superior de cualquiera de las varias páginas que permiten la toma de instantáneas (por ej. Visualización de datos, Funciones de control, etc.). Cuando haya terminado de grabar, seleccione Terminar grabación de instantánea. Ahora la instantánea estará disponible para reproducirla.

Reproducción de instantáneas

La función de reproducción de instantáneas permite revisar datos del vehículo previamente grabados en instantáneas. Esta información se puede utilizar para analizar problemas del vehículo y para detectar condiciones que ocasionen errores. La función gráfica se puede usar para comparar parámetros y ver si un componente está funcionando adecuadamente. Las instantáneas grabadas se pueden seleccionar para revisarlas con uno de los siguientes tres métodos:

    • Selección de un archivo de instantánea desde la Página principal del vehículo, haciendo clic en Revisar.
    • Haga un doble clic en la instantánea seleccionada desde la Página principal del vehículo.
    • Entrar en selección de Herramientas desde Menú y seleccione Reproducción de instantáneas. A continuación, abra el Administrador de instantáneas. Haga un doble clic en el archivo de la instantánea que usted desea revisar o haga clic en la selección Cargar instantánea y seleccione un archivo de instantánea.

Independientemente del método seleccionado, aparecerá una ventana de Archivo cargado desde la pantalla de Reproducción de instantáneas. Seleccione OK desde la ventana que haya aparecido.

Funciones para reproducir una instantánea

Existen varias funciones de reproducción que se usan para visualizar instantáneas grabadas

    • La función de Reproducción se usa para mover una instantánea hacia delante haciendo clic en esta selección. Se moverá un marcador por la pantalla cuando se use esta selección. Todas las selecciones de esta área se vuelven grises cuando están activas.
    • La función de Pausa se usa para detener temporalmente la reproducción de la instantánea. Cuando se hace clic en esta opción por segunda vez, la instantánea continuará reproduciéndose desde el lugar donde se detuvo.
    • La función de Parada se usa para detener la reproducción de la instantánea. La instantánea iniciará desde el principio, si se vuelve a seleccionar Reproducción después de usar la función de Parada.

Control del tiempo

El control del tiempo se usa para controlar la reproducción de instantáneas. Esta función permite la selección de un periodo de tiempo específico, mostrar diferentes vistas, y controlar la velocidad de la pantalla. Se pueden seleccionar las siguientes funciones directamente desde Control del tiempo.

    • La selección de Marca le permite añadir un comentario específico en la pantalla actual y volver a este punto en el futuro. Cuando se selecciona esta opción, aparece una ventana. Entre el nombre de la marca y seleccione OK.
    • Se puede usar el Cuaderno de notas para añadir comentarios al archivo de reproducción de la instantánea en dicho punto en el tiempo. Cuando se selecciona esta opción, aparece una ventana. Entre un comentario para el archivo de instantánea y seleccione OK.
    • La función de Carga de instantánea (carpeta azul) abre un archivo de instantánea y lo prepara para analizarlo más detenidamente. Esta función también está disponible haciendo clic en un archivo de instantánea que esté disponible en el Administrador de instantáneas.
    • La flecha Hacia abajo indica todas las opciones de Control del tiempo al seleccionarse. También se pueden seleccionar opciones individuales haciendo clic en el artículo que usted desea.
    • La velocidad de la pantalla de instantáneas se puede controlar usando esta opción. Se pueden seleccionar velocidades bajas de 1/8X, 1/4X y 1/2X. También se puede seleccionar la velocidad en tiempo real en 1. Además, se pueden obtener velocidades más altas, de 2X, 4X y 8X, usando esta selección.

Selecciones de visualización

Cada Selección de visualización ofrece una forma distinta de visualizar los datos del vehículo para su análisis. Se pueden seleccionar las siguientes funciones bajo Vistas disponibles:

Nota: El nombre del parámetro aparecerá en idioma original usado para la captura si no hay traducción disponible para dicho nombre.

    • Vista de lista de parámetros -- Muestra información que viene del archivo de instantáneas. El valor actual de los parámetros se muestra en la pantalla.
    • Vista de registro -- Contiene toda la información del archivo de instantáneas. Todos los valores de parámetros están listados en el orden en que se registraron. Se pueden añadir o borrar parámetros específicos de una lista haciendo clic en el artículo correspondiente.
    • Vista de estado -- Muestra los valores de todos los parámetros en un periodo de tiempo específico. Estos valores se pueden usar para comparar con la vista de disparadores.
    • Vista de disparadores -- Muestra todos los disparadores del sistema y los definidos por el usuario. Esta función se puede usar cuando hay una combinación de factores implicados en un problema de diagnóstico. Cuando se cumple una condición de disparo aparece un triángulo de advertencia en la pantalla.

Disparadores:

    • Disparadores para cargar -- Le permiten cargar un archivo que contenga definiciones de disparadores.
    • Disparadores para grabar -- Le permiten guardar el disparador en el disco duro de su ordenador.
    • Disparador para borrar--Le permite borrar el disparador actual. Ya no estará presente.
    • Editar disparador -- Muestra una ventana donde usted puede modificar los límites y parámetros contenidos en un disparador.
    • Añadir disparador -- Muestra una pantalla para configurar un disparador.
    • Vista de DTC -- Tras seleccionar la función Reproducción y esta opción, esta función abre la lista de DTC grabados.
    • Vista de gráfico de líneas -- Muestra datos que se pueden visualizar en relación al eje de tiempo. Desde esta vista se pueden seleccionar parámetros específicos para ver gráficos haciendo clic en la casilla bajo Seleccionado. En la parte inferior del gráfico hay cuatro funciones, Acercar (aumentar tamaño), Alejar (disminuir tamaño), Marcas y Pausar avance de pantalla. Cada una de estas funciones se puede acceder haciendo clic sobre ellas.
