Información de la herramienta de diagnóstico del módulo antirrobo

Controles de salida de la herramienta de diagnóstico del módulo antirrobo

Control de salida

Descripción

Programación de la llave

Este control de salida se usa para programar una nueva llave y se debería realizar al sustituir todas las llaves, un inmovilizador o ambos inmovilizadores y el ECM.

Programación de una llave adicional

Este control de salida se usa para programar una nueva llave.

Modo neutro

Este control de salida se usa para programar un nuevo ECM para el inmovilizador y debería realizarse cuando se sustituye sólo el ECM.
