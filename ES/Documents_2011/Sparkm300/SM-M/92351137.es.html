Descripción del circuito/sistema

El conjunto del cuadro de instrumentos (IPC) enciende momentáneamente el indicador del sistema Stabilitrak, durante la comprobación de la bombilla, cuando se CONECTA el encendido. Cuando se pulsa el interruptor del sistema de control de tracción, el EBCM envía un mensaje de datos serie al IPC para que se encienda a fin de indicar que el control de tracción está desactivado. Cuando el EBCM detecta un fallo, envía un mensaje de datos serie al IPC para que encienda el indicador.

Referencias del módulo de control para información de herramienta de diagnóstico.

Verificación del circuito/sistema
  1. Con una herramienta de exploración conectada, ACTIVE y DESACTIVE la comprobación de la pantalla del IPC. El indicador del sistema Stabilitrak debe ENCENDERSE y APAGARSE.
    2. ⇒Si el indicador del sistema Stabilitrak no se ENCIENDE y APAGA, sustituya el IPC.
  2. Observe el parámetro "interruptor de control de tracción" en la herramienta de diagnóstico del EBCM mientras pulsa y suelta el interruptor de control de tracción. El valor debería cambiar entre Activo e Inactivo.

Comprobación del circuito/sistema
  1. Con el encendido DESCONECTADO, desenchufe el conector del mazo de cables del interruptor del sistema de control de tracción.
  2. Compruebe si hay menos de 1 ohmio entre el terminal 4 del circuito de masa y masa.
    3. ⇒Si es superior al intervalo especificado, compruebe si el circuito de masa tiene una resistencia abierta/alta.
  3. Con el encendido CONECTADO, compruebe que en la herramienta de exploración el parámetro Interruptor del sistema de control de tracción esté activado.
    4. ⇒Si no tiene el valor especificado, compruebe si el terminal 3 del circuito de señales tiene un cortocircuito a masa. Si la comprobación del circuito indica resultados normales, sustituya el EBCM.
  4. Monte un cable de puente con fusible de 3A entre el terminal 3 del circuito de señal y el terminal 4 del circuito de masa. Compruebe que en la herramienta de exploración el parámetro Interruptor del sistema de control de tracción esté activado.
    5. ⇒Si no tiene el valor especificado, compruebe el circuito de señal para ver si hay alguna interrupción / resistencia alta. Si la comprobación del circuito indica resultados normales, sustituya el EBCM.
  5. Si la comprobación de todos los circuitos indica resultados normales, compruebe o sustituya el interruptor del sistema de control de tracción.

Comprobación de componentes
  1. Con el encendido DESCONECTADO, desenchufe el conector del mazo de cables del interruptor del sistema de control de tracción.
  2. Compruebe si la resistencia es infinita entre el terminal de señal 3 el terminal 4 de masa con el interruptor en la posición de apertura.
    3. ⇒Si no tiene el valor especificado, sustituya el interruptor del sistema de control de tracción.
  3. Compruebe que haya una resistencia inferior a 3 ohmios entre el terminal de señal 3 y el terminal de toma de masa 4, con el interruptor cerrado.
    4. ⇒Si el valor es superior al especificado, sustituya el interruptor del sistema de control de tracción.
