Descripciones de DTC

Códigos de avería C0561 71: Recibidos datos de serie inválidos - Almacenada información de sistema deshabilitado

Descripción del circuito/sistema

El módulo de control de freno de mano cuenta con un motor interno, actuador de aplicación, actuador de liberación y sensor de temperatura. El módulo de control del freno de mano también contiene la lógica para aplicar y liberar el freno de mano ordenado mediante el interruptor de control del freno de mano. Cuando el módulo de control del freno de mano recibe una señal del interruptor, se comprueba la temperatura interna de la placa de circuito para verificar si se encuentra en el intervalo operativo antes de que el módulo de control lleve a cabo la operación solicitada. El módulo de freno de mano verificará que tiene una señal válida de otros módulos antes de realizar la operación solicitada.

Condiciones para la activación de DTC
    • Encendido en modo OFF, ACCESSORY o RUN.
    • La tensión de la batería está entre 9-16 V

Condiciones para ajustar el DTC

El módulo de control del freno de mano recibe un mensaje de datos de serie inválido o no recibe ningún mensaje de datos de serie de otro módulo.

Acción realizada cuando el DTC se ajusta
    • Podrá mostrarse un mensaje y/o indicador de aviso.
    • La funcionalidad del freno de estacionamiento eléctrico está degradada.

Condiciones para el borrar el DTC
    • El módulo de control del freno de mano borra el DTC de historial cuando no se detecta un DTC actual en 40 ciclos de encendido.
    • El módulo de control del freno de mano recibe señales válidas de otros módulos.

Verificación del circuito/sistema
  1. Mediante la herramienta de diagnóstico, localice el parámetro inválido en la pantalla de datos.
  2. Consulte el módulo adecuado para diagnosticar la señal de datos inválida. No sustituya el módulo de control del freno de estacionamiento eléctrico.
