Descripción del circuito/sistema

El módulo electrónico de control de la carrocería (BCM) controla el interruptor del control de crucero. Cuando el interruptor on/off del control de crucero está en la posición ON, el interruptor está cerrado y el circuito de señal es bajo. Cuando el interruptor on/off del control de crucero está en la posición OFF, el interruptor está abierto y el circuito de señal es elevado.

Condiciones para la activación de DTC

El encendido está en la posición ON.

Condiciones para ajustar el DTC
    • El BCM detecta un exceso de corriente en circuito de señal del control de crucero/indicador de calentamiento de las ruedas.
    • La condición anterior se da durante más de 5 segundos.

Acción realizada cuando el DTC se ajusta
    • El BCM almacena el DTC B0835 en la memoria.
    • El sistema de control de velocidad de crucero deja de funcionar.

Condiciones para el borrar el DTC
    • Un DTC pasa al historial cuando las condiciones de activación del DTC dejan de existir.
    • El historial de DTC se borra después de 50 ciclos de calentamiento sin un fallo de funcionamiento.
    • El BCM recibe un comando de código de borrado de la herramienta de diagnóstico.

Ayuda al diagnóstico

La utilización de los datos de "Failure Records" (registros de averías) puede ayudar a localizar un fallo intermitente. Si no se puede duplicar el DTC, la información almacenada en "Failure Records" (registros de averías) puede ayudarle a determinar cuántas millas se han recorrido desde que se puso el DTC. El "Fail Counter" (contador de fallos) y el "Pass Counter" (contador de aprobados) pueden ayudar a determinar de cuántos ciclos de encendido ha informado el test de diagnóstico que aprobó o falló.

Verificación del circuito/sistema

Con el encendido activado, observe el parámetro del interruptor del control de crucero con la herramienta de exploración mientras presiona el interruptor del control de crucero ON y OFF. La lectura debe cambiar entre el estado ACTIVADO y DESACTIVADO.

Comprobación del circuito/sistema
  1. Con el encendido desactivado, desconecte el conjunto del interruptor del control de crucero.
  2. Con el encendido activado, verifique que una luz de comprobación se enciende entre el circuito del control de crucero/indicador de calentamiento de las ruedas y la tierra.
    3. ⇒Si la luz de comprobación no se ilumina, sustituya el conjunto del interruptor del control de crucero.
  3. Repare el cortocircuito de la tensión en el circuito de señal del control de crucero/indicador de calentamiento de las ruedas.
    4. ⇒Si la prueba del circuito sale bien, sustituya el BCM.
