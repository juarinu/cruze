Descripción del circuito/sistema

Los sensores de objeto son sensores de 3 cables que se utilizan para determinar la distancia entre el vehículo y un objeto de interés. El módulo de control de aparcamiento asistido suministra 8 V a los sensores de objeto mediante el circuito de referencia de 8 V y ofrece masa mediante el circuito de referencia baja. El módulo de control de aparcamiento asistido dispara los sensores en un ciclo secuencial. Después de que cada sensor transmita, el módulo de control de aparcamiento asistido utiliza el eco del sensor recibido a través del circuito de señal para calcular la distancia y posicionar un objeto.

Condiciones para la activación de DTC
    • Encendido ON
    • El aparcamiento asistido está activado. La activación toma lugar colocando la marcha atrás u operando el interruptor del estacionamiento asistido.

Condiciones para ajustar el DTC

B0954 01, B0955 01, B0956 01 o B0957 01

El módulo de control del aparcamiento asistido avanzado ha detectado que la tensión del circuito del sensor es superior a 11,5 V o el sensor en cuestión no está conectado a masa.

B0954 06, B0955 06, B0956 06 o B0957 06

El módulo de control de aparcamiento asistido ha detectado que la tensión del circuito sensor es superior a 0,5 V.

B0954 08, B0955 08, B0956 08 o B0957 08

El módulo de control de aparcamiento asistido ha recibido una señal inválida.

B0954 21, B0955 21, B0956 21 o B0957 21

El sensor de objetos no determina ningún cambio en la posición de un objeto mientras el vehículo está en movimiento.

B0954 3A, B0955 3A, B0956 3A o B0957 3A

El módulo de control de aparcamiento asistido determina que se ha instalado un sensor de tipo incorrecto.

Acción realizada cuando el DTC se ajusta
    • El estacionamiento asistido está desactivado.
    • El cuadro de instrumentos muestra el símbolo de servicio del sistema de aparcamiento asistido.

Condiciones para el borrar el DTC

La anomalía que dio lugar al código de avería (DTC) ya no está presente.

Verificación del circuito/sistema
  1. Verificar que no se ha registrado el DTC B138A.
    2. ⇒Si existe un DTC, consulte DTC B138A .
  2. Verificar que no hay un DTC B0954 01, B0955 01, B0956 01 o B0957 01.
    3. ⇒ Si hay un DTC, comprobar el sensor apropiado para ver si está contaminado por barro, hielo o nieve. Si no se diagnostica contaminación, consulte las pruebas del sistema/circuito.
  3. Verificar que no hay un DTC B0954 21, B0955 21, B0956 21 o B0957 21.
    4. ⇒ Si hay un DTC, comprobar el sensor respectivo para ver si está contaminado por barro, hielo o nieve. Si no se diagnostica contaminación, cambiar el sensor.
  4. Verificar que no hay un DTC B0954 3A, B0955 3A, B0956 3A o B0957 3A.
    5. ⇒ Si hay un DTC, sustituir el sensor respectivo con un sector del tipo correcto.
  5. Con el encendido conectado y la posición DRIVE seleccionada en el cambio, verificar en la herramienta de diagnóstico que el parámetro Estado del sistema de aparcamiento asistido muestra Habilitado.

Comprobación del circuito/sistema

Nota: La herramienta de exploración debe estar desconectada en los pasos 1 y 2 para permitir al módulo apagarse.

  1. Con el encendido desconectado durante 1 minuto, desenchufar el conector del mazo de cables del sensor apropiado de objeto delantero B78.
  2. Compruebe si hay menos de 5 Ω entre el terminal 2 del circuito referencia baja y la masa.
    3. ⇒Si es mayor que el valor especificado, compruebe si el circuito de referencia baja tiene una resistencia abierta/alta. Si todas las comprobaciones de los circuitos son normales, sustituir el módulo de control de aparcamiento asistido avanzado K86.
  3. Encendido conectado, activar el estacionamiento asistido, comprobar que haya 7,5-9,0 V entre el terminal 1 del circuito de referencia de 8 V y masa.
    4. ⇒Si es menor que el margen especificado, compruebe si hay un circuito abierto/resistencia alta en el circuito de referencia 8 V. Si todas las comprobaciones de los circuitos son normales, sustituir el módulo de control de aparcamiento asistido avanzado K86.
  4. Encendido desactivado, retirar el sensor de objetos que está fallando y cambiar su emplazamiento con otro sensor de objeto frontal que esté funcionando correctamente. Conectar el conector del mazo de cables en ambos sensores.
  5. Encendido conectado, verificar que el DTC se ha reposicionado para la nueva ubicación donde está instalado el sensor que tiene un fallo de funcionamiento.
    6. ⇒Si el DTC no se ha reposicionado para la nueva ubicación, comprobar el terminal 3 del circuito de señal para ver si existe un cortocircuito de tensión, un cortocircuito a masa o una resistencia alta/abierta. Si todas las comprobaciones de los circuitos son normales, sustituir el módulo de control de aparcamiento asistido avanzado K86.
  6. Si el DTC se reposiciona para la nueva ubicación en la que se encuentra el sensor con el fallo de funcionamiento, sustituir el sensor de objeto frontal apropiado.
