Descripción del circuito/sistema

El módulo de control del motor (ECM) suministra tensión a cada inyector de combustible en los circuitos de control de tensión positiva de los inyectores. El ECM excita cada inyector de combustible conectando a masa el circuito de control de dicho inyector. El ECM supervisa el estado de los circuitos de control de tensión positiva de los inyectores y los circuitos de control de los inyectores de combustible. Los inyectores están separados en los dos grupos siguientes:

    • Grupo 1 - Inyectores 1 y 4
    • Grupo 2 - Inyectores 2 y 3

Si el ECM detecta una condición en el circuito del inyector de combustible, el inyector afectado se desactivará y puede desactivar el grupo de inyectores de combustible afectado.

Condiciones para el funcionamiento de DTC
    • El motor está funcionando.
    • El ECM supervisa si se da una condición una vez por revolución del árbol de levas.

Condiciones para ajustar el DTC

El ECM detecta que el estado ordenado por el conductor y el estado real del circuito de control no coinciden.

Acción realizada cuando se registra el DTC

Los DTC P1248 00, P1249 00, P122A 00 y P122B 00 son DTC de tipo A.

Condiciones para el borrar el DTC

Los DTC P1248 00, P1249 00, P122A 00 y P122B 00 son DTC de tipo A.

Verificación del circuito/sistema
  1. Con el encendido conectado, observe la información del DTC con una herramienta de exploración. No deben activarse los DTC P2147 00, P2148 00, P2150 00 o P2151 00.
    2. ⇒Si se ha registrado alguno, consulte Lista de códigos de diagnóstico de avería (DTC) - Vehículo .
  2. Conduzca el vehículo en las condiciones de ejecución de los DTC para verificar que el DTC no reaparece. También puede conducir el vehículo en las condiciones que se han capturado en la lista de datos Registros de fallos/cuadro congelado.

Comprobación del circuito/sistema
  1. Con el encendido desactivado, desconecte el conector del mazo de cables en el inyector de combustible afectado.
  2. Con el encendido conectado, compruebe si hay 2,0-2,5 V entre el terminal 1 del circuito de control y la masa.
    3. ⇒Si es menor que el intervalo especificado, compruebe si el circuito de control tiene una resistencia abierta/alta o un cortocircuito a masa. Si todas las pruebas de circuitos resultan bien, sustituya el K20 módulo de control del motor.
    ⇒Si es superior al margen especificado, compruebe si el circuito de control tiene un cortocircuito a voltaje. Si todas las pruebas de circuitos resultan bien, sustituya el K20 módulo de control del motor.
  3. Con el encendido desconectado, desconecte el conector del mazo de cables X1 del módulo de control del motor K20.
  4. Compruebe si hay menos de 5 Ω entre el terminal 2 del circuito de control bajo de la válvula de inyección de combustible afectada y el terminal del módulo de control del motor K20 tal y como se muestra a continuación:
    5. • Terminal 26 para la válvula de inyección 1
    • Terminal 74 para la válvula de inyección 2
    • Terminal 75 para la válvula de inyección 3
    • Terminal 25 para la válvula de inyección 4
    ⇒Si el valor es superior al intervalo especificado, compruebe si el circuito de control bajo tiene una resistencia abierta/alta
  5. Si según la comprobación todos los circuitos están bien, lleve a cabo la Comprobación de componentes antes de sustituir el módulo de control del motor K20.

Comprobación de componentes

Prueba estática

  1. Desconecte el conector de mazo de cables del inyector de combustible afectado.

    2. Nota: El DMM y las conexiones de prueba deben calibrarse a 0 Ω con el fin de evitar diagnósticos incorrectos.

  2. Compruebe si hay 0,2-0,7 Ω entre el terminal 2 de control bajo y el terminal 1 de control alto.
    3. ⇒Si no se encuentra dentro del rango especificado, sustituya el inyector de combustible.
  3. Compruebe si hay resistencia infinita entre cada terminal y la masa.
    4. ⇒Si no es el valor especificado, sustituya el inyector de combustible.
