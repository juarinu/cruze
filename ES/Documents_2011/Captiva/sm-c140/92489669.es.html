Descripción del circuito/sistema

El cuadro de instrumentos vigila el sensor de temperatura de aire ambiente con un circuito de baja referencia y un circuito de señal de 5 V. El cuadro de instrumentos vigila la caída de voltaje en el sensor, que es proporcional a la temperatura. Cuando las temperaturas de aire ambiente son frías, la resistencia del sensor es elevada y las señales de tensión son altas. Cuando las temperaturas de aire ambiente son calientes, la resistencia de los sensores es baja y las señales de tensión son bajas. El cuadro de instrumentos convierte el valor de tensión a un valor de temperatura de grados Celsius o Fahrenheit que se mostrará en el centro de información del conductor.

Condiciones para el funcionamiento de DTC

La tensión del sistema está entre 9-16 V.

Condiciones para ajustar el DTC

B0158 02

El cuadro de instrumentos detecta que el circuito de señal del sensor está a más de 88°C (190°F).

B0158 05

El cuadro de instrumentos detecta que el circuito de señal del sensor está a menos de -40°C (-40°F).

Acción realizada cuando se registra el DTC

El cuadro de instrumentos usa un valor de temperatura de aire predeterminado para otros cálculos. El centro de información del conductor muestra --- °C (--- °F).

Condiciones para el borrar el DTC

El DTC será historia si el cuadro de instrumentos ya no detecta una avería.

Verificación del circuito/sistema

Con el encendido conectado, observe el parámetro de temperatura del aire exterior de la herramienta de diagnóstico. El valor debe estar entre -40 °C (-40 °F) y +88 °C (+190 °F) y debe modificarse según los cambios de temperatura del aire.

Comprobación del circuito/sistema
  1. Con el encendido apagado, desenchufe el conector del mazo de cables del sensor de temperatura del aire exterior.
  2. Monte un cable de puente protegido con fusible de 3 A entre el terminal 3 del circuito de señal y el terminal 1 del circuito de referencia baja. Verifique en la herramienta de diagnóstico si el parámetro de temperatura del aire exterior es superior a 54°C (130°F).
    3. ⇒Si el valor es inferior al margen especificado, compruebe si el circuito de referencia baja tiene un cortocircuito a tensión o un circuito abierto/resistencia alta. Si la prueba del circuito sale bien, sustituya el panel de instrumentos.
  3. Con el encendido conectado, compruebe si hay 4,6-5,4 V entre el terminal 1 del circuito de señal y masa.
    4. ⇒Si es superior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a voltaje. Si la prueba del circuito sale bien, sustituya el panel de instrumentos.
    ⇒Si el valor es inferior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a masa o un circuito abierto/resistencia alta. Si la prueba del circuito sale bien, sustituya el panel de instrumentos.
  4. Verifique la precisión del sensor de temperatura del aire exterior. Consultar Resistencia del sensor de temperatura del aire ambiente .
    5. ⇒Si no está dentro del intervalo, sustituya el sensor de temperatura del aire exterior.
  5. Si la comprobación de todos los circuitos es correcta, compruebe o sustituya el sensor de temperatura del aire exterior.

Comprobación de componentes

Prueba estática

  1. Compruebe el sensor de temperatura del aire exterior variando el sensor de temperatura mientras vigila la resistencia del sensor.
  2. Compare las lecturas con el Resistencia del sensor de temperatura del aire ambiente .
    3. ⇒Si no está dentro del intervalo, sustituya el sensor de temperatura del aire exterior.
