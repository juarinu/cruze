Descripción de DTC

DTC P0496: Flujo del sistema de emisión de vapores sin purga

Descripción del circuito/sistema

Este DTC comprueba la existencia de vacío incorrecto en el colector de admisión al sistema de emisiones de vapores (EVAP). El módulo de control sella el sistema EVAP ordenando la desactivación de la válvula solenoide de purga del depósito EVAP y activando la válvula solenoide de ventilación de dicho depósito. El módulo de control controla el sensor de presión del depósito de combustible (FTP) para determinar si hay vacío en el sistema EVAP. Si el vacío en el sistema EVAP es superior a un valor predeterminado dentro de un intervalo de tiempo predeterminado, aparece este DTC.

Condiciones para la activación de DTC
    • No se han ajustado DTC P0107, P0108, P0112, P0113, P0117, P0118, P0125, P0128, P0442, P0443, P0446, P0449, P0452, P0453, P0445, P1106, P1107, P1111, P1112, P1114, P1115, P1516.
    • La tensión de encendido se sitúa entre 10-18 voltios.
    • La presión barométrica (BARO) es superior a 74 kPa.
    • El nivel de combustible está entre 15-85 por ciento.
    • La temperatura del refrigerante del motor (ECT) está entre 4-30 °C (39-86 °F).
    • La temperatura del aire de admisión (IAT) está entre 4-30°C (39-86°F).
    • El ECT y IAT iniciales están dentro del margen de 9°C (16°F) entre sí.
    • El sensor de velocidad del vehículo (VSS) es inferior a 121 km/h (75 mph).
    • El sensor FTP está en menos de -2,5 pulgadas H2O con el encendido conectado.
    • El DTC P0496 aparece una vez por cada arranque frío durante 96 segundos.

Condiciones para ajustar el DTC

El módulo de control detecta vacío durante una fase sin purga.

Acción realizada cuando el DTC se ajusta

El DTC P0496 es un DTC de tipo B.

Condiciones para borrar el MIL/DTC

El DTC P0496 es un DTC de tipo B.

Comprobación del circuito/sistema
  1. Encendido conectado, motor desconectado.
  2. Selle el sistema EVAP utilizando la función de Purga/Sellado con una herramienta de exploración.
  3. Arranque el motor.
  4. Observe el sensor FTP con una herramienta de exploración. El parámetro del sensor de presión del depósito de combustible debe ser inferior a 1,7 voltios después de 90 segundos.
    5. ⇒Si el parámetro del sensor de presión del depósito de combustible es superior a los valores especificados, sustituya la válvula solenoide de purga del recipiente del EVAP.
