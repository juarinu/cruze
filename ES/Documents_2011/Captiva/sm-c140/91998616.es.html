1. La herramienta de diagnóstico no se comunica con el dispositivo GMLAN de baja velocidad

Descripción del circuito/sistema

Los módulos conectados a los circuitos de datos de serie GMLAN de alta y baja velocidad controlan comunicaciones de datos serie durante el funcionamiento normal del vehículo. Entre los módulos se intercambian comandos e información operativa. Los módulos disponen de información almacenada previamente sobre los mensajes que es necesario intercambiar en los circuitos de datos serie para cada una de las redes virtuales. Se supervisan los mensajes y, además, el módulo receptor utiliza algunos mensajes periódicos como indicación de la disponibilidad del módulo transmisor. Todos los mensajes contienen el número de identificación del módulo transmisor.

El módulo de control de la carrocería (BCM) activa el circuito de habilitación de la comunicación de datos en serie aplicando tensión cuando la llave de encendido está en ACC, ACTIV. o ARRANQUE. El circuito de habilitación de la comunicación de datos en serie activa el módulo de control del freno electrónico (EBCM), el módulo de tracción a las cuatro ruedas y el módulo de control de la dirección asistida electrónica, si los hay, para la comunicación por bus.

Condiciones para la activación de DTC
    • La tensión del sistema está entre 9-16 V.
    • El módulo maestro de modo de potencia del vehículo requiere que se produzca la comunicación de datos serie.

Condiciones para ajustar el DTC

No se ha recibido un mensaje supervisado periódicamente que incluye la disponibilidad del módulo transmisor.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio de 50 sin que se repita una avería.

Comprobación del circuito/sistema
  1. Mediante la descripción anterior de DTC y la ayuda de diagnóstico, determine el módulo que no está comunicando y que debería comprobarse.
  2. Con el encendido OFF, desconecte el conector del mazo de cables del módulo que no se está comunicando.

    3. Nota: Sólo los módulos GMLAN de alta velocidad tienen un circuito de habilitación de la comunicación de datos en serie. Consulte los esquemas de módulos para identificar qué módulos tienen estos circuitos. Si el módulo que no se está comunicando no dispone de uno de estos circuitos, proceda al paso 4.

  3. Con el encendido conectado, verifique que se ilumina una lámpara de prueba entre el circuito de habilitación de la comunicación de datos en serie y masa.
    4. ⇒Si la lámpara de prueba no se ilumina, repare el circuito si hay un circuito abierto/resistencia alta. Si la prueba de circuitos sale bien, sustituya el BCM.
  4. Con el encendido conectado, verifique que se ilumina una lámpara de prueba entre todos los circuitos de tensión positiva de la batería y masa:
    5. ⇒Si la lámpara de prueba no se ilumina, repare el circuito si hay un cortocircuito a tierra, un circuito abierto o una resistencia alta. Si el fusible del circuito está abierto, compruebe también si las salidas de voltaje positivo del módulo están cortocircuitados a tierra. Si la prueba de circuitos sale bien, sustituya el módulo defectuoso.
  5. Verifique que se ilumina una lámpara de pruebas entre todos los circuitos de tensión de encendido y la masa.
    6. ⇒Si la lámpara de prueba no se ilumina, repare el circuito si hay un cortocircuito a tierra, un circuito abierto o una resistencia alta. Si el fusible del circuito está abierto, compruebe también si las salidas de voltaje positivo del módulo están cortocircuitados a tierra. Si la prueba de circuitos sale bien, sustituya el módulo defectuoso.
  6. Compruebe si hay menos de 1 Ω entre el circuito de marcha del módulo y la masa.
    7. ⇒Si hay más de 1 Ω, repare el circuito de masa si está abierto.
  7. Use Referencias de enlace de datos para determinar si el módulo que no se está comunicando debería usar datos en serie GMLAN de baja velocidad.
    8. ⇒Si el módulo es GMLAN de alta velocidad solamente, cambie el módulo.
  8. Compruebe que existe menos de 1 Ω en el circuito de datos en serie GMLAN de baja velocidad entre el módulo que no comunica y el módulo que muestra el DTC.
    9. ⇒Si es superior a 1 Ω, compruebe si el circuito de datos en serie GMLAN de baja velocidad tiene un circuito abierto/resistencia alta.
  9. Si todos los circuitos están bien, sustituya el módulo que no se está comunicando.
