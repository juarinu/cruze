Descripción de DTC

DTC P06B8 00: Funcionamiento de la memoria de acceso aleatorio del módulo de control

Descripción del circuito/sistema

Este DTC se refiere a las condiciones de integridad interna del microprocesador dentro del módulo de control del motor (ECM). Este diagnóstico también trata el problema del ECM no programado.

Condiciones para el funcionamiento de DTC
    • El encendido está en la posición ON.
    • Este DTC se reproduce continuamente cuando se cumple la condición anterior.

Condiciones para ajustar el DTC

El ECM detecta una condición interna durante más de 1 seg.

Acción realizada cuando se registra el DTC

El DTC P06B8 00 es un DTC de tipo A.

Condiciones para el borrar el DTC

El DTC P06B8 00 es un DTC de tipo A.

Comprobación del circuito/sistema

Programe el ECM. Si el DTC P068B 00 se activa después de realizar la programación, lleve a cabo los pasos siguientes:

    • Asegúrese de que todas las conexiones son seguras.
    • Asegúrese de que el equipo de programación funciona correctamente.
    • Asegúrese de que se utiliza el software correcto y la calibración correcta.
    • Intente programar el módulo de control del motor K20.
       ⇒Si se activa el DTC P06B8 00, sustituya el módulo de control del motor K20.
