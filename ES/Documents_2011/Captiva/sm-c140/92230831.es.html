Descripción del sistema de encendido electrónico

El sistema de encendido electrónico (EE) produce y controla un encendido secundario de alta energía. Este encendido se utiliza para encender la mezcla de aire comprimido/combustible exactamente en el momento preciso. Ello ofrece un recorrido óptimo, un ahorro de combustible, y el control de los gases de escape. El sistema de encendido utiliza una bobina individual para cada cilindro. Las bobinas de encendido se montan en el centro de la tapa de cada árbol de levas con fuelles cortos integrados que conectan las bobinas a las bujías. El módulo de control del motor (ECM) enciende y apaga los módulos del accionamiento en cada bobina de encendido. El ECM utiliza de manera primaria el régimen de revoluciones del motor, la señal del sensor MAF y la información de los sensores de la posición de cigüeñal y la posición del árbol de levas. Esto controla la secuencia, el intervalo y el calado de la distribución del encendido. El sistema EE consta de los siguientes componentes:

Sensor de posición del cigüeñal

El sensor de posición del cigüeñal trabaja junto con el reluctor de 58 dientes en el cigüeñal. El módulo de control del motor (ECM) efectúa un seguimiento de la frecuencia de la tensión en el circuito de señales del sensor de posición del cigüeñal. Cuando un diente del reluctor pasa por el sensor, el sensor crea un pulso digital Activado/Desactivado. Esta señal digital es procesada por el ECM. Los dientes del reluctor presentan una separación de 6 grados. Dado que tiene solo 58 dientes, queda un hueco de 12 grados que no está cortado. Esto crea un patrón de firma que permite al ECM determinar la posición del cigüeñal. El ECM utiliza la señal para determinar qué par de cilindros se acercan al punto muerto superior basándose únicamente en la señal de posición del cigüeñal. Las señales del sensor de posición del árbol de levas se utilizan para determinar cuál de estos 2 cilindros está en la carrera de encendido y cuál en la carrera de escape. El ECM utiliza esta información para sincronizar adecuadamente el sistema de encendido, los inyectores de combustible y el control de picado. Este sensor también se utiliza para detectar averías en el encendido.

Sensor de posición del árbol de levas

Este motor utiliza un sensor de posición del árbol de levas para cada árbol de levas. Las señales del sensor de posición del árbol de levas son de pulsos digitales activados/desactivados, y se emiten 4 veces por cada vuelta del árbol de levas. El sensor de posición del árbol de levas no afecta directamente el funcionamiento del sistema de encendido. La información del sensor de posición del árbol de levas es utilizada por el módulo de control del motor (ECM) para determinar la posición del árbol de levas en relación con la posición del cigüeñal. Efectuando un seguimiento de las señales de posición del árbol de levas y de posición del cigüeñal, el ECM puede cronometrar de manera precisa el funcionamiento de los inyectores de combustible. El ECM suministra al sensor de posición del árbol de levas un circuito de referencia de 5 V y un circuito de referencia baja. Las señales del sensor de posición del árbol de levas se introducen en el ECM. Estas señales también se utilizan para detectar la alineación del árbol de levas con el cigüeñal.

Sensor de picado

El sistema del sensor de picado permite que el módulo de control controle la sincronización del encendido para obtener el máximo recorrido, protegiendo al mismo tiempo el motor de unos niveles de detonación, también llamada picado de chispa, que podrían dañarlo. El sistema de sensor de picado utiliza uno de 2 sensores del cable doble de respuesta plana. El sensor utiliza tecnología de cristal piezoeléctrico que produce una señal de tensión CA de amplitud y frecuencia variables de acuerdo con la vibración del motor o el nivel de ruido. La amplitud y la frecuencia dependen del nivel de picado detectado por el sensor de picado. El módulo de control recibe la señal del sensor de picado a través del circuito de señal. El módulo de control suministra la masa del sensor de picado a través del circuito de referencia baja.

El módulo de control aprende un nivel de ruido mínimo, o ruido de fondo, al ralentí desde el sensor de picado y utiliza valores calibrados para los demás intervalos de r.p.m. El módulo de control utiliza el nivel de ruido mínimo para calcular un canal de ruido Una señal del sensor de picado normal irá por dentro del canal de ruido. Como la carga y el régimen del motor cambian, los parámetros superiores e inferiores del canal de ruido variarán para ajustarse a la señal del sensor de picado normal, manteniendo la señal dentro del canal. Para determinar qué cilindros golpetean, el módulo de control sólo utiliza la información de señal del sensor de picado cuando todos los cilindros están cerca del punto muerto superior (TDC) de la carrera de encendido. Si se produce un picado, la señal se saldrá del canal de ruido.

Si el módulo de control determina que se está produciendo un picado, retrasará la sincronización del encendido para intentar eliminar el picado. El módulo de control siempre intenta volver a un nivel de compensación cero o sin retardo de encendido. Una señal del sensor de picado anormal se mantendrá fuera del canal de ruido o no se producirá. Los diagnósticos del sensor de picado están calibrados para detectar fallos en los circuitos del sensor de picado dentro del módulo de control, el cableado del sensor de picado o la salida de tensión del sensor de picado. Algunos diagnósticos también se calibran para detectar un ruido constante proveniente del exterior, como un componente que se ha aflojado o roto o un ruido mecánico excesivo del motor.

Bobinas de encendido

Cada bobina de encendido contiene un módulo de accionamiento de estado sólido como elemento primario. El módulo de control del motor (ECM) ordena al excitador de la bobina que inicie un evento de encendido aplicando la tensión al circuito de control de encendido (IC) durante el tiempo o intervalo adecuados. Cuando se quita la tensión, la bobina acciona la bujía.

Módulo de control del motor (ECM)

El módulo de control del motor (ECM) controla todas las funciones del sistema de encendido, y corrige constantemente la sincronización de la chispa de la bujía. El ECM efectúa un seguimiento de la información de varias entradas de sensores que podría incluir los siguientes componentes, si fuera pertinente:

    • Sensor de posición del acelerador
    • El sensor de temperatura del refrigerante del motor (ECT).
    • El sensor del flujo y masa de aire (MAF)
    • El sensor de la temperatura de aire de entrada (IAT)
    • El sensor de velocidad del vehículo (VSS)
    • Los sensores de posición de marcha de la transmisión o información de intervalo
    • Sensores de picado del motor
    • El sensor de presión ambiente (BARO)

Información significativa sobre el encendido

El sensor de posición del árbol de levas de admisión del cilindro 1 se utiliza para sincronizar el sistema de encendido y la válvula de inyección. Si la señal del sensor CMP es intermitente, el vehículo se calará y no se activará ningún DTC. Compruebe todos los circuitos del sensor de posición del árbol de levas de admisión del cilindro 1 por si las conexiones fueran defectuosas.
