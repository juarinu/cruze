Descripción y funcionamiento del control de crucero

Funcionamiento del sistema de control de crucero

La aplicación del sistema de control de crucero es mantener el vehículo a una velocidad constante que ha programado el conductor. Cunado se activa el control de crucero, un módulo de control del motor (ECM) mantiene o aumenta la velocidad.

Si las condiciones de conducción requieren una aceleración repentina después de haber seleccionado el control de crucero, la velocidad se puede aumentar de la manera habitual, pisando el acelerador manualmente. El control de crucero se desactiva si se pisa el freno (o el embrague, con un eje de transmisión manual).

La velocidad mínima para seleccionar el control de crucero es de 19 mph (30 km/h) Cuando el control de crucero está en funcionamiento, la luz indicadora CRUISE aparecerá en el cuadro de instrumentos.

El ECM realiza un seguimiento de los circuitos de señales de los interruptores del control de crucero. El ECM utiliza el circuito de señal del interruptor del control de crucero para determinar el momento de establecer y mantener la velocidad del vehículo. El ECM realiza un seguimiento del circuito de señal del sensor de velocidad del vehículo para mantener la velocidad del vehículo. El ECM utiliza el motor del control del actuador del acelerador (TAC) para controlar la velocidad del vehículo.

Interruptor del control de velocidad de crucero

Tras activar el interruptor principal y después de que el actuador del control de crucero detecte la posición neutral del interruptor, se pueden realizar las siguientes operaciones mediante el interruptor del control de crucero:

Establecer

Para activar el sistema de control de crucero, presione el botón de On/Off del sistema y sitúelo en la posición ON, a continuación presione el botón configurar/dejar rodar cuando la velocidad sea 19 mph (30 km/h). El módulo de control del motor (ECM) confirmará que se han cumplido los criterios de control de crucero. El ECM registrará la velocidad del vehículo e iluminará el indicador de crucero.

Cuando se pisa el pedal del acelerador con el sistema de control de crucero activado, el conductor podrá desactivar este sistema para acelerar el vehículo por encima de la velocidad establecida. Cuando se suelte el pedal del acelerador, el vehículo decelerará y volverá a la velocidad establecida.

Rodar en punto muerto

Si el control de crucero se apaga con el interruptor principal, todas sus funciones se detendrán. Cuando se apaga el control de crucero o el encendido, se borrará la memoria de la velocidad del control de crucero programada.

Acelerar

Al activar y mantener pulsado el interruptor de REANUDAR/ACC, cuando el sistema de control de crucero está activado, el vehículo acelerará a una velocidad superior a la programada actualmente. Cuando se suelte el interruptor REANUDAR/ACC, el ECM registrará la velocidad del vehículo y la mantendrá como la nueva velocidad establecida. Cuando se active el sistema de control de crucero, si se activa momentáneamente el botón de REANUDAR/ACC el vehículo podrá acelerar en incrementos de 1 mph (1,6 km/h) cada vez que este botón se active momentáneamente.

Reanudar

Si el control de crucero está puesto y se ha desactivado mediante el uso del freno o el embrague, si se supera la velocidad máxima y no se mantiene la mínima, la última velocidad memorizada se puede reprogramar con el botón de REANUDAR/ACC. Si se activa el botón de REANUDAR/ACC momentáneamente, tras haber desactivado el sistema de control de crucero mediante el pedal del freno, se recordará la velocidad previa seleccionada que se había registrado en el ECM.

Si el control de crucero se apaga con el interruptor principal, todas sus funciones se detendrán. Cuando se apaga el control de crucero o el encendido, se borrará la memoria de la velocidad del control de crucero programada.
