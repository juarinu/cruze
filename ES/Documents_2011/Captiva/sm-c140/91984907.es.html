Descripción de DTC

Códigos de avería B1019 00: Error de configuración del sistema

Descripción del circuito/sistema

El módulo de diagnóstico y detección de sistemas de retención inflables (SDM) ejecuta comprobaciones de diagnóstico del sistema de retención de forma continuada. Cuando el encendido está conectado, el SDM comprobará si están montados los componentes correctos para la configuración del vehículo. Si el SDM detecta que hay demasiados o demasiados pocos componentes en el sistema, se registrará el DTC B1019.

Condiciones para la activación de DTC

La tensión del encendido esté entre 9 y 16 voltios.

Condiciones para ajustar el DTC
    • El SDM se ha programado con las opciones equivocadas para el vehículo.
    • Se ha montado el SDM equivocado para el contenido del vehículo.

Acción realizada cuando el DTC se ajusta
    • El SDM activa el testigo AIR BAG mediante la comunicación de datos serie.
    • El SDM cancela todas las activaciones.

Condiciones para el borrar el DTC
    • Se debe montar el SDM correcto.
    • Se deben montar los componentes correctos.

Ayuda al diagnóstico

Si se ha sustituido el SDM, asegurarse de que se ha montado el número de pieza correcto para este vehículo.

Información de referencia

Referencia de información eléctrica

    •  Prueba del circuito
    •  Reparaciones de conectores
    •  Compruebe si hay estados intermitentes o conexiones deficientes
    •  Reparaciones de los cableados

Comprobación del circuito/sistema
  1. Con el encendido conectado, emplear la herramienta de diagnóstico para solicitar los DTC SIR. Verificar que se registra el DTC B1019.
    2. ⇒Si el DTC B1019 es histórico, borrar los códigos y volver a comprobarlos.
  2. Con la herramienta de diagnóstico, comprobar que el BCM está programado con el VIN. Verificar que el BCM está programado con el número de VIN correcto en la placa de VIN del vehículo.
    3. ⇒Si el BCM tiene el número de VIN correcto, consultar Programación y configuración del módulo de control de la carrocería para la programación.
  3. Con la herramienta de diagnóstico llevar a cabo el procedimiento de configuración del número de pieza del SDM en el BCM. Consultar Referencias del módulo de control para la programación.
  4. Con el encendido conectado, emplear la herramienta de diagnóstico para borrar los DTC. A continuación, cortar el encendido y volver a conectarlo. Verificar que se borra el DTC B1019.
    5. ⇒Si se registra el DTC B1019, sustituir el SDM.

Instrucciones de reparación

Realice Verificación de la reparación por diagnóstico después de completar el procedimiento de diagnóstico.
