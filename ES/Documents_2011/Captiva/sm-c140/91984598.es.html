Descripción de DTC

Códigos de avería C0280: Sistema de estabilidad activo demasiado tiempo

Descripción del circuito/sistema

El módulo de control del freno electrónico (EBCM) detecta un fallo de funcionamiento interno.

Condiciones para la activación de DTC

El interruptor de encendido debe estar en la posición de activado.

Condiciones para ajustar el DTC
    • El módulo de control detecta un fallo de escritura interno.
    • El módulo de control detecta un fallo de suma de control interno.

Acción realizada cuando el DTC se ajusta

Pueden producirse una o varias de las siguientes acciones:

    • El sistema antibloqueo de frenos (ABS) y el sistema de dosificación dinámica trasera (DRP) se encuentran desactivados.
    • Los indicadores del ABS se encienden.
    • Se enciende el testigo rojo del sistema de frenos.
    • El LED del sistema de control de tracción (TCS) se apaga.
    • La luz de actividad del ESP parpadea durante al menos 15 segundos.

Condiciones para el borrar el DTC
    • Ya no existe la situación que activa el DTC y éste se borra con una herramienta de exploración.
    • El EBCM automáticamente borra el DTC de historial cuando no se detecta un DTC actual en 100 ciclos de conducción consecutivos.

Ayuda al diagnóstico

Este DTC puede almacenarse como un DTC de historial sin afectar al funcionamiento del módulo. Si se almacena únicamente como un DTC de historial y no se recupera como un DTC actual, no sustituya el módulo.

Si se recupera el DTC como un DTC actual y del historial, sustituya el módulo.

Verificación del circuito/sistema
  1. Si el DTC no se activó como un DTC actual, consulte las ayudas al diagnóstico.
  2. Si el DTC se activó como un DTC actual.
    3. ⇒Sustituya el EBCM.

Instrucciones de reparación

Realice Verificación de la reparación por diagnóstico después de completar el procedimiento de diagnóstico.
