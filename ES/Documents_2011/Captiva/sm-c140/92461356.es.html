Descripción de circuito

El sensor de la temperatura del refrigerante del motor (ECT) es un resistor variable que mide la temperatura del refrigerante del motor. El módulo de control del motor (ECM) suministra 5 voltios al circuito de señales de ECT y suministra una masa para el circuito de referencia baja. El ECM efectúa un seguimiento del recorrido del sensor ECT. Si el ECM detecta un ruido excesivo y unas fluctuaciones rápidas en la señal de ECT, se activa este DTC.

Condiciones para la activación de DTC
    • No se activa el DTC P0117 o P0118.
    • El encendido está en ON o el motor está funcionando.
    • El DTC P0119 se activa continuamente cuando se cumplen las anteriores condiciones.

Condiciones para ajustar el DTC
    • El ECM detecta que el ECT ha cambiado ±5 °C (±9 °F) más que el valor esperado durante el intervalo de tiempo calibrado.
    • La condición se produce durante aproximadamente 1 minuto.

Acción realizada cuando el DTC se ajusta

El DTC P0119 es un DTC de tipo B.

Condiciones para borrar el MIL/DTC

El DTC P0119 es un DTC de tipo B.

Ayuda al diagnóstico
    • Compruebe el sensor ECT en varios niveles de temperatura para evaluar la posibilidad de que el sensor sea asimétrico. Un sensor asimétrico puede provocar un DTC o afectar la conducción. Consultar Temperatura frente a resistencia - Sensor de temperatura del refrigerante del motor .
    • Si el vehículo ha reposado durante la noche, los valores del sensor IAT y del sensor ECT debe ser de 3 °C (5 °F).
    • Después de poner en marcha el motor en frío, la temperatura del sensor ECT debe subir lentamente, después estabilizarse cuando se abra el termostato.
    • Una resistencia alta en el circuito de señales del sensor ECT o en el circuito de referencia baja del sensor ECT puede activar un DTC.

Verificación del circuito/sistema
  1. Observe en la herramienta de diagnóstico el parámetro del sensor de temperatura del refrigerante del motor (ECT), mientras se mueve el conector del sensor ECT y el conector del ECM K20. El parámetro del sensor ECT no debe fluctuar cuando se mueven los relativos mazos de cables y conectores.
    2. ⇒Repare el mazo de cables o el conector sospechoso si el parámetro de ECT fluctúa más de 3 °C (5 °F) cuando se mueven los relativos mazos de cables y conectores.
  2. Revise el nivel del refrigerante del motor. Compruebe si el sistema de refrigeración funciona correctamente.
    3. ⇒Si cree que hay algún problema en el sistema de refrigeración del motor, consulte Pérdida de refrigerante .
  3. Maneje el vehículo en las condiciones de ejecución de los DTC para verificar que el DTC no reaparece. También puede conducir el vehículo en las condiciones que ha observado de los datos de Registros de fallos/cuadro congelado.

Comprobación del circuito/sistema
  1. Con el encendido desactivado, desconecte el conector del mazo de cables en el sensor B34 de ECT.
  2. Con el encendido desactivado, extraiga el fusible que alimenta B+ al terminal del circuito56 X2 del K20 ECM.

    3. Atención: NO utilice una lámpara de pruebas para comprobar la continuidad del circuito. Un consumo de corriente excesivo podría producir daños en el módulo de control.

    Nota: El módulo de control o el sensor pueden estar dañados si el circuito está cortocircuitado a una tensión positiva de la batería.

  3. Compruebe si hay menos de 5 Ω entre el terminal 2 del circuito referencia baja y la masa.
    4. ⇒Si es superior al intervalo especificado, compruebe si el circuito de referencia baja presenta una resistencia abierta/alta, o un cortocircuito a tensión. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  4. Coloque el fusible que alimenta B+ al terminal del circuito56 X2 del K20 ECM.
  5. Con el encendido conectado, compruebe que en la herramienta de diagnóstico el parámetro del sensor ECT es de -40 °C (-40 °F).
    6. ⇒Si la temperatura es superior al valor especificado, compruebe si el terminal 1 del circuito de señales tiene un cortocircuito a masa. Si la prueba del circuito sale bien, sustituya el K20 ECM.

    Nota: Si se abre el fusible en el cable con puente, el circuito de señales se cortocircuita a tensión y el sensor puede dañarse.

  6. Con el encendido conectado, monte un cable de puente con fusible de 3A entre el terminal 1 del circuito de señales y el terminal 2 del circuito de referencia baja. Compruebe que el parámetro del sensor ECT sea superior a 142 °C (288 °F).
    7. ⇒Si es inferior al valor especificado, compruebe si el circuito de señal tiene un cortocircuito de voltaje o una resistencia abierta/alta. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  7. Si todas las comprobaciones de los circuitos son normales, compruebe o sustituya el sensor B34 de ECT.

Comprobación de componentes
  1. Con el encendido desactivado, desconecte el conector del mazo de cables en el sensor B34 de temperatura del refrigerante del motor (ECT).

    2. Nota: Puede utilizarse un termómetro para comprobar el sensor fuera del vehículo.

  2. Compruebe el sensor ECT variando la temperatura del sensor mientras efectúa un seguimiento de la resistencia del sensor. Compare las lecturas con tabla Temperatura frente a resistencia - Sensor de temperatura del refrigerante del motor y verifique que la resistencia está dentro del 5 por ciento del intervalo especificado.
    3. ⇒Si no está en el intervalo especificado, sustituya el sensor B34 de ECT.
