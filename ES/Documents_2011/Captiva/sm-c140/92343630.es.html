Descripción del circuito/sistema

Los módulos conectados a los circuitos de datos en serie del bus CAN son módulos de datos en serie de alta velocidad que monitorizan las comunicaciones de datos en serie durante el funcionamiento normal del vehículo. Entre los módulos se intercambian comandos e información operativa. Los módulos disponen de información preregistrada acerca de los mensajes que es necesario intercambiar en los circuitos de datos de serie. Se supervisan los mensajes y, además, el módulo receptor utiliza algunos mensajes periódicos como indicación de la disponibilidad del módulo transmisor. Todos los mensajes contienen el número de identificación del módulo transmisor.

El módulo de control del freno electrónico (EBCM) activa los otros módulos conectados a los circuitos de datos en serie CAN aplicando tensión cuando la llave de encendido está en ACC, Activ. o ARRANQUE. El EBCM, el sensor de velocidad de derrape y el sensor de ángulo de dirección son los únicos módulos conectados a los circuitos de datos en serie del bus CAN. Para más información sobre datos en serie de bus CAN consulte Descripción y funcionamiento de las comunicaciones de enlace de datos .

Condiciones para la activación de DTC
    • La tensión del sistema está entre 9-16 V.
    • El módulo maestro de modo de potencia del vehículo requiere que se produzca la comunicación de datos serie.

Condiciones para ajustar el DTC

Existe un problema de no comunicación en los datos en serie de bus CAN para el EBCM.

Acción realizada cuando el DTC se ajusta
    • El sistema de mejora de la estabilidad del vehículo (VSES), si está disponible, y el sistema de control de tracción (TCS) no funcionarán.
    • Las lámparas de desconexión de sistema se encenderán para estos sistemas.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio sin que se repita una avería.

Ayuda al diagnóstico
    • Un circuito abierto en un circuito de bus CAN 1 no provocará un DTC U1500.
    • Si hay varios módulos que no se comunican escoja el que esté más alejado del EBCM.

Comprobación del circuito/sistema
  1. Con el encendido desactivado, desconecte el conector del mazo de cables del sensor de velocidad de derrape.
  2. Con el encendido conectado, intente con una herramienta de diagnóstico rescatar los datos del sensor de ángulo del volante en el módulo de control del freno electrónico/chasis. Debería haber datos disponibles.
    3. ⇒Si no hay datos disponibles, compruebe los circuitos de datos en serie por si hay un cortocircuito a masa o un cortocircuito a tensión. Si la comprobación de los circuitos da un resultado normal, sustituya el sensor de ángulo del volante.
  3. Cambie el sensor de velocidad de derrape.

Instrucciones de reparación

Realice Verificación de la reparación por diagnóstico después de completar el procedimiento de diagnóstico.

    •  Reparaciones de cableado GMLAN
    •  Sustitución del sensor de guiñada del vehículo
    • Referencias del módulo de control para sustituir, configurar y programar el módulo.
