Aclarado del sistema de refrigeración

Nota: Este procedimiento se ha diseñado únicamente para el sistema de refrigeración del motor. No utilice nunca este procedimiento para lavar el sistema de refrigeración híbrido.

Nota: No realice un aclarado químico.

Guarde el refrigerante usado de la forma adecuada; por ejemplo, en un depósito con refrigerante de un motor usado. No eche el refrigerante usado por un desagüe. El anticongelante de etilenoglicol es una sustancia química muy tóxica. No eche el refrigerante por una alcantarilla o en aguas subterráneas. Es ilegal y perjudicial para el medio ambiente.

Se pueden usar varios métodos y equipos para aclarar el sistema de refrigerante. Si se usa un equipo especial, como una baldeadora, observe las instrucciones del fabricante. Antes de lavar el sistema de refrigeración, extraiga siempre el termostato.

Si el sistema de refrigeración se ha contaminado, se deberá lavar a fondo para eliminar los contaminantes antes de que se produzca un daño serio al motor.

  1. Vacíe el sistema de refrigeración. Consultar Drenaje y llenado del sistema de refrigeración .
  2. Retire el depósito de expansión. Consultar Sustitución del vaso de expansión del radiador : Diesel → Motores de gasolina .
  3. Lave y enjuague el depósito de compensación con agua potable limpia.
  4. Monte el depósito de expansión. Consultar Sustitución del vaso de expansión del radiador : Diesel → Motores de gasolina .
  5. Siga el procedimiento de vaciado y llenado y utilice sólo agua potable limpia. Consultar Drenaje y llenado del sistema de refrigeración .
  6. Ponga el motor en marcha durante 20 minutos.
  7. Pare el motor.
  8. Vacíe el sistema de refrigeración. Consultar Drenaje y llenado del sistema de refrigeración
  9. Repita el procedimiento si es necesario, hasta que el líquido sea prácticamente incoloro.
  10. Llene el sistema de refrigeración. Consultar Drenaje y llenado del sistema de refrigeración .
