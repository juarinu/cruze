Descripción y funcionamiento del sistema antirrobo (CTD)

El sistema antirrobo de contenido (CTD) está diseñado para impedir que se robe el contenido del vehículo encendiendo las bocinas y las luces exteriores durante unos 30 segundos cuando se detecta una entrada no autorizada en el vehículo. No obstante, el sistema CTD no afecta al arranque del motor.

Con el sistema CTD activado, se puede considerar como acceso no autorizado cualquiera de los siguientes:

    • Cuando se abre el compartimento trasero sin emplear el sistema del mando a distancia
    • Cuando se abre un puerta sin desbloquearla empleando transmisor del sistema del mando a distancia

Los componentes del sistema son:

    • El módulo electrónico de la carrocería (BCM)
    • Los interruptores de puerta abierta
    • El interruptor de la tapa del maletero abierta

Activación del sistema CTD

Importante: El sistema antirrobo de contenido (DTC) se puede activar con las ventanillas o la capota abiertas.

Utilice el siguiente procedimiento para activar el sistema:

  1. Apague el encendido.
  2. Salir del vehículo.

    3. Importante: El sistema no se activa si las puertas se bloquean de forma manual. Se debe emplear el transmisor de entrada remota sin llave para activar el sistema CTD.

  3. Cierre las puertas pulsando el botón de bloqueo (LOCK) del transmisor. El sistema está en modo pendiente de ser activado y no pondrá en marcha el temporizador de activación hasta que se cierren todas las puertas. El indicador de seguridad parpardeará rápidamente. En este momento el sistema CTD está en estado abierto. La transición del sistema CTD al estado cerrado tiene lugar cuando se cierra la última puerta.

    4. El indicador de seguridad se ilumina de forma permanente cuando el sistema se encuentra en estado bloqueado. Si se pulsa el botón LOCK (bloquear) después de haber cerrado las puertas del vehículo, el estado de apertura se elude y el sistema pasará directamente al estado bloqueado con el indicador iluminado de forma permanente.

  4. El sistema se activará aproximadamente 30 segundos a partir del momento en que el sistema pase a estado bloqueado. Cuando el testigo parpadee lentamente significa que el sistema está activado. Además, si se pulsa por segunda vez el botón LOCK (bloquear) del transmisor de entrada remota sin llave, el sistema se activará inmediatamente, ignorando el retardo de 30 segundos.

Bloqueo del vehículo sin activar el sistema CTD

El bloqueo del vehículo se puede efectuar sin activar el sistema CTD bloqueando el vehículo mediante las cerraduras de las puertas de forma manual.

Desactivación de un sistema activado/apagado de la alarma

Si se ha usado el transmisor del mando a distancia de las puertas para solicitar la activación del sistema, éste deberá desactivarse.

Importante: El desbloqueo manual de las puertas no inhabilita el modo de activación del sistema antirrobo de contenido (CTD). Además, el desconectar la batería o quitar fusibles no deshabilita el modo de activación, ya que el módulo electrónico de la carrocería (BCM) guarda el estado CTD en la memoria.

Para desactivar el sistema DTC, llevar a cabo lo siguiente:

  1. Pulse el botón UNLOCK (Desbloqueo) del transmisor de entrada sin llave.
  2. Introducir una llave válida en el interruptor del encendido y arrancar el vehículo.

Descripción del circuito CTD

A continuación figura una descripción de cada uno de los componentes utilizados en el sistema antirrobo de contenido (CTD):

Módulo electrónico de la carrocería (BCM)

El sistema CTD es una función interna del módulo electrónico de la carrocería (BCM) que emplea datos en serie y señales de entrada de diferentes interruptores para efectuar las funciones CTD. Cuando el BCM detecta una entrada no autorizada, activa las bocinas y las luces exteriores. Cuando el BCM detecta una entrada no autorizada, el BCM pasará primero a modo disuasorio. El modo disuasorio hacer sonar pitidos breves del claxon. Si después de 10 segundos se produce una entrada no autorizada, el BCM pasa al modo alarma. La bocina sonará y los faros se iluminarán, de forma intermitente, durante 30 segundos. Después del período de alarma de 30 segundos, se produce una pausa de 10 segundos. A continuación, el sistema se volverá a activar.

Interruptores de puerta abierta

El sistema CTD emplea interruptores de puerta abierta en las jambas de las puertas como un método para activar la alarma.

El BCM supervisa los interruptores de puerta abierta a través de una señal de entrada discreta de cada interruptor de puerta abierta. Si el BCM recibe una señal de masa de un interruptor de puerta abierta cuando el sistema CTD está activado, el BCM activa la alarma.

Señales de entrada

El BCM supervisa las siguientes señales de entrada:

    • Los interruptores de puerta abierta del conductor y acompañante
    • El interruptor de la tapa del maletero abierta
    • Los botones del transmisor LOCK/UNLOCK (bloquear/desbloquear); una función interna del BCM
    • El estado de llave maestra. El BCM utiliza el estado de llave maestra para desactivar el sistema o silenciar la alarma cuando se utiliza la llave correcta para arrancar el vehículo.

Salidas

El BCM controla lo siguiente:

    • El relé de la bocina
    • El alumbrado exterior
