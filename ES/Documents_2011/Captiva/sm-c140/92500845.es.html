Descripción de DTC

Códigos de avería P0219 00: Velocidad excesiva del motor

Descripción del circuito/sistema

Nota: Este DTC sólo tiene carácter informativo. Este DTC NO indica que exista un problema en el sensor del cigüeñal o en el módulo de control del motor (ECM).

El DTC se utiliza para indicar que el régimen de revoluciones del motor ha superado el límite de corte de combustible en marcha. Este DTC puede aparecer si el cambio se sitúa en una marcha baja mientras se conduce por una inclinación considerable o se viaja a gran velocidad.

Condiciones para la activación de DTC
    • El motor está funcionando.
    • El DTC se ejecuta continuamente cuando se cumple la condición anterior.

Condiciones para ajustar el DTC

El ECM detecta que el régimen de revoluciones del motor es superior al umbral durante más de 0,5 segundos.

Acción realizada cuando el DTC se ajusta

El DTC P0219 00 es un DTC de tipo C.

Condiciones para el borrar el DTC

El DTC P0219 00 es un DTC de tipo C.
