Parámetros de los datos de la herramienta de diagnóstico del módulo de control del freno de estacionamiento

Parámetro

Valor previsto

Descripción

Condiciones de servicio: Con la llave de encendido en posición ON, quite el freno de mano

Hardware base

Varía

La herramienta de diagnóstico muestra el número de hardware base.

Señal del pedal de freno BCM

Válido

La herramienta de diagnóstico indica Válido o No válido. Este parámetro muestra el estado válido o no válido de la señal recibida por el pedal del freno del módulo de control de la carrocería. Si la señal recibida no es válida, el módulo de control del freno de estacionamiento se desactivará.

Temperatura del sistema calculada

Varía

La herramienta de diagnóstico muestra en grados centígrados la temperatura del módulo de control del freno de estacionamiento.

Calibraciones

Varía

La herramienta de diagnóstico muestra el número de calibración.

Señal de habilitación del freno del estacionamiento de emergencia EBCM

Válido

La herramienta de diagnóstico indica Válido o No válido. Este parámetro muestra si el estado de la señal de activación del freno de mano de emergencia recibida por el módulo de control del freno electrónico es válido o no válido. Si la señal recibida no es válida, el módulo de control del freno de estacionamiento se desactivará.

Señal de posición del pedal del acelerador ECM

Válido

La herramienta de diagnóstico indica Válido o No válido. Este parámetro muestra si el estado de la señal de posición del pedal del acelerador recibida por el módulo de control del motor es válido o no válido. Si la señal recibida no es válida, el módulo de control del freno de estacionamiento se desactivará.

Señal de posición del pedal del embrague ECM

Válido

La herramienta de diagnóstico indica Válido o No válido. Este parámetro muestra si el estado de la señal de posición del pedal del embrague recibida por el módulo de control del motor es válido o no válido. Si la señal recibida no es válida, el módulo de control del freno de estacionamiento se desactivará.

Señal de velocidad del vehículo ECM

Válido

La herramienta de diagnóstico indica Válido o No válido. Este parámetro muestra si el estado de la señal de velocidad del vehículo recibida por el módulo de control del motor es válido o no válido. Si la señal recibida no es válida, el módulo de control del freno de estacionamiento se desactivará.

Modelo final

Varía

La herramienta de diagnóstico muestra el número de modelo final.

SO/software

Varía

La herramienta de diagnóstico muestra el número de software.

Estado de calibración del freno de estacionamiento

Aprobado

La herramienta de diagnóstico indica Aprobado o Fallido. Este parámetro indica si la calibración de freno de estacionamiento se programó correctamente en el módulo de control del freno de estacionamiento.

Posición del cable del freno de estacionamiento

1

La herramienta de diagnóstico muestra 1, 2 o 3. Este parámetro muestra el valor de ciclo del sistema. Cuando el freno de estacionamiento no esté aplicado, se mostrará el valor 1. Cuando el freno de estacionamiento esté aplicado, se mostrará el valor 3. Cuando el freno de estacionamiento se esté aplicando o soltando, aparecerá el valor 2.

Comando del motor del freno de estacionamiento

Desactivado

La herramienta de diagnóstico indica On (Activado) u Off (Desac.). Este parámetro indica el estado del motor de freno de estacionamiento.

Dirección del motor del freno de estacionamiento

Inactivo

La herramienta de diagnóstico indica Accionado, Soltado o Inactivo. Este parámetro indica el estado de la dirección del motor del freno de estacionamiento.

Comando ciclo de trabajo del motor del freno de estacionamiento

Varía

La herramienta de diagnóstico muestra el ciclo modulado por ancho de pulso como un porcentaje.

Motivo de cancelación del freno de estacionamiento

Ninguno

La herramienta de diagnóstico indica Ninguno, No conocido Cíclico, Velocidad Umbral, o Freno aplicado. Este parámetro indica el motivo por el que se desbloqueó el freno de mano.

Estado del freno de estacionamiento

Soltado

La herramienta de diagnóstico indica Accionado, Soltado, Mal funcionamiento o No válido. Este parámetro indica el estado actual del freno de mano.

Interruptor del freno de estacionamiento

Inactivo

La herramienta de diagnóstico indica Accionado, Soltado, Inactivo o No válido. Este parámetro indica el estado actual del interruptor del freno de mano.

Señal del nivel de marcha actual TCM

Válido

La herramienta de diagnóstico indica Válido o No válido. Este parámetro muestra si el estado de la señal de marcha seleccionada recibida por el módulo de control del cambio es válido o no válido. Si la señal recibida no es válida, el módulo de control del freno de estacionamiento se desactivará.

VIN

Varía

La herramienta de diagnóstico muestra el número de identificación del vehículo.

Controles de salida de la herramienta de diagnóstico del módulo de control del freno de estacionamiento

Control de salida

Descripción

Reinicio del servicio de la zapata del freno de mano

El módulo de freno de mano eléctrico reiniciará el contador de freno dinámico. Cuando se seleccione, el contador EPB se establecerá en cero.

Calibración del freno de estacionamiento

El módulo de freno de mano eléctrico activará el procedimiento de programación de la calibración del freno de estacionamiento cuando se seleccione. Durante el procedimiento de programación, los frenos traseros girarán varias veces.

Aplicación del cable del freno de estacionamiento

El módulo de freno de mano eléctrico accionará la tensión del cable del freno de mano para diagnosticar un posible fallo en el interruptor EPB.

Desbloqueo del cable del freno de estacionamiento

El módulo de freno de mano eléctrico liberará el cable del freno de mano para diagnosticar un posible fallo en el interruptor EPB.

Sustitución del cable del freno de estacionamiento

El módulo de freno de mano eléctrico anulará el cable de freno de mano a sustituir o revisar cuando se seleccione.

Reinicio de DTC del actuador del freno de estacionamiento

El módulo de freno de mano eléctrico reiniciará el actuador del freno de estacionamiento cuando así se solicite.

Procedimiento del interruptor del freno de mano

El módulo de freno de mano eléctrico comprobará las señales del interruptor del freno de mano para verificar si funcionan correctamente o si existe un circuito abierto.
