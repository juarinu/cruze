Descripción del sistema de encendido electrónico

Funcionamiento del sistema de encendido electrónico (EE)

El sistema de encendido electrónico (EI) produce y controla un encendido secundario de alta energía. Esta chispa enciende la mezcla comprimida de aire/combustible precisamente en el momento correcto, proporcionando rendimiento óptimo, economía de combustible y control de las emisiones del escape. El módulo de control del motor (ECM) recoge principalmente información de los sensores de posición del cigüeñal y del árbol de levas, para controlar la secuencia, ángulo de contacto dwell y sincronización de la chispa.

Sensor de posición del cigüeñal

Los circuitos del sensor de posición del cigüeñal contienen un circuito de referencia de 5 V alimentado por el módulo de control del motor (ECM), un circuito de referencia baja y un circuito de señal de salida. El sensor del cigüeñal es un dispositivo sensor a circuito integrado, de salida digital, con polarización magnética interna. El sensor detecta el cambio del flujo magnético de los dientes y ranuras de una rueda reluctora de 58 dientes del cigüeñal. Cada diente en la rueda reluctora se encuentra a una separación equivalente para 60 dientes, con 2 dientes que faltan en una separación de referencia. El sensor del cigüeñal genera una tensión de CC ON/OFF de frecuencia variable, con 58 pulsos de salida por revolución del cigüeñal. La frecuencia de la salida del sensor de posición del cigüeñal depende de la velocidad del cigüeñal. El sensor del cigüeñal envía una señal digital que representa una imagen de la rueda reluctora del cigüeñal al ECM a medida que cada diente de la rueda gira frente al sensor de posición del cigüeñal. El ECM utiliza cada pulso de señal de posición del cigüeñal para determinar la velocidad del cigüeñal y decodifica la separación de referencia de la rueda reluctora del cigüeñal para identificar la posición del cigüeñal. Esta información se utiliza para determinar los puntos óptimos de inyección y encendido del motor. El ECM también utiliza la información de la salida del sensor de posición del cigüeñal para determinar la posición relativa del árbol de levas con el cigüeñal y detectar fallos de encendido en los cilindros.

Rueda reluctora del cigüeñal

La rueda reluctora del cigüeñal es parte de éste. La rueda reluctora consta de 58 dientes y una separación de referencia. Cada diente en la rueda reluctora está separado 6 grados y con un espacio de 12 grados en la separación de referencia. Al pulso de la separación de referencia se le conoce como pulso de sincronismo. El pulso de sincronismo se usa para sincronizar la secuencia del encendido de las bobinas con la posición del cigüeñal, mientras los otros dientes proporcionan la localización en el cilindro durante una revolución.

Sensor de posición del árbol de levas

El sensor de posición del árbol de levas se activa mediante rueda reluctora con muescas, construida sobre el piñón del árbol de levas de admisión. El sensor de posición del árbol de levas proporciona cuatro pulsos de señal en cada revolución del árbol de levas. Cada muesca o rasgo de la rueda reluctora tiene una medida diferente, lo cual se usa para identificar la carrera de compresión en cada cilindro y para habilitar la inyección secuencial de combustible. El sensor de posición del árbol de levas está conectado al módulo de control del motor (ECM) por los siguientes circuitos:

    • Referencia de 5 V
    • Referencia baja
    • Señal

Sensor de picado

El sistema del sensor de picado permite que el módulo de control controle la sincronización del encendido para obtener el máximo recorrido, protegiendo al mismo tiempo el motor de unos niveles de detonación, también llamada picado de chispa, que podrían dañarlo. El sistema de sensor de picado utiliza uno de 2 sensores del cable doble de respuesta plana. El sensor utiliza tecnología de cristal piezoeléctrico que produce una señal de tensión CA de amplitud y frecuencia variables de acuerdo con la vibración del motor o el nivel de ruido. La amplitud y la frecuencia dependen del nivel de picado detectado por el sensor de picado. El módulo de control recibe la señal del sensor de picado a través del circuito de señal. El módulo de control suministra la masa del sensor de picado a través del circuito de referencia baja.

El módulo de control aprende un nivel de ruido mínimo, o ruido de fondo, al ralentí desde el sensor de picado y utiliza valores calibrados para los demás intervalos de r.p.m. El módulo de control utiliza el nivel de ruido mínimo para calcular un canal de ruido Una señal del sensor de picado normal irá por dentro del canal de ruido. Como la carga y el régimen del motor cambian, los parámetros superiores e inferiores del canal de ruido variarán para ajustarse a la señal del sensor de picado normal, manteniendo la señal dentro del canal. Para determinar qué cilindros golpetean, el módulo de control sólo utiliza la información de señal del sensor de picado cuando todos los cilindros están cerca del punto muerto superior (TDC) de la carrera de encendido. Si se produce un picado, la señal se saldrá del canal de ruido.

Si el módulo de control determina que se está produciendo un picado, retrasará la sincronización del encendido para intentar eliminar el picado. El módulo de control siempre intenta volver a un nivel de compensación cero o sin retardo de encendido. Una señal del sensor de picado anormal se mantendrá fuera del canal de ruido o no se producirá. Los diagnósticos del sensor de picado están calibrados para detectar fallos en los circuitos del sensor de picado dentro del módulo de control, el cableado del sensor de picado o la salida de tensión del sensor de picado. Algunos diagnósticos también se calibran para detectar un ruido constante proveniente del exterior, como un componente que se ha aflojado o roto o un ruido mecánico excesivo del motor.

Bobina /módulo de encendido

Cada bobina/módulo de encendido tiene los siguientes circuitos:

    • Tensión de encendido
    • Toma de tierra
    • Control de encendido (IC)
    • Referencia baja

El módulo de control del motor (ECM) controla las bobinas individuales transmitiendo impulsos de sincronización en el circuito IC de cada bobina / módulo de encendido para activar una chispa.

Las bujías están conectadas a cada bobina por un fuelle corto. El fuelle contiene un muelle, que conduce la energía para la chispa desde la bobina hasta la bujía. El electrodo de la bujía está adicionado con platino para una larga vida y mayor eficiencia.

Módulo de control del motor (ECM)

El módulo de control del motor (ECM) controla todas las funciones del sistema de encendido, y corrige constantemente la sincronización de la chispa de la bujía. El ECM efectúa un seguimiento de la información de varias entradas de sensores que incluyen los siguientes elementos:

    • El sensor de posición del acelerador (TP)
    • El sensor de temperatura del refrigerante del motor (ECT).
    • El sensor del flujo y masa de aire (MAF)
    • El sensor de la temperatura de aire de entrada (IAT)
    • El sensor de velocidad del vehículo (VSS)
    • Sensores de picado del motor
    • El sensor de presión absoluta de colector (MAP)

Modos de funcionamiento

Durante el funcionamiento normal, el módulo de control del motor (ECM) controla todas las funciones de encendido. Si se pierde la señal del sensor de posición del cigüeñal o del sensor de posición del árbol de levas, el motor continuará funcionando porque el ECM entrará por defecto a un modo de operación de emergencia, usando la entrada del sensor que quede. Cada bobina está protegida internamente contra daños por tensión excesiva. Si una o más bobinas fallara de esta manera, el resultado sería un fallo de encendido. Los códigos de diagnóstico de averías están disponibles para diagnosticar acertadamente el sistema de encendido con la herramienta de diagnóstico.
