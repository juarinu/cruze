Descripción del circuito/sistema

Los módulos conectados a los circuitos de datos en serie GMLAN de alta velocidad controlan comunicaciones de datos serie durante el funcionamiento normal del vehículo. Entre los módulos se intercambian comandos e información operativa. Los módulos disponen de información almacenada previamente sobre los mensajes que es necesario intercambiar en los circuitos de datos serie para cada una de las redes virtuales. Se supervisan los mensajes y, además, el módulo receptor utiliza algunos mensajes periódicos como indicación de la disponibilidad del módulo transmisor. Todos los mensajes contienen el número de identificación del módulo transmisor.

Condiciones para la activación de DTC
    • La tensión del sistema está entre 9-16 V.
    • El módulo maestro de modo de potencia del vehículo requiere que se produzca la comunicación de datos serie.

Condiciones para ajustar el DTC

No se ha recibido un mensaje supervisado periódicamente que incluye la disponibilidad del módulo transmisor.

Acción realizada cuando el DTC se ajusta

No existe transmisión de señales con ellos módulos, por lo que el los subsistemas específicos no funcionarán.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio de 50 sin que se repita una avería.

Ayuda al diagnóstico
    • Utilice la lista de Descripciones de DTC anterior para determinar el módulo que no comunica.
    • Si hay varios módulos que no comunican, elija el más cercano al conector de enlace de datos (DLC).
    • Use Referencias de enlace de datos para determinar si el módulo usa comunicaciones de datos de serie GMLAN de alta o baja velocidad.
    • Algunos módulos pueden no tener protección interna para salidas de tensión específicas y pueden abrir un fusible de fuente de tensión de encendido o tensión positiva de batería. Si está abierto un fusible de entrada de tensión y no se encuentra ningún cortocircuito en ese circuito, antes de sustituir el módulo asegúrese de que ningún circuito de tensión de salida del módulo presenta un cortocircuito a masa.
    • Este diagnóstico puede utilizarse para cualquier módulo que deba comunicar con datos de serie GMLAN de alta o baja velocidad siempre que el vehículo esté equipado con la opción que utiliza ese módulo.

Comprobación del circuito/sistema
  1. Mediante la descripción anterior de DTC y la ayuda de diagnóstico, determine el módulo de control que no está comunicando y que debería comprobarse.
  2. Con el encendido OFF, desenchufe el conector del mazo de cables en el módulo de control que no se está comunicando.
  3. Pruebe con menos de 10 Ω entre cada terminal del circuito de masa y masa.
    4. ⇒Si es superior al intervalo especificado, compruebe si el circuito de masa tiene una resistencia abierta/alta.
  4. Si está disponible, con el encendido conectado, verifique que se ilumina una lámpara de prueba entre cada terminal del circuito de B+ y masa.
    5. ⇒Si la luz de prueba no se ilumina, pruebe el circuito B+ para ver si hay un cortocircuito a tierra o una resistencia abierta/alta. Si el fusible del circuito está abierto, compruebe también si las salidas de voltaje positivo del módulo de control están cortocircuitados a tierra. Si la prueba de circuitos sale bien, sustituya el módulo de control.
  5. Si está equipado, encendido ON, verifique que la luz de prueba entre cada terminal del circuito de encendido y toma de tierra.
    6. ⇒Si la luz de prueba no se ilumina, compruebe si en el circuito de encendido hay un cortocircuito a masa o una resistencia abierta/alta. Si el fusible del circuito está abierto, compruebe también si las salidas de voltaje positivo del módulo de control están cortocircuitados a tierra. Si la prueba de circuitos sale bien, sustituya el módulo de control.

    Nota: Sólo los módulos GMLAN de alta velocidad tienen un circuito de habilitación de comunicación de datos de serie o un circuito de datos de serie de activación accesorio, y el BCM es la salida de estos circuitos. Consulte los esquemas de módulos de control para identificar qué módulos tienen estos circuitos.

  6. Si está equipado, con el encendido activado, verifique que la luz de prueba se ilumina entre el terminal del circuito que activa la comunicación o el terminal del circuito de activación del accesorio y a masa.
    7. ⇒Si la luz de prueba no se enciende, compruebe todo el circuito, incluido el circuito B+ al BCM, por si presenta una resistencia abierta/alta o un cortocircuito a masa. Si la prueba de circuitos sale bien, sustituya el BCM.
  7. Con el encendido OFF durante 60 segundos, pruebe con menos de 5 Ω entre los terminales del circuito de datos en serie y el terminal DLC adecuado indicado a continuación:
    8. • Terminal 2 del circuito de datos de serie de clase 2
    • Terminal 1 de datos de serie GMLAN de baja velocidad
    • Terminal 6 ó 14 de datos de serie GMLAN de alta velocidad
    ⇒Si es superior al intervalo especificado, compruebe si el circuito de datos de serie tiene una resistencia abierta/alta entre el módulo de control que no se comunica y un módulo que establezca el DTC o un paquete de empalmes de datos de serie.
  8. Si todos los circuitos están bien, sustituya el módulo de control que no se está comunicando.
