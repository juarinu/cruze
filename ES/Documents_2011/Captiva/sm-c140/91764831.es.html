Descripción del circuito/sistema

El módulo de control de la carrocería (BCM) supervisa el circuito de control del relé de alta velocidad del limpiaparabrisas únicamente cuando el modo de alta velocidad del limpiaparabrisas está activo. Cuando se solicita el modo de alta velocidad del limpiaparabrisas, el BCM responde aplicando una masa, a través del circuito de control del relé de alta velocidad del limpiaparabrisas, al lado de la bobina del relé, activando así el relé de velocidad del limpiaparabrisas.

Condiciones para la activación de DTC

El DTC sólo puede activarse si el BCM solicita activamente la salida.

Condiciones para ajustar el DTC

Este DTC se activa si el BCM detecta un cortocircuito a tensión de batería en el circuito de control del relé de alta velocidad del limpiaparabrisas.

Acción realizada cuando el DTC se ajusta

El BCM no activará la salida.

Condiciones para el borrar el DTC
    • El DTC actual pasará al historial cuando se elimine la solicitud de la salida o cuando se haya corregido la condición que activa la avería.
    • Se borrará el historial de DTC una vez que se hayan producido 50 ciclos de encendido consecutivos sin fallos.

Comprobación del circuito/sistema
  1. Con el encendido desconectado, desemborne el enchufe del mazo de cables J2 del bloque de fusibles del motor.
  2. Conecte una lámpara de prueba entre el terminal 21 del circuito de control y la tensión de batería.
  3. Ordene el encendido y apagado del relé de alta velocidad del limpiaparabrisas con una herramienta de exploración. La lámpara de pruebas deberá encenderse y apagarse, cuando se cambie entre los dos estados comandados.
    4. ⇒Si la luz de comprobación está siempre encendida, compruebe si el circuito de control tiene un cortocircuito a masa. Si según la comprobación el circuito está bien, sustituya el módulo de control de la carrocería.
    ⇒Si la luz de prueba está siempre apagada, pruebe el circuito de control para ver si hay un cortocircuito a masa o un circuito abierto/resistencia alta. Si según la comprobación el circuito está bien, sustituya el módulo de control de la carrocería.
  4. Si según la comprobación todos los circuitos están bien, compruebe o sustituya el bloque de fusibles del motor.
