Descripción de DTC

Códigos de avería C0561 71: Información almacenada sobre la desactivación del sistema

Descripción del circuito/sistema

El módulo de control del freno electrónico (EBCM) desactiva el ABS cuando otros módulos de control electrónico activan DTCs para componentes que afectan al funcionamiento del sistema de control de tracción.

Condiciones para la activación de DTC
    • El interruptor de encendido debe estar en la posición de activado.
    • La tensión de encendido sea superior a 8 voltios.

Condiciones para ajustar el DTC

El módulo de control del tren motriz (PCM) o el módulo de control de la carrocería (BCM) diagnostica una situación que evita el funcionamiento de la sección de control del motor de la función de control de tracción y envía un mensaje de datos en serie al EBCM indicando que no se puede llevar a cabo la reducción del par motor. El PCM o el BCM normalmente indicarán un DTC y el EBCM activará dicho DTC.

Acción realizada cuando el DTC se ajusta

Pueden producirse una o varias de las siguientes acciones:

    • El EBCM desactiva el sistema de control de tracción hasta que el DTC pasa a formar parte del historial.
    • El indicador de desactivación del control de tracción se enciende.
    • El sistema antibloqueo de frenos (ABS) permanece operativo.

Condiciones para el borrar el DTC
    • La anomalía que dio lugar al código de avería (DTC) ya no está presente.
    • El EBCM automáticamente borra el DTC de historial cuando no se detecta un DTC actual en 100 ciclos de conducción consecutivos.

Verificación del circuito/sistema

Este DTC sólo tiene carácter informativo. Como ayuda para el técnico, este DTC indica que no existen problemas en el sistema ABS/TCS.
