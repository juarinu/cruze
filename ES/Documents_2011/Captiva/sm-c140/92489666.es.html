Descripción del circuito/sistema

Aplicaciones de combustible E85

Para combustible sin plomo normal, el ECM controla una señal constante. Cuando el vehículo tiene etanol de aproximadamente el 20% o superior para aplicaciones E85, el ECM controla una señal de ciclo de impulso donde controla la señal y calcula el porcentaje de nivel de combustible sólo durante el impulso de aproximadamente 100 ms.

Aplicaciones de depósito de combustible sencillo

Para una aplicación de depósito sencillo, el sensor de nivel de combustible cambia la resistencia en respuesta al nivel de combustible. El módulo de control del motor (ECM) monitoriza el circuito de señal del sensor del nivel de combustible y el tipo de combustible para determinar el nivel de combustible.

Cuando el depósito de combustible está lleno, la resistencia del sensor es baja y el ECM detecta una tensión de señal baja. Cuando el depósito de combustible está vacío, la resistencia del sensor es alta y el ECM detecta una tensión de señal alta. El ECM utiliza el circuito de señal del sensor de nivel de combustible para calcular el porcentaje del combustible restante en el depósito. El ECM envía el porcentaje de nivel de combustible mediante el circuito de datos de serie al cuadro de instrumentos para controlar el indicador de combustible.

Aplicaciones de depósito de combustible doble

El sensor del nivel de combustible primario y el sensor del nivel de combustible secundario cambian la resistencia según el nivel de combustible. El módulo de control del motor (ECM) monitoriza los circuitos de señal del sensor de combustible primario y el sensor de combustible secundario para determinar el nivel de combustible. Cuando el depósito de combustible está lleno, las resistencias de ambos sensores de nivel de combustible están bajas y el ECM detecta una tensión de señal baja en ambos circuitos de señal del sensor de nivel de combustible primario y el sensor de combustible secundario. Cuando el depósito del combustible está vacío, las resistencias de los sensores de nivel de combustible están altas y el ECM detecta una tensión de señal alta. El ECM utiliza los circuitos de señal del sensor de nivel de combustible primario y el sensor de nivel de combustible secundario para calcular el porcentaje del combustible restante en el depósito. El ECM envía el porcentaje de nivel de combustible mediante el circuito de datos de serie al cuadro de instrumentos para controlar el indicador de combustible.

Condiciones para el funcionamiento de DTC
    • El motor está funcionando
    • La tensión del sistema está entre 11-16 V

Condiciones para ajustar el DTC

P0461 o P2066

    • El ECM detecta una diferencia entre el combustible consumido por el motor y cambia la señal de nivel de combustible de menos de 1 L (0,26 gal) sobre un rango de 30 L (7,92 gal) o 45,6% del depósito de combustible primario.
    • La condición anterior está presente durante unos 300 segundos.

P0462 o P2067

    •  La tensión de señal es menor de 0,25 V.
    • Las condiciones anteriores se deben cumplir durante 5 segundos.

P0463 o P2068

    •  La tensión de señal es mayor de 4,7 V.
    • Las condiciones anteriores se deben cumplir durante 5 segundos.

P0464

    • El cambio en el nivel de combustible es superior al 10%.
    • Las condiciones anteriores se deben cumplir durante 30 segundos.
    • El DTC P0464 se activa y falla 2 de 3 ciclos de prueba.

Acción realizada cuando se registra el DTC
    • P0461, P0462, P0463, P0464, P2066, P2067 y P2068 son DTC de tipo B
    • La galga de combustible indica vacío por defecto
    • El indicador de combustible bajo se ilumina

Condiciones para el borrar el DTC
    • P0461, P0462, P0463, P0464, P2066, P2067 y P2068 son DTC de tipo B
    • Un DTC pasa al historial cuando las condiciones de activación del DTC dejan de existir.
    • El historial de DTC se borra después de 40 ciclos de calentamiento sin un fallo de funcionamiento.

Verificación del circuito/sistema
  1. Con el encendido conectado, observe los parámetros "Sensor del nivel de combustible" del ECM en la herramienta de diagnóstico. La lectura debe situarse entre 0,5-3,5 V y variar con el nivel de combustible.
  2. Encendido conectado, realizar la comprobación Barrido de medidores del cuadro de instrumentos en la herramienta de diagnóstico, comprobar que la galga de nivel de combustible barre desde su posición baja a su posición alta
    3. ⇒Si el indicador de nivel de combustible no se mueve, sustituya el cuadro de instrumentos P16.
  3. Comandar que todos los indicadores se comprueben en Activado y Desac. con la herramienta de diagnóstico, verificar que el indicador de nivel de combustible bajo se coloque en Activado y en Desac.
    4. ⇒Si el indicador de nivel de combustible bajo siempre permanece activado o desactivado, sustituya el cuadro de instrumentos P16.

Comprobación del circuito/sistema
  1. Con el encendido en OFF, desenchufe el conector del mazo de cables del sensor del nivel de combustible que corresponda.
  2. Pruebe con menos de 10 Ω en el terminal 3 del circuito referencia baja.
    3. ⇒Si es mayor que el intervalo especificado, compruebe si el circuito de baja referencia tiene una resistencia abierta/alta. Si todas las pruebas de circuitos resultan bien, sustituya el K20 módulo de control del motor.
  3. Con el encendido en ON, compruebe si hay 4,8-5,2 V entre el terminal 4 del circuito de señal y masa.
    4. ⇒Si es superior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a voltaje. Si todas las pruebas de circuitos resultan bien, sustituya el K20 módulo de control del motor.
    ⇒Si el valor es inferior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a masa o un circuito abierto/resistencia alta. Si todas las pruebas de circuitos resultan bien, sustituya el K20 módulo de control del motor.
  4. Si todas las comprobaciones de circuitos son correctas, sustituya el sensor de nivel de combustible correspondiente.
