DTC P0616

Descripción de circuito

El módulo de control del motor (ECM) suministra una vía de masa para el relé del motor de arranque cuando se solicita la habilitación del arranque. El ECM monitoriza el circuito para si existen condiciones que sean incorrectas para el estado ordenado. Si el ECM detecta una condición incorrecta del sistema, el DTC P0616 del relé del motor de arranque se activará.

Descripción de DTC

Este procedimiento de diagnóstico soporta el siguiente DTC:

DTC P0616 Tensión baja en el circuito de control del relé del motor de arranque

Condiciones para el funcionamiento de DTC
    • El encendido está en la posición ON.
    • La tensión del sistema está entre 8-16 voltios.
    • El motor está funcionando.

Condiciones para ajustar el DTC
    • El ECM detecta un nivel de tensión incorrecto en el circuito de control que controla el relé del motor de arranque.
    • El problema existe durante al menos 2 segundos.

Acción realizada cuando se registra el DTC
    • Ahora el ECM no encenderá la luz indicadora de avería (MIL).
    • El ECM almacenará los problemas existentes cuando se activó el DTC únicamente como datos de registros de fallos.
    • Se guarda un DTC de historial.

Condiciones para borrar el MIL/DTC
    • El historial del DTC se eliminará tras 40 ciclos de calentamiento consecutivos sin que se produzca una avería.
    • El DTC se puede borrar usando la función Borrar información de DTC en la herramienta de diagnóstico.
