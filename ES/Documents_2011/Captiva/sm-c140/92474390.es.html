El fusible es el procedimiento más común de protección de un circuito de cableado de automóvil. Si existe una cantidad excesiva de corriente a través de un circuito, el elemento de fusible se fundirá y generará un circuito abierto o incompleto. Los fusibles son dispositivos de protección de un solo uso y deben sustituirse cada vez que se sobrecarga el circuito. Para determinar si un fusible está abierto, extraiga el fusible sobre el que se tienen dudas y examine si el elemento del fusible está roto (2). Si no está roto (1), compruebe también la continuidad mediante un multímetro digital (DMM) o un aparato verificador de continuidad. Si el elemento está roto o la continuidad es dudosa, sustituya el fusible por uno de igual corriente nominal.
