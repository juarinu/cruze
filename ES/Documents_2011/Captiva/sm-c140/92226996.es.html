Descripción de circuito

El sensor de presión absoluta del colector (MAP) tiene un circuito de referencia de 5 V, un circuito de referencia baja y un circuito de señal. El módulo de control del motor (ECM) alimenta con 5 V el sensor MAP en el circuito de referencia de 5 V, y proporciona una masa al circuito de baja referencia. El sensor MAP proporciona una señal de tensión al ECM en el circuito de señal relacionada con los cambios de presión en el colector de admisión.

Condiciones para la activación de DTC
    • El DTC P0697 no está activado.
    • El motor está funcionando.
    • Este DTC se ejecuta continuamente cuando se producen las condiciones de habilitación.

Condiciones para ajustar el DTC

P0107

El ECM detecta que la tensión del sensor MAP es inferior a 0,20 V durante más de 5 s.

P0108

El ECM detecta que la tensión del sensor MAP es superior a 4,80 V durante más de 5 s.

Acción realizada cuando el DTC se ajusta

Los DTC P0107 y P0108 son DTC de tipo B.

Condiciones para borrar el MIL/DTC

Los DTC P0107 y P0108 son DTC de tipo B.

Verificación del circuito/sistema
  1. Compruebe que no se activan los DTC P0641, P0651 ó P0697.
    2. ⇒Si se activa alguno de estos DTC, consulte DTC P0641, P0651, P0697, o P06A3 .
  2. Con el encendido conectado, observe el parámetro del sensor MAP en la herramienta de diagnóstico. La lectura debería estar entre 0,20 y 4,80 V.
  3. Maneje el vehículo en las condiciones de ejecución de los DTC para verificar que el DTC no reaparece. También puede conducir el vehículo en las condiciones que ha observado de los datos de Registros de fallos/cuadro congelado.

Comprobación del circuito/sistema
  1. Con el encendido desconectado durante 90 segundos, desemborne el conector del mazo de cables del sensor MAP B74.
  2. Compruebe si hay menos de 5 Ω entre el terminal 2 del circuito referencia baja y la masa.
    3. ⇒Si es mayor que el intervalo especificado, compruebe si el circuito de baja referencia tiene una resistencia abierta/alta. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  3. Con el encendido conectado, compruebe que existen 4,8-5,2 V entre el terminal 1 del circuito de referencia de 5 V y la conexión a masa.
    4. ⇒Si el valor es inferior al margen especificado, compruebe si el circuito de referencia de 5 V tiene un cortocircuito a masa o un circuito abierto/resistencia alta. Si la prueba del circuito sale bien, sustituya el K20 ECM.
    ⇒Si el valor es superior al margen especificado, compruebe si el circuito de referencia de 5 V tiene un cortocircuito a voltaje. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  4. Compruebe en la herramienta de diagnóstico si el parámetro del sensor MAP es inferior a 1 kPa (0,2 psi).
    5. ⇒Si es superior al intervalo especificado, compruebe si el terminal 3 del circuito de señales tiene un cortocircuito a tensión. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  5. Monte un cable de puente con fusible de 3 A entre el terminal 3 del circuito de señales y el terminal 1 del circuito de referencia de 5 V. Verifique en la herramienta de diagnóstico que el parámetro del sensor MAP es superior a 127 kPa (18,4 psi).
    6. ⇒Si el valor es inferior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a masa o un circuito abierto/resistencia alta. Si la prueba del circuito sale bien, sustituya el K20 ECM.
  6. Si la comprobación de los circuitos es normal, compruebe o sustituya el sensor MAP B74.

Comprobación de componentes

Nota: Debe realizar la comprobación del circuito / sistema para verificar la integridad de los circuitos del sensor MAP antes de proceder con la comprobación de componentes.

Comprobación de señal errática

  1. Encendido desconectado, desmonte el sensor MAP B74.
  2. Monte un cable de puente protegido con fusible de 3 A entre el terminal 1 del circuito de referencia de 5 V y el terminal correspondiente del sensor MAP B74.
  3. Monte un cable de puente entre el terminal 2 del circuito de referencia baja del sensor MAP B74 y masa.
  4. Instale un cable de puente en el terminal 3 del sensor MAP B74.
  5. Conecte un DMM entre el cable de puente del terminal 3 del sensor MAP B74 y masa.
  6. Con el encendido en ON y con la herramienta Mityvac J 23738-A o el Mityvac metálico J 35555, aplique lentamente vacío al sensor mientras observa la tensión en el DMM. La tensión debe variar entre 0-5 V, sin picos ni caídas.
    7. ⇒Si la lectura de tensión es errática, sustituya el sensor MAP B74.
