Descripción de DTC

Códigos de avería B1016 00: Información sobre el número de identificación del vehículo (VIN) no programada

Descripción de circuito

Al restaurar la alimentación de la batería después de haberse cortado, el módulo del sistema antirrobo (TDM) efectuará una comprobación del número de identificación del vehículo (VIN) almacenado en el vehículo. Si el VIN es CORRECTO, el TDM activará el temporizador del VIN durante un período de tiempo programable, normalmente 10 minutos. El TDM enviará un mensaje a través del circuito de datos serie al cuadro de instrumentos (IPC) para iluminar el indicador SEGURIDAD. Una vez que se haya agotado el tiempo programado el temporizador del VIN, se apagará el indicador de seguridad. Si el VIN es INCORRECTO, el TDM enviará la contraseña para desactivar el combustible al módulo de control del motor (ECM) a través del circuito de datos serie. El sistema inmovilizador considera que se ha producido una manipulación indebida y no permitirá el arranque del vehículo. El TDM enviará un mensaje a través del circuito de datos serie al IPC para iluminar de forma intermitente el indicador SEGURIDAD.

Condiciones para la activación de DTC

Encendido ON

Condiciones para ajustar el DTC
    • El indicador de comprobación del VIN del TDM es verdadero.
    • El TDM ha establecido que se ha vuelto a conectar la batería después de una desconexión.

Acción realizada cuando el DTC se ajusta
    • El vehículo no arrancará.
    • El TDM envía un mensaje de desactivación del combustible a través del circuito de datos serie al ECM.
    • El TDM envía un mensaje al IPC para iluminar el indicador SEGURIDAD.

Condiciones para el borrar el DTC

El TDM borra el DTC actual cuando se detecta la información del VIN correcto.
