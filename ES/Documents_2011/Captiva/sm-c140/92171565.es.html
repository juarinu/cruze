Descripción de DTC

Consultar Lista de códigos U del módulo de control .

Descripción del circuito/sistema

Algunos módulos de control reciben constantemente información de otros módulos de control mediante la red de comunicación de datos en serie. El código de datos no válido se establecerá cuando un módulo de control receptor detecte una discrepancia en la información que reciba de otro módulo de control, lo que provocará que se cuestione su integridad. El número de síntoma indicado en la Descripción de DTC sólo es una referencia para los ingenieros. No está relacionado con ningún diagnóstico de circuito externo.

Condiciones para la activación de DTC

La tensión de la batería está entre 9-16 V y las comunicaciones de enlace de datos funcionan con normalidad.

Condiciones para ajustar el DTC

El módulo de control no está configurado correctamente.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio sin que se repita una avería.

Verificación del circuito/sistema
  1. Motor en marcha durante 10 segundos.
  2. Con el encendido conectado y el motor apagado, verifique que no se active el DTC U0400-U05FF.
    3. ⇒Si el DTC U0400-U05FF está activado al mismo tiempo que otros DTC, diagnostique todos los otros códigos primero. Consultar Lista de los códigos diagnósticos de avería (DTC) - Vehículo .
    ⇒Si el DTC U0400-U05FF está activado sin que hayan otros DTC activados, programe el módulo de control especificado mediante la descripción de DTC. Si se vuelve a activar el DTC, sustituya el módulo de control apropiado.
