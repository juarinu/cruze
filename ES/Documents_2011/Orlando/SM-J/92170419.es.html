* Una interrupción entre el conector de enlace de datos (DLC) y el primer empalme sólo permitirá la comunicación con la herramienta de diagnóstico. Los módulos de control del vehículo seguirán comunicando.

1. Ninguna comunicación en ningún módulo GMLAN de velocidad media.

2. Ninguna comunicación en ningún módulo GMLAN de alta velocidad del chasis.

3. Una interrupción en sólo un circuito de datos de serie GMLAN de alta velocidad del chasis o de velocidad media puede permitir una comunicación degradada entre las unidades de mandos.

Descripción del circuito/sistema

Los módulos de control conectados a los circuitos de datos serie GMLAN de alta velocidad del chasis o de velocidad media controlan comunicaciones de datos serie durante el funcionamiento normal del vehículo. Entre los módulos de control se intercambia información y comandos operativos cuando el interruptor de encendido está en cualquier posición distinta de OFF. El bus de datos de serie GMLAN de alta velocidad del chasis o de velocidad media utiliza una resistencia de terminación que está en paralelo a los circuitos GMLAN (+) y (-) de alta velocidad del chasis o de velocidad media.

Condiciones para la activación de DTC

La tensión del sistema está entre 9-16 V.

Condiciones para ajustar el DTC

No se ha recibido un mensaje supervisado periódicamente que incluye la disponibilidad del módulo transmisor.

Acción realizada cuando el DTC se ajusta

Subsistemas específicos no funcionarán.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio de 50 sin que se repita una avería.

Ayuda al diagnóstico
    • Utilice Referencias de enlace de datos para identificar los módulos de datos de serie GMLAN de alta velocidad del chasis o de velocidad media.
    • Algunos módulos de control pueden no tener protección interna para salidas de tensión específicas y pueden abrir un fusible de fuente de tensión de encendido o tensión positiva de batería. Si está abierto un fusible de entrada de tensión y no se encuentra ningún cortocircuito en ese circuito, antes de sustituir el módulo de control asegúrese de que ningún circuito de tensión de salida del módulo de control presenta un cortocircuito a masa.
    • Un fusible abierto en el terminal 5 del circuito de masa DLC permitirá que la herramienta de diagnóstico funcione para configurar el vehículo en la herramienta y después no comunicará con el vehículo. Cuando la herramienta de diagnóstico está en el punto de comunicación con el vehículo, un mensaje en la herramienta de diagnóstico indicará "no se ha detectado el módulo CANdi" y no comunicará.

Verificación del circuito/sistema
  1. Verifique que no se ha activado ningún otro DTC de código U.
    2. ⇒Si se han activado otros DTC de código U, diagnostíquelos primero. Consultar Lista de códigos U del módulo de control .
  2. Consulte Referencias de enlace de datos para determinar si el DTC U0074 de ajuste del módulo de control está en los circuitos de datos de serie de alta velocidad del chasis o de velocidad media.

Comprobación del circuito/sistema

Nota: Puede ser necesario desconectar cada uno de los módulos de control para aislar un circuito defectuoso. Use los esquemas para identificar lo siguiente:

   • Los módulos de control con los que está equipado el vehículo
   • Resistencias de terminación del circuito de datos de serie de alta velocidad del chasis o de velocidad media
   • Ubicaciones de módulo de control en los circuitos de datos de serie
   • Cada uno de los terminales del circuito de datos de serie de alta velocidad del chasis o de velocidad media

  1. Intente comunicarse con todos los módulos de control en el circuito de datos de serie de alta velocidad del chasis o de velocidad media. Consultar Referencias de enlace de datos . Debería no ser posible la comunicación con dos o más módulos de control en el circuito de datos de serie de alta velocidad del chasis o de velocidad media.
    2. ⇒Si únicamente un módulo de control no comunica, diagnostique únicamente ese módulo de control. Consultar DTC U0100-U02FF .
  2. Desconecte la herramienta de diagnóstico del DLC. Deberán realizarse las siguientes comprobaciones en el conector DLC.
  3. Con todas las puertas cerradas y el encendido en OFF durante 60 segundos, compruebe que existen menos de 10 Ω entre el terminal 5 de circuito a masa y la masa.
    4. ⇒Si es superior al intervalo especificado, compruebe si el circuito de masa tiene una resistencia abierta/alta.
  4. Con el encendido conectado, compruebe que existen menos de 3 V entre los circuitos de datos de serie indicados abajo y la masa:
    5. • Terminal 3 de datos de serie GMLAN de velocidad media
    • Terminal 11 de datos de serie GMLAN de velocidad media
    • Terminal 12 de datos de serie GMLAN de alta velocidad del chasis
    • Terminal 13 de datos de serie GMLAN de alta velocidad del chasis
    ⇒Si es superior al intervalo especificado, compruebe si el circuito de datos de serie presenta un cortocircuito con la tensión. Consulte Comprobar si existe cortocircuito con la tensión en circuitos de datos de serie
  5. Con el encendido en OFF durante 60 segundos, compruebe que existen más de 100 Ω entre los circuitos de datos de serie indicados abajo y la masa:
    6. • Terminal 3 de datos de serie GMLAN de velocidad media
    • Terminal 11 de datos de serie GMLAN de velocidad media
    • Terminal 12 de datos de serie GMLAN de alta velocidad del chasis
    • Terminal 13 de datos de serie GMLAN de alta velocidad del chasis
    ⇒Si no se corresponde con el valor especificado, compruebe si el circuito de datos de serie tiene un cortocircuito a masa. Consulte "comprobar si existe cortocircuito a masa en circuitos de datos de serie".
  6. Compruebe si existen entre 50 y 70 Ω entre los circuitos de datos de serie que se indican a continuación:
    7. • Terminal 3 y terminal 11 de los circuitos de datos de serie GMLAN de velocidad media
    • Terminal 12 y terminal 13 de los circuitos de datos de serie GMLAN de alta velocidad del chasis
    ⇒Si es inferior a 35 Ω, compruebe si existe un cortocircuito entre los circuitos de datos de serie. Consulte Comprobar si existe cortocircuito entre los circuitos de datos de serie
    ⇒Si existen 35-50 Ω, puede haber una tercera resistencia de terminación entre los circuitos de datos de serie. Esto puede suceder si se ha instalado un módulo de control incorrecto. Algunos módulos de control están disponibles con y sin resistencias de terminación instaladas para reducir la necesidad de resistencias de terminación en el mazo de cables.
    ⇒Si es superior a 70 Ω pero no es infinita, compruebe si el circuito de datos de serie tiene una resistencia alta/abierta. Consulte Comprobar si existe una resistencia alta/abierta en los circuitos de datos de serie.
    ⇒Si es infinita, compruebe si los circuitos de datos de serie tienen una resistencia abierta/alta entre el DLC y la primera conexión al circuito de datos de serie.
  7. Si según la comprobación los circuitos están bien, consulte Comprobación del circuito de encendido.

Comprobar si existe cortocircuito con la tensión en circuitos de datos de serie

  1. Con el encendido desconectado, desemborne los conectores del mazo de cables con los circuitos de datos de serie de alta velocidad del chaisis o de velocidad media en un módulo de control al que pueda accederse fácilmente.
  2. Con el encendido conectado, compruebe si existen más de 3 V entre los circuitos de datos de serie y la masa. Compruebe si uno o varios circuitos de datos de serie son superiores a 3 V.
    3. ⇒Si es inferior al intervalo especificado para todos los circuitos de datos de serie, sustituya el módulo de control que se desconectó.
  3. Con el encendido desconectado, desemborne los conectores del mazo de cables con los circuitos de datos de serie de alta velocidad del chaisis o de velocidad media en otro módulo de control en la dirección del circuito con cortocircuito con la tensión.
  4. Con el encendido conectado, compruebe si existen más de 3 V entre los circuitos de datos de serie y la masa. Compruebe si uno o varios circuitos de datos de serie son superiores a 3 V.
    5. ⇒Si es inferior al intervalo especificado para todos los circuitos de datos de serie, sustituya el módulo de control que se acaba de desconectar.
  5. Repita el paso 3 hasta que se aísle una de las siguientes condiciones:
    6. • Un cortocircuito con la tensión en el circuito de datos de serie entre 2 módulos de control
    • Un cortocircuito con la tensión en el circuito de datos de serie entre un módulo de control y una resistencia con terminación

Comprobar si existe cortocircuito a masa en circuitos de datos de serie

  1. Con el encendido desconectado, desemborne los conectores del mazo de cables con los circuitos de datos de serie de alta velocidad del chaisis o de velocidad media en un módulo de control al que pueda accederse fácilmente.
  2. Con el encendido en OFF durante 60 segundos, compruebe si existen menos de 100 Ω entre los circuitos de datos de serie y la masa. Verifique si uno o varios circuitos de datos de serie son inferiores a 100 Ω.
    3. ⇒Si es superior al intervalo especificado para todos los circuitos de datos de serie, sustituya el módulo de control que se desconectó.
  3. Desemborne los conectores del mazo de cables con los circuitos de datos de serie de alta velocidad del chaisis o de velocidad media en otro módulo de control en la dirección del circuito con cortocircuito a masa.
  4. Con el encendido en OFF durante 60 segundos, compruebe si existen menos de 100 Ω entre los circuitos de datos de serie y la masa. Verifique si uno o varios circuitos de datos de serie son inferiores a 100 Ω.
    5. ⇒Si es superior al intervalo especificado para todos los circuitos de datos de serie, sustituya el módulo de control que se acaba de desconectar.
  5. Repita el paso 3 hasta que se aísle una de las siguientes condiciones:
    6. • Un cortocircuito a masa en el circuito de datos de serie entre 2 módulos de control
    • Un cortocircuito a masa en el circuito de datos de serie entre un módulo de control y una resistencia de terminación

Comprobar si existe cortocircuito entre los circuitos de datos de serie

  1. Con el encendido desconectado, desemborne los conectores del mazo de cables con los circuitos de datos de serie de alta velocidad del chaisis o de velocidad media en un módulo de control al que pueda accederse fácilmente y que no se esté comunicando.
  2. Con el encendido en OFF durante 60 segundos, compruebe que existen menos de 110 Ω entre cada par de circuitos de datos de serie. Verifique que un par de circuitos de datos de serie son inferiores a 110 Ω.
    3. ⇒Si es superior al intervalo especificado para cada par de circuitos de datos de serie, sustituya el módulo de control que se desconectó.
  3. Conecte los conectores del mazo de cables en el módulo de control que se desconectó.
  4. Desemborne los conectores del mazo de cables con los circuitos de datos de serie de alta velocidad del chaisis o de velocidad media en otro módulo de control en la dirección de los circuitos cortocircuitados juntos.
  5. Con el encendido en OFF durante 60 segundos, compruebe que existen menos de 110 Ω entre cada par de circuitos de datos de serie. Verifique que un par de circuitos de datos de serie son inferiores a 110 Ω.
    6. ⇒Si está en el intervalo especificado en cada uno de los pares de circuitos de datos de serie, sustituya el módulo de control que acaba de desconectarse.
  6. Repita el paso 3 hasta que se aísle una de las siguientes condiciones:
    7. • Circuitos de datos de serie cortocircuitados juntos entre 2 módulos de control
    • Circuitos de datos de serie cortocircuitados juntos entre un módulo de control y una resistencia de terminación
    • Una resistencia de terminación cortocircuitada

Comprobar si existe una resistencia alta/abierta en los circuitos de datos de serie.

  1. Con el encendido desconectado, desemborne los conectores del mazo de cables con los circuitos de datos de serie de alta velocidad del chaisis o de velocidad media en un módulo de control al que pueda accederse fácilmente y que no se esté comunicando.
  2. Con el encendido en OFF durante 60 segundos, compruebe que existen más de 130 Ω entre cada par de circuitos de datos de serie. Verifique que un par de circuitos de datos de serie son superiores a 130 Ω.
    3. ⇒Si es inferior al intervalo especificado para cada par de circuitos de datos de serie, sustituya el módulo de control que se desconectó.
  3. Conecte los conectores del mazo de cables en el módulo de control que se desconectó.
  4. Desemborne los conectores del mazo de cables con los circuitos de datos de serie de alta velocidad del chaisis o de velocidad media en otro módulo de control en la dirección del circuito abierto.
  5. Con el encendido en OFF durante 60 segundos, compruebe que existen más de 130 Ω entre cada par de circuitos de datos de serie. Verifique que un par de circuitos de datos de serie son superiores a 130 Ω.
  6. Repita el paso 3 hasta que se aísle una de las siguientes condiciones:
    7. • Una resistencia abierta/alta en el circuito de datos de serie entre 2 módulos de control
    • Una resistencia abierta/alta en el circuito de datos de serie entre un módulo de control y una resistencia de terminación
    • Una resistencia de terminación con resistencia abierta/alta

Comprobación del circuito de encendido

  1. Con el encendido desconectado, desemborne los enchufes del mazo de cable con los circuitos de encendido en un módulo de control de fácil acceso que no esté comunicando.
  2. Con el encendido desconectado, compruebe que se enciende una lámpara de prueba entre cada terminal del circuito de encendido y la masa en el conector del módulo de control que se acaba de desembornar.
    3. ⇒Si se enciende la lámpara de prueba, compruebe si el circuito de encendido presenta un cortocircuito a tensión. Si según la comprobación los circuitos están bien, sustituya el módulo de control que se acaba de desconectar.
  3. Con el encendido conectado, verifique que la lámpara de prueba entre cada terminal del circuito de encendido y la masa.
    4. ⇒Si la luz de prueba no se ilumina, compruebe si en el circuito de encendido hay un cortocircuito a masa o una resistencia abierta/alta. Si según la comprobación los circuitos están bien, sustituya el módulo de control que se acaba de desconectar.
  4. Con el encendido desconectado, conecte los conectores del mazo de cables en el módulo de control que se acaba de desconectar.
  5. Repita el paso 1 en otro módulo de control hasta encontrar un cortocircuito a tensión, un cortocircuito a masa o una resistencia abierta/alta en los circuitos de encendido.
