Descripción de DTC

Códigos de avería P1678: Sistema del inmovilizador. Identificación incorrecta del módulo de control del motor (ECM)

Descripción del circuito/sistema

Cuando se programa y configura el módulo de control del motor (ECM) durante la instalación, el ECM memoriza un identificador de inmovilizador específico que es exclusivo de ese vehículo. El identificador de inmovilizador se utiliza para impedir el intercambio de módulos entre vehículos. El módulo de control de la carrocería (BCM) es que el alberga el identificador del inmovilizador. Durante el funcionamiento del vehículo, el ECM compara este identificador del inmovilizador con el BCM. Si los identificadores coinciden, se permite el arranque del vehículo. Si el identificador del inmovilizador no coincide, se desactiva el arranque del vehículo.

Condiciones para la activación de DTC

El encendido esté en la posición RUN o ACCESSORY.

Condiciones para ajustar el DTC

El ECM recibe un identificador del inmovilizador incorrecto.

Acción realizada cuando el DTC se ajusta
    • Se encenderá el indicador de seguridad del cuadro de instrumentos.
    • Se desactivará el arranque del motor.

Condiciones para el borrar el DTC

Un DTC actual se borrará cuando el ECM obtenga un identificador del inmovilizador.
