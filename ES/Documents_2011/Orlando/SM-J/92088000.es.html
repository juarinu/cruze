Descripción y funcionamiento del control de crucero

El control de crucero es un sistema de control de velocidad que mantiene una velocidad deseada en condiciones normales de conducción cuando se marcha a más de 40 km/h (25 millas por hora). La inclinación puede causar variaciones en la velocidad seleccionada del vehículo.

Los siguientes son los principales componentes del sistema de control de crucero:

    • El pedal del acelerador
    • El sensor de posición del pedal del freno (BPP)
    • El módulo electrónico de la carrocería (BCM)
    • El interruptor de encendido/apagado del control de velocidad
    • El interruptor de cancelación del control de crucero
    • El interruptor + RES
    • El interruptor - SET
    • El módulo de control del motor (ECM)
    • El motor de control del actuador de la mariposa (TAC)
    • El sensor de velocidad del vehículo (VSS)

El módulo electrónico de la carrocería (BCM) vigila el circuito de señales de los interruptores de control de crucero, que se ubican en el volante. EL BCM envía el estado del interruptor de control de crucero al módulo de control del motor (ECM) a través del circuito de datos de serie. EL ECM usa el estado del interruptor de control de crucero para determinar cuándo capturar y mantener la velocidad del vehículo. El ECM vigila el circuito de señal de velocidad del vehículo para determinar la velocidad deseada del vehículo. El ECM usa el motor TAC para mantener la velocidad del vehículo. Si desea más información sobre el sistema TAC, consulte Descripción del sistema de control del actuador del acelerador (TAC) .

Se suministra la tensión al interruptor de control de crucero a través del circuito de tensión de referencia del interruptor de control de volante suministrado por el BCM. Los interruptores de función del control de crucero están ubicados en un diseño de resistencias en escalera, teniendo cada interruptor un valor de resistencia distinto. EL BCM detecta un valor de tensión específico que está asociado al interruptor de función de control de crucero que se activa. Cuando el interruptor on/off de control de crucero que está abierto normalmente está encendido (ON), el interruptor se cerrará y el BCM suministrará un circuito indicador de interruptor de control de crucero encendido cuando se ilumine. El BCM envía un mensaje de datos de serie GMLAN al ECM indicando que el interruptor on/off está encendido. De forma similar, cuando están pulsados el interruptor + RES, normalmente abierto, o el - SET, normalmente abierto, se cierra el interruptor y el BCM detecta la señal de tensión predeterminada en el circuito de señales del interruptor reanudar/acelerar y configurar/dejar rodar del control de crucero. El BCM envía un mensaje de datos de serie GMLAN al ECM indicando que el interruptor + RES o el - SET están activos. El interruptor + RES o - SET seguirán inactivos, cuando el BCM no haya recibido la señal de tensión predeterminada del interruptor activado/desactivado.

Control de crucero activado

El sistema de control de crucero activará y ajustará las velocidades del vehículo basándose en la activación de los siguientes controles de crucero:

    • Activado/desactivado
    • + RES
    • - SET

Para activar el sistema de control de crucero, asegúrese de que la velocidad del vehículo está por encima de 40,2 km/h (25 mph), gire el interruptor activado/desactivado de crucero a activado y pulse un momento el interruptor - SET. El ECM activará el sistema de control y registrará la velocidad del vehículo. El ECM envía un mensaje de datos de serie GMLAN al cuadro de instrumentos (IPC) para iluminar el indicador de crucero activado en el IPC.

Al apretar el pedal del acelerador cuando está activado el sistema de control de crucero, el conductor sustituirá al sistema de control de crucero para acelerar por encima de la velocidad establecida. Cuando se suelte el pedal del acelerador, el vehículo decelerará y volverá a la velocidad establecida. El conductor también podrá variar la velocidad establecida del vehículo con el interruptor - SET y + RES. Cuando esté activado el sistema de control de crucero, si se pulsa y mantiene el interruptor - SET, el vehículo desacelerará de la velocidad establecida sin desactivar el sistema de control de crucero. Cuando se haya soltado el interruptor - SET, el ECM registrará la velocidad del vehículo y mantendrá ésta como la nueva velocidad establecida. Cuando el sistema de control de crucero esté activado, al pulsar momentáneamente el interruptor - SET, el vehículo desacelerará en pasos de 1,6 km/h (1 mph) cada vez que se pulse el - SET, hasta una velocidad mínima de 37 km/h (23 mph).

Al pulsar y mantener pulsado el interruptor + RES, cuando el sistema de control de crucero está activado, el vehículo acelerará a una velocidad superior a la establecida actualmente en el vehículo. Cuando se haya soltado el interruptor + RES, el ECM registrará la velocidad del vehículo y mantendrá ésta como la nueva velocidad establecida. Cuando el sistema de control de crucero está activado, al pulsar un momento el interruptor + RES, el vehículo acelerará en pasos de 1,6 km/h (1 milla por hora) cada que se pulse un momento el interruptor + RES, hasta una aceleración máxima total de 16 km/h (10 millas por hora) sobre la velocidad establecida actualmente. Si se activa un momento el interruptor + RES, se recordará la velocidad anterior, después de que el sistema de control de crucero se haya desactivado apretando el pedal del freno o el interruptor CANCELAR.

Control de crucero desactivado

El módulo de control de motor (ECM) desactiva el funcionamiento del control de crucero basándose en las señales de los siguientes interruptores:

    • El sensor de posición del pedal del freno (BPP)
    • El interruptor de encendido/apagado del control de velocidad
    • El interruptor de cancelación del control de crucero

El sistema de control de crucero se desactivará cuando se presione el pedal del freno. El módulo electrónico de la carrocería (BCM) vigila el sensor BPP a través del circuito de señal del sensor BPP cuando la señal de tensión aumenta mientras el pedal alcanza la posición de tope máximo. El ECM vigila la señal BPP a través de una entrada discreta y una señal de mensaje de datos de serie GMLAN del BCM que indica el estado del freno. Cuando las dos señales indiquen que se ha apretado el pedal del freno, el ECM desactivará el sistema de control de crucero. Si desea más información sobre el sensor BPP, consulte Descripción y funcionamiento de los sistemas de iluminación exterior o Calibración del sensor de posición del pedal del freno .

El sistema de control de crucero también se desactivará cuando el interruptor on/off de control de crucero se apague (OFF) o se active el interruptor de cancelación de control de crucero. El módulo electrónico de la carrocería (BCM) determina cuándo se activa el interruptor de cancelación del control de crucero. Cuando está cerrado el interruptor de cancelación, normalmente abierto, el BCM detecta la señal de tensión predeterminada en el circuito del interruptor de la función del control de crucero. La velocidad del vehículo almacenada en la memoria del módulo de control del motor se borrará cuando el interruptor on/off del control de crucero se apague (OFF) o se apague el interruptor de encendido. El BCM envía un mensaje de datos de serie GMLAN al ECM para desactivar el sistema de control de crucero. El sistema de control de crucero se desactivará cuando el ECM detecte que una función de sustitución del conductor ha estado activa unos 60 segundos. Cuando se haya desactivado el sistema de control de crucero, la ECM envía un mensaje de serie GMLAN al cuadro de instrumentos (IPC) para apagar el indicador de crucero activado.

Cada vez que se desactive el sistema de control de crucero, el ECM registrará cuál es la razón de la desactivación del sistema. Las últimas 8 razones de desactivación se registrarán en la memoria ECM. La herramienta de diagnóstico mostrará los últimos 8 parámetros del historial de desactivación de crucero; en cada uno de ellos se mostrará una de las aprox. 50 razones posibles. Para que se muestre la razón de desactivación en el parámetro de la herramienta de diagnóstico o el sistema de control de crucero estará activo y se solicitará la desactivación o se solicitará la activación del sistema, pero habrá un error. Si desea obtener una lista de todas las razones de desactivación junto a sus definiciones, consulte Información de la herramienta de diagnóstico del módulo de control del motor : LNP  → LDE o 2H0 .

Control de crucero inhibido

El módulo de control del motor (ECM) inhibe el funcionamiento del control de crucero cuando ocurre una de las siguientes posibilidades:

    • Se ha ajustado un DTC del sistema de control de crucero.
    • La velocidad del vehículo es inferior a 40,2 km/h (25 millas por hora).
    • La velocidad del vehículo es demasiado elevada.
    • El vehículo está en ESTACIONAMIENTO, MARCHAS ATRÁS, PUNTO MUERTO o primera marcha.
    • Las r.p.m. del motor son demasiado bajas.
    • Las r.p.m. del motor son demasiado elevadas.
    • El voltaje del sistema no está entre 9-16 voltios.
    • El sistema de frenos antibloqueo (ABS)/sistema de control de tracción (TCS) está activo durante más de 2 segundos.
