Referencias de enlace de datos

Esta tabla identifica el enlace de datos de serie que utiliza un módulo concreto para la transmisión de datos en el vehículo. Algunos módulos pueden utilizar más de un enlace de datos para comunicar. Algunos módulos pueden tener múltiples circuitos de comunicación que pasan a través de ellos sin comunicar activamente en ese enlace de datos. Esta tabla se utiliza para ayudar a corregir un fallo de comunicación. Consulte en Descripción y funcionamiento de las comunicaciones de enlace de datos la descripción y el funcionamiento de estos circuitos de comunicación de datos de serie.
