Descripción de DTC

Códigos de avería B1015 00: Fallo de funcionamiento de la información sobre el número de identificación del vehículo

Descripción del circuito/sistema

Cuando el encendido se activa, el módulo de control de aparcamiento asistido recibe el número de identificación del vehículo (VIN) del módulo de control de la carrocería. El módulo de control de aparcamiento asistido comparará el VIN recibido con el VIN almacenado en la memoria.

Condiciones para la activación de DTC

La tensión de encendido está entre 9-16 V.

Condiciones para ajustar el DTC

El VIN almacenado en el módulo de control de aparcamiento asistido no concuerda con el VIN recibido.

Acción realizada cuando el DTC se ajusta
    • El estacionamiento asistido está desactivado.
    • El centro de información del conductor muestra el mensaje REPARAR EL SISTEMA DE APARCAMIENTO ASISTIDO.

Condiciones para el borrar el DTC

El VIN almacenado en el módulo de control de aparcamiento asistido coincide con el VIN almacenado en el módulo de control de la carrocería.

Verificación del circuito/sistema

Con el encendido conectado, programe el módulo de control de aparcamiento asistido K41 y verifique que el código de avería no se vuelve a activar.

⇒Si el código de avería se vuelve a activar, sustituya el módulo de control de aparcamiento asistido K41
