Descripción de DTC

Códigos de avería P1649: Código de seguridad del inmovilizador no programado

Descripción del circuito/sistema

Cuando aprende un código de seguridad, así como la secuencia de la señal y la respuesta, el módulo de control del motor (ECM) se pone en modo de aprendizaje. Una vez aprendidos el código de seguridad y la señal/respuesta, se saldrá del modo de aprendizaje cuando el motor arranque con éxito.

Condiciones para la activación de DTC

El ECM está en modo de aprendizaje.

Condiciones para ajustar el DTC

El ECM no está programado.

Acción realizada cuando el DTC se ajusta

Se encenderá el indicador de seguridad del cuadro de instrumentos.

Condiciones para el borrar el DTC
    • Un DTC P1649 actual se borrará tras un arranque con éxito del motor después de salir del modo de aprendizaje.
    • Se borrará del historial de DTC después de 100 ciclos consecutivos de encendido sin fallos o cuando se use una herramienta de diagnóstico para borrar los DTCs.

Nota: Debido a las comunicaciones normales de sistema entre el BCM y el ECM, todos los códigos B de inmovilización deben diagnosticarse antes del diagnóstico de los códigos P.

Llevar a cabo el procedimiento de reaprendizaje del inmovilizador. Consultar Programación de componentes del sistema del inmovilizador .

⇒Si el ECM no completa la programación o si el arranque del vehículo sigue desactivado, sustituya el módulo de control del motor K20.
