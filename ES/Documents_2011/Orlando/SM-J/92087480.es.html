Iconos generales

Símbolo

Descripción



Icono de la lista de componentes principal

Este icono se usa en los esquemas para enlazar con la lista de componentes eléctricos principal.



Icono de descripción y funcionamiento

Este icono se usa en los esquemas para enlazar con la descripción y funcionamiento de ese sistema específico.



Icono de programación informática

Este icono se usa en los esquemas para enlazar a las referencias del módulo de control que identifican qué componentes hay que programar cuando se sustituyen.



Icono de siguiente página de esquemas

Este icono se usa en los esquemas para ir al siguiente esquema en el subsistema.



Icono de página de esquemas anterior

Este icono se usa en los esquemas para ir al esquema anterior en el subsistema.



Icono de sujeción inflable suplementaria (SIR) o sistema de sujeción suplementario (SRS)

Este icono se usa para alertar al técnico de que el sistema contiene componentes SIR/SRS que requieren ciertas precauciones antes de su manipulación.



Icono de información

Este icono se usa para avisar al técnico de que hay información adicional que le ayudará a manipular un sistema.



Peligro - Icono de alto voltaje

Este icono se usa para avisar al técnico de que un componente/sistema contiene circuitos de 300 voltios.



Icono de alto voltaje

Este icono se usa para avisar al técnico de que el componente/sistema contiene un voltaje superior a 42 voltios, pero menor de 300 voltios.



Icono de precaución

Este icono se usa para avisar al técnico de que tenga cuidado cuando esté manipulando el componente.



Comunicación de datos de serie de funcionamiento

Este icono se usa para mostrar al técnico que el detalle del circuito de datos de serie no se muestra completo. También proporciona un enlace activo a los Esquemas de comunicación de datos en los que se muestra el circuito completo.
