Descripción del circuito/sistema

El sensor de posición del pedal del freno se utiliza para detectar la acción del conductor sobre el pedal del freno. El sensor de posición del pedal del freno proporciona una señal de tensión analógica que aumentará a medida que se pisa el pedal del freno. El módulo de control de la carrocería proporciona una señal de referencia baja y una tensión de referencia de 5 V al sensor de posición del pedal de freno. Cuando la señal variable alcanza un umbral de tensión que indica que se han aplicado los frenos, el BCM aplicará tensión de batería al circuito de control de las luces de freno y al circuito de control de la luz de freno central instalada en alto. Cuando el circuito de control recibe la energía, se iluminan las luces de freno.

Condiciones para la activación de DTC
    • El voltaje de la batería debe situarse en 9-16 V.
    • Frenos aplicados.

Condiciones para ajustar el DTC

C0277 06 o C0890 03

    • El BCM detecta un cortocircuito a masa, resistencia alta o interrupción en el circuito de señal del sensor de posición del pedal de freno.
    • El BCM detecta un cortocircuito a masa, resistencia alta o interrupción en el circuito de referencia de 5 V del sensor de posición del pedal de freno.

C0277 07 o C0890 07

    • El BCM detecta un cortocircuito a la tensión de batería en el circuito de señal del sensor de posición del pedal de freno.
    • El BCM detecta una resistencia abierta/alta en el circuito de referencia baja del sensor de posición del pedal de freno.

C0277 09

Se activará un código de avería cuando el BCM detecte que la señal del sensor de posición del pedal de freno cambia más rápido de lo que es posible en condiciones normales de funcionamiento.

C0277 4B

Se establecerá un código de avería cuando no se haya programado la posición inicial del sensor de posición del pedal de freno.

Acción realizada cuando el DTC se ajusta

Luces de freno siempre encendidas.

Condiciones para el borrar el DTC

La anomalía que dio lugar al DTC ya no está presente.

Ayuda al diagnóstico

C0277 4B

Si se establece el código de avería, recalibre el sensor de posición del pedal de freno.

C0277 09

Si se establece el código de avería, recoloque el sensor de posición del pedal de freno. Si el DTC se vuelve a registrar, sustituya el BCM.

Verificación del circuito/sistema

Encendido conectado, observar el parámetro Interruptor de freno en la herramienta de diagnóstico. La lectura debe estar entre Inactivo/Activo y cambiar cuando el pedal de freno se pisa o se suelta.

Comprobación del circuito/sistema

C0277 06 o C0277 07

  1. Con el encendido en OFF, desconecte el conector del mazo de cables en el sensor de posición del pedal de freno B22.
  2. Desconecte el terminal negativo de la batería C1.
  3. Con el encendido en OFF, compruebe que hay menos de 5 Ω entre el terminal C del circuito de referencia baja y la toma de masa.
    4. ⇒Si es mayor que el intervalo especificado, compruebe si el circuito de baja referencia tiene una resistencia abierta/alta. Si tras la comprobación el circuito está normal, sustituya el módulo de control de la carrocería K9.
  4. Vuelva a conectar el terminal negativo a la batería C1.
  5. Con el encendido conectado, compruebe si hay 4,8-5,2 V entre el terminal A del circuito de referencia de 5 V y la masa.
    6. ⇒Si el valor es inferior al margen especificado, compruebe si el circuito de referencia de 5 V tiene un cortocircuito a masa o un circuito abierto/resistencia alta. Si tras la comprobación el circuito está normal, sustituya el módulo de control de la carrocería K9.
    ⇒Si el valor es superior al margen especificado, compruebe si el circuito de referencia de 5 V tiene un cortocircuito a voltaje. Si tras la comprobación el circuito está normal, sustituya el módulo de control de la carrocería K9.
  6. Compruebe para menos de 0,3 V entre el terminal B del circuito de señal y masa.
    7. ⇒Si es superior al margen especificado, compruebe si el circuito de señal tiene un cortocircuito a voltaje. Si tras la comprobación el circuito está normal, sustituya el módulo de control de la carrocería K9.
  7. Pruebe para 4,8-5,2 V entre el terminal A del circuito de referencia de 5 voltios y el terminal B del circuito de señal.
    8. ⇒Si es menor que el intervalo especificado, compruebe si el circuito de señal tiene una resistencia abierta/alta. Si tras la comprobación el circuito está normal, sustituya el módulo de control de la carrocería K9.
  8. Encendido desactivado, desconectar el conector del mazo de cables X2 en el módulo de mando de control de la carrocería K9.
  9. Compruebe si existe una resistencia infinita entre el terminal B del circuito de señal y la masa:
    10. ⇒Si es inferior al intervalo especificado, compruebe si el circuito de señales tiene un cortocircuito a masa.
  10. Si todos los circuitos parecen estar normales, compruebe o sustituya el sensor de posición del pedal del freno B22.

C0277 09

  1. Si se establece el código de avería, sustituya el sensor de posición del pedal del freno B22.
  2. Llevar a cabo el procedimiento de recalibración del sensor de posición del pedal de freno. Consultar Calibración del sensor de posición del pedal del freno
  3. Maneje el vehículo en las condiciones de ejecución y activación de los DTC y verifique que el DTC no se vuelve a activar.
    4. ⇒Si se vuelve a activar el DTC, sustituya el módulo de control de la carrocería K9.

C0277 4B

Si se establece el código de avería, recalibre el sensor de posición del pedal del freno B22.

  1. Verificar que el DTC se fija en las condiciones de funcionamiento y fijación.
  2. Llevar a cabo el procedimiento de recalibración del sensor de posición del pedal de freno. Consultar Calibración del sensor de posición del pedal del freno
  3. Maneje el vehículo en las condiciones de ejecución y activación de los DTC y verifique que el DTC no se vuelve a activar.
    4. ⇒Si se reactiva el código de avería, sustituya el sensor de posición del pedal del freno B22.

Comprobación de componentes

Prueba estática

  1. Con el encendido en OFF, desconecte el conector del mazo de cables en el sensor de posición del pedal de freno B22.
  2. Prueba para 2,7-3,1 kΩ entre el terminal A de la referencia de 5 V y el terminal C de la referencia baja.
    3. ⇒Si no está en el intervalo especificado, sustituya el sensor de posición del pedal del freno B22.
  3. Barra el sensor en todo el rango mientras supervisa la resistencia entre el terminal B de señal y el terminal C de referencia baja. La resistencia debería variar entre 1,5-2,3 k Ω sin picos ni caídas.
    4. ⇒Si no está en el intervalo especificado o actúa de forma imprevisible, sustituya el sensor de posición del pedal del freno B22.

Prueba dinámica

  1. Instalar un cable de cierre con fusible de 3 A entre el terminal A de la referencia de 5 V y 5 V. Instalar un cable de cierre entre el terminal C de referencia baja y masa.
  2. Barrer el sensor en todo el rango mientras supervisa la tensión entre el terminal B de señal y el terminal C de referencia baja. La tensión debería variar entre 0,8-2,1 V sin picos ni caídas.
    3. ⇒Si no está en el intervalo especificado o actúa de forma imprevisible, sustituya el sensor de posición del pedal del freno B22.
