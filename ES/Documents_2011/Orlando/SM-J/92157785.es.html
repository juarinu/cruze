1. Fallo de funcionamiento del sensor de humedad exterior del parabrisas

Descripción del circuito/sistema

Se suministra tensión positiva de la batería (B+) al sensor de humedad exterior del parabrisas. Siempre que la llave de encendido está en la posición ON, o de accesorios, el módulo de control de la carrocería (BCM) envía el estado del interruptor multifunción/de los intermitentes mediante una señal modulada por ancho de pulso (PWM). Cuando es preciso un ciclo de barrido, el sensor de humedad exterior del parabrisas envía una señal de tensión PWM al BCM iniciando el accionamiento del limpiaparabrisas. El BCM ordena los ciclos de barrido del motor del limpiaparabrisas. Si en cualquier momento se produce alguna avería en el sensor de humedad exterior del parabrisas y el BCM, éste usa las entradas del interruptor multifunción/de los intermitentes en las posiciones de retardo para accionar el motor del limpiaparabrisas a intervalos variables continuos.

Condiciones para la activación de DTC
    • Encendido ON.
    •  La tensión del sistema está entre 9-16 V.

Condiciones para ajustar el DTC

El BCM recibe una señal no válida del sensor de humedad exterior del parabrisas.

Acción realizada cuando el DTC se ajusta
    • Si el BCM pierde comunicación con el sensor de humedad exterior del parabrisas mientras el sensor de humedad exterior del parabrisas solicitaba barridos sueltos, no podrá suavizar el funcionamiento de los limpiaparabrisas a manual - intermitente para los limpiaparabrisas delanteros con un tiempo fijo de parada de 4 segundos.
    • Si el BCM pierde comunicación con el sensor de humedad exterior del parabrisas mientras el sensor de humedad exterior del parabrisas solicitaba de forma continua 'bajo', no podrá suavizar el funcionamiento de los limpiaparabrisas al modo limpiaparabrisas delanteros manual - bajo.
    • Si el BCM pierde comunicación con el sensor de humedad exterior del parabrisas mientras el sensor de humedad exterior del parabrisas solicitaba de forma continua 'alto', no podrá suavizar el funcionamiento de los limpiaparabrisas al modo limpiaparabrisas delanteros manual - alto.

Condiciones para el borrar el DTC

Un DTC actual se borra cuando se realiza y se supera un diagnóstico.

Verificación del circuito/sistema

Con el encendido conectado, el interruptor de limpia/lavaparabrisas S80 en posición baja. Los limpiaparabrisas deberían funcionar a baja velocidad.

⇒Si los limpiaparabrisas no funcionan, consulte Fallo de funcionamiento del sistema limpiaparabrisas .

Comprobación del circuito/sistema

  1. Con el encendido desactivado, desconecte el conector del sensor de lluvia B117.
  2. Pruebe con menos de 5 Ω entre el terminal 2 de circuito de toma de tierra y masa.
    3. ⇒Si es mayor que el valor especificado, compruebe si el circuito de toma de tierra tiene una resistencia abierta/alta.
  3. Con el encendido en ON, verifique que una luz de comprobación se ilumina entre el terminal 1 del circuito de voltaje B+ y masa.
    4. ⇒Si la luz de prueba no se ilumina, pruebe el circuito de voltaje B+ para ver si hay un cortocircuito a masa o un circuito abierto/resistencia alta.
  4. Con el encendido conectado, compruebe si hay 4-12 V entre el terminal 3 del circuito de señal y masa.
    5. ⇒Si es menor que el intervalo especificado, compruebe si el circuito de señal tiene una resistencia abierta/alta. Si tras la comprobación el circuito está normal, sustituya el módulo de control de la carrocería K9.
  5. Si tras la comprobación todos los circuitos están normales, sustituya el sensor de lluvia B117.

Instrucciones de reparación

Realice Verificación de la reparación por diagnóstico después de completar el procedimiento de diagnóstico.

    •  Sustitución del sensor de humedad exterior del parabrisas
    • Referencias del módulo de control para sustituir, configurar y programar el BCM.
