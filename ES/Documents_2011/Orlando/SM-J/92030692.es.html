Acero de fase dual extra-resistente

Esta información proporciona recomendaciones de reparación y orientación general para acero clasificado como acero de fase doble de resistencia ultra-alta (acero de fase doble con una resistencia a la tracción superior a 800 MPa), también conocido como DPX.

General Motors recomienda lo siguiente al reparar o sustituir este tipo de acero durante las reparaciones por colisión.

Nota: 

   • No se recomienda reparar este tipo de acero.
   • Este tipo de acero se deberá sustituir sólo en uniones de fábrica. No se recomienda seccionarlo o sustituirlo parcialmente.
   • No se recomienda el uso de calor para reparar daños para este tipo de acero.
   • No se recomienda la soldadura de puntada para este tipo de acero.
   • Esta clasificación de acero no puede emplearse como un refuerzo para la parte posterior o un casquillo para una unión de seccionamiento, a menos que esté disponible un procedimiento específico.

Reparaciones recomendadas:

    • Puede utilizarse la soldadura de resistencia por puntos para sustituir a los puntos de soldadura de fábrica, donde proceda.
    • La soldadura de tapón MIG se puede usar para sustituir soldaduras por puntos de fábrica.
    • La soldadura MIG se puede usar para sustituir soldaduras por puntos de fábrica.
