Descripción de DTC

DTC P2299 00: Posición del pedal de freno - Posición del pedal del acelerador no admisible

Descripción del circuito/sistema

La posición del pedal de freno es una entrada variable al controlador basada en la aplicación o liberación del pedal de freno. La posición del pedal del acelerador es una entrada variable al controlador basada en la aplicación o liberación del pedal del acelerador.

Condiciones para el funcionamiento de DTC
    • El encendido está activado o el motor está en marcha.
    • El DTC se activa continuamente cuando se cumplen las condiciones anteriores.

Condiciones para ajustar el DTC

El módulo de control del motor (ECM) detecta que el pedal del acelerador y el freno se han accionado simultáneamente, o que el pedal del acelerador se ha accionado antes que el freno.

Acción realizada cuando se registra el DTC

EL DTC P2299 5A es un DTC de tipo B.

Condiciones para el borrar el DTC

EL DTC P2299 5A es un DTC de tipo B.

Verificación del circuito/sistema
  1. Con el encendido conectado, observe la información de los DTC con una herramienta de diagnóstico. Compruebe que no se activan los DTC P2122, P2123, P2127, P2128 o P2138.
    2. ⇒Si se activa cualquiera de estos DTC, consulte DTC P2122, P2123, P2127, P2128, o P2138 para una ulterior diagnosis.
  2. Compruebe que la luz de freno funciona correctamente.
    3. ⇒Si las luces de freno no funcionan correctamente, consulte Avería de lámparas de parada para continuar con el diagnóstico.
  3. El DTC P2299 00 es un DTC informativo. Si no hay presentes otros DTC de los controles del motor, consulte Lista de códigos de diagnóstico de avería (DTC) - Vehículo para continuar con el diagnóstico.
