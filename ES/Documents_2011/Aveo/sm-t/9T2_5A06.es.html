DTC U0001 -

Comunicaciones del CAN de alta velocidad

Condiciones para activar el DTC

  • Las revoluciones del sensor de velocidad del motor son 400 rpm o más.
  • Las revoluciones del sensor de velocidad de entrada son 400 rpm o más.
  • El TCM recibe el estado "Bus OFF" desde el controlador del CAN.

Condiciones para la activación del DTC

  • La detección anterior se produce 7 veces de forma continuada.

Acción realizada cuando se activa el DTC

  • El TCM hará que se encienda la MIL y almacenará el DTC cuando detecte un fallo en el primer ciclo de encendido.
  • El TCM juzga que el acelerador se abre al 0% para el control de punto de cambio y de punto de bloqueo.
  • El TCM juzga que el acelerador se abre al 100% para el control de presión de funcionamiento.
  • No hay bloqueo
  • No hay control de automemorización
  • No se selecciona la 4ª marcha
  • El TCM juzga que la velocidad del motor es igual a 0 rpm.

Condiciones para borrar el DTC

  • El TCM apagará la MIL cuando no se detecten más fallos en tres ciclos de encendido consecutivos.
  • El DTC puede borrarse del historial del TCM con un scanner.

Ayudas para el diagnóstico

  • Este DTC no se puede recuperar con un estado actual. El diagnóstico de un DTC actual se consigue a través del síntoma, "El scanner no se comunica con el dispositivo GMLAN".
  • Una condición intermitente puede ser causada por un cortocircuito en los circuitos de datos de serie CAN. Utilice el procedimiento "El scanner no se comunica con el dispositivo GMLAN" para aislar una condición intermitente.

DTC U0100 -

Comunicación perdida con ECM/PCM

Condiciones para activar el DTC

  • Estado de fallo del bus CAN desactivado.
  • La velocidad del motor sea igual o superior a 400 rpm.
  • Las revoluciones del sensor de velocidad de entrada son 400 rpm o más.
  • El TCM no puede detectar el marco del ECM durante 0,2 segundos de forma continuada.

Condiciones para la activación del DTC

  • La detección anterior se produce de forma continuada en 10 ocasiones.

Acción realizada cuando se activa el DTC

  • El TCM hará que se encienda la MIL y almacenará el DTC cuando detecte un fallo en el primer ciclo de encendido.
  • El TCM juzga que el acelerador se abre al 0% para el control de punto de cambio y de punto de bloqueo.
  • El TCM juzga que el acelerador se abre al 100% para el control de presión de funcionamiento.
  • No hay bloqueo
  • No hay control de automemorización
  • No se selecciona la 4ª marcha
  • El TCM juzga que la velocidad del motor es igual a 0 rpm.

Condiciones para borrar el DTC

  • El TCM apagará la MIL cuando no se detecten más fallos en tres ciclos de encendido consecutivos.
  • El DTC puede borrarse del historial del TCM con un scanner.

Ayudas para el diagnóstico

  • Una mala conexión en el módulo inoperativo puede causar la activación de este código.
  • Un módulo mal alimentado puede provocar la activación de este código.
  • Si existen múltiples módulos sin comunicación, escoja el más cercano al conector de enlace de datos (DLC). Ciertos módulos pueden no disponer de protección interna para salidas de tensión específicas y podrían abrir una tensión positiva de la batería o un fusible de la fuente de tensión del encendido. Si el fusible de entrada de tensión presenta un circuito abierto y no se detecta ningún cortocircuito, asegúrese de que ningún circuito de tensión de salida del módulo está cortocircuitado a masa antes de sustituir el módulo. Este diagnóstico se puede utilizar para cualquier módulo que debería comunicarse con los datos de serie CAN, siempre y cuando al vehículo esté equipado con la opción que utiliza ese módulo.

DTC U0401

Datos no válidos recibidos desde el ECM/PCM

Condiciones para activar el DTC

  • La tensión de encendido sea superior a 9 V.
  • Sin estado de fallo en el bus CAN desactivado.
  • El TCM recibe la señal de velocidad del motor durante 0,2 segundos de forma continuada.

Condiciones para la activación del DTC

  • La detección anterior se produce de forma continuada en 10 ocasiones.

Acción realizada cuando se activa el DTC

  • El TCM solicitará la iluminación de la luz de retención.
  • No hay bloqueo
  • No se selecciona la 4ª marcha
  • No hay control de automemorización
  • No se controla la reducción del par motor
  • No hay control de presión de engrane
  • No hay control del solenoide de sincronización para N-D
  • El TCM estima que las revoluciones del motor son 0 rpm.

Condiciones para borrar el DTC

  • El TCM desactiva la luz de retención si no se detectan más fallos durante tres ciclos consecutivos de encendido.
  • El DTC puede borrarse del historial del TCM con un scanner.
  • El TCM borra el DTC de la memoria del historial del TCM después de cuarenta ciclos consecutivos de calentamiento sin fallo.
  • El TCM recibe la señal de velocidad del motor durante 2 segundos.

Causa de la avería

  • ECM
  • Piezas asociadas al motor
