Luces de circulación diurna

Ayudas para el diagnóstico

El sistema de las luces de circulación diurna (DRL) no funcionará si el freno de estacionamiento está puesto. Este sistema está diseñado para funcionar sólo cuando el motor esté en marcha y el freno de estacionamiento no esté aplicado. Si el circuito del freno de estacionamiento está derivado a masa o el interruptor permanece cerrado con el motor en marcha, el sistema de DRL no funcionará.
