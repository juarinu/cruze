Sustitución del sensor de posición del árbol de levas

Procedimiento de desmontaje
  1. Abra el capó.


    2. Nota: El gráfico muestra el sensor de posición del árbol de levas de escape. Utilice el mismo procedimiento para el sensor de posición del árbol de levas de admisión.

  2. Desconecte el enchufe del mazo de cables del sensor de posición del árbol de levas (1).


    3. Nota: El gráfico muestra el sensor de posición del árbol de levas de escape. Utilice el mismo procedimiento para el sensor de posición del árbol de levas de admisión.

  3. Retire el tornillo del sensor de posición del árbol de levas (3).
  4. Desmonte el sensor de posición del árbol de levas (2) y el anillo de junta (1).

Procedimiento de montaje


    Nota: El gráfico muestra el sensor de posición del árbol de levas de escape. Utilice el mismo procedimiento para el sensor de posición del árbol de levas de admisión.

  1. Monte el sensor de posición del árbol de levas (2) y el anillo de junta (1).

    2. Precaución:Consulte Precaución con las fijaciones en la sección Prólogo.

  2. Monte el tornillo del sensor de posición del árbol de levas (3) y apriételo hasta 8 N·m (71 lib. pulg.).


    3. Nota: El gráfico muestra el sensor de posición del árbol de levas de escape. Utilice el mismo procedimiento para el sensor de posición del árbol de levas de admisión.

  3. Conecte el enchufe del mazo de cables del sensor de posición del árbol de levas (1).
  4. Tras sustituir un sensor de posición del árbol de levas, utilice una herramienta de exploración para que la calibración sea más rápida. Consultar Programación y configuración del módulo de control del motor → LNP → LUW, LUJ → LLW → LXT → LXV, 2H0
  5. Cerrar el capó.
