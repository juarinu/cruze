Para obtener mejores resultados, utilice combustible Diesel número 2-D por encima y por debajo de temperatura de congelación, ya que las petroleras mezclan el combustible número 2-D para las distintas diferencias climáticas. Se puede usar combustible diésel número 1-D a temperaturas muy bajas, cuando la temperatura se mantiene por debajo de -18 °C (0 °F). Sin embargo, el combustible tendrá una pérdida económica de potencia y combustible. El uso del combustible Diesel número 1-D en climas cálidos o muy calientes puede hacer que se cale y arranque mal cuando el motor está caliente y puede dañar el sistema de inyección de combustible.

El combustible Diesel puede soltar espuma cuando se está llenando el depósito. Esto puede provocar que se cierre el difusor de la bomba automático, incluso si el depósito no está lleno. Si ocurre esto, espere a que deje de formarse espuma y luego continúe llenando el depósito.

Servicio con tiempo muy frío

Si el vehículo se conduce a temperaturas muy frías y no se puede obtener un número 2-D invernal que se haya adaptado al tiempo frío o un número 1-D, use un galón de queroseno por cada 2 galones de combustible Diesel. Una vez que añada el queroseno, haga funcionar el motor varios minutos para que se mezclen los combustibles. Sólo se debe añadir queroseno cuando la temperatura baje de -18 °C (0 °F) porque las propiedades lubricantes y el consumo de combustible del queroseno no son tan buenas como las del combustible diésel.

Con tiempo frío, el filtro de combustible se puede obstruir (encerar). Para eliminar la obstrucción del filtro, lleve el vehículo a una zona de taller cálida y caliente el filtro a una temperatura entre 0-10°C (32-50°F). No hará falta sustituir el filtro.

Prueba de gravedad específica del combustible

Use un aparato de verificación de calidad de combustible Diesel para medir la gravedad específica del combustible (clasificación de calidad API). Siga las instrucciones de la herramienta para obtener el valor ajustado de temperatura correcto. Esta información debe ser precisa para obtener un diagnóstico correcto del sistema de combustible.

Flujo de retorno del inyector de combustible y presiones de combustible

El retorno de combustible de los inyectores al depósito variará dependiendo del valor API del combustible. Mida la clasificación API del combustible con el aparato de verificación de calidad del combustible Diesel. Por esta razón, los valores del diagnóstico del sistema de combustible - lado de alta presión variarán para identificar un problema del inyector o de la bomba de combustible. Use las tablas siguientes cuando el diagnóstico le remita a ellas. La primera tabla se debe usar durante el diagnóstico inicial para identificar los peores inyectores. Una vez se haya quitado la cápsula de los inyectores que han fallado en la primera parte de la prueba, se deberá medir de nuevo el flujo de retorno de cada inyector sin cápsula. Esto es porque el sistema de combustible está devolviendo menos combustible al depósito y, por lo tanto, la presión de combustible es mayor durante la nueva prueba. Si no se usa la tabla correcta, habrá que sustituir unos inyectores que están bien.

Agua en el combustible

Algunas veces, se puede bombear agua al depósito de combustible junto al combustible Diesel. Este puede ocurrir si la estación de servicio no comprueba y limpia regularmente sus depósitos de combustible o éste se contamina en los proveedores de las estaciones de servicio.

Si se bombea agua al depósito de combustible, se iluminará el agua en la luz del combustible. Si se ilumina el agua en la luz del combustible, el exceso de agua se vaciará del sistema de combustible al vehículo.
