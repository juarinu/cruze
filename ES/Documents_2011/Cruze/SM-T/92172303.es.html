Habilitar comunicación de datos de serie del tren motriz

1. El vehículo no arrancará.

Descripción del circuito/sistema

El módulo de control de la carrocería (BCM) activa el circuito de habilitación de comunicación de datos de serie cuando la llave de encendido está en ACC, ON o START. El circuito de habilitación de comunicación de datos de serie activa el módulo de control del motor (ECM) para la comunicación por bus de datos de serie.

Condiciones para el funcionamiento de DTC
    • El voltaje del sistema está entre 9-16 voltios.
    • El módulo maestro de modo de potencia del vehículo requiere que se produzca la comunicación de datos serie.

Condiciones para ajustar el DTC

El BCM detecta un cortocircuito a masa en el circuito de habilitación de comunicación de datos de serie.

Acción realizada cuando se registra el DTC
    • El comando de salida está desactivado mientras esté presente el fallo de funcionamiento.
    • Los módulos utilizan un valor por defecto para los parámetros faltantes hasta el siguiente ciclo de encendido.
    • No existe transmisión de señales con el(los) módulo(s), por lo que el (los) subsistema(s) específico(s) no funcionará(n).
    • El vehículo no arrancará mientras el circuito tenga un cortocircuito a masa.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio de 50 sin que se repita una avería.

Ayuda al diagnóstico
    • Utilice la función DMM MIN/MAX para capturar/localizar estados intermitentes.
    • El cortocircuito a masa también activará múltiples códigos de no comunicación de historial para módulos GMLAN de alta velocidad.

Comprobación del circuito/sistema

Nota: Utilice los esquemas para determinar qué módulos de control están en el circuito 22 X4 de habilitación de comunicaciones del BCM.

  1. Con el encendido desconectado, desemborne el enchufe del mazo de cables del módulo de control más fácilmente accesible del circuito de habilitación de comunicaciones del BCM K9.
  2. Con el encendido conectado, vuelva a comprobar los códigos de avería actuales. El DTC U1814 debería permanecer actual.
    3. ⇒Si el DTC pasa sólo al registro, sustituya el módulo de control que se acaba de desconectar.
  3. Con los anteriores módulos de control desconectados, repita los pasos 1 y 2 para cada módulo de control del circuito excepto el BCM K9.
  4. Con el encendido en OFF, desconecte el conector del mazo de cables X4 en el BCM K9.
  5. Compruebe si existe resistencia infinita entre el terminal 22 del circuito habilitado de comunicación de datos de serie X4 y la masa.
    6. ⇒Si la resistencia no es infinita, repare el cortocircuito a masa del circuito de habilitación de comunicaciones.
  6. Si todas las pruebas de los circuitos salen bien, sustituya el BCM K9.
