Descripción de DTC

Códigos de avería P1629: Señal de activación de paso de combustible tras el inmovilizador no recibida

Descripción del circuito/sistema

El módulo de control de la carrocería (BCM) envía la contraseña pre-editada al módulo de control del motor (ECM) a través del circuito de datos serie. El ECM envía entonces una señal al BCM. Tanto el ECM como el BCM realizan un cálculo en la recepción de esta señal. Si la respuesta calculada procedente del BCM es igual al cálculo realizado por el ECM, el ECM permitirá el arranque del vehículo.

Condiciones para la activación de DTC

El encendido esté en la posición RUN o ACCESSORY.

Condiciones para ajustar el DTC

El ECM recibe una contraseña pre-editada incorrecta del BCM.

Acción realizada cuando el DTC se ajusta
    • Se encenderá el indicador de seguridad del cuadro de instrumentos.
    • Se desactivará el arranque del motor.

Condiciones para el borrar el DTC
    • Un DTC P1629 actual se borrará cuando se haya recibido una contraseña de activación de la alimentación de combustible correcta.
    • Se borrará del historial de DTC después de 40 ciclos consecutivos de encendido sin fallos o cuando se use una herramienta de diagnóstico para borrar los DTCs.

Verificación del circuito/sistema

Nota: Debido a las comunicaciones normales de sistema entre el BCM y el ECM, todos los códigos B de inmovilización deben diagnosticarse antes del diagnóstico de los códigos P.

Llevar a cabo el procedimiento de reaprendizaje del inmovilizador. Consultar Programación de componentes del sistema del inmovilizador . El ECM debe completar con éxito el aprendizaje de la contraseña y el vehículo debe arrancar.

⇒Si el ECM no completa el aprendizaje o sigue desactivado el arranque del vehículo, sustituir el módulo de control del motor (ECM) K20.
