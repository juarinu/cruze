Descripciones de DTC

DTC P0324 00: Funcionamiento del módulo del sensor de picado

DTC P0326 00: Funcionamiento del sensor de picado

DTC P06B6 00: Rendimiento del procesador 1 del sensor de picado del módulo de control

Descripción del circuito/sistema

El sensor de picado permite que el módulo de control del motor (ECM) controle la sincronización del encendido para obtener el máximo recorrido y protege al mismo tiempo el motor de unos niveles de detonación que podrían dañarlo. El sensor de picado produce una señal de tensión de corriente alterna (CA) que varía dependiendo del nivel de vibraciones durante el funcionamiento del motor. El ECM ajusta el reglaje de chispa basándose en la amplitud y en la frecuencia de la señal del sensor de picado. El ECM recibe la señal del sensor de picado a través de un circuito de señal. El ECM proporciona un circuito de masa al sensor de picado a través de un circuito de referencia baja. El ECM también proporciona un circuito de masa apantallado al sensor de picado. El ECM aprende un nivel de ruido mínimo del sensor de picado al ralentí y utiliza valores calibrados para el resto de la gama de revoluciones del motor. El ECM debe supervisar si hay una señal normal del sensor de picado dentro del canal de ruido.

Condiciones para el funcionamiento de DTC
    • El régimen de revoluciones del motor es superior a 700 r.p.m.
    • El régimen del motor está entre 700 y 2 500 r.p.m. para el DTC P06B6 00.
    • El flujo de aire hacia el motor está entre 40 y 2.000 mg/cilindro
    • Este DTC se ejecuta continuamente cuando se producen las condiciones de habilitación.
    • Los DTC P0324 00, P0326 00 y P06B6 00 se reproducen continuamente cuando se cumplen las condiciones anteriores.

Condiciones para ajustar el DTC

P0324 00 ó P0326 00

El módulo de control detecta un sensor de picado hiperactivo.

P06B6 00

El módulo de control detecta un fallo de la circuitería interna.

Acción realizada cuando se registra el DTC

Los DTC P0324 00, P0326 00 y P06B6 00 son DTC de tipo C.

Condiciones para el borrar el DTC

Los DTC P0324 00, P0326 00 y P06B6 00 son DTC de tipo C.

Ayuda al diagnóstico
    • Compruebe si existen daños físicos en el sensor de picado y si está correctamente instalado. Un sensor de picado que se ha caído o está dañado puede hacer que se active un DTC.
    • Compruebe si el sensor de picado está bien montado. Un sensor de picado suelto o demasiado apretado puede hacer que se active un DTC. El sensor de picado no debe presentar sellador de roscas.
    • La superficie de montaje del sensor de picado debe estar libre de rebabas, restos de fundición y material extraño.

Comprobación del circuito/sistema
  1. Con el motor al ralentí, observe la información de los DTC con una herramienta de diagnóstico. Verifique que no esté activado el DTC P06B6 00.
    2. ⇒Si se activa el DTC, sustituya el módulo de control del motor K20.
  2. Con el encendido desconectado, compruebe lo siguiente:
    3. • Compruebe si el sensor de picado B68 tiene daños físicos. Si el sensor de picado B68 se cae o está dañado, se puede activar un DTC.
    • Compruebe si el sensor de picado B68 está bien montado. Si el sensor de picado B68 está flojo o apretado en exceso, se puede activar un DTC. El sensor de picado B68 debería estar libre de sellador de roscas. La superficie de montaje del sensor de picado B68 debe estar libre de rebabas, restos de fundición y material extraño
    • El sensor de picado B68 debe estar separado de tubos flexibles, soportes y cables eléctricos del motor
    • Compruebe si hay soportes de accesorios flojos
    • Compruebe el desgaste de la correa de transmisión y del tensor
    • Compruebe si hay un problema mecánico del motor Síntomas: Mecánica del motor
         ⇒Si se detecta un problema, efectúe la reparación.
  3. Si no hay ningún problema mecánico, compruebe o sustituya el sensor de picado B68.
  4. Conduzca el vehículo en las condiciones de ejecución de los DTC para verificar que el DTC no reaparece. También puede conducir el vehículo en las condiciones que ha observado en los datos de Registros de fallos/cuadro congelado.
