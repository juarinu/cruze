Descripción de DTC

DTC P0630 00: VIN no programado o no coincidente - Módulo de control del motor (ECM)

Descripción del circuito/sistema

Este DTC se activa si el módulo de control del motor (ECM) ha determinado que el número de identificación del vehículo (VIN) no se ha almacenado en la memoria del ECM. Si se sustituye el módulo de control, debe programarse el VIN en la memoria del módulo de control.

Condiciones para el funcionamiento de DTC
    • El módulo de control está activado o despierto.
    • El DTC se ejecuta continuamente cuando se cumple la anterior condición.

Condiciones para ajustar el DTC

El ECM detecta que no hay ningún VIN almacenado en la memoria.

Acción realizada cuando se registra el DTC

El DTC P0630 00 es un código de avería de tipo C.

Condiciones para el borrar el DTC

El DTC P0630 00 es un código de avería de tipo C.

Verificación del circuito/sistema

Nota: El ECM no puede determinar si el VIN programado en el módulo de control es válido. Verifique siempre que el VIN que se visualiza en la herramienta de diagnóstico es correcto para el vehículo.

  1. Encendido conectado, observe en la herramienta de diagnóstico la información del vehículo, en la sección Información del sistema de ID. La información de VIN debe coincidir con el VIN del vehículo.
    2. ⇒Si un VIN no se visualiza, prográmelo con una herramienta de diagnóstico.
    ⇒Si se visualiza un VIN, borre los DTC con una herramienta de diagnóstico. Si se activa el DTC P0630 00 después de borrar los DTC, sustituya el módulo de control del motor K20.
  2. Conduzca el vehículo en la condiciones para que se ejecute el DTC. También podría conducir el vehículo en las condiciones que se haya observado de los datos del registro de fallos/cuadro congelado.
