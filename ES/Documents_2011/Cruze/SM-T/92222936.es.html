Descripción del sistema del sensor de picado

Descripción del circuito/sistema

El sistema del sensor de picado permite que el módulo de control del motor (ECM) controle la sincronización del encendido para obtener el máximo recorrido, protegiendo al mismo tiempo el motor de unos niveles de detonación que podrían dañarlo. El ECM utiliza el sistema del sensor de picado para comprobar si existe un ruido anormal en el motor que podrían indicar que se está produciendo una detonación, también llamada golpeteo de chispa.

Descripción del sensor

El sistema de sensor de picado utiliza uno de dos sensores de dos cables de respuesta plana. El sensor utiliza tecnología de cristal piezoeléctrico que produce una señal de tensión de corriente alterna de amplitud y frecuencia variables de acuerdo con la vibración del motor o el nivel de ruido. La amplitud y la frecuencia dependen del nivel de picado detectado por el sensor de picado. El ECM recibe la señal del sensor de picado a través de 2 circuitos de señal aislados.

Si el ECM determina que se está produciendo un picado, retrasará la sincronización del encendido para intentar reducir el picado. El ECM puede controlar el retraso del encendido de acuerdo con el cilindro individual. El ECM siempre intenta volver a un nivel de compensación cero o sin retardo de encendido. El diagnóstico del sensor de picado está calibrado para detectar errores en los circuitos del sensor de picado en el ECM, el cableado del sensor de picado o la salida de voltaje del sensor de picado. Algunos diagnósticos también se calibra para detectar un ruido constante desde una influencia exterior, como un componente aflojado o roto o un ruido mecánico del motor excesivo.
