Descripción de DTC

DTC U2101: Lista de configuración máxima de bus CAN

Descripción del circuito/sistema

Algunos módulos de control deben programarse con software que identifica de forma específica el tipo y número correcto de módulos en el bus de datos de serie basándose en la configuración RPO. Si un módulo de control no se configuró adecuadamente tras la instalación, puede activarse el código U correspondiente para ese módulo. No está relacionado con ningún diagnóstico de circuito externo.

Condiciones para el funcionamiento de DTC
    • La tensión alimentada al módulo está dentro del intervalo de tensión operativa normal.
    • El modo de potencia del vehículo requiere que se produzca la comunicación de datos serie.

Condiciones para ajustar el DTC
    • El módulo de control no está configurado correctamente.
    • La unidad de control detecta un error de programación.
    • La configuración de bus CAN es inválida.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio sin que se repita una avería.

Verificación del circuito/sistema
  1. Verifique que no se activa el DTC correspondiente.
    2. ⇒Si se ha activado el DTC junto con otros códigos, diagnostique primero los demás códigos. Si no se han activado otros códigos, programe el módulo de control especificado mediante la descripción de DTC.
  2. Con una herramienta de diagnóstico, borre los códigos de avería.
  3. Con el encendido en OFF, extraiga la herramienta de diagnóstico, abra y cierre la puerta del conductor.
  4. Espere 60 segundos, después compruebe que no se ha activado el DTC.
    5. ⇒Si se ha activado el DTC, sustituya el módulo de control correspondiente.
