Descripción y operación del teléfono móvil

El módulo de control del teléfono móvil ofrece la oportunidad de mantener conversaciones por teléfono utilizando un teléfono móvil mediante un micrófono del vehículo y el sistema de altavoces del vehículo y operar las funciones más importantes del teléfono móvil mediante el sistema infotainment del vehículo. El módulo de control de teléfono móvil está disponible en dos configuraciones, basadas en diferentes tipos de radio, un módulo de control de teléfono móvil bajo para una radio base con CD y un módulo de control de teléfono móvil alto para radios con CD y sistemas de navegación. Ambos módulos ofrecen básicamente las mismas funciones:

    • Lista de llamadas
    • Gestión de llamadas
    • Control desde el volante
    • Mensajes de estado
    • Agenda
    • Bluetooth
    • Reconocimiento de voz, incluyendo algoritmo de adaptación al eco y de reducción de ruido

Hay disponible como opción una placa base y un soporte para teléfonos móviles específicos. Permite cargar el teléfono móvil y conectarlo a una antena exterior. Toda interacción con el teléfono móvil y transmisión de datos al módulo de control del teléfono móvil se realiza por Bluetooth. Para crear una conexión entre el módulo de control del teléfono móvil y el sistema infotainment, el módulo de control del teléfono móvil debe estar conectado y el teléfono móvil debe estar accesible. El módulo de control del teléfono móvil se auto conecta y desconecta mediante el encendido. Si el encendido está desactivado en el módulo de control del teléfono móvil se puede conectar y desconectar mediante el sistema infotainment. Cuando hay activa una conexión telefónica, el sonido de la radio se desconecta. Cuando termina la conexión telefónica, el sonido de la radio se reactiva.

Reconocimiento de voz

El reconocimiento de voz posibilita operar algunas funciones del teléfono móvil mediante la voz. Reconoce comandos y secuencias de números independientemente de la persona que hable. Los comandos y secuencias de números se pueden decir sin ninguna pausa entre cada palabra. También es posible almacenar números telefónicos utilizando un nombre al azar escogido por el usuario. Se puede crear una conexión telefónica utilizando el nombre propio.
