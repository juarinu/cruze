Descripción de DTC

DTC P0401 00: Flujo insuficiente de recirculación de gases de escape (EGR)

DTC P0402 00: Flujo excesivo de recirculación de gases de escape (EGR)

Descripción del circuito/sistema

El módulo de control del motor (ECM) realiza una prueba de flujo insuficiente en el sistema de recirculación de gases de escape (EGR) durante la deceleración y una prueba de flujo excesivo durante el arranque del motor.

La prueba de flujo insuficiente es cuando el ECM ordena momentáneamente a la válvula EGR que se abra mientras que monitoriza la señal del sensor de presión absoluta del colector (MAP) y el sensor de posición EGR. Si la señal MAP es incorrecta para la posición del tetón EGR, el ECM registra la cantidad de la diferencia MAP detectada y ajusta un contador de fallo calibrado hasta el nivel de umbral de fallo calibrado. El número de pruebas de flujo EGR requerido para exceder el umbral de fallo puede variar dependiendo de la cantidad del error de flujo EGR detectado. El ECM sólo permitirá una comprobación del flujo de EGR durante un ciclo de encendido. Después de que se borre un código, el ECM realizará múltiples pruebas de flujo EGR.

La prueba de flujo excesivo es cuando el ECM monitoriza la posición de la válvula EGR durante el arranque del motor. La válvula EGR no debe estar abierta durante el arranque del motor.

Condiciones para el funcionamiento de DTC

P0401 00

    • No están activados los DTC  P0106 00, P0107 00, P0108 00, P0112 00, P0113 00, P0117 00, P0118 00, P0122 00, P0123 00, P0351 00, P0352 00, P0402 00, P0404 00, P0405 00, P0406 00, P0502 00, P0506 00, y P0507 00.
    • La prueba de flujo EGR se realiza en modo de interrupción de combustible en la deceleración con las siguientes condiciones presentes antes de que se produzca la deceleración:
       - El embrague del compresor de A/C no cambia de estado durante la deceleración.
       - El sensor de temperatura del aire de entrada (IAT) está entre -3 y +120 °C (de +27 a +248 °F).
       - La presión barométrica (BARO) es superior a 82 kPa (11,89 PSI).
       - La velocidad del vehículo es superior a 30 km/h (19 MPH) antes de la deceleración.
       - La posición del acelerador es inferior al 1,5%.
    • La prueba de flujo EGR se realiza cuando se cumplen las siguientes condiciones durante el modo de interrupción de combustible en la deceleración:
       - La posición EGR es menos de 1%.
       - El MAP está entre 11-55 kPa (1,5-7,9 PSI).
       - El régimen de revoluciones del motor está entre 1.600 y 3.200 r.p.m.
       - El MAP no varía más de 3 kPa (0,4 PSI).
    • La prueba de flujo EGR se realiza una vez por ciclo de ignición.
    • Este DTC corre varias veces durante el primer ciclo de ignición después de que se borre un DTC de la memoria ECM.

P0402 00

    • El motor está arrancando.
    • La tensión de encendido se sitúa entre 10-16 V.
    • Este DTC se ejecuta continuamente cuando se cumplen las anteriores condiciones.

Condiciones para ajustar el DTC

P0401 00

Los cambios de MAP monitorizados por el ECM durante las pruebas de flujo EGR indican una cantidad insuficiente de flujo EGR.

P0402 00

El ECM detecta flujo EGR excesivo debido a que el tetón de la válvula EGR está abierto más del 55% durante el arranque.

Acción realizada cuando se registra el DTC
    • El DTC P0401 00 es un código de avería de tipo C.
    • El DTC P0402 00 es un DTC de tipo B.

Condiciones para el borrar el DTC
    • El DTC P0401 00 es un código de avería de tipo C.
    • El DTC P0402 00 es un DTC de tipo B.

Verificación del circuito/sistema
  1. Con el motor en marcha, observe la información de los DTC con una herramienta de diagnóstico. Compruebe que no se activan los DTC  P0106 00, P0107 00, P0108 00, P0403 00, P0404 00, P0405 00 o P0406 00.
    2. ⇒Si se activa alguno de estos DTC, consulte Lista de códigos de diagnóstico de avería (DTC) - Vehículo
  2. Observe la información de DTC con una herramienta de diagnóstico. No deberán activarse los DTC P0401 00 o P0402 00.
  3. Con el encendido conectado, observe el parámetro del Sensor de Posición EGR en la herramienta de diagnóstico. La lectura deberá estar entre el 0 y el 4%.
  4. Conduzca el vehículo en las condiciones de ejecución de los DTC para verificar que el DTC no reaparece. También puede conducir el vehículo en las condiciones que ha observado en los datos de Registros de fallos/cuadro congelado.

Comprobación del circuito/sistema
  1. Compruebe que no se producen ninguna de las siguientes condiciones:
    2. • Válvula EGR incorrecta para la aplicación del motor
    • Una fuga de vacío entre la válvula EGR y el colector de admisión
    • Señales de fuga externa normalmente indicadas por una acumulación de carbón en las superficies de conexión de los componentes o un ruido de escape audible.
    • Limitaciones en los pasajes o en la válvula EGR causadas por depósitos de carbón o rebabas de fundición.
    • Fugas o limitaciones en el sensor MAP

       Nota: Las limitaciones pueden causar una contrapresión excesiva y un bajo vacío del motor.

    • Si la instalación de escape tiene fugas, limitaciones o se ha revisado - Consulte Síntomas - Escape del motor
    • Fallo mecánico del motor, como segmentos de pistón desgastados, árbol de levas o instalación de una correa dentada incorrecta - Consulte Síntomas: Mecánica del motor
  2. Con el encendido desconectado, desconecte el conector de mazo de cables de la válvula de Recirculación de Gases de Escape Q14.
  3. Con el encendido desactivado, compruebe que la luz de comprobación no se enciende entre el terminal del circuito de control C y B+.
    4. ⇒Si la luz de prueba se ilumina, compruebe el circuito de control para ver si hay un cortocircuito a masa. Si todas las pruebas de circuitos resultan bien, sustituya el K20 módulo de control del motor.
  4. Si la comprobación de todos los circuitos es normal, compruebe o sustituya la válvula de recirculación de gases de escape Q14.
