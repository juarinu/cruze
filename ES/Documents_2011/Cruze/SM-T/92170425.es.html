Descripción de DTC

Códigos de avería U0001 00: Bus de comunicación CAN de alta velocidad

Descripción de circuito

Módulos conectados a los circuitos de datos de serie GMLAN controlan comunicaciones de datos de serie en la red GMLAN durante el funcionamiento normal del vehículo. Entre los módulos se intercambian comandos e información operativa. Cada módulo de la red GMLAN mantiene un contador de errores de transmisión (TEC) y un contador de errores de recepción (REC). Los valores de contador aumentan con errores detectados y se reducen con mensajes sin errores. Si el valor TEC es superior a 255, el controlador se retira de la red y se activará un DTC U0001.

Condiciones para el funcionamiento de DTC
    • La tensión alimentada al módulo está dentro del intervalo de tensión operativa normal.
    • El modo de potencia del vehículo requiere que se produzca la comunicación de datos serie.

Condiciones para ajustar el DTC

El módulo ha detectado un determinado número de mensajes inválidos transmitidos en los circuitos de datos de serie GMLAN.

Acción realizada cuando se registra el DTC
    • El módulo suspende la transmisión de todos los mensajes.
    • El módulo utiliza valores por defecto para todos los parámetros recibidos en los circuitos de datos de serie GMLAN.
    • El módulo impide que se activen todos los demás códigos de avería de comunicación GMLAN.

Condiciones para el borrar el DTC
    • Un DTC actual se borra cuando ya no existe la avería.
    • Un DTC del historial se borra cuando el contador del ciclo de encendido del módulo alcanza el umbral de reinicio sin que se repita una avería.

Ayuda al diagnóstico
    • Este DTC no puede recuperarse con un estado actual. El diagnóstico del DTC actual se realiza a través del síntoma "la herramienta de diagnóstico no se comunica con el dispositivo GMLAN". Consultar La herramienta de diagnóstico no se comunica con el dispositivo GMLAN de alta velocidad .
    • Es probable que un estado intermitente se deba a un cortocircuito en los circuitos de datos de serie GMLAN. Utilice el procedimiento "la herramienta de diagnóstico no se comunica con el dispositivo GMLAN" para aislar un estado intermitente. Consultar La herramienta de diagnóstico no se comunica con el dispositivo GMLAN de alta velocidad .
