Descripción de DTC

Consultar Lista de códigos U del módulo de control .

Descripción del circuito/sistema

Los datos en serie se transmiten por un bus de circuito de red de cable simple LIN (Local Interconnect Network, o red de interconexión local) entre un módulo de control maestro y otros módulos dentro de un subsistema particular. Si se pierde la comunicación de datos de serie entre cualquiera de los módulos de control en la red de buses LIN, se activará un código de no comunicación en relación con el módulo de control que no comunica. El módulo de control maestro es el que proporciona el código de no comunicación. Nota: un DTC de pérdida de comunicación de datos serie no significa un fallo del módulo de control maestro que lo activa.

Ayuda al diagnóstico
    • Un módulo de control puede tener un código U almacenado en el historial que no requiera reparación alguna. Puede haber temporalmente problemas con mensajes tardíos o corruptos entre módulos de control sin un síntoma o avería aparente; no significa que el módulo de control esté averiado. No sustituya un módulo de control basándose sólo en un código U de historial.
    • No sustituya un módulo de control que informe sobre un código U. Éste identifica qué módulo de control se debe diagnosticar por un problema de comunicación.
    • Estará disponible la comunicación entre el módulo de control maestro y la herramienta de diagnóstico si existe una pérdida de las comunicaciones con cualquiera de los otros módulos de control de la red de bus LIN.
    • Una interrupción en el circuito de datos de serie de bus LIN entre el paquete de empalmes y el módulo sólo afectará a ese módulo específico. Este tipo de fallo activará un DTC de pérdida de comunicación para cada módulo de control afectado y los otros módulos de control seguirán comunicando.

Comprobación del circuito/sistema

Nota: 

   • Use los esquemas para identificar lo siguiente:
      - Módulos de control y dispositivos LIN con los que está equipado el vehículo
      - Ubicaciones de módulo de control y dispositivos LIN en el circuito de datos en serie
      - Los terminales del circuito de datos en serie del módulo de control y el dispositivo LIN B+, el encendido, la masa y los terminales de circuito de datos en serie
   • Utilice la lista de códigos U del módulo de control para determinar el módulo de control que no se comunica con el módulo de control maestro. Consultar Lista de códigos U del módulo de control .
   • Un DTC de pérdida de comunicación de datos en serie no significa un fallo del módulo que lo ha activado.

  1. Con el encendido desactivado, desemborne el conector del mazo de cables en el dispositivo LIN donde no se está comunicando.
  2. Pruebe con menos de 5 Ω entre cada terminal del circuito de masa y masa.
    3. ⇒Si es mayor que el margen especificado, compruebe si hay un circuito abierto/resistencia alta en el circuito de masa.
  3. Si está equipada, verifique que la lámpara de prueba se ilumina entre cada terminal del circuito B+ y la toma de tierra.
    4. ⇒Si la luz de prueba no se ilumina, pruebe el circuito B+ para ver si hay un cortocircuito a tierra o una resistencia abierta/alta. Si el fusible del circuito está abierto, compruebe si los circuitos de control del dispositivo LIN están cortocircuitados a masa. Si la prueba de circuitos sale bien, sustituya el dispositivo LIN.
  4. Si está equipado, encendido ON, verifique que la luz de prueba entre cada terminal del circuito de encendido y toma de tierra.
    5. ⇒Si la luz de prueba no se ilumina, compruebe si en el circuito de encendido hay un cortocircuito a masa o una resistencia abierta/alta. Si el fusible del circuito está abierto, compruebe si los circuitos de control del dispositivo LIN están cortocircuitados a masa. Si la prueba de circuitos sale bien, sustituya el dispositivo LIN.
  5. Con el encendido conectado, compruebe si existen 2-11,5 V entre el circuito de datos en serie LIN y masa.
    6. ⇒Si es superior al intervalo especificado, compruebe si el circuito de datos en serie LIN presenta un cortocircuito con la tensión. Si las pruebas de circuito salen bien, sustituya el módulo de control maestro.
    ⇒Si el valor es inferior al margen especificado, compruebe si el circuito de de datos en serie LIN tiene un cortocircuito a masa o un circuito abierto/resistencia alta. Si las pruebas de circuito salen bien, sustituya el módulo de control maestro.
  6. Si todas las pruebas de circuitos salen bien, sustituya el dispositivo LIN.
