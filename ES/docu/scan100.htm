1.       Descripción general

 

El software de diagnóstico que se está usando en la red de talleres para SCAN-100 debe actualizarse de forma regular con un nuevo software asociado para poder llevar a cabo el diagnóstico de nuevos modelos de coches y sistemas.  Por esa razón, se ha desarrollado "GMDAT Loader" para suministrar sin contratiempos programas de SCAN-100 a la red de talleres; éste viene con un CD-ROM o DVD para el usuario del scanner.

 

 

2.        Componentes de GMDAT Loader

 

(1)   "GMDAT Loader" se presenta como un paquete completo de programas que contiene diversas aplicaciones para la comunicación con SCAN-100.  El software incluido es el siguiente:

 

(2)   iSPS (Interface Service Programming System): Para reprogramar el software de una unidad de control electrónico (ECU), seleccionar el software para el vehículo asociado por modelo u opción que está almacenada en la base de datos de iSPS y descargar a continuación en la R-Card (tarjeta R) de SCAN-100.  Este es el método que hay que utilizar con el vehículo C100 (WINSTORM, CAPTIVA).

 

(3)   SL - SCANLOAD: Esta aplicación permite la descarga de programas de diagnóstico en la D-card (tarjeta D) de SCAN-100.

 

(4)   KWPSend: Para reprogramar software de una unidad de control electrónico (ECU); este software hace falta para descargar programas en la R-card que se suministra según el tipo de coche u opción aparte de la red de servicio.

 

(5)    DAP (Snapshot): Permite descargar datos de instantáneas al PC.

 

(6)   SA (Security Access): Durante determinadas operaciones de Scan-100 en el vehículo, es posible que se solicite un acceso de seguridad. Este programa proporciona el acceso de seguridad a Scan-100.

 

 

3.        Instalación de GMDAT Loader y especificaciones recomendadas para el PC

 

(1)      Especificaciones básicas

CPU: Pentium 3, 500 MHz

RAM: 256 MB

Disco duro: 20 GB

VGA: Tarjeta VGA con más de 16 MB de memoria VRAM

Otras: Puerto (serie) RS-232C, ratón PS/2 o USB

Sistema operativo: MS Windows 2000, MS Windows XP, MS Windows 2003

 

(2)      Especificaciones recomendadas

CPU: Pentium 4, 1,4 GHz

RAM: 512 MB

Disco duro: 40 GB

VGA: Tarjeta VGA con más de 32 MB de memoria VRAM

Otras: Puerto (serie) RS-232C, ratón PS/2 o USB

Sistema operativo: MS Windows 2000 SP4, MS Windows XP SP2, MS Windows 2003

6.       iSPS

 

(1)      Descripción general

Al igual que el modelo C100, al crearse directamente los datos de programación (software, calibración y componentes opcionales, etc.) en cada base de datos de la ECU en la línea de producción, la ECU que se suministra a la red de talleres para el servicio también permitirá cada una de sus funciones cuando sus datos de programación asociados deban introducirse mediante reprogramación.  Para ello, la información de producto del vehículo y los datos de programación según el sistema/modelo del coche se guardan en la carpeta "iSPS" asociada, con lo que se detecta automáticamente el número de identificación del vehículo (VIN) y, a continuación, se seleccionan los últimos datos y se descargan en la R-card de Scan-100.

 

(2) Ejecución

Haga clic en el botón iSPS en "GMDAT Loader" o ejecute el icono de acceso directo de iSPS en Inicio >> Programas >> GMDSCAN >> iSPS.

 

Figura 7  Ejecutar la aplicación iSPS en el Escritorio de Windows

 

Figura 8  Ejecutar la aplicación iSPS en la pantalla principal de Loader

(Haga clic en el botón [iSPS] marcado con el recuadro de color rojo)

 

(3)      Pantalla principal de iSPS

La pantalla principal de iSPS muestra información sobre el número VIN del vehículo y el número de serie del software de la ECU al hacer el diagnóstico con SCAN-100; también permite la búsqueda de los datos de reprogramación.

Figura 9  Pantalla principal

 

El menú principal consta de los siguientes apartados: Información del vehículo, Lectura de VIN, Descargar y Sistema.  La información sobre cada parte se da de la siguiente manera:

 

A. Información del vehículo

-     ECU Type (Tipo de ECU):

Selección de tipo de ECU o Visualización, el usuario puede verlo en el modo iSPS de SCAN-100 o seleccionar directamente el tipo.

-     VIN:

Entrada de VIN o Visualización, el usuario puede verlo en el modo iSPS de SCAN-100 o introducir directamente el número VIN.

-     Car Model (Modelo de coche):

Elegir el tipo de coche o el usuario puede seleccionarlo directamente.

 

B. Read VIN (Lectura de VIN)

-     Connect (Conectar):

Para conectarse a SCAN-100

-     Load Vehicle Information (Cargar información del vehículo):

Leer tipos de ECU, VIN y números de piezas de ECU almacenados en SCAN-100 -> Si la lectura es correcta, el contenido de la información del vehículo se actualiza automáticamente.

Nota: Primero debe hacerse clic en Connect (Conectar) y luego en Load Vehicle Information (Cargar información del vehículo).  Sin embargo, también se puede ir a la conexión automática con SCAN-100 haciendo clic en Load Vehicle Information (Cargar información del vehículo).

 

C. Programación de descarga de datos

-     Modo normal:

Este modo de descarga sólo se usa cuando los tipos de ECU y la información de número VIN en SCAN-100 son válidos, o cuando la información de nº VIN que introduce el usuario es válida.  En este modo iSPS selecciona automáticamente el mapa de datos y genera el archivo para descargar (modo recomendado).

-     Modo manual:

Este modo de descarga sólo se usa cuando los tipos de ECU y la información de número VIN en SCAN-100 no son válidos, o cuando no puede utilizarse el modo automático.  En este modo el usuario selecciona el mapa de datos y genera el archivo para descargar.

 

D. Sistema

-     Configuration (Configuración):

Para ir a la pantalla de configuración de iSPS

-     Exit:

Para salir de iSPS

 

(4)      Configuración del sistema iSPS

Esta pantalla es para configurar la comunicación entre iSPS y SCAN-100 y para configurar la carpeta de iSPS.

 

Figura 10  Pantalla de configuración

 

El paquete iSPS se distribuye con los valores óptimos reconfigurados, que son compatibles con la mayoría de ordenadores personales, de forma que no se recomienda cambiar los parámetros de configuración salvo en circunstancias especiales, como casos de emergencia, sustitución de equipos o sustitución de puertos de comunicaciones.

 

A. Communication Settings (Parámetros de comunicación)

-     Port (Puerto):

Establece el puerto de comunicaciones.  Admite desde puertos COM1 a COM4.

-     Baud (Baudio):

Para configurar la velocidad de comunicación (SCAN-100 admite sólo 38400 a fecha de abril de 2006).

-     Data bit (Bit de datos):

Para configurar los bits de datos (SCAN-100 sólo admite 8 bits a fecha de abril de 2006).

-     Stop bit (Bit de parada):

Para configurar el bit de parada (actualmente SCAN-100 sólo admite 1 bit a fecha de abril de 2006).

-     Parity (Paridad):

Para configurar el bit de paridad (actualmente SCAN-100 no admite bit de paridad a fecha de abril de 2006).

 

B. Working Directory (Directorio de trabajo)

El usuario puede definir la vista de la pantalla con la carpeta de iSPS.  Se recomienda mantener la carpeta predeterminada para garantizar un funcionamiento óptimo.

-     Browse (Examinar):

Para ver la pantalla de selección de la carpeta.

 

Figura 11  Pantalla de selección de carpeta (hacer clic en OK tras seleccionar la carpeta de iSPS).

(5)      Conexión

La interfaz del proceso de conexión es necesaria para usar el modo normal o manual durante la descarga.

 

A. Para conectar al puerto de comunicaciones entre SCAN-100 y el PC.

B. Haga clic en Connect (Conectar)

C. Haga clic en Load Vehicle Information (Cargar información del vehículo)

 

 

(6)      Modo normal

El modo normal permite a iSPS generar automáticamente el archivo para descargar utilizando la información de número VIN y de ECU en SCAN-100 al hacer clic en Load Vehicle Information (Cargar información del vehículo), permitiendo al usuario descargar el archivo generado.

 

Nota: Se recomienda a los usuarios que utilicen el modo normal.  Use el modo manual solamente en casos especiales (ej., cuando no se pueda utilizar el modo normal o haya otros motivos).

 

[Cómo utilizar el modo normal]

 

A. Realice el proceso de conexión descrito en el apartado anterior en "Conexión" (5).

 

B. Seleccione el modo normal (Normal Mode) en la pantalla principal.

 

C. Aparecerá la siguiente pantalla cuando sean válidos el número VIN, que había en SCAN-100, o la información introducida por el usuario.

 

Nota: Si el número VIN no es válido, aparecerá un mensaje de error pidiendo al usuario que vuelva a introducir el nº VIN.  En tal caso, no puede utilizarse el modo normal.

 

Figura 12  Pantalla de modo normal

D. Seleccione el nombre del coche en Car Model

 

E.   Haga clic en Download (Descargar) o Make KWP (Crear KWP)

 

Nota: Al hacer clic en Download, el usuario puede descargar datos de mapa en la R-CARD de SCAN-100.  Por otra parte, Make KWP sólo se pulsa cuando el usuario desea descargar el archivo a través de KWPSend.

 

F.   Aparecerá la barra de progreso de la descarga o el mensaje "KWP file generation" (Generación de archivo KWP).

 

(7)      Modo manual

El modo manual sólo debe usarse cuando el número VIN cargado en SCAN-100 no coincida con los datos de versión de producción de la aplicación iSPS, o cuando el usuario deba seleccionar el archivo del mapa para sus propias necesidades.

 

Nota: No se recomienda a los usuarios que utilicen el modo manual.  Solamente debe accederse a éste cuando no sea posible usar el modo normal.  El fabricante no se hará responsable de ningún defecto o problema que pueda surgir por un uso descuidado del modo manual.

 

El modo manual consta de diversas secciones, a saber, información del vehículo, vista de utilidad, vista de software, vista de calibración y vista de datos de mapa, los cuales se describen a continuación:

 

A. Información del vehículo

-     ECU Type (Tipo de ECU):

Selección de tipo de ECU o Visualización (el usuario puede verlo en el modo iSPS de SCAN-100 o seleccionar directamente el tipo).

-     VIN:

Entrada de VIN o Visualización (el usuario puede verlo en el modo iSPS de SCAN-100 o introducir directamente el número VIN).

-     Car Model (Modelo de coche):

Elegir el tipo de coche (o seleccionarlo directamente).

 

B. Utility view (Vista de utilidad):

Seleccione un archivo en la vista

 

C. Software view (Vista de software):

Seleccione un archivo en la vista

 

D. Calibration view (Vista de calibración):

Seleccione un archivo de acuerdo con las opciones de configuración del vehículo

 

E. Map data view (Vista de datos de mapa):

Muestra finalmente los datos combinados que se van a descargar.

Figura 13  Pantallas de modo manual nº 1

 

[Cómo utilizar el modo manual]

 

A. Realice el proceso de conexión descrito en el apartado "Conexión" (5).

 

B. Haga clic en el modo manual (Manual Mode) en la pantalla principal.

 

C. Aparecerá la siguiente pantalla si se han realizado correctamente los pasos anteriores.

 

Figura 14  Pantallas de modo manual nº 2

 

D. Introduzca el número VIN en la sección VIN de la parte de información del vehículo (este paso puede omitirse si aparece el nº VIN al hacer clic en Load Vehicle Information (Cargar información del vehículo) en la pantalla principal, o cuando se introduzca un VIN válido en la pantalla principal).

 

E.   Seleccione el tipo de ECU en la parte de información del vehículo (este paso puede omitirse si aparece el tipo de ECU al hacer clic en Load Vehicle Information (Cargar información del vehículo) en la pantalla principal, o cuando ECU está seleccionada en la pantalla principal).

 

F.   Haga clic en ADD (Agregar) tras seleccionar el archivo en la vista de utilidad (al hacer clic en ADD se muestra el archivo seleccionado en la vista de datos de mapa).

J.  Aparecerá la barra de progreso de la descarga o el mensaje "KWP file generation" (Generación de archivo KWP).
