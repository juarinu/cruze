DTC C0292

descripción del circuito

El EBCM proporciona 5 V al circuito de referencia del conjunto del sensor de nivel de oscilación y una masa al circuito de referencia de baja del sensor de nivel de oscilación, que comparte con el sensor de ángulo del volante. El EBCM, el conjunto del sensor de nivel de oscilación y el sensor de ángulo del volante comparten un circuito común de tensión de alimentación de 12 V.
El sensor de nivel de oscilación produce una tensión de salida de señal equivalente a la rotación del vehículo sobre su eje vertical.

Descriptor de DTC

Este procedimiento de diagnóstico admite el DTC C0292.
Circuitos del sensor de combinación VSES (Sistema de mejora de la estabilidad del vehículo.

Condiciones para la activación del DTC

La llave de contacto se encuentra en la posición ON.

Condiciones para la activación del DTC

Si el EBCM detecta un fallo en la salida de tensión de la señal del sensor de nivel de oscilación.

Acción realizada cuando se activa el DTC

  • El EBCM desactiva el ESC durante el ciclo de encendido.
  • La luz de aviso del ABS, el icono de aviso del ABS y el icono de tracción desactivada (Trac-off) se activan.

Condiciones para borrar el DTC

  • Deja de aparecer la condición para el DTC.
  • Borre los DTCs utilizando el scanner.
  • El EBCM borra automáticamente el historial de DTC cuando no se detecta un DTC en 100 ciclos de conducción consecutivos.

Descripción de la prueba

Los números indicados más abajo corresponden a los números de los pasos de la tabla de diagnóstico.

DTC C0460

descripción del circuito

El sensor de ángulo del volante comparte una tensión de alimentación de encendido de 12 V con el EBCM y el sensor de nivel de oscilación. El EBCM aplica una masa al circuito de referencia baja del sensor de ángulo del volante, que se comparte con el sensor de nivel de oscilación.
El sensor de ángulo del volante proporciona una salida de señal de datos serie al EBCM que representa el grado de giro del volante. El EBCM utiliza esta señal para calcular la dirección de conducción pretendida por el conductor.

Descriptor de DTC

Este procedimiento de diagnóstico admite el DTC C0460 - Sensor de posición de la dirección.

Información adicional

Un fallo eléctrico en los circuitos eléctricos compartidos puede activar DTCs en los componentes o sensores que comparten el circuito defectuoso. Compruebe el circuito eléctrico de los sensores o componentes correspondientes para aislar el fallo.

Condiciones para la activación del DTC

La llave de contacto se encuentra en la posición ON.

Condiciones para la activación del DTC

Si el EBCM detecta el siguiente fallo en el sensor de ángulo del volante,
  • fallo interno en el sensor de ángulo del volante,
  • señal fuera de intervalo en la salida del sensor de ángulo del volante o
  • un sensor de ángulo del volante sin calibrar.

Acción realizada cuando se activa el DTC

  • El EBCM desactiva el ABS-TCS/ESC durante el ciclo de encendido.
  • Uno de los siguientes indicadores de advertencia puede activarse:
    • Indicador de advertencia del ABS-TCS,
    Visualización de advertencia del ABS-TCS,
    Visualización de advertencia del ESC,
    Indicador de fallo de frenos, y/o
    Visualización de advertencia de desconexión de tracción.

Condiciones para borrar el DTC

  • El DTC actual pasa al historial de DTCs cuando no hay ningún DTC almacenado durante la autocomprobación del EBCM.
  • Utilice el scanner para borrar el DTC.

Descripción de la prueba

Los números indicados más abajo corresponden a los números de los pasos de la tabla de diagnóstico.
  1. Verifica si fallo que activa este DTC persiste.
  2. Compruebe el circuito compartido de encendido de 12 V.
  3. Compruebe el circuito compartido de masa de referencia.

DTC C0550

descripción del circuito

La detección de fallos internos se realiza en el interior del módulo de control. No hay ningún circuito externo implicado.

Descriptor de DTC

El procedimiento de diagnóstico admite los siguientes DTCs:
DTC C0550 - Rendimiento de la Unidad de control electrónica (ECU)

Condiciones para la activación del DTC

El microprocesador ejecuta el programa para detectar un fallo interno cuando se ordena la entrada de alimentación. Los únicos requisitos son tensión y una masa. Este programa se ejecuta incluso cuando la tensión se encuentra fuera del intervalo de funcionamiento válido.

Condiciones para la activación del DTC

  • El módulo de control detecta un fallo de escritura interno.
  • El módulo de control detecta un fallo de suma de control interno.

Acción realizada cuando se activa el DTC

Si cuenta con los dispositivos especificados, pueden darse las siguientes acciones específicas del módulo:
  • El testigo del sistema antibloqueo de frenos (ABS) se ilumina.
  • El indicador de aviso de frenos se ilumina.
  • El LED del sistema de control de tracción (TCS) se apaga.

Condiciones para borrar el DTC

  • Deja de aparecer la condición para el DTC.
  • Borre los DTCs utilizando el scanner.
  • El EBCM borra automáticamente el historial de DTC cuando no se detecta un DTC en 100 ciclos de conducción consecutivos.

Ayudas para el diagnóstico

  1. Este DTC puede almacenarse como un DTC de historial sin afectar al funcionamiento del módulo. Si se almacena sólo como un DTC de historial y no se recupera como un DTC actual, no sustituya el módulo.
  2. Si este DTC se recupera tanto como DTC de historial como actual, sustituya el módulo.

DTC C0551

descripción del circuito

El módulo electrónico de control de frenos (EBCM) memoriza las opciones del modelo que está instalado utilizando la información de ID del vehículo desde el módulo de control de la carrocería (BCM) a través de la línea de datos serie de clase 2. Cada vez que el vehículo se encuentra en la posición RUN, el BCM comunica esta información a través de la clase 2. El EBCM compara internamente el VIN y la información opcional con la información almacenada en el BCM.

Descriptor de DTC

El procedimiento de diagnóstico admite:
DTC C0551 - Error de configuración de opciones

Condiciones para la activación del DTC

  • El EBCM detecta el dígito 2-9 del VIN y que no son ceros.
  • La condición debe darse cuando el interruptor de contacto se coloca por primera vez en la posición ON.

Condiciones para la activación del DTC

  • El EBCM detecta la información del BCM acerca de las opciones incorrectas.
  • La condición debe darse cuando el interruptor de contacto se coloca por primera vez en la posición ON.

Acción realizada cuando se activa el DTC

Si cuenta con estos dispositivos, se dan las siguientes acciones:
  • El EBCM desactiva el ABS y el sistema de control de tracción (TCS) durante el ciclo de encendido.
  • Se activa un DTC de avería.
  • El testigo del ABS se ilumina.
  • El indicador de control de tracción se activa.
  • La luz de aviso roja de freno se ilumina.

Condiciones para borrar el DTC

  • El EBCM reconoce esa información del BCM como información válida.
  • Un DTC en historial se borrará después de 100 ciclos de encendido consecutivos si la condición de la avería ya no aparece.

Ayudas para el diagnóstico

  1. El DTC C0551 se activa cuando el EBCM detecta un desequilibrio entre sí mismo y el BCM programado del vehículo, mediante el control del mensaje de clase 2 del BCM.
  2. Si este DTC se recupera tanto como DTC de historial como actual, sustituya el módulo.

DTC C0561

descripción del circuito

El módulo electrónico de control de frenos (EBCM) desactiva el control de tracción cuando otros módulos electrónicos de control activan DTCs en componentes que afectan al funcionamiento del Sistema de control de tracción (TCS).

Descriptor de DTC

El procedimiento de diagnóstico admite los siguientes DTCs:
DTC C0561 - Información de sistema desactivado almacenada
Este vehículo tiene DTCs que incluyen síntomas de DTC. Para más información acerca de síntomas de DTC, consulte la Descripción de síntomas de DTCs en la Información de DTCs del vehículo.

Condiciones para la activación del DTC

  • La llave de contacto está en posición ON.
  • La tensión de encendido sea superior a 8 V.

Condiciones para la activación del DTC

El módulo de control de la carrocería (BCM) detecta un estado que evita que la parte del control del motor del control de tracción funcione y envía un mensaje de datos serie al EBCM indicando que la reducción del par no está permitida. El BCM activará un DTC y el EBCM activará este DTC.

Acción realizada cuando se activa el DTC

  • El EBCM desactiva el TCS hasta que el DTC pase al historial de DTCs.
  • El indicador de desactivación de tracción se activa.
  • El sistema antibloqueo de frenos (ABS) permanece funcional.

Condiciones para borrar el DTC

  • La condición del DTC ya no está presente y el DTC se borra con el scanner.
  • El módulo electrónico de control de frenos (EBCM) borra automáticamente el DTC del historial cuando un DTC actual no se detecta en 100 ciclos de conducción consecutivos.

Ayudas para el diagnóstico

Este DTC es meramente informativo. Como ayuda para el técnico, este DTC indica que no hay problemas en el sistema ABS/TCS.

DTC C0899

descripción del circuito

El módulo electrónico de control de frenos (EBCM) controla el nivel de tensión disponible para el funcionamiento del sistema. Una tensión baja no permite que el sistema funciona correctamente.

Descriptor de DTC

El procedimiento de diagnóstico admite:
DTC C0899 - Tensión baja de dispositivos

Condiciones para la activación del DTC

  • La velocidad del vehículo sea superior a 8 km/h (5 mph).
  • El relé del sistema está activado.

Condiciones para la activación del DTC

  • Durante la inicialización o cuando el sistema no está activo, la tensión del sistema es inferior a 10,8 voltios.
  • Durante el funcionamiento del sistema, la tensión del mismo es inferior a 9,0 voltios.

Acción realizada cuando se activa el DTC

Si cuenta con estos dispositivos, se dan las siguientes acciones:
  • El EBCM desactiva el ABS durante el ciclo de encendido.
  • El testigo del ABS se ilumina.
  • El indicador de control de tracción se activa.
  • El DTC C0899 queda almacenado.

Condiciones para borrar el DTC

  • La condición del DTC ya no está presente y el DTC se borra con el scanner.
  • El EBCM borra automáticamente el historial de DTC cuando no se detecta un DTC en 100 ciclos de conducción consecutivos.

Ayudas para el diagnóstico

  • Compruebe el sistema de carga.
  • Las siguientes condiciones pueden ser la causa de este DTC:
  1. Una batería débil o descargada.
  2. Un contacto deficiente en los terminales de la batería.
  3. Funcionamiento excesivo de la batería de fábrica o equipos accesorios.
  4. Una conexión defectuosa a masa de la batería en el bloque o el chasis.
  5. Un contacto deficiente de los terminales del EBCM.
  6. Una conexión a masa defectuosa del EBCM.
  7. Un fallo de funcionamiento en el sistema de carga (alternador).
  • Compruebe de forma completa el cableado y los conectores. La no realización de una comprobación completa y cuidadosa del cableado y de los conectores puede dar lugar a la sustitución de piezas sin haber reparado verdaderamente la avería.

DTC C0900

descripción del circuito

El módulo electrónico de control de frenos (EBCM) controla el nivel de tensión disponible para el funcionamiento del sistema. Si el nivel de tensión es demasiado alto, pueden producirse daños en el sistema. Cuando el EBCM detecta un estado de tensión alta, el EBCM desactiva el relé del sistema, lo que elimina la tensión de batería de las válvulas solenoides y del motor de la bomba.

Descriptor de DTC

El procedimiento de diagnóstico admite los siguientes DTCs:
DTC C0900 - Tensión alta de dispositivos

Condiciones para la activación del DTC

La velocidad del vehículo sea superior a 8 km/h (5 mph).

Condiciones para la activación del DTC

La tensión del sistema sea superior a 17 voltios durante 0,72 segundos.

Acción realizada cuando se activa el DTC

Si cuenta con estos dispositivos, se dan las siguientes acciones:
  • El EBCM desactiva el ABS durante el ciclo de encendido.
  • El testigo del ABS se ilumina.
  • El indicador de control de tracción y activación de control está iluminado.
  • El relé de solenoide se abre cortando la corriente al motor de la bomba y a los solenoides.
  • El DTC C0900 queda almacenado.

Condiciones para borrar el DTC

  • La condición del DTC ya no está presente y el DTC se borra con el scanner.
  • El EBCM borra automáticamente el historial de DTC cuando no se detecta un DTC en 100 ciclos de conducción consecutivos.

Ayudas para el diagnóstico

Una posible causa de este DTC es la sobrecarga.

DTC U2105, U2106, U2142, U2143

descripción del circuito

Los módulos conectados a los circuitos de datos serie GMLAN de alta velocidad controlan las comunicaciones de datos serie durante el funcionamiento normal del vehículo. La información de funcionamiento y los comandos se intercambian entre los módulos. Los módulos cuentan con información pregrabada acerca de qué mensajes deben intercambiarse en los circuitos de datos serie para cada red virtual. Los mensajes se supervisan y también se utilizan algunos mensajes periódicos por parte del módulo de recepción como indicación de disponibilidad del módulo transmisor.

Descriptor de DTC

El procedimiento de diagnóstico admite los siguientes DTCs:
  • DTC U2105 - Pérdida de comunicación con el ECM
  • DTC U2106 - Pérdida de comunicación con el TCM
  • DTC U2142 - Pérdida de comunicación con el conjunto de sensores
  • DTC U2143 - Pérdida de comunicación con el sensor de ángulo del volante

Condiciones para la activación del DTC

  • La tensión suministrada a los módulos se encuentra dentro del intervalo de tensión de funcionamiento normal.
  • El modo de alimentación del vehículo requiere que se produzca la comunicación de datos serie.

Condiciones para la activación del DTC

Un mensaje periódico supervisado que incluye la disponibilidad del módulo transmisor no se ha recibido.

Acción realizada cuando se activa el DTC

El módulo utiliza un valor predeterminado para el parámetro que falta.

Condiciones para borrar el DTC

  • Un DTC actual se borra cuando la avería ya no está presente.
  • Un DTC de historial se borra cuando el contador de ciclos de encendido del módulo alcanza el umbral de reinicio, sin que se repita la avería.

Ayudas para el diagnóstico

  • Una mala conexión en el módulo no operativo puede provocar la activación de este código.
  • Un módulo con mala alimentación puede provocar la activación de este código.

DTC U2100, U1500, C0730

descripción del circuito

Los módulos conectados al circuito de datos serie GMLAN controlan las comunicaciones de datos serie de la red GMLAN durante el funcionamiento normal del vehículo. La información de funcionamiento y los comandos se intercambian entre los módulos. El módulo de control de la carrocería (BCM) mantiene un contador de errores de transmisión (TEC) y un contador de errores de recepción (REC). Los valores del contador aumentan con los errores detectados y disminuyen con los mensajes sin errores.

Descriptor de DTC

El procedimiento de diagnóstico admite los siguientes DTCs:
  • DTC U1500 - Fallo de comunicación (bus desactivado) del bus CAN 2 (conjunto de sensores)
  • DTC U2100 - Fallo de comunicación (bus desactivado) del bus CAN
  • DTC C0730 - Error en la señal del engranaje CAN

Condiciones para la activación del DTC

  • La tensión suministrada a los módulos se encuentra dentro del intervalo de tensión de funcionamiento normal.
  • El modo de alimentación del vehículo requiere que se produzca la comunicación de datos serie.

Condiciones para la activación del DTC

El módulo detecta un número determinado de mensajes transmitidos no válidos en los circuitos de datos serie GMLAN. El motor no arranca cuando hay un fallo total de los circuitos de datos serie GMLAN de alta velocidad mientras el motor no está en funcionamiento. Los siguientes estados puede provocar una pérdida total de comunicación de datos GMLAN de alta velocidad.
  • Un cortocircuito entre los circuitos GMLAN de alta velocidad (+) y GMLAN de alta velocidad (-)
  • Derivación a masa o cortocircuito con tensión en los circuitos de datos serie GMLAN de alta velocidad.
  • Un fallo interno en el módulo que provoca un cortocircuito con tensión o masa en los circuitos GMLAN de alta velocidad.
  • Circuito abierto en cualquiera de los circuitos de datos serie GMLAN de alta velocidad

Acción realizada cuando se activa el DTC

  • El módulo suspende la transmisión de todos los mensajes.
  • El módulo utiliza los valores predeterminados para todos los parámetros recibidos en los circuitos de datos serie GMLAN de alta velocidad.
  • El módulo inhibe la activación de todos los demás DTCs de comunicación del GMLAN de alta velocidad.

Condiciones para borrar el DTC

  • Un DTC actual se borra cuando la avería ya no está presente.
  • Un DTC de historial se borra cuando el contador de ciclos de encendido del módulo alcanza el umbral de reinicio, sin que se repita la avería.

Ayudas para el diagnóstico

El BCM puede activar este DTC como actual ya que es el enlace entre los circuitos de datos serie de clase 2 y GMLAN de alta velocidad, y el diagnóstico del BCM se realiza en el enlace de datos de clase 2.
El bus de datos serie GMLAN de alta velocidad utiliza dos resistencias de finalización de 120 ohmios situadas en paralelo con los circuitos (+) y (-) GMLAN de alta velocidad. Una de las resistencias está conectada al BCM y la otra al módulo de control del motor (ECM). Cuando se comprueba si hay cortocircuitos entre ambos circuitos del GMLAN, es normal obtener una lectura de 60 ohmios. Si hay un circuito abierto en el circuito de datos serie GMLAN de alta velocidad, la lectura de resistencia será de unos 120 ohmios.
